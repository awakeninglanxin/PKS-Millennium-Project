# UF20 三角不等式高级版 (Advanced TIA) — 正M vs 逆M

## 公式

$$r_n = \frac{|z_{n+1} - z_n|}{|z_{n+1}|},\quad \text{TIA}_{\text{adv}} = \frac{1}{N}\sum_{n=0}^{N-1} w_n \cdot r_n$$

与 UF11 (三角不等式平均) 的区别: 加入了**加权系数** $w_n$, 对靠近边界的迭代步赋予更高权重, 增强边界区域的纹理对比。

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 轨道比值 | $r_n = \|Δz\|/\|z_{n+1}\|$ | **相同** |
| 加权累加 | `Σ(w_n × r_n)/N` | **相同** |
| 色板 | plasma (对数归一化) | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸 = `ext`) |
| **未着色区域** | 集合内部 (黑) | 水滴外部 (黑) |

## 与 UF11 的差异

| | UF11 (普通版) | UF20 (高级版) |
|:---|:---|:---|
| 权重 | 等权 $w_n=1$ | 加权（靠近边界权重更大） |
| 纹理 | 均匀 | 边界增强 |
| 归一化 | log1p | log1p |
