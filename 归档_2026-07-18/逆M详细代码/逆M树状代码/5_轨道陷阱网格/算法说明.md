# UF5 轨道陷阱网格 — 正M vs 逆M 对照

## 正M公式

$$z_{n+1} = z_n^2 + c,\quad d_{\min} = \min_n \text{dist}(z_n, \text{trap})$$

## 逆M公式

$$z_{n+1} = z_n^2 + c_{\text{eff}},\quad d_{\min} = \min_n \text{dist}(z_n, \text{trap}) \quad \text{(相同)}$$

## 对照表

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:---|:---|:---|
| 复数映射 | `c = pixel` | `c_eff = 1/c` |
| 迭代公式 | `z = z² + c` | `z = z² + c_eff` |
| 陷阱形状 | 网格线 `min(\|Re(z)\|,\|Im(z)\|)` | **相同** |
| 距离追踪 | `d_min = min\|z_n-trap\|` | **相同** |
| 灰度映射 | `g = 1 - clip(d_min·2.5, 0, 1)` | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸) |
| **未着色** | 集合内部 (中灰) | 水滴外部 (中灰) |

### 视觉效果差异

- **正M**: 网格线在M集外部扩散, 大量开放空间
- **逆M**: 网格线压缩进水滴内部 → 挤压变形 → 同心皱褶纹理

### 管线

```python
1. 每次迭代追踪: gd = min(|Re(z)|, |Im(z)|)
2. mtd[n] = min(gd[0..n])  # 历史最小值
3. 逃逸后: g = 1 - clip(mtd·2.5, 0, 1)
4. 未逃逸(水滴外部) → 中灰 0.5
```
