> 更新日期：2026-07-12
> 涵盖：正M vs 逆M 数学对照 + 本目录下 6 个 py 变体的算法说明

---

## 一、核心数学对照

### 正M公式 (Standard Mandelbrot)

$$z_{n+1} = z_n^2 + c,\quad c = \text{pixel}$$

### 逆M公式 (Inverted Mandelbrot)

$$z_{n+1} = z_n^2 + c_{\text{eff}},\quad c_{\text{eff}} = \|c\|^{-1} e^{-i\arg(c)} = c^{\alpha},\ \alpha=-1$$

### 正M vs 逆M 逐项目对照

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:------|:--------------|:--------------|
| 复数映射 | `c = pixel` (直接采样) | `c_eff = c^α, α=-1` (极坐标幂) |
| 迭代公式 | `z = z² + c` | **相同** `z = z² + c_eff` |
| 逃逸判定 | `\|z\| > bailout` | **相同** (本目录统一 bailout=50) |
| 外部角度 | `θ = arg(z)/(2π·2ⁿ) mod 1` | **相同** |
| 连续势能 | `pot = MAXITER-1-log₂(log₂\|z\|)` | **相同** |
| 棋盘格逻辑 | `(⌊θN⌋%2)==(⌊pot/Δ⌋%2)` | 旧版相同；新版改为边缘检测 |
| **棋盘格位置** | 集合外部 (逃逸区) | 水滴内部 (逃逸区) |
| **未着色区域** | 集合内部 (黑色) | 水滴外部 (白色) |
| bailout | 可大 (100+) | **50** (小, 反演空间早逃逸) |

**视觉效果差异**：逆M 的 `1/c` 反演把原本在 M 集四周扩散的棋盘格压缩进水滴"内部"，格点被挤压成三角鳞片状；外部反而成为空白逃逸区。

## 二、引擎实现要点 (Engine)

所有变体共用同一套迭代引擎，关键约定：

1. **逆映射**：`ce = |c|^α · exp(i·α·arg(c))`，`α=-1`，`|c|≈0` 时置 `1e6` 防爆。
2. **旋转修正**：输出统一 `np.rot90(..., k=3)`，使实部轴水平、虚部轴垂直，匹配参考图朝向。
3. **视窗固定**：`Re[-1.833, 4.500]`，`Im[-2.124, 2.124]`（即 `B=-4/3, TIP=4.0, H=1.62427191, M=0.5`）。
4. **分辨率**：`W=2400`，`H` 按比例算出（约 3576）。
5. **⚠️ 关键发现**：逆M 集的 `pot`（平滑迭代）值**恒≈246**（全部外部点高迭代），**不能用 `pot` 阈值判边界**。判边界必须用**原始逃逸整数 `esc_iter`**（main.py 新增返回）。

### 1. `main.py` — 树状连线（满意版）

**算法**：外部角度场阴阳分界线 + 逃逸迭代边界着色

```python
# 1) 高精度角度量化
rz = floor(er * ANGLE_BINS)          # ANGLE_BINS=512
# 2) 相邻像素角度跳变检测 → 树状分岔线
edge_h = (rz[:,1:] != rz[:,:-1]) & ext邻域
edge_v = (rz[1:,:] != rz[:-1,:]) & ext邻域
tree_edge = edge_h | edge_v
# 3) 形态学膨胀加粗 (LINE_WIDTH=2)
# 4) 边界蓝紫: esc_iter < 80 (非 pot!)
```

**原理**：逆M 集的外部角场 `er` 在 Mini-bulb 处产生不连续性，相邻像素角度跳变即对应"阴阳分界线"。检测这些跳变即可画出从实数轴向水滴边缘分三岔的分形连线，完全不依赖势函数 → 无同心环。

**图层**：白底 → 深细线 `[0.15,0.12,0.28]` → 淡蓝紫边界 `[0.55,0.50,0.82]` (`esc_iter<80`) → 内部纯白。

**关键参数**：`ANGLE_BINS=512`, `LINE_WIDTH=2`, `esc_iter<80`, `MAXITER=250`, `BAILOUT=50`。

---

### 2. `main_旧版_同心环.py` — 旧版归档

**算法**：角度⊕势函数双维度 XOR 棋盘（产生同心环的原始方案）

```python
rz = floor(er * RAY_DIV)             # RAY_DIV=64
pz = floor(pot / POT_STEP)           # POT_STEP=0.12 → 数百个同心带!
cb = ((rz%2==0) == (pz%2==0)) & ext
```

**问题根源**：`pot/0.12` 在全量程（0~246）上产生约 2000 个同心带，渲染成 bullseye 辐射环，淹没树状纹理。

**保留目的**：与满意版对照，证明"环是势函数维度造成的"。输出 `UF1_二分棋盘格素描_旧版_同心环.png`。

---

### 3. `UF1_去环棋盘格.py` — 取模势能去环

**算法**：势能取模截断，保留 XOR 树杈纹理但消除远距离环累积

```python
pz = floor((pot % POT_MOD) / POT_STEP)   # POT_MOD=1.0, POT_STEP=0.5
cb = ((rz%2==0) == (pz%2==0)) & ext
```

**原理**：`pot % 1.0` 把势能限制在单位区间内，任何位置的势函数都只产生 2 个 XOR 带（`1.0/0.5=2`），环不再随距离累积。但 XOR 纹理仍依赖势函数，靠近边界处仍有轻微带状感。

---

### 4. `UF1c_辐射棋盘格.py` — 平滑波去环

**算法**：用 `sin(pot)` 连续波替代硬 `floor(pot)`，软化环边缘

```python
pot_wave = sin(sm * 6) > 0
cb = ((rz%2==0) != pot_wave) & ext
```

**原理**：`sin` 波让 XOR 切换在势函数维度上平滑渐变，硬同心环消失，但树杈纹理被正弦调制成辐射状波纹。适合想要"柔和外晕"风格时使用。

---

### 5. `UF1b_壳线描边.py` — 黑白素描 + 外壳描边

**算法**：纯黑白 XOR 棋盘 + Sobel 梯度外壳 + 解析心形/圆轮廓线

```python
cb = (re == pe) & ext               # 标准 XOR 棋盘
# Sobel 梯度 → 高梯度阈值 → 形态学开/膨胀 → 外壳黑线
shell = binary_opening(binary_dilation(gm > th))
# 解析逆心形: c = 0.5e^{iθ} - 0.25e^{2iθ}; 描 -1/c 轮廓
```

**特点**：最接近"手绘素描"风格——纯黑 XOR 格 + Sobel 提取的集边界黑壳 + 解析逆心形曲线叠加。无色彩，靠线条密度表现结构。依赖 `scipy.ndimage`。

---

### 6. `main_阴阳adam.py` — 阴阳Adam（黑底 + 灰阶棋盘 + DEM金色边）

**算法**：基于逆M引擎的 `c^α` 迭代 + **新增 `dz/dz₀` 导数追踪**（用于 DEM 距离估计），渲染层叠加灰阶 XOR 棋盘 + 金色边界辉光。

```python
# 引擎层: 导数链式法则追踪 (DEM 必需)
dz[alive] = 2*z[alive]*dz[alive] + 1

# 渲染层1: 黑底
img = np.zeros((h,w,3))

# 渲染层2: 灰阶XOR棋盘 (粗量化→大块棋盘, 参考图风格)
rz = floor(er * RAY_DIV)          # RAY_DIV=16
pz = floor(pot / POT_STEP)        # POT_STEP=8.0 (很粗→极少环)
chess = ((rz%2==0) != (pz%2==0)) & ext
img[chess] = [0.95,0.93,0.90]     # 暖白色格
img[ext & ~chess] = [0.18,0.16,0.14]  # 深灰格

# 渲染层3: DEM橙黄辉光边 (仅 interior↔ext 交界像素!)
#   Mu-Ency: dem = 0.5*log(|z|)*|z|/|dz|
boundary = 与interior相邻的ext像素
dn = dem[boundary] / dem.max()
img[boundary] = [0.90+0.10*dn, 0.40+0.50*dn, 0.05+0.03*dn]  # 橙→亮黄

# 渲染层4: Mini-bulb 内部保持黑底 (无需操作)
```

**与满意版 `main.py` 的本质区别**：
- `main.py`：**完全去掉势函数维度**，靠角度场边缘检测画树状细线，白底 + 蓝边界。
- `main_阴阳adam.py`：**保留势函数维度**做粗 XOR 棋盘（POT_STEP=8 压制环密度），黑底 + 灰阶大块棋盘 + DEM 金色边，整体更接近「逆M阴阳 adam.jpg」的厚重棋盘风格。

**本次更新（2026-07-12 22:30）**：移除了原「层3 中心蓝紫 colormap」（`PuBu` 覆盖 `|c|<2.0` 区域）。用户反馈右侧黑白棋盘格正确，**仅要求去掉左侧蓝区**。现已删除该层，整张图统一为灰阶 XOR 棋盘 + 金色边 + 黑底。

**关键参数**：`RAY_DIV=16`, `POT_STEP=8.0`, `DEM_SCALE=20.0`, `MAXITER=250`, `BAILOUT=50`。

**关键发现**：逆M 不能用 `esc_iter` 判中心区域（分布异常），需用 `|c|` 原点距离——但本版已移除中心 colormap，故此约束不再影响输出。

**输出**：`UF1_阴阳Adam.png`。

## 四、去环技术演进总结

| 方案 | 核心思路 | 残留问题 | 适用场景 |
|:-----|:---------|:---------|:---------|
| 旧版 `pot/0.12` | 双维度硬 XOR | 密集同心环 (bullseye) | 仅对照 |
| 取模 `pot%1/0.5` | 势函数截断 | 近边界轻微带 | 实验 |
| `sin(pot)` 波 | 平滑调制 | 辐射波纹 | 柔和外晕 |
| **角度边缘检测** | 完全弃用势函数 | 无（最佳） | ✅ 满意版 |

**结论**：同心环的本质是"势函数维度在全量程上的硬量化"。彻底去掉该维度（改用角度场边缘检测）才能得到参考图「树状连线.png」那样的纯净三岔分形连线。
