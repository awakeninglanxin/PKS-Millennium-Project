# UF18 多陷阱形状 (Multi-Trap) — 正M vs 逆M

## 公式

$$d_{\min} = \min_{n, k} \text{dist}(z_n, \text{trap}_k)$$

同时追踪迭代轨道 $z_n$ 到多个几何陷阱的距离, 取所有陷阱全局最小值。陷阱形状包括: 心形曲线、圆形、水平/竖直线。

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 轨道追踪 | $z_n$ 全序列 | **相同** |
| 陷阱集 | 心形 + 圆 + 十字线 | **相同** |
| 距离 | `d_min = min dist(z_n, all traps)` | **相同** |
| 色板 | `d_min` → twilight | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸 = `ext`) |
| **未着色区域** | 集合内部 (黑) | 水滴外部 (黑) |

## 视觉效果

- 多种陷阱形状混合: 心形纹理、圆形年轮、十字交错
- 在轨道密集区域形成复合纹理
