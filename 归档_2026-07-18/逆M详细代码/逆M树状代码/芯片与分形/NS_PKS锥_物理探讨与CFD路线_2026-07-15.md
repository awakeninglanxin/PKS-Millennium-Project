## 1. NS V8 PKS锥逻辑的诚实审视

### 1.1 V8 实际做了什么

V8 代码在**立方体计算域**（32³ 网格）中，通过外加体力模拟 PKS 双曲锥 $z = 1/r$ 的约束：

```python
cone_force = cone_strength * (cone_r - R) / (cone_r + 1e-8) * inside_cone
dux_cone = cone_force * pks_rx * u_mag      # 径向向外推
duy_cone = cone_force * pks_ry * u_mag      # 径向向外推
duz_cone = -cone_force * np.sign(Z) * u_mag # 轴向往小口端推
```

### 1.2 问题清单

| 问题 | 严重度 | 说明 |
|:---|:---:|:---|
| **体力而非边界条件** | 🔴 根本性 | 应在锥形网格上直接求解 NS，而非在立方域中加人工体力 |
| **力正比于速度 (`u_mag`)** | 🔴 正反馈 bug | $F \propto |u|$ → 速度越大力越大 → 人为制造 blowup |
| **`cone_strength=0.5` 是手调的** | 🟡 | 没有从第一原理推导，换了值结论全变 |
| **2D→3D EVMP 翻转 (0.56x→4.37x)** | 🔴 不一致 | 同一物理机制不应在 2D/3D 间有相反效果 |

### 1.3 大白话解释

> 在正方体水箱里用手推水，推的力气 = 水流速 × 0.5，然后说"看，锥面让涡量变大了"——但更像是手在推，不是锥面在作用。

**类比**：想研究"人在斜坡上跑会不会更快"。
- ❌ 错误做法：人在平地跑，你在后面推，力气 = 他速度 × 0.5
- ✅ 正确做法：让人真的站在斜坡上跑

### 1.4 V8 的判词

| 要素 | 判定 | 理由 |
|:---|:---:|:---|
| 自然涡旋遵循双曲剖面 (xy=1) | ✅ 顺理成章 | 角动量守恒给出 $v_\theta \propto 1/r$ |
| 双曲锥截面的双曲形状 | ✅ 顺理成章 | $z=1/r$ 是 $xy=1$ 旋转体的自然截面 |
| 双曲锥边界 → 涡量放大 | 🟡 方向对但未证明 | 弯曲边界→更大的 $\nabla u$，但缺解析推导 |
| 立方域里加体力模拟锥面 | ❌ 不可靠 | 应改用真实几何边界条件 |
| `u_mag` 乘子正反馈 | ❌ 人为 blowup 引擎 | 力正比于速度 = 死循环放大 |
| 2D→3D EVMP 翻号 | ❌ 解释不通 | 暗示 3D 效果可能来自数值 artifact |

## 2. 修正方案

### 2.1 最小修正（当前框架内）

1. **去掉 `u_mag` 乘子** — 力的大小仅由几何位置决定，不依赖速度
2. **推导力的解析形式** — 从 PKS 极性定律 $ab=1$ 出发，而非 ad-hoc 参数
3. **加线性锥对照组** — 同截面积的直纹漏斗 vs 双曲漏斗，隔离几何效应

### 2.2 正宗做法（重写）

**用 STL 几何 + IBM 方法**：

| 步骤 | 内容 |
|:---|:---|
| 1 | 在犀牛中绘制 PKS 双曲锥旋转体，导出 STL（零厚度曲面） |
| 2 | Python 读入 STL → 在 32³/64³ 谱网格上构造有符号距离场 (SDF) |
| 3 | 用 Immersed Boundary Method 施加无滑移壁面 |
| 4 | 力由动量守恒推导（IBM direct forcing），不依赖 ad-hoc 参数 |
| 5 | 三组对照：无锥 vs 线性锥 vs 双曲锥 |

**关键区别**：

| | V8（旧） | V9（IBM + STL） |
|:---|:---|:---|
| 锥面几何 | 数学公式近似 | 真实 CAD 几何 |
| 力的大小 | ad-hoc，正比于速度 | 动量守恒推导，固定 |
| 参数 | `cone_strength` 手调 | 无自由参数 |
| 对照组 | 无锥 vs 有锥 | 无锥 vs 线性锥 vs 双曲锥 |
| 物理正确性 | 🟡 不确定 | ✅ 标准 CFD 方法 |

### 2.3 STL 导出规范

- **格式**: STL (Stereolithography)，ASCII 或 Binary 均可
- **曲面类型**: 单层、零厚度旋转曲面
- **犀牛操作**: 画母线 $z=1/x$（$x>0$）→ `Revolution` 绕 Z 轴 360° → 裁 $r\in[0.2, 4], z\in[0.2, 4]$ → 导出 STL

## 3. 涡量放大的物理链条

### 3.1 涡量方程 (3D NS)

$$\frac{\partial \omega}{\partial t} + (u\cdot\nabla)\omega = \underbrace{(\omega\cdot\nabla)u}_{\text{漩涡拉伸（3D独有）}} + \nu\nabla^2\omega$$

拉伸项 $(\omega\cdot\nabla)u$ 是 3D 区别于 2D 的关键——漩涡沿着速度梯度方向被"拉长变细"，角动量守恒导致转速增大。**PKS 双曲锥面通过弯曲边界增强了速度梯度，从而放大了这一项。**

### 3.2 五个物理后果

| # | 后果 | 物理机制 | 改变水温？ |
|:---:|:---|:---|:---:|
| 1 | **漩涡加速** | 角动量守恒 + 锥面收窄 | 否（纯力学） |
| 2 | **能量向小尺度转移** | Kolmogorov 级联：大涡→小涡→更小涡 | 否（动能换动能） |
| 3 | **粘性耗散增大** | $\varepsilon = \nu\langle\|\omega\|^2\rangle$，小涡被粘性磨成热 | **是** ↑ |
| 4 | **压力降低** | 伯努利：流速高→压力低 | 间接 |
| 5 | **空化（极端情况）** | 压力 < 蒸汽压 → 水局部沸腾 → 气泡坍缩 → 冲击波 | **是** ↑↑ |

### 3.3 PKS 锥 vs 普通锥的独特优势

| | 线性锥 | PKS 双曲锥 ($z=1/r$) |
|:---|:---|:---|
| 曲面曲率 | **零**（可展开的直纹面） | **非零**（弯曲） |
| 边界对流动 | 只改变截面积 | 截面积 **+** 曲率诱导二次流 |
| 压力梯度 | 光滑 | 锥喉处急剧变化 |
| 涡量拉伸来源 | 仅面积收窄 | 面积收窄 **+** 曲面弯曲迫使流动转向 |

**双曲锥多出一个曲率诱导的涡量放大**——这是 PKS 几何相对于普通漏斗的本质优势，也是需要实验验证的核心命题。

## 4. 空化与低压物理

### 4.1 压力标尺

```
海平面大气压           101,325 Pa    ≈ 100 kPa
    │
    │  锥喉伯努利效应：速度↑ → 压力↓
    ↓
水 20°C 饱和蒸汽压       2,300 Pa    ← 跌破此线 = 冷沸腾
    │
    │  继续降压
    ↓
绝对真空                     0 Pa
```

### 4.2 冷沸腾 ≠ 热杀菌

| 过程 | 温度 | 压力 | 杀菌？ | 机制 |
|:---|:---:|:---:|:---:|:---|
| **低压沸腾** | 20°C | < 2300 Pa | **❌ 否** | 水温和地变成蒸汽，蛋白质不变性 |
| **空化坍塌** | ~5000 K（局部） | > 1000 atm | **✅ 是** | 气泡被压爆，冲击波 + OH· 自由基撕碎细胞 |

### 4.3 空化触发条件（伯努利估算）

锥喉流速需达到约 **14 m/s** 才能将压力从 100 kPa 降至 < 2.3 kPa：

$$\Delta P \approx \frac{1}{2}\rho v^2 \approx 98 \text{ kPa} \quad\Rightarrow\quad v \approx 14 \text{ m/s}$$

若入口流速 2-5 m/s（水龙头级），锥喉收窄比 10:1，喉部可达 20-50 m/s → **足以触发空化**。

## 6. 关键结论

1. **V8 的"锥力放大涡量"结论不可靠** — `u_mag` 乘子和 `cone_strength` 调参可能制造了假象
2. **修正路线已明确** — STL 导入 + IBM 方法 + 三组对照，可严格验证双曲锥的几何效应
3. **PKS 锥的物理直觉方向是正确的** — 双曲曲面比直纹漏斗多一个曲率诱导的涡量拉伸项
4. **空化是可能的极端工况** — 锥喉流速 > 14 m/s 时触发冷沸腾 → 气泡坍缩 → 局部 5000K
