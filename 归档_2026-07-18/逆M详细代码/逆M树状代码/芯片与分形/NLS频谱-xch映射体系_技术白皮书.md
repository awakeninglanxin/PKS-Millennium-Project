## 摘要

本研究通过对NLS Metatron GR Hunter系统的xch二进制文件和频谱图截图的深度分析，建立了完整的"xch患者特异性值 → 频谱图N/S值"的数学映射体系。核心发现包括：

1. **W标准库公式**: `W ≈ 0.30 + 0.40×(N̄-S̄) + 0.30×(1-corr(N,S))` (R²=0.770)
2. **频谱-xch转换公式**: `频谱值 ≈ 0.45 × xch + b` (W<0.6时a=0.45固定)
3. **a-W函数关系**: a系数与W值正相关，W>1.0时a变大
4. **患者特异性数据提取**: 两人对比法从93.8%标准库噪音中提取真实患者数据

### 2.2 四大核心公式

#### 公式1: W标准库计算

```
W ≈ 0.30 + 0.40×(N̄ - S̄) + 0.30×(1 - corr(N,S))

其中:
  N̄ = N曲线9点均值
  S̄ = S曲线9点均值
  corr(N,S) = N与S的皮尔逊相关系数
```

**物理含义**: W = 频谱差异化度量。正常组织(N≠S, 形态差异大)→W高；病变组织(N≈S, 趋同)→W低。

#### 公式2: 频谱-xch转换 (W<0.6时)

```
频谱值 ≈ 0.45 × xch患者特异性值 + b

其中:
  a = 0.45 ± 0.04 (系统放大常数, W<0.6时固定)
  b = 3.2 ~ 3.6 (器官基线偏移, 可能与W相关: b/W ≈ 5.6-6.6)
  xch = 患者组织与标准库etalon的偏差度量 (0~2范围)
```

**物理含义**: NLS频谱图 = xch偏差 × 0.45放大 + b偏移 后的可视化呈现。

#### 公式3: a-W函数关系 (初步)

```
a ≈ 0.45 × (1 + k×(W - 0.5))

其中:
  W < 0.6: a ≈ 0.45 (k≈0, 稳定区)
  W = 0.6~0.8: a ≈ 0.5~0.9 (过渡区, k>0)
  W > 1.0: a ≈ 0.9~1.5 (高W区, k较大)
  k值待过渡区数据确定
```

#### 公式4: xch插值法 (配对算法)

```
将m个xch值线性插值到9个点:
  xch_interp[i] = xch_sorted[lo]×(1-frac) + xch_sorted[hi]×frac
  
其中:
  t = i / 8 × (m - 1)
  lo = int(t), hi = min(lo+1, m-1)
  frac = t - lo

然后与9个排序后的频谱值全量线性回归。
```

## 三、技术方法论

### 3.1 频谱图数据提取

从NLS软件截图中提取：
- N曲线9点值 (蓝色输入信号)
- S曲线9点值 (红色组织响应)
- W标准库值 (右上角显示)

### 3.2 xch患者特异性值提取

**两人对比法**:
1. 解析两人xch文件 (xch_parver.py)
2. 过滤大垃圾值: `abs(v) < 50000` (batch_parse.py)
3. 对比两人共有参数: 93.8%相同=标准库值, 6.2%不同=患者特异性
4. 有效值过滤: `0 <= v <= 6`
5. 结果: 92个患者特异性差异参数

### 3.3 频谱-xch配对算法

**排序配对法** (原始方法, 有偏差):
1. xch值按大小排序 ↔ 频谱值按大小排序
2. 从9个频谱值中选n个(n=xch数量), 使MSE最小
3. ⚠️ 铁律70: 当n远小于9时, 会去掉极端频谱值→压缩Y范围→a被压低

**xch插值法** (修正方法, 推荐):
1. 将n个xch值线性插值到9个点
2. 9个xch插值点排序 ↔ 9个频谱值排序
3. 全量线性回归
4. ✅ 无子集选择偏差

### 3.4 线性回归

手工实现高斯消元法 (不依赖numpy):
```
y = a×x + b
a = cov(x,y) / var(x)
b = ȳ - a×x̄
MSE = Σ(yi - a×xi - b)² / n
R² = 1 - SS_res / SS_tot
```
