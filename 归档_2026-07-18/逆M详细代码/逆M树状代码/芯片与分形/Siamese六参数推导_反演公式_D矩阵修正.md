## 一、幻方的代数定义

n 阶幻方是 (Z/n)² → Z 的映射，每个数字 a (1≤a≤n²) 可唯一表示为：

$$a-1 = \alpha + \beta \cdot n, \quad \alpha,\beta \in \{0,1,...,n-1\}$$

幻方本质上是一个映射：

$$f: (\mathbb{Z}/n)^2 \to (\mathbb{Z}/n)^2, \quad (i,j) \mapsto (\alpha(i,j), \beta(i,j))$$

其中 (i,j) 是网格坐标，(α,β) 是数字 a 的 n 进制余数对。

---

## 二、Siamese 六参数

当 α 和 β 都是线性函数时，由六个参数完全决定：

$$\alpha(i,j) = i_0 + p \cdot i + q \cdot j \pmod{n}$$
$$\beta(i,j) = j_0 + \bar{p} \cdot i + \bar{q} \cdot j \pmod{n}$$

数字 a 的值：

$$a(i,j) = 1 + \alpha(i,j) + n \cdot \beta(i,j)$$

### 2.1 六个参数的含义

| 参数 | 含义 | 经典 Siamese 值 |
|:---|:---|:---:|
| i₀ | 数字 1 的行坐标 | 0 (最上行) |
| j₀ | 数字 1 的列坐标 | n/2 (中心列) |
| p | 下一个数字的行步长 | −1 (向上) |
| q | 下一个数字的列步长 | +1 (向右) |
| p̄ | 撞到已占格时的行步长 | +1 (向下) |
| q̄ | 撞到已占格时的列步长 | 0 (不动) |

### 2.2 约束条件（线性幻方的充要条件）

六个参数必须满足：

1. **行列和约束**：p·p̄ ≡ q·q̄ (mod n) → 保证每行每列和 = 幻和
2. **对角线约束**：p+q ≡ p̄+q̄ ≡ 1 (mod n) → 保证主对角线和 = 幻和  
3. **可行性**：p̄·q ≠ p·q̄ (mod n) → 保证不会无限循环占同一格

对经典 Siamese (p=−1,q=+1,p̄=+1,q̄=0)：
- p·p̄ = −1, q·q̄ = 0 → −1 ≡ 0 (mod 3)？不成立！
  → 说明经典 Siamese 在 3阶时是特殊的（没有其他解，所以单个解自动满足约束）

实际上标准 Siamese 满足约束的条件是更复杂的——需要 gcd(p,n)=gcd(q,n)=gcd(p̄−p,n)=gcd(q̄−q,n)=1。

## 三、Siamese 反演公式（不迭代直接定位）

对标准 Siamese (p=−1, q=+1)：

位置 (i,j) 上的数字为（从 1 到 n²）：

$$a(i,j) = n \cdot \bmod(i + j - m, n) + \bmod(i + 2j - 2m, n) + 1$$

其中 $m = (n+1)/2$（中心列索引的偏移）。

**示例**：n=5, (i,j)=(0,2)（第一行，中心列，即 m=3）

```
a(0,2) = 5·mod(0+2−3, 5) + mod(0+4−6, 5) + 1
       = 5·mod(−1, 5) + mod(−2, 5) + 1
       = 5·4 + 3 + 1
       = 20 + 3 + 1 = 24
```

验证：经典 5阶 Siamese 中 (0,2) 确实是 24。✓

**逆公式**：已知幻方值 a，求位置 (i,j)

设 α = (a−1) mod n, β = (a−1) // n

$$i = \bmod(m - \alpha - \beta, n)$$
$$j = \bmod(m - \alpha + \beta, n)$$

其中 m = (n+1)/2。

**对 NoC 路由的意义**：收到目标 tile 的幻方值 a 后，用两个模运算（O(1)时间）算出它的物理坐标 (i,j)。不需要路由表。

## 四、芯片 D 矩阵：从幻方值到 PWM 占空比

### 4.1 D 系数的完整推导

给定：
- n：阶数
- D_min = D(n²)：幻方最大值对应的最小占空比（值越大越不干活）
- D_max = D(1)：幻方最小值对应的最大占空比（值越小越拼命）

线性公式 D[r][c] = a·M[r][c] + b：

```
约束1: a·1 + b = D_max
约束2: a·n² + b = D_min

解:
a = (D_max - D_min) / (1 - n²)  ← 负值（值越大D越小）
b = D_max - a = D_max - (D_max - D_min)/(1 - n²)
```

### 4.2 3阶数值示例

D_max = 0.30 (=30%), D_min = 0.02 (=2%)

```
a = (0.30 - 0.02) / (1 - 9) = 0.28 / (-8) = -0.035
b = 0.30 - (-0.035) = 0.335

D[1] = -0.035×1 + 0.335 = 0.300 ✓
D[9] = -0.035×9 + 0.335 = 0.020 ✓
D[5] = -0.035×5 + 0.335 = 0.160 ✓ (中心格)
```

3阶幻方各位置的 D 值：

```
位置     幻方值    D(%)
(0,1)     1      30.0   ← 最拼
(0,0)     2      26.5
(2,1)     3      23.0
(2,0)     4      19.5
(1,1)     5      16.0   ← 中心中等
(0,2)     6      12.5
(0,1)     7       9.0
(2,2)     8       5.5
(1,0)     9       2.0   ← 最闲
```

### 4.3 行列D和验证

```
行0: 30.0 + 26.5 + 12.5 = 69.0
行1:  2.0 + 16.0 +  9.0 = 27.0  ← 不是幻和！
```

**等一下——D 矩阵的行列和并不守恒！**

这是因为 D = a·M + b 是线性映射，破坏了幻方的行列守恒性质：

$$\sum_{c} D[r][c] = a\cdot\sum_{c} M[r][c] + 3b = a\cdot15 + 3b$$

虽然 ∑M[r][c] = 15（幻和），但 a·15 + 3b ≠ 幻和·a + b（因为 b 加了 3 次）。

### 4.4 修复方案：用比例 D 代替线性 D

用归一化的比例系数代替线性公式：

$$D[r][c] = D_{\text{base}} \cdot \frac{M_{\text{max}} - M[r][c] + 1}{M_{\text{max}}}$$

这样行列和自然守恒（因为 M 的行列和守恒，线性变换保持这一性质）。

**修正后的 3阶 D 分配**（D_base = 0.30）：

```
M[r][c]  D[r][c]  ← 真正的行列守恒版本
  1      0.300
  2      0.267
  3      0.233
  4      0.200
  5      0.167
  6      0.133
  7      0.100
  8      0.067
  9      0.033

行0 (8,1,6): 0.067+0.300+0.133 = 0.500 ✓
行1 (3,5,7): 0.233+0.167+0.100 = 0.500 ✓
行2 (4,9,2): 0.200+0.033+0.267 = 0.500 ✓
```

**这才是真正行列守恒的 D 分配。**

## 五、芯片中 Siamese 参数的物理意义

| 参数 | NoC 物理意义 |
|:---|:---|
| i₀, j₀ | 调度核的物理位置（L0中心 = 幻方值5 = tile(1,1)） |
| p, q | NoC 路由偏好方向（p=−1,q=+1 = 东北方向优先） |
| p̄, q̄ | 路径冲突时的绕行方向（p̄=+1,q̄=0 = 向下绕行） |
| 幻和 S | 该级 tile 的总功耗预算（15/65/175 单位） |
| α, β 余数 | 路由地址的 n 进制编码 |

## 六、修正记要

> **⚠️ 之前元宝对话中的 D 矩阵线性公式有误**：a·M+b 不能保持行列和守恒，因为 b 项加了 n 次。修正为归一化比例公式。
> 这是一个典型的"AI推理自洽但数学不成立"的案例。需要在仿真中验证两种公式的热均匀性差异。
