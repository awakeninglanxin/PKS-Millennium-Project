## 一、物理层特征（Cole-Cole 介电指纹）

### 核心公式
```
ε*(ω) = ε_∞ + Σᵢ Δεᵢ/(1 + (jωτᵢ)^(1-αᵢ)) + σ_s/(jωε₀)
```

你1.4-4.9GHz能稳拟合的参数:
- **γ极** Δε₂ + τ₂（水含量+自由离子 主贡献）—— 最稳
- **β尾** 残根（膜面积 残存信号）—— 1.4GHz起跳只剩尾巴

### 提取的4极参数 → 每通道5-9维parametric向量
```
Cole_left = [Δε₂_L, τ₂_L, α₂_L, Δε₁_L, τ₁_L, ...]
Cole_right = [Δε₂_R, τ₂_R, α₂_R, ...]
ΔCole = |Cole_left - Cole_right|  → 甲状腺占位时ΔCole拉大
```

## 二、深度模型层（三种联合架构）

### ① DFARK — 双通道频点自适应残差核
```
输入: [2, N, 351, 2]  (ch1+ch2 × N方位角 × 351频点 × 2档功率)
架构: Conv3d(通道维2 × 方位角维3 × 频点维自适应5-15)
      → 通道间差卷积 + 频点轴Self-Attention
物理hook: ch1左耳/ch2右耳 → 跨通道差分捕获甲状腺单侧占位的不对称
```

### ② PG-GNN — 物理图神经网络（双图+跨通道边）
```
改法1(只有S11):
  每通道独立建图 → 节点=频点, 边权=|∂ε/∂f| Cole-Cole
  加跨通道边: ch1_freq_i ↔ ch2_freq_i
  边权 = 几何衰减模型(耳间距15-20cm, 组织ε锚定)
  图级读出 = Readout(ch1_GNN) ∥ Readout(ch2_GNN) ∥ Readout(cross_GNN)

改法2(有S21):
  节点特征扩为 [|S11_ch1|, ∠S11_ch1, |S11_ch2|, ∠S11_ch2, |S21|, ∠S21]
  加跨模态边 S11↔S21↔S11 → 单端+双端混合约束

融合: Cole_Δ + DFARK_feat + GNN_readout → XGBoost 分类
```

### ③ P-SFS — 物理引导敏感频点筛选
```
原版: 单通道轻量CNN → 挑Top-K频点(351→20)
双通道版: 联合敏感频点——损失同时考虑ch1/ch2梯度
  → 挑"ch1和ch2差异最大"的频点
物理动机: 癌变在γ上升沿(3-4.5GHz) 水↑ → 同侧S11强/对侧衰减后弱
  → 差异最大频点正好在γ上升沿 → 与Cole-Cole self-consistent
```

### ④ 功率非线性特征（新增）
```
NI(f) = |S11(f, P_high) - S11(f, P_low)| / |S11(f, P_low)|
→ NI(f)曲线 喂1D-CNN 或 并入PG-GNN节点特征
俄系"非线性"的真物理hook可能在这里
```

## 三、Ch1/Ch2通道验证（关键！）

元宝反复追问一个硬件问题——**你的手环是真双通道同时激励还是固件时分虚拟通道？**

| 验证方法 | 真双通道 | 虚拟双通道 |
|----------|----------|------------|
| 同时开ch1=2G ch2=4G看频谱仪 | 两峰同时 | 只有时分残影 |
| ch1/ch2功率独立调，互调产物IM3 | 能测到 | 无 |
| 硬件框图有无两颗VCO/PLL | 双/真双输出 | 单VCO时分 |

**决定路线**:
- 真双通道 → 形态C(quasi-bistatic S21) + DFARK + 跨通道GNN全吃满
- 虚拟双通道 → 只能形态A(左右耳S11时分) + 通道间差分(噪声底比真双通道高)

## 四、甲状腺首发最小协议（双通道版）

```
硬件层:
  ch1=左耳, ch2=右耳
  每侧: 16方位角 × 351频点 × 2档功率
  输出: ch1_S11[16,351,2], ch2_S11[16,351,2]

物理层:
  Cole-Cole 4极拟合 → 左右各5-9维 → 左右差ΔCole

DL流:
  DFARK([2,16,351,2]→通道间差卷积+频点self-attn)
  + PG-GNN 双图+跨通道边
  + P-SFS 联合挑Top-20频点
  + 融合 Cole_Δ + DFARK + GNN_readout → XGB

预期AUC: 0.80-0.85
(甲状腺单侧占位的不对称杠杆 > MammoWave垂杯双侧对称)
```
