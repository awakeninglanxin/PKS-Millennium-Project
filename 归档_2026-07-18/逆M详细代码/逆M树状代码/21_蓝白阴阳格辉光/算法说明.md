# UF21 蓝白阴阳格辉光 — 正M vs 逆M

## 算法组合

此算法为混合渲染: 将 UF1(二分棋盘格) 的 XOR 网格与 UF4(DEM距离估计) 的金色边界辉光组合, 并采用蓝白配色。

## 核心公式

### 网格着色 (水滴外部)

$$\text{chess} = \lfloor \theta \cdot N_{\text{ang}} \rfloor \% 2 \oplus \lfloor \log_2(\text{dist}) / \Delta \rfloor \% 2$$

- $\theta$ = 复平面上点的极角
- $\text{dist}$ = 点到原点的对数距离 ($\log_2(|c|)$)
- $N_{\text{ang}}$ = 角度分区数 (ANG_DIV), $v4c=240$ 细100倍
- $\Delta$ = 距离环间距 (DIST_LOG_STEP), $v4c=0.035$

### 边界辉光 (水滴边界)

$$d = \log(|z|^2) \cdot \frac{|z|}{|dz|},\quad \text{glow} = 1 - \text{clip}(d, 0, 1)$$

金橙色渐变沿集合边界薄层发光。

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 棋盘 XOR | `⌊θ·N⌋%2 ⊕ ⌊log₂(dist)/Δ⌋%2` | **相同** |
| DEM 辉光 | `d = log∣z∣²·∣z∣/∣dz∣` | **相同** |
| 辉光颜色 | 橙→金色 | **相同** |
| **网格区域** | 集合外部 (逃逸) | 水滴**外部** (逃逸 = `ext`) |
| **内部颜色** | 集合内部 (暗) | 水滴内部 (深蓝纯色) |

## 版本迭代

| 版本 | 文件 | 说明 |
|:---|:---|:---|
| v1 | `main.py` | 初版, 矩形 XOR 棋盘 (太细密) |
| v2 | `main_v2.py` | 纯极坐标扇形 (有扇形缺环) |
| v4 | `main_v4.py` | XOR 阴阳格 + 环 (着色于水滴内部) |
| v4b | `main_v4b.py` | XOR 改到水滴外部 (CHESS_MODE='A'|'B'|'C') |
| **v4c** | **`main_v4c.py`** | **ANG_DIV=240, DIST_LOG_STEP=0.035** → 瓷砖细100倍 |

## 关键参数 (v4c 细100倍版)

| 参数 | 值 | 说明 |
|:---|:---|:---|
| ANG_DIV | 240 | 角度分区 (原 v4: 24, 细10倍) |
| DIST_LOG_STEP | 0.035 | 距离环间距 (原 v4: 0.35, 细10倍) |
| bailout | 50 | 逃逸半径 |
| MAXITER | 200 | 最大迭代 |
| DEM_SCALE | 20 | 辉光强度 |
