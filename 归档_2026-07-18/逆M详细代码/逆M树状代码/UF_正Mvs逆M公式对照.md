## 算法公式对照

### 1. Smooth (Mandelbrot) — 连续势能平滑

| | 正M | 逆M (UF2) |
|:---|:---|:---|
| C | `c = pixel` | `c_eff = \|c\|^{-1}e^{-i⋅arg(c)}` |
| 迭代 | `z = z² + c` | `z = z² + c_eff` |
| 势能 | `ν = n - log₂(log₂\|z\|)` | **相同** |
| 着色区域 | 集合外部 (所有逃逸点) | 水滴**内部** (所有逃逸点) |
| 视觉 | 外部多色渐变 | 水滴内部 plasma 色板 |

### 2. Binary Decomposition — 二分棋盘格

| | 正M | 逆M (UF1) |
|:---|:---|:---|
| 角度 | `θ = arg(z)/(2π·2ⁿ) mod 1` | **相同** |
| 势能 | `pot = log\|z\| / 2ⁿ` | **相同** |
| 棋盘格 | `(⌊θN⌋%2) == (⌊pot/Δ⌋%2)` | **相同** |
| 着色区域 | 集合外部 | 水滴**内部** |
| 未着色 | 集合内部 (黑) | 水滴外部 (白) |

### 3. Decomposition — 分解着色

| | 正M | 逆M |
|:---|:---|:---|
| C | `c = pixel` | `c_eff = 1/c` |
| 角度 | `θ = arg(z)/(2π)` | **相同** |
| 着色 | 按 θ 分6区 | **相同** |
| 着色区域 | 外部 | 水滴**内部** |

### 4. Distance Estimator — 距离估计 (DEM/M)

| | 正M | 逆M (UF4) |
|:---|:---|:---|
| 导数 | `dz = 2z·dz + 1` | **相同** |
| 距离 | `d = log\|z\|² ⋅ \|z\| / \|dz\|` | **相同** |
| 映射 | `index = -k·log(d)` | **相同** |
| 着色区域 | 集合边界 (所有逃逸点) | 水滴**边界** (逃逸点) |
| 内部 | 集合内部 (不参与) | 水滴外部 (不参与) |

### 5. Orbit Traps — 轨道陷阱

| | 正M | 逆M (UF5) |
|:---|:---|:---|
| 轨道 | 追踪 `z_n` | **相同** |
| 陷阱形状 | 圆/线/网格 | **相同** |
| 距离 | `d_min = min dist(z_n, trap)` | **相同** |
| 着色区域 | 外部 (轨道穿过陷阱) | 水滴**内部** |
| 内部 | 集合内部 (无轨道) | 水滴外部 (中灰) |

### 6. Exponential Smoothing — 指数平滑

| | 正M | 逆M |
|:---|:---|:---|
| 指数 | `ratio = (log\|z\|)/(2^n)` | **相同** |
| 平滑 | `s = n - log₂(log\|z\|)` | **相同** |
| 着色区域 | 集合外部 | 水滴**内部** |

### 7. Lighting — 光照效果

| | 正M | 逆M |
|:---|:---|:---|
| 法线 | 由 `dz` 计算梯度 | **相同** |
| 光源 | 指定方向 | **相同** |
| 着色 | 漫反射 + 高光 | **相同** |
| 着色区域 | 集合外部 | 水滴**内部** |

### 8. Emboss — 浮雕效果

| | 正M | 逆M |
|:---|:---|:---|
| 梯度 | Sobel 检测跳变 | **相同** |
| 着色 | 灰度 = 梯度方向 | **相同** |
| 着色区域 | 集合边界 | 水滴**边界** |

## 核心结论

在 $c \to 1/c$ 反演下, 所有着色算法的数学公式不变, 但**着色区域发生内外反转**:

```
正M: 集合外部 (逃逸) → 彩色/纹理/渐变
     集合内部 (不逃逸) → 黑/白/单色

逆M: 水滴内部 (逃逸) = 正M外部 → 彩色/纹理/渐变
     水滴外部 (不逃逸) = 正M内部 → 黑/白/单色
```

| 图像区域 | 正M | 逆M |
|:---|:---|:---|
| 心形体内 | 逃逸 → 着色 | **未逃逸 → 单色** |
| 心形体外 | 未逃逸 → 单色 | **逃逸 → 着色** |

这就是为什么所有 UF 算法在逆M中"翻转"了。
