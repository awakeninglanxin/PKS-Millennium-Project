# UF6 拓扑交错瓦片 — 正M vs 逆M 对照

## 正M公式

$$u = \lfloor \theta \cdot N \rfloor,\quad v = \left\lfloor \frac{\text{pot}}{\Delta} \right\rfloor,\quad i = (u + v) \bmod K$$

## 逆M公式

所有公式相同, 仅着色区域内外反转。

## 对照表

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:---|:---|:---|
| 复数映射 | `c = pixel` | `c_eff = 1/c` |
| 外部角度 | `θ = arg(z)/(2π·2ⁿ) mod 1` | **相同** |
| 势能 | `pot = log\|z\|/2ⁿ` | **相同** |
| 索引 | `i = (⌊θN⌋ + ⌊pot/Δ⌋) % K` | **相同** |
| 色板 | K=4 级灰度 | **相同** |
| **着色区域** | 集合**外部** | 水滴**内部** |
| **未着色区域** | 集合内部 (白) | 水滴外部 (白) |

### 与 UF1 的区别

| | UF1 二分棋盘格 | UF6 拓扑交错 |
|:---|:---|:---|
| 判定 | XOR `(u%2)==(v%2)` → 2色 | `(u+v)%K` → **K色** |
| 色数 | 2 | K=4 |
| 镶嵌 | 黑白交替 | 多层嵌套 |

### 视觉效果差异

- **正M**: K色镶嵌在集合外部扩散
- **逆M**: 1/c反演压缩进水滴内部 → 多层弧形穹顶感
