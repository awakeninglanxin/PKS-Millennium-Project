# UF16 渐变着色 (Gradient) — 正M vs 逆M

## 公式

$$\nu = n - \log_2(\log_2\|z\|)$$

将平滑势能 $\nu$ 归一化后映射到 HSV 彩虹色板, 形成从边界到远处的连续渐变。

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 势函数 | `ν = n − log₂(log₂∣z∣)` | **相同** |
| 归一化 | `ν → [0,1]` → HSV | **相同** |
| 色板 | HSV 彩虹 | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸 = `ext`) |
| **未着色区域** | 集合内部 (黑) | 水滴外部 (黑) |

## 视觉效果

- 彩虹渐变, 从边界暖色(红/橙)向深层冷色(蓝/紫)过渡
- ν 小(靠近边界) → 暖色; ν 大(远离边界) → 冷色
