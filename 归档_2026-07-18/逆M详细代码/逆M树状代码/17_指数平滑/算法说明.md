# UF17 指数平滑 (Exponential Smoothing) — 正M vs 逆M

## 公式

$$\nu = n - \log_2(\log_2\|z\|),\quad s = \exp(-\nu \times 0.3)$$

逃逸势能以指数速率快速衰减, 使外部区域迅速变暗, 仅边界附近保留高亮细节。

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 势函数 | `ν = n − log₂(log₂∣z∣)` | **相同** |
| 指数衰减 | `s = exp(−ν × 0.3)` | **相同** |
| 色板 | 灰度 `s → [0,1]` | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸 = `ext`) |
| **未着色区域** | 集合内部 (黑) | 水滴外部 (黑) |

## 视觉效果

- 边界锐利高亮, 远场迅速消隐
- 灰度单色, 衰减系数 0.3 控制消失速度
