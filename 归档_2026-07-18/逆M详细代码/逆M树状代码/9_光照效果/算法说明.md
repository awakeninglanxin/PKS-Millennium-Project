# UF9 光照效果 (Lighting) — 3D法线打光

## 正M vs 逆M 对照

| 对比项 | 正M | 逆M |
|:---|:---|:---|
| 映射 | `c = pixel` | `c_eff = 1/c` |
| 距离场 | DEM: `d = log\|z\|²·\|z\|/\|dz\|` | **相同** |
| 法线 | Sobel(gx,gy) → `n = (-gx,-gy)/|g|` | **相同** |
| 光源 | 方向向量 `(lx,ly)` | **相同** |
| 着色 | `shade = clamp(n·l, 0, 1)` | **相同** |
| **着色区域** | 集合**外部** | 水滴**内部** |

### 视觉效果

- 仿3D光照效果: 边界凸起, 外部凹陷
- 灰色调, 无颜色
