# UF2 连续势能流光 — 正M vs 逆M 对照

## 正M公式

$$z_{n+1} = z_n^2 + c,\quad \nu = n - \log_2(\log_2\|z\|)$$

## 逆M公式

$$z_{n+1} = z_n^2 + c_{\text{eff}},\quad c_{\text{eff}} = c^{-1}$$

$$\nu = n - \log_2(\log_2\|z\|) \quad \text{(与正M相同)}$$

## 对照表

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:---|:---|:---|
| 复数映射 | `c = pixel` | `c_eff = 1/c` |
| 迭代公式 | `z = z² + c` | **相同** `z = z² + c_eff` |
| 势函数 | `ν = n - log₂(log₂\|z\|)` | **完全相同** |
| 平滑作用 | 抹平 |z|跃变阶梯 | **相同** |
| **着色区域** | 集合**外部** (逃逸) | 水滴**内部** (逃逸) |
| **未着色区域** | 集合内部 (黑) | 水滴外部 (黑) |
| **ν 大 →** | 远离集合 (外部亮) | 水滴深处 (plasma亮色) |
| **ν 小 →** | 靠近边界 (暗) | 靠近水滴边缘 (暗色轮廓) |

### 视觉效果差异

- **正M**: 外部亮色渐变 → 集合内部全黑
- **逆M**: 水滴内部 plasma 色板渐变 → 水滴外部全黑 (内外翻转)

### 着色 (逆M)

归一化 `ν` 到 [0,1] → matplotlib plasma 色板:

```python
norm = percentile_clip(ν, 2%, 98%)
img[ext] = plasma(norm)
```

### 参数

| 参数 | 值 |
|:---|:---|
| bailout | 50 |
| 色板 | plasma |
| 归一化 | 2%-98% 百分位 |
