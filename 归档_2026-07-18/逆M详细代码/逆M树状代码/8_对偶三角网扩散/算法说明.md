# UF8 对偶三角网扩散 — 正M vs 逆M 对照

## 算法定位

介于 UF1 (纯二分棋盘格) 和 UF2 (纯势能渐变) 之间的混合体:
- 水滴**内部**(逃逸): 高频棋盘格 XOR → 三角网 (UF1风格)
- 水滴**外部**(未逃逸): 深蓝聚光灯渐变 (UF2风格)
- 边界: 金色蕾丝过渡

## 对照表

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:---|:---|:---|
| 复数映射 | `c = pixel` | `c_eff = 1/c` |
| 迭代公式 | `z = z² + c` | `z = z² + c_eff` |
| **三角网区域** | 集合**外部** (XOR)→扩散 | 水滴**内部** (XOR)→挤压成三角 |
| **渐变区域** | 集合内部 (渐变) | 水滴**外部** (渐变) |
| 金边位置 | 集合**边界**外缘 | 水滴**边界**内壁 |
| RAY_DIV | 256 (高频) | **相同** |
| POT_STEP | 0.04 (细密) | **相同** |

### 三层渲染结构

| 层 | 逆M位置 | 正M对应 | 说明 |
|:---|:---|:---|:---|
| 内部(XOR三角网) | 水滴**内** | M集外部 | RAY_DIV=256 → 高频挤压→三角鳞片 |
| 外部(蓝渐变) | 水滴**外** | M集内部 | `\|z\|`模长 → 深蓝聚光灯 |
| 金边 | 水滴**边界** | M集边界 | `pot<0.05` → 橙金淡出 |

### 颜色

```python
# 外部渐变
img[~ext] = 深蓝渐变 (R=0.02-0.12, G=0.35-0.80, B=0.75-1.0)

# 内部三角网 (蓝紫基调)
img[ext] = bg * [0.08, 0.35, 0.65] + sin调制

# 金边
img[gold] = orange-gold blend: (R=1.0, G=0.65-1.0, B=0.08)
```

### 参数

| 参数 | 值 |
|:---|:---|
| RAY_DIV | 256 (极高频→三角网) |
| POT_STEP | 0.04 (细密等势) |
| bailout | 50 |
