# UF3 有理外部射线 — 正M vs 逆M 对照

## 正M公式

$$z_{n+1} = z_n^2 + c,\quad \theta = \frac{\arg(z_{\text{final}})}{2\pi \cdot 2^n} \pmod 1$$

## 逆M公式

$$z_{n+1} = z_n^2 + c_{\text{eff}},\quad \theta = \frac{\arg(z)}{2\pi \cdot 2^n} \pmod 1 \quad \text{(相同)}$$

## 对照表

| 对比项 | 正M (Standard) | 逆M (Inverted) |
|:---|:---|:---|
| 复数映射 | `c = pixel` | `c_eff = 1/c` |
| 迭代公式 | `z = z² + c` | `z = z² + c_eff` |
| 外部角度 | `θ = arg(z)/(2π·2ⁿ) mod 1` | **相同** |
| 射线检测 | `\|θN - round(θN)\| < ε` | **相同** |
| **射线位置** | 集合**外部** (向外辐射) | 水滴**内部** (向内压缩) |
| **背景区域** | 集合外部 (浅蓝) | 水滴内部 (浅蓝) |
| **内部** | 集合内部 (深色) | 水滴外部 (深色) |
| n范围 | 10-250 | 0-15 (小bailout导致早期逃逸) |

### 视觉效果差异

- **正M**: 射线从M集向外笔直辐射
- **逆M**: 1/c反演把外部射线压缩进水滴内部 → 射线弯曲折射 → 刺入心形体和周期泡芽

### 管线

```python
1. 追踪逃逸步数 n 和最终 z
2. θ = arg(z) / (2π · 2ⁿ) mod 1
3. ray = |θ·N - round(θ·N)| < ε
4. 水滴内部: 射线=深蓝, 背景=浅蓝
5. 水滴外部: 深色单色
```
