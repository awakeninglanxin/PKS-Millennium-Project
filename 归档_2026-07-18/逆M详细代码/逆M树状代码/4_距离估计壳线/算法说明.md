# UF4 距离估计壳线 DEM (逆M版本) — 生成逻辑

## 数学原理 (Mu-Ency DEM/M)

正M和逆M的DEM公式相同, 仅着色区域内外反转。

标准 Mandelbrot 迭代只追踪 $z_n$。DEM 同时追踪导数 $dz_n$:

$$z_{n+1} = z_n^2 + c_{\text{eff}}, \quad dz_{n+1} = 2 \cdot z_n \cdot dz_n + 1$$

在逆M空间中, $c_{\text{eff}} = c^{-1}$。

逃逸后的连续距离指数 (Mu-Ency):

$$d = \log(|z|^2) \cdot \frac{|z|}{|dz|} = 2 \ln|z| \cdot \frac{|z|}{|dz|}$$

### 关键: 连续光晕映射 (不是硬边界线)

不同于简单二值阈值, 真实 DEM 将 $d$ 映射为连续指数:

$$s = -\log(d)$$

$d$ 越小(越靠近边界) → $s$ 越大 → 越亮。形成从边界向外**逐渐衰减的光晕**。

### 着色逻辑

```python
# Mu-Ency 公式 (无 0.5 因子)
d = log(|z|²) * |z| / |dz|

# 连续光晕映射
s = clip(-log(d) / 3, 0, 1)

# 绿色渐变: 边界高亮
R = s * 0.15 + 0.02   # 暗红
G = s * 0.60 + 0.10   # 主绿色
B = s * 0.20 + 0.02   # 暗蓝

# 额外高亮细线: d < pixel_width → 亮白绿
stroke = white_green
```

### 与标准M对照

| | 标准 Mandelbrot | 逆 Mandelbrot |
|:---|:---|:---|
| c | pixel | `c^α α=-1` |
| 距离公式 | `d = log\|z\|²⋅\|z\|/\|dz\|` | **相同** |
| 光晕映射 | `s = -log(d/3)` | **相同** |
| 着色区域 | 集合**外部** (逃逸) | 水滴**内部** (逃逸) |
| 光晕位置 | 集合**外部**边界 | 水滴**内壁**边界 |
| 内部(不参与) | 集合内部 → 黑 | 水滴外部 → 黑 |

### 视觉效果

- **黑色背景**: 经典 DEM 风格
- **绿色光晕**: 从边界向外辐射状衰减, 精细结构清晰可见
- **亮白色细线**: 最外层 d≈0 处 → 精确勾勒轮廓
- **水滴内部**: 光晕在逃逸区平滑衰减, 非硬边界

### DEM 的优势

- 物理距离 $d$ 与 $n$ 无关 → 全图等宽细线
- $-\log(d)$ 映射放大近边界差异 → 细丝结构清晰
- 连续光晕 = 无阶梯, 无棋盘格伪影
