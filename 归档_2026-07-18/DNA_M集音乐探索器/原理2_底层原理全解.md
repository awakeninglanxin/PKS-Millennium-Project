```
  基因名(BRCA1)
      │  ① 位编码 (每字符4碱基, 2bit/碱基)
      ▼

                                       │
                    ┌──────────────────┼──────────────────┐

                    ▼                   │                   ▼
             主心形泡中心 c              │              氨基酸(20+终止)
             c = e^{iθ}/2 − e^{2iθ}/4    │                   │
             θ = 2π·p/q, 周期 = q        │              极性分类
                    │                   │           疏水/亲水/正电/负电
              ③ 轨道→声音               │                   │
```

**核心思想**：把生命信息（DNA）编码进 Mandelbrot 集的几何结构（Farey 泡），再用分形动力学（轨道）发声。基因序列 = 在正/逆 M 图上点击的一串位置，每次点击的轨道就是一个音符。

## 一、第①层：基因名 → DNA 碱基序列

### 1.1 位编码规则

每个字符取 ASCII 码，按 2 bit 一组拆成 4 个碱基（A=00, T=01, G=10, C=11）：

```
   字符 'B' = 0x42 = 0100 0010(bin)
   拆成 4 组 2bit:  01 | 00 | 00 | 10
                    T  | A  | A  | G
```

```js
const BASE = 'ATGC';                    // A=0 T=1 G=2 C=3
for (const ch of name){
  const b = ch.charCodeAt(0);
  for (const s of [6,4,2,0]) seq += BASE[(b>>s)&3];
}
```

- 每字符 → 4 碱基；`BRCA1`(5字符) → 20 碱基 → 6 密码子 → 6 音符。
- 名字越长旋律越长；这是**确定性编码**（同名必同曲）。

### 1.2 切密码子

每 3 个碱基一组（生物学密码子长度）：

```
   TAAG TTAG TAAC …  →  TAA | GTT | AGT | AAC …
```

## 二、第②层：密码子 → M集泡中心（Farey 树 p/q）

### 2.1 Stern-Brocot / Farey 树生成 64 个分数

64 密码子 (4³) ↔ 64 个既约分数 p/q ∈ (0,1)，用 Farey 树 BFS 生成：

```
                  1/2 (周期2)
                 /          \
            1/3 (周3)       2/3 (周3)
           /      \        /      \
        1/4     2/5      3/5      3/4  …
```

```js
function buildFarey(n){
  const nodes=[], queue=[[0,1, 1,1]];       // 区间端点 a/b, c/d
  while(nodes.length<n && queue.length){
    const [a,b,c,d]=queue.shift();
    const p=a+c, q=b+d;                       // Farey 中位数
    nodes.push([p,q]);
    queue.push([a,b,p,q]); queue.push([p,q,c,d]);
  }
  return nodes.slice(0,n);
}
```

密码子 6bit 值 (0–63) 直接索引第 value 个 Farey 分数：

```js
val = Σ base[i]·4^i     (i=0,1,2)   →   FAREY[val] = [p, q]
```

### 2.2 泡中心：主心形附着点公式

Farey 分数 p/q 对应主心形上角度 θ=2π·p/q 处的 **q 周期泡**，其附着点为精确复数：

```
   c(θ) = e^{iθ}/2 − e^{2iθ}/4
        = [cos θ/2 − cos 2θ/4] + i[sin θ/2 − sin 2θ/4]
```

> **数学根基（Devaney 1999）**：M 集主心形上的泡排列与 Farey 树**同构**——相邻两泡 a/b 与 c/d 之间最大的中间泡，其旋转数 = 它们的 Farey 和 (a+c)/(b+d)，周期 = b+d。周期 q 决定该泡内轨道的周期，从而决定音高。

### 2.3 特征外角对（Farey p/q 外角）

每个 p/q 泡的根部有两条外部射线（external rays）landing，其角度对为：

```
   下角 t₋: 二进制 b_i = ⌊i·p/q⌋ − ⌊(i−1)·p/q⌋ , i=1…q ,  t₋ = V/(2^q−1)
   上角 t₊ = t₋ + 1/(2^q−1)
```

**验证**（与经典结果一致）：

| p/q | 泡周期 | 特征外角对 | 经典值 |
|---|---|---|---|
| 1/2 | 2 | 1/3, 2/3 | ✅ 周期2泡的外角正是 1/3, 2/3 |
| 1/3 | 3 | 1/7, 2/7 | ✅ 周期3(顶)泡外角 1/7, 2/7 |

外角在软件"当前音符"面板显示，标注该泡在 M 集边界上的精确锚定。

## 三、第③层：泡中心 c → 轨道 → 声音

> 完整推导见配套文档 **《原理1_分形轨道降频发声算法.md》**。此处提炼要点：

```
   c(泡中心) ──迭代 z²+c──▶ 周期 q 轨道 ──差分向量──▶ 波表 ──循环读取──▶ 音调
   
   音高  f₀ = f_max / q         (steps = sr/f_max, 把频率压扩进可听带)
   波形  L=dx, R=dy             (归一化差分向量, 去直流+限幅+立体声)
   八度  f_max(s) = 4200·2^(−s)  (s=0阳明亮 / s=1阴低八度)
```

- 周期 q ∈ {2,3,5,8,13,21…} → 音高 {2100,1400,840,525,323,200} Hz（阳）。
- 音色来自轨道形状；音高来自 Farey 周期 q。

## 四、第④层：氨基酸极性 → 音色（振幅 + 相位调制）

### 4.1 标准遗传密码表

密码子（DNA 义链，T 代 U）→ 20 种氨基酸 + 终止，内置 64 项查表。

### 4.2 极性四分类 → Web Audio 音色链

```
   BufferSource(循环波表)
        │
        ▼  WaveShaper   ← 极性决定波形整形(奇次谐波量)
        ▼  BiquadFilter ← 高通/低通 + 截止频率
        ▼  StereoPanner ← 相位/声像(charge sign)
        ▼  GainNode     ← ADSR包络 + 振幅
        ▼  (LFO调制)    ← 颤音/震音
        ▼  destination
```

| 极性 | 氨基酸 | 音色处理 | 调制类型 |
|---|---|---|---|
| 🟠**疏水** | A V L I P F M W G | WaveShaper(奇次谐波,drive高)+高通380Hz | 明亮尖锐 |
| 🟢**亲水** | S T C Y N Q | 低通1100Hz柔化,drive低 | 振幅偏低(0.62) |
| 🔴**正电(碱性)** | K R H | 颤音 vibrato: LFO→detune(+16cent, 6.5Hz) | **相位/频率调制**, 声像右偏(+0.55) |
| 🔵**负电(酸性)** | D E | 震音 tremolo: LFO→gain(5Hz) | **振幅调制**, 声像左偏(−0.55) |
| ⚪**终止** | * | 短促·低幅 | 停顿感 |

> **对应用户需求"疏水/亲水/电荷调制振幅和相位"**：
> - **振幅调制**：负电氨基酸用 gain-LFO（震音）；各类别基础振幅 amp 不同（疏水0.85/亲水0.62/带电0.95）。
> - **相位调制**：正电用 detune-LFO（颤音=瞬时频率/相位调制）；带电氨基酸用 StereoPanner 分左右（相位声像）。

### 4.3 波形整形器（奇次谐波）

疏水氨基酸用 tanh 型 WaveShaper 生成奇次谐波（方波感，明亮）：

```js
curve[i] = (1+k)·x / (1 + k·|x|)     // k=drive, x∈[-1,1]
```

## 五、正逆M对偶：s 滑块的数学 (s=0 正M / s=1 逆M)

### 5.1 幂变换

分形背景渲染用**幂变换**统一正逆 M（来自"正逆M滑动变换"工具）：

```
   c_eff = c^(1−2s)          α = 1 − 2s
     s=0   → α=+1 → z²+c        (标准正 M)
     s=1   → α=−1 → z²+1/c      (逆 M 水滴)
     s=0.5 → α= 0 → z²+1        (统一退化点)
```

### 5.2 c·(1/c)=1 极化原理（阴阳对偶）

正 M 与逆 M 通过复反演 `c ↔ 1/c` 互为对偶（ℤ₂ 群对合）：

```
   正M 原点(0) ←──反演──→ 逆M 无穷(∞)
   正M 泡中心 c_bulb ←──→ 逆M 位置 1/c_bulb
```

### 5.3 音符位置随 s 的"呼吸"动画

密码子的家 = 泡中心 c_bulb。它在**当前 s 视图**下的显示位置：

```
   c_marker(s) = c_bulb^( 1/(1−2s) )
       s=0  → c_bulb        (正M 主心形边缘)
       s=1  → 1/c_bulb      (逆M 水滴，复反演)
```

> 拖动 s 滑块，64 个泡标记在正 M ↔ 逆 M 之间平滑迁移——**同一段基因旋律，音符不变，只是"点击位置"在阴阳两张图间镜像流动**。

## 六、分形渲染引擎（Canvas）

### 6.1 平滑逃逸着色（消色带）

普通迭代数是整数会产生色带断层；用对数平滑：

```
   ν = log₂( log₂|z| )         (连续化迭代数)
   smooth = i + 1 − ν          → 循环色图取模 → 丝滑彩环
```

### 6.2 轨道陷阱同心圆（内部不死黑）

未逃逸区追踪轨道到原点的历史最小距离 trap，log 取模 → 围绕原点的同心圆环：

```
   field_interior = ( log(trap) × 0.55 ) mod 1
```

### 6.3 等轴比（equal axis）— 防压扁

**关键修复**：正方形缓冲被 CSS 拉伸到非正方形舞台会压扁逆 M 水滴。解决：

```
   asp   = 显示区高/宽 (dr.height/dr.width)
   缓冲   W = quality,  H = quality × asp        (缓冲宽高比 = 显示区宽高比)
   跨度   spanX = view.w,  spanY = view.w × asp   (每像素代表相同复平面单位)
```

于是 Re 轴与 Im 轴**单位/像素严格相等**，水滴呈正圆、心形不变形；点击/拖拽也用统一的 `GEOM{xmin,ymin,spanX,spanY}` 映射，缩放拖拽全程等轴。

## 十、一句话总结

> **DNA · 正逆M集 · 音乐探索器** 把基因序列编码成 Mandelbrot 主心形上的一串 Farey 周期泡：**每个密码子是逆 M 图上的一次点击，其轨道的周期 q 决定音高、氨基酸极性决定音色、正逆 M 滑块决定阴阳八度**。分形几何的数学之美（Farey 树、周期泡、正逆对偶）与生命信息（密码子、氨基酸极性）在此合一，化作可听、可视、可交互的音乐。
