# DNA · 正逆M集 · 音乐探索器

融合两大工具的优点，创造的交互式软件：
- **逆分形声音探索器** → 点击复平面任意点，听见该点轨道的声音（轨道差分向量→PCM立体声）
- **正逆M滑动变换** → s滑块 `c_eff = c^(1-2s)`，**s=0 正M (z²+c)**，**s=1 逆M (z²+1/c)**
- **7族内生序列 + v4引擎** → 64密码子→Farey泡中心，DNA旋律=在M图上点击一串泡

## 启动
双击 `启动_DNA_M集音乐探索器.bat`（用默认浏览器打开，纯离线，无需联网/Python）。

## 四层映射链

```
基因名 ──编码──▶ DNA碱基 ──3合1──▶ 密码子(64种)
   │                                    │
   │                          ┌─────────┴─────────┐
   │                     Farey p/q            遗传密码表
   │                          │                    │
   │                  主心形泡中心 c            氨基酸(20+终止)
   │                  c=e^{iθ}/2−e^{2iθ}/4         │
   │                  θ=2π·p/q, 周期=q          极性分类
   │                          │                    │
   ▼                          ▼                    ▼
 旋律序列 ◀──── 音高=maxFreq/q ────  音色(滤波/整形/颤音/震音)
```

### ① 64密码子 → M集泡中心（Farey树 p/q 外角）
- Stern-Brocot/Farey 树 BFS 生成前64个既约分数 p/q ∈ (0,1)
- 每个 p/q 对应主心形 θ=2π·p/q 处的 **q周期泡**，泡中心 `c = e^{iθ}/2 − e^{2iθ}/4`
- 特征外角对：下角二进制=floor差分 `⌊ip/q⌋−⌊(i−1)p/q⌋`，上角=下角+1/(2^q−1)
  （验证：1/2泡→1/3,2/3；1/3泡→1/7,2/7 ✓）
- 密码子6bit值(0-63)直接索引第value个Farey分数

### ② 氨基酸极性 → 音色（振幅+相位调制）
| 极性 | 颜色 | 音色处理 |
|------|------|----------|
| 疏水 | 橙 | 波形整形(奇次谐波)+高通，明亮尖锐 |
| 亲水 | 青 | 低通柔化，正弦感，振幅偏低 |
| 正电(碱性) | 粉 | **颤音**(pitch LFO)+detune上扬，相位右偏 |
| 负电(酸性) | 蓝 | **震音**(amp LFO)+detune下沉，相位左偏 |
| 终止 | 灰 | 短促·停顿 |

### ③ 音高 = maxFreq/q（周期泡定音高）
- 阳(s=0) maxFreq=4200：周期2→2100Hz，3→1400，5→840，8→525…
- 阴(s=1) 降一个八度 maxFreq=2100
- s滑块只改八度（阴阳），不改音符——因为泡周期q决定音高

### ④ 正逆对偶可视化
- 密码子在s视图下的位置：`c = c_bulb^(1/(1-2s))`
- s=0：泡在正M主心形边缘；s=1：位置反演到 1/c_bulb（逆M水滴）
- 拖滑块看每个音符的位置在正M↔逆M间"呼吸"

## 操作
- **▶演奏**：当前基因整段旋律，序列条+分形图同步高亮
- **下拉/导航/随机**：选基因即自动播放
- **点击分形图**：听任意点的轨道声音（探索模式）
- **点击64网格/序列条**：单独试听某密码子
- **s滑块**：☀正M ↔ 逆M☽
- 滚轮缩放 · 拖拽平移 · 双击复位 · 配色切换
