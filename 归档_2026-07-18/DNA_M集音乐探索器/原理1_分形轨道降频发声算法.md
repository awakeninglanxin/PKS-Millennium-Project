# DNA 音乐 · 分形轨道降频发声算法原理

> **主题**：如何把 Mandelbrot 分形轨道从"人耳听不见"变成"清晰可听"。
> **定位**：`DNA · 正逆M集 · 音乐探索器` 的声音合成核心（第③层）。
> **关键词**：频率压扩 (frequency companding)、波表合成 (wavetable synthesis)、`steps = sr/f_max`。
> **日期**：2026-07-08

---

## 一、问题：分形轨道天生"不在可听带"

### 1.1 人耳可听范围

```
次声波(听不见)  |         人耳可听带           |  超声波(听不见)
────────────────┼──────────────────────────────┼────────────────
      < 20 Hz   |   20 Hz  ～  20,000 Hz       |   > 20 kHz
                |   (钢琴/人声主要在 80–2000Hz) |
```

采样率 `sr = 44100 Hz` 时，奈奎斯特上限 = `sr/2 = 22050 Hz`。

### 1.2 分形轨道是什么

点击复平面参数 c，迭代 `z_{n+1} = z_n² + c`（z₀=0），得到一条轨道：

```
轨道点序列:  z₀ → z₁ → z₂ → z₃ → … → z_N
复平面坐标:  (x₀,y₀)(x₁,y₁)(x₂,y₂) …
```

- **有界轨道**（c 在 M 集内）：轨道最终锁定到一个**周期 q 的循环** → 天然的"周期信号" → 可当波形。
- **逃逸轨道**（c 在 M 集外）：轨道发散 → 噪声感。

**周期 q 从哪来？** 本系统中 c 取 Farey 树 p/q 泡中心，其轨道周期**恰好 = q**（周期 2/3/5/8/13/21…）。这是可控音高的数学根基。

### 1.3 两种"听不见"的失败模式

把轨道直接变声音，会掉进两个陷阱：

| 失败模式 | 做法 | 结果 | 后果 |
|---|---|---|---|
| **① 太低 (v2/v3 的 bug)** | 每个轨道点占 0.03s = 1323 采样 | 基频 = `1/(q×0.03) = 33/q` Hz，q=2 时仅 **16.7 Hz** | 次声波，**听不见** |
| **② 太高 (朴素 1点/采样)** | 每个轨道点占 1 采样 | 基频 = `sr/q`，q=2 时高达 **22050 Hz** | 超声波/接近奈奎斯特，**听不见** |

> 两个极端都出局：一个跌进次声波，一个冲上超声波。**必须把轨道的"读取速率"精确控制，使基频落在可听带正中央。**

## 二、核心算法：`steps = sr / f_max`（频率压扩）

### 2.1 关键洞察

声音的音高由波形**重复的快慢**决定，而不是轨道点的绝对数值。所以我们引入一个**读取步长** `steps`：

> **每个轨道点在音频里持续 `steps` 个采样**（而不是 1 个，也不是 1323 个）。

于是一个周期 q 的轨道，其一个完整波形周期 = `q × steps` 个采样，基频为：

```
                sr              sr           
   f₀  =  ───────────  =  ─────────────  =  f_max / q
           q × steps        q × (sr/f_max)
```

### 2.2 steps 的取值 — 把天花板钉在 f_max

设定一个**目标上限频率 `f_max`**（本系统阳=4200Hz），令：

```
   steps = round( sr / f_max )      // 44100/4200 ≈ 10.5 → 11 采样/轨道点
```

代入 f₀ = f_max/q，得到**音高天花板 = f_max**，且随周期 q 逐级下降：

| 周期 q (Farey) | 阳 f_max=4200 → f₀ | 阴 f_max=2100 → f₀ | 音区 |
|---|---|---|---|
| 2  | 2100 Hz | 1050 Hz | 高音 |
| 3  | 1400 Hz | 700 Hz  | 高音 |
| 5  | 840 Hz  | 420 Hz  | 中高 |
| 8  | 525 Hz  | 263 Hz  | 中音 |
| 13 | 323 Hz  | 162 Hz  | 中低 |
| 21 | 200 Hz  | 100 Hz  | 低音 |

> **全部落在 100–2100 Hz** —— 正是人声/钢琴最敏感的黄金频段。**这就是"降频处理"的本质：不是滤波，而是选对读取步长，让轨道的固有周期性正好落进可听带。**

### 2.3 本质：这是波表合成 (Wavetable Synthesis)

换个专业视角，整个机制等价于经典的**波表合成**：

```
   ┌─────────────────────────────┐
   │  波表 = 一个周期的轨道波形    │   长度 = q × steps 采样
   │  [seg₀ … seg_{q-1}] 各扩 steps│
   └─────────────────────────────┘
              │ 以采样率 sr 循环读取
              ▼
       输出音调 f₀ = sr / 表长 = f_max / q
```

- **波表内容** = 轨道的形状（决定音色/谐波）
- **波表长度** = `q × steps`（决定音高）
- 循环读取 = 持续音（sustain）

## 三、三个配套的信号处理细节

### 3.1 差分向量 + 归一化（去直流、限幅）

**不能直接用轨道坐标 (xₙ, yₙ) 当波形**——它们有巨大直流偏移、且逃逸时数值爆炸。改用**相邻点的差分向量并归一化**：

```
   d_n = (z_{n+1} − z_n) / |z_{n+1} − z_n|     // 单位速度向量
```

- **去直流**：差分消除了绝对位置的 DC 偏置，波形自然围绕 0 振荡。
- **限幅**：归一化后 |d|=1，振幅恒定有界，绝不削波爆音。
- **立体声**：左声道 = d_x，右声道 = d_y —— 轨道在复平面的"横/纵运动"分离成左右耳，产生空间感。

### 3.2 余弦插值（抗混叠、消爆音）

相邻差分向量之间若硬切，会产生高频"咔哒"声（阶跃→宽频谱→混叠）。用**余弦等功率过渡**在 `steps` 个采样内平滑：

```
   t_i = 0.5 − 0.5·cos(π · j/steps)      j = 0…steps-1   (Hann 型渐变)
   w   = t_i · d_{n+1} + (1 − t_i) · d_n
```

这是一种轻量**带限**处理，把方波般的阶跃磨成平滑曲线，抑制混叠。

### 3.3 s 滑块 → 八度控制（阴阳）

`f_max` 不是常数，而是随正逆M滑块 s 变化：

```
   f_max(s) = 4200 × 2^(−s)
        s=0 (☀阳/正M) → 4200 Hz  (明亮)
        s=1 (☽阴/逆M) → 2100 Hz  (低一个八度, 柔和)
```

> 关键：**s 只改八度（f_max），不改音符**——因为音高 = f_max/q，而 q 由密码子的 Farey 周期锁定。所以拖动 s，整段旋律**整体升降八度**，音程关系不变（正逆M 阴阳对偶在听觉上 = 同一旋律的高/低八度镜像）。

## 四、完整伪代码

```python
def build_note(c_eff, q, f_max, sr, duration):
    steps = max(2, round(sr / f_max))          # ① 降频关键: 读取步长

    # ② 算轨道 (z²+c_eff, z0=0)
    orbit = [(0,0)]; z = 0+0j
    for _ in range(220):
        z = z*z + c_eff; orbit.append((z.real, z.imag))
        if abs(z) > 10: break                  # 逃逸半径

    # ③ 差分向量 + 归一化 (去直流/限幅)
    D = []
    for i in range(len(orbit)-1):
        dx = orbit[i+1][0]-orbit[i][0]
        dy = orbit[i+1][1]-orbit[i][1]
        m  = hypot(dx,dy) + 1e-12
        D.append((dx/m, dy/m))

    # ④ 波表合成: 每段扩 steps 采样, 余弦插值, 强制周期 q 循环
    total = max(q*steps, int(duration*sr))
    L, R = [], []; seg = 0; prev = D[0]
    while len(L) < total:
        a = D[seg % len(D)]; b = D[(seg+1) % len(D)]
        for j in range(steps):
            ti = 0.5 - 0.5*cos(pi * j/steps)   # ⑤ 余弦抗混叠
            L.append(ti*b[0] + (1-ti)*a[0])    # 左 = dx
            R.append(ti*b[1] + (1-ti)*a[1])    # 右 = dy
        seg = (seg + 1) % q                     # ⑥ 周期 q 循环 → 音高 = f_max/q
    return stereo(L, R)                          # → f₀ = f_max/q, 落在可听带
```

## 五、频率关系速查

```
   ┌──────────────────────────────────────────────┐
   │   f₀  =  sr / (q × steps)  =  f_max / q        │  ← 音高公式
   │   steps = round( sr / f_max )                  │  ← 降频旋钮
   │   f_max(s) = 4200 · 2^(−s)                      │  ← 八度(阴阳)
   │   0.03s/点(=1323采样) → f₀=33/q Hz ✗ 太低      │  ← v2/v3 bug
   │   1采样/点            → f₀=sr/q Hz ✗ 太高      │  ← 朴素错误
   │   steps≈11采样/点     → f₀=f_max/q ✓ 可听      │  ← 正解
   └──────────────────────────────────────────────┘
```

## 六、结论

| 环节 | 技术 | 作用 |
|---|---|---|
| 降频核心 | `steps = sr/f_max` | 把音高天花板钉在 f_max，全部 q 落入可听带 |
| 音高定量 | `f₀ = f_max/q` | Farey 周期 q 直接决定音符 |
| 波形来源 | 归一化差分向量 | 去直流+限幅，音色来自轨道形状 |
| 抗混叠 | 余弦插值 | 消除阶跃爆音 |
| 立体声 | L=dx, R=dy | 复平面横纵运动→左右耳 |
| 八度 | `f_max·2^(−s)` | 正逆M 阴阳 = 高/低八度镜像 |

**一句话原理**：分形轨道本身是个"周期 q 的信号"，但它的固有频率要么太高（22kHz）要么太低（17Hz）。我们通过**波表合成**——把一个周期铺成 `q×steps` 个采样、以采样率循环读取——精确地把基频**压扩**到 `f_max/q`，稳稳落在人耳 100–2100 Hz 的黄金频段，于是"看得见的分形"变成了"听得见的音乐"。

## 七、音乐化层（好听 = 忠实 + 4 步润色）

> **重要认知**：以上流程让分形"能听见"，但"能听见 ≠ 好听"。原始 `f_max/q` 是**次谐波序列**，不落在音乐音阶上（跑调感），且轨道波形泛音不谐和（嗡沙声）。需要在保持 DNA→M集映射不变的前提下，加一层"音乐化"。

### 7.1 音阶量化（最关键）— `quantizeHz`
把每个音的频率**吸附到大调五声音阶**（`SCALE_ROOT=220Hz A3`，`PENTA=[0,2,4,7,9]` 半音），五声音阶天然无半音冲突、永远协和：

```
semi  = 12·log₂(hz / SCALE_ROOT)          # 距根音的半音数
target= 最近的  SCALE_ROOT·2^(scaleDegree/12)   # 跨八度找最近五声音
```

用 `src.playbackRate = target / baseNat` 把**分形音色整体重新定音**到音阶——既保留分形"指纹"音色，又保证每个音在调内。这是"跑调→好听"的决定性一步。

### 7.2 暖音色 — 纯音叠加
分形轨道音色降为 0.45 的"色彩层"，叠加**纯正弦基频 + 八度泛音**作为主旋律听感（明确音高、温暖）；波形整形 drive×0.4、滤波 Q 0.6、指数衰减尾音。

### 7.3 混响母线 — 空间感
所有音符汇入 `master()`：反馈延迟（`delay 0.19s + feedback 0.30 + lowpass 3000`），dry 0.82 / wet 0.26，制造厅堂空间，音符之间有余韵衔接。

### 7.4 低音持续层（7族混音基座）— `startDrone`
演奏整段旋律时，底层铺一个**根音(110Hz) + 纯五度(165Hz)** 的 triangle 持续音（drone），给旋律一个稳定的和声地基。这是"7族混音"的第一层——后续可用 7 族 M 集内生序列（Feigenbaum/Fibonacci/…）各驱动一个声部，扩展成多层混音。

| 润色步骤 | 技术 | 听感提升 |
|---|---|---|
| 音阶量化 | playbackRate 吸附五声音阶 | 跑调 → 协和 ✅ |
| 暖音色 | 正弦基频+八度泛音叠加 | 嗡沙 → 温暖 |
| 混响 | 反馈延迟母线 | 干瘪 → 有空间 |
| 低音drone | 根音+五度持续层 | 单薄 → 有地基 |

> **结论**：DNA→M集→分形轨道 提供"独一无二的旋律骨架"，音乐化层提供"悦耳的皮肉"。二者结合，才既是"这段 DNA 专属的曲子"，又"好听"。
