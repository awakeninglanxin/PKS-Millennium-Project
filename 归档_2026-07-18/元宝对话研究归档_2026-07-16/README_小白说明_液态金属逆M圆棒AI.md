# 液态金属 + 逆M + 圆棒AI — 小白版

> **一句话**: 三篇元宝对话聊了三件事——用液态金属代替风扇散热、用分形数学安排GPU的"作息表"、把AI塞进充电宝大小的圆棒里。它们其实是在讲同一个道理：**让物理自己算，比用电脑模拟物理快得多。**

---

## 十分钟看懂三件事

### 第一件事：液态金属是什么？为什么能做计算？

| 概念 | 大白话 |
|------|--------|
| **Galinstan** | 镓+铟+锡的合金，室温下是液体，像水银但不是毒的 |
| **趋肤效应** | 电流只走电线表面。液态金属同理——流动也走壁面 |
| **MONIAC** | 1949年的"水力计算机"——用水流模拟经济，水的高度=GDP |
| **16bands 漏斗** | 一个螺旋形的通道，水从16个方向扭着往下流，每扭一圈就加一次速 |

核心思路：**普通人用电脑仿真水流要算几小时，但如果你直接造一个真的水流通道，它自己就"算"出答案了——一秒不到。**

就像你不用算"水往低处流"——你打开水龙头，水自己就流了。自然界早就会"算"了，你只是给它一条跑道。

---

### 第二件事：GPU为什么发热烧毁？幻立方怎么帮忙？

| 概念 | 大白话 |
|------|--------|
| **GPU共振** | GPU里几百亿个晶体管同时开关，像几亿个人同时跺脚→电流尖峰→瞬间局部烧毁 |
| **幻立方（魔方阵）** | 一个数学结构，每条线加起来都一样。用在芯片上=所有晶体管的工作量均匀分摊 |
| **逆M cardioid** | 水滴形状的数学曲线。把它上面的"拐点"标记出来=晶体管的最优工作顺序 |
| **Farey 分数** | 一种分数序列(1/2, 1/3, 2/3...)，恰好可以给晶体管排"作息表"——永远不会同时开工 |

核心思路：**不让所有人同时跺脚。给每个晶体管一个微小的"延迟"，像音乐里的轮唱——你唱完我才唱，电流永远不会一起冲。**

逆M分形天然提供这个"轮唱表"。Douglas-Peucker拐点序列(40,78,82,122...)就是最优的工作节奏。

## ⚡ 关键认知：两种水流，两种排名

这是我们今天刚刚纠正的一个错误认知：

```
情况A: 水从上方竖直落下（像淋浴）→ 均匀填满整个漏斗
  → 🥇 30°直锥最好（缓坡长跑道，慢慢加速）
  → 16bands 排第3（螺旋纹路被平均掉了）

情况B: 水贴着墙壁螺旋往下流（像滑滑梯）
  → 🥇 16bands 最好（16圈螺旋=16次加速助推）
  → 30°直锥排第5（没有螺旋，纯靠重力不够快）

液态金属属于情况B——因为电驱动 + 趋肤效应，液态金属天然走壁面。
```

**同一面墙，在情况A是"阻力"，在情况B是"跑道"。这就是双流态的核心。**
