## 核心公式链

```
a · b = 1          ← 第一性原理(极性/开普勒/毕达哥拉斯)
    ↓
x · y = 1          ← 超双曲锥 (音调塔)
    ↓
直圆锥(电性/∇p) ⟷ 超双曲锥(磁性/ν∇²u)  ← NS方程拆解
    ↓
阻尼项 f(u) = -α|u|^{β-1}u          ← NSD = PKS锥面约束
    ↓
β ≥ 1 ↔ BKM PASS ↔ k_E ≥ 1         ← 三体统一
    ↓
三档SEG: 安全(β=3) / 推荐(β=2) / 进取(β=1.5)
```
