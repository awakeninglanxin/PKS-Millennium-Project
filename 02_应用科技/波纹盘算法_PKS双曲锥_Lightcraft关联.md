## 一、波纹盘的 Python 算法——两个核心文件

### 1.1 `平面环绕sin波.py` — 波纹调制螺旋锥管

```python
# 核心算法 (行 19-29)
x = radius * cos(t)
y = radius * sin(t)
z = (i / total_points) * height

# 波纹效果 — 径向正弦调制
ripple = sin(t * bands) * 5
x += ripple * cos(t)
y += ripple * sin(t)
```

**数学表达**：

$$r(t) = R_0 + A \cdot \sin(n \cdot t)$$

- $R_0$：基准半径
- $A=5$：波纹振幅
- $n$（`bands`）：波纹周期数——对应 Repulsine 铜盘上的同心波纹圈数

> **这不是"装饰性波纹"——$n$ 对应盘面上从中心到外缘的同心波纹槽数目。每个波纹槽在水流离心加速过程中完成一次"压缩→释放"循环。**

### 1.2 `舒式螺旋羚羊角管圆盘.py` — 锥面投影 + 蛋形截面扫掠

这是项目的**核心几何引擎**，以黄金比 $\phi = (\sqrt{5}-1)/2$ 为对数基，将螺旋投影到锥面上：

```python
# get_cone_profile (行 152-162) — 锥面映射（关键！）
def get_cone_profile(self, x, y):
    r_xy = sqrt(x² + y²)
    z_axis = log(0.5)/log(φ) + 
             (log(bottom_r / r_xy) * user_high) / (log(turns_num/2) - log(0.5))
    return z_axis, r_xy
```

**数学表达**：

$$z(r) = \frac{\ln(0.5)}{\ln\phi} + \frac{\ln(R_0/r) \cdot H}{\ln(N/2) - \ln(0.5)}$$

化简为 $z(r) = C_1 - C_2 \cdot \ln r$，即：

$$r(z) = R_0 \cdot e^{-(z - C_1)/C_2}$$

### 1.3 锥面与 $xy=1$ 的精确对应

将 $r = \sqrt{x^2 + y^2}$ 代入锥面方程，在固定高度 $z_0$ 处：

$$\sqrt{x^2 + y^2} = R_0 \cdot e^{-z_0/C_2}$$

这是一个**圆**。但用倾斜平面 $z = kx + z_0$ 切割锥面：

$$\sqrt{x^2 + y^2} = R_0 \cdot e^{-(kx + z_0)/C_2}$$

当 $k \neq 0$ 时，截面为**蛋形**——这是 PKS 蛋形的直接来源。

**$xy=1$ 在哪？** 这是双曲锥的对偶形式。锥面 $z \propto \ln r$ 的**反函数**是 $r \propto e^{-z}$，等价于 $xy = \text{常数}$ 绕 $x=0$ 旋转生成的双曲锥体。两者是**同一几何在两个坐标系中的表述**：

| 坐标 | 锥面形式 | 截面 |
|:---|:---|:---|
| 柱坐标 $(r,\theta,z)$ | $r = R_0 e^{-\alpha z}$（对数锥） | 水平切→圆 |
| 笛卡尔 $(x,y,z)$ | $xy = \text{常数}$（双曲锥） | 水平切→双曲线 |
| **共同** | **同一曲面，两种参数化** | **斜切→蛋形** |

## 二、波纹盘的完整物理链条——从代码到等离子体

### 2.1 水的轨迹

```
水面在中心低压区（z=0, r→0）
    ↓ 盘旋转 → 离心力
水沿波纹槽向外加速（每过一个波纹槽完成一次压缩-释放循环）
    ↓ 到达盘边缘（r=R_max）
超音速水速（DOCX 记载：舒伯格圆盘边缘产生超音速水速）
    ↓ 波面局部真空 + 铜汞合金催化
H₂O → HHO (Brown's gas) → 冷等离子体
    ↓ 等离子体沿锥面焦散线喷射
反作用推力 + 反重力效应
```

### 2.2 每个波纹槽的作用

波纹槽的数学本质是**径向正弦调制** $r(t) = R_0 + A\sin(nt)$：

| 槽相位 | 物理作用 | 流体力学 |
|:---|:---|:---|
| 波峰 ($\sin=+1$) | 水流被压缩 → 压力↑ → 速度↑ (Bernoulli) | 局部马赫数峰值 |
| 波谷 ($\sin=-1$) | 水流释放 → 压力↓ → 微空化气泡形成 | 局部负压 → 水分子解离 |

$n$ 个波纹槽 = $n$ 次压缩-释放循环 = $n$ 次水分子被"锤击" → 到达盘边缘时已累积足够的能量用于 HHO 分解。

### 2.3 为什么波纹盘的效率远超 Lightcraft 抛物面

| 机制 | Lightcraft 抛物面 | Schauberger 波纹盘 |
|:---|:---|:---|
| 能量输入 | 外部激光（需地面站） | 水流离心加速（自持） |
| 聚焦方式 | 一次反射聚焦 | $n$ 次波纹槽逐级压缩 |
| 等离子体产生 | 激光直接加热空气 | 水经 $n$ 次波纹锤击 → HHO → 内爆 |
| 推力密度 | 1 次爆轰/脉冲 | 连续等离子体喷射 |
| $xy=1$ 的作用 | **无**（抛物面是 $y=x^2$） | **锥面剖面 = 等离子体涡旋的自然约束面** |

> **核心**：Lightcraft 的抛物面只是"光线聚焦器"——它不参与工质的相变。波纹盘的每个槽都参与水→HHO→等离子体的物理过程。$xy=1$ 锥面是等离子体喷射的**自然几何约束**——等离子体沿锥面焦散线（曲率最大处）出射，形成定向推力。
