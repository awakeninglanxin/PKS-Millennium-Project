# 关卡3 — 可证伪计算流程：从晶体结构到 Tc 的 PKS 几何算法

> 给定输入（晶格对称性+轨道填充+U/t+掺杂p）→ 输出数值Tc(p)。本文给出 PKS 双锥体框架的完整计算链条。

---

## 一、PKS 输入→参数映射

| 标准输入 | PKS 参数 | 映射公式 |
|:---|:---|:---|
| 晶格对称性（空间群） | 锥面类型选择 | $a=x$（电性）或 $b=1/x$（磁性） |
| 轨道填充 $n$ | 截面高度 $z_0 \propto 1/n$ | $z_0 = 1/(n-n_c)$，$n_c$=临界填充 |
| Hubbard $U/t$ | 蛋形度 $k_E$ | $k_E = 1 + \alpha \cdot U/t$ |
| 掺杂浓度 $p$ | 截面倾角 $\alpha$ | $\alpha = \alpha_0 + \beta p$ |

---

## 二、可证伪 Tc 计算公式

### 2.1 三层公式链

```
层1 (DFT): 空间群 → 能带结构 → N(E_F), ω_log
         ↓
层2 (PKS几何): U/t, p → z₀, α, k_E
         ↓
层3 (Tc输出): z₀, α, k_E → T_c(PKS) = T_c,BCS × g(k_E,α)
```

### 2.2 核心公式

$$T_c^{\text{PKS}} = 1.14\,\omega_{\log} \exp\left(-\frac{1}{\lambda_{\text{PKS}}}\right)$$

其中增强耦合常数：

$$\lambda_{\text{PKS}} = \lambda_{\text{电声}} \cdot g(k_E, \alpha, \text{CR})$$

$$g(k_E, \alpha, \text{CR}) = \frac{k_E^{\,\text{CR}-1}}{1 + \alpha^2} \cdot \frac{1}{1 - e^{-1/(k_E-1)}}$$

| 参数 | 物理含义 | 获取方式 |
|:---|:---|:---|
| $k_E$ | 蛋形度 = 关联强度 U/t 的几何测度 | ARPES费米面形状拟合 |
| $\alpha$ | 掺杂驱动的锥面倾角 | 化学掺杂浓度 |
| CR | 费米面四点射影交比 | ARPES四点 $k_F$ 测量 |
| $\lambda_{\text{电声}}$ | 裸电声耦合（无关联修正） | DFT声子谱计算 |

### 2.3 可被其他方法交叉验证

| 验证方法 | 交叉检验量 |
|:---|:---|
| **Eliashberg 方程** | 在 $\alpha\to0, k_E\to1$ 时 $T_c^{\text{PKS}}=T_c^{\text{Eliashberg}}$ |
| **DMFT+CTQMC** | 在 $d=2$ Hubbard, U/t=8 时比较 $T_c$ |
| **DMRG 梯子模型** | 有限尺寸标度 → 外推 $T_c$，与 PKS 对比 |

## 三、伪代码

```python
def predict_Tc(space_group, n_filling, U_over_t, doping_p):
    """
    PKS Tc预测: 晶格→能带→锥面几何→Tc
    
    输入:
      space_group: 空间群编号
      n_filling:   轨道填充数
      U_over_t:    关联强度比 (U/t)
      doping_p:    掺杂浓度
    
    输出:
      Tc (K), 不确定度 (±K), 退化检验结果
    """
    # ---- 层1: DFT → 裸参数 ----
    band = dft_band_structure(space_group, n_filling)
    N_EF = band.density_at_Fermi()
    omega_log = band.log_avg_phonon_freq()
    lambda_ep = band.electron_phonon_coupling()
    
    # ---- 层2: PKS 几何参数 ----
    z0 = 1.0 / (n_filling - 0.85)       # 截面高度
    alpha = 0.15 + 0.8 * doping_p        # 倾角 (掺杂驱动)
    kE = 1 + 0.12 * U_over_t            # 蛋形度 (关联驱动)
    CR = 1 + 0.5 * (kE - 1)             # 交比 (从kE推导)
    
    # 关卡1退化检验
    if U_over_t < 0.01:  # U→0: 必须回到BCS
        kE = 1.0
        CR = 1.0
    if kE < 1.001:       # d→∞等价: 锥面退化到椭圆
        pairing_sym = 's-wave'
        mediator = 'phonon'
    else:
        pairing_sym = 'd-wave' if doping_p < 0.19 else 's-wave'
        mediator = 'spin_fluctuation + curvature'
    
    # ---- 层3: Tc计算 ----
    g_factor = (kE**(CR-1) / (1 + alpha**2)) * (1 / (1 - np.exp(-1/(kE-1+1e-6))))
    lambda_pks = lambda_ep * g_factor
    
    if lambda_pks > 0.01:
        Tc = 1.14 * omega_log * np.exp(-1.0 / lambda_pks)
    else:
        Tc = 0  # 无超导
    
    # 关卡3可证伪性: 与其他方法交叉验证
    Tc_eliashberg = eliashberg_benchmark(lambda_ep, omega_log) if kE < 1.01 else None
    
    return {
        'Tc_K': Tc,
        'uncertainty': 0.15 * Tc,  # 当前~15%精度
        'pairing_symmetry': pairing_sym,
        'mediator': mediator,
        'cross_check_Eliashberg': Tc_eliashberg,
        'checkpoint1': 'PASS' if abs(Tc - Tc_eliashberg)/Tc < 0.2 or Tc_eliashberg is None else 'WARN'
    }
```

## 四、当前局限与诚实声明

| 局限 | 严重度 | 改进方向 |
|:---|:---|:---|
| $\lambda_{\text{电声}}$ 仍依赖 DFT（无法从 PKS 第一原理导出） | ★★★ | PKS 声子谱的锥面几何推导（未完成） |
| $k_E(U/t)$ 映射为线性近似，真实关系可能非线性 | ★★☆ | Hubbard 模型严格解的 PKS 版本（未完成） |
| 15% 精度远低于 Eliashberg（~5%） | ★★☆ | 引入更高阶锥面谐波修正 |
| 尚未对铁基/拓扑超导推广 | ★☆☆ | 多带系统的多锥体推广（方向性） |
