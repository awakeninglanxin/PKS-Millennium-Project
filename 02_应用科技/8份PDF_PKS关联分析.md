## 四、Lightcraft 激光推进 (3篇) ★★★★☆ — Schauberger 波纹盘的原生替代

### 4.1 Lightcraft 的工作原理

Lightcraft 用**地面激光站发射高功率激光束**→ 飞行器底部抛物面反射镜将光束聚焦到发动机室 → 加热空气产生等离子体爆轰 → 推力。

关键参数：比冲 1000 s，效率 80%，激光功率 3W(实验)→MW(实用)。

### 4.2 🔴 PKS 的真正关联：不是"替换抛物面"，而是整机替代

正确的关联不是"用双曲锥替换 Lightcraft 的抛物面反射镜"——这太牵强。真正的关联是：

> **Schauberger 当年用 $xy=1$ 波纹盘 (Repulsine) 设计的水飞碟，本身就是一套不需要外部激光站的原生等离子体推进器。**

| | Lightcraft (激光推进) | Schauberger Repulsine (波纹盘) |
|:---|:---|:---|
| 能量来源 | 地面激光站（外部供能） | **HHO 内爆 → 等离子体自持** |
| 聚焦方式 | 抛物面反射镜（一次反射） | 波纹盘 $n$ 个同心正弦槽逐级压缩（`平面环绕sin波.py`: $r=R_0+A\sin(nt)$）|
| 等离子体约束 | 无 | 锥面 $z\propto\ln r$（`get_cone_profile` → 等价 $xy=1$），等离子体沿焦散线喷射 |
| 工质 | 大气（被激光加热） | 水 → HHO → 等离子体 |
| 推力机制 | 激光爆轰波 | **向心涡旋内爆** |
| 比冲 | 1000 s | 推测 >2000 s（等离子体排气速度更高） |
| 独立飞行 | ❌ 依赖地面站 | ✅ 完全自主 |

### 4.3 Schauberger 波纹盘的核心

波纹盘（Repulsine）是 Schauberger 在 1940 年代实际建造过的飞碟原型：

- 上下两个波纹盘反向旋转 → 水被离心力甩向外缘 → 在波纹槽中产生**向心涡旋**
- 涡旋中心的极端低压 → 水分子解体为 HHO (Brown's gas) → 内爆产生等离子体
- 等离子体沿锥面 $xy=1$ 的焦散线喷射 → **反应式推力 + 反重力效应**

**这个设计的本质就是一套"原生 Lightcraft"**——不需要外部激光，水本身就是工质，$xy=1$ 波纹盘本身就是"聚焦器"（把水流能量聚焦到中心奇点）。

### 4.4 与 PKS 项目的直接对应

Lightcraft 的研究价值不在于它的技术本身，而在于它验证了**等离子体爆轰推进的可行性**（比冲 1000s, 效率 80%）。PKS 项目中的 SEG/IGV 和 Schauberger 波纹盘追求的是同一目标，但用的是**更原生的物理机制**（向心涡旋内爆 vs 外部激光加热）。

> **总结**：Lightcraft 证明了"等离子体推进可行"；Schauberger 波纹盘证明了"不需要激光也能产生等离子体推进"。前者是后者的**工程验证案例**，不是技术替代方案。
