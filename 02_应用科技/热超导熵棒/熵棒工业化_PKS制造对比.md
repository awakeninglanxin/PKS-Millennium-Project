# 熵棒工业化路线图 × PKS 制造工艺对比

> **来源**：IMA 笔记 7454077076128404（五大模块工业化流程）
> **日期**：2026-06-13

## 一、五大模块

| 模块 | 核心工序 | 与 SEG 制造对比 |
|:---|:---|:---|
| **一：原料预处理** | 磁渗透（≥37min 电磁共振）+ 铅浴变性（850°C/750°C） | SEG 磁粉 DC+AC 复合磁化（150-180A + 1-3MHz） |
| **二：三层涂层制备** | 垂直浸涂 → 低温干燥(5-8°C, 8h) → 中温(60°C, 4h) → 粉末注入 | SEG 四层材料模压（200-400 bar, 150-200°C, >20min） |
| **三：定向热极化** | 梯度加热(A端40-120°C, B端常温) → 真空注粉 → 蓝光监测 | SEG 滚筒→定子磁极定向排列（充磁工艺） |
| **四：检测分级** | 红外快速筛查(1min) + 抽样精密测试 + 加速老化(400°C) | SEG 尺寸公差 + 磁通量一致性检测 |
| **五：包装追溯** | 惰性气体封装 + 区块链数字护照 | — |

## 二、"热极化" = SEG "充磁"的对应物

| 熵棒热极化 | SEG 充磁 |
|:---|:---|
| 温度梯度 40-120°C | DC 150-180A + AC 1-3MHz |
| 真空注粉 → 快速抽气 | 模压 200-400 bar |
| 激光密封 | 等重量原理验收 |
| 目标："雕刻"从A到B的定向传热通道 | 目标：建立非均匀磁畴结构 |

**两者本质上都是 "通过外力在材料中建立永久不对称结构"**——熵棒用热极化建立热不对称，SEG 用充磁建立磁不对称。

## 三、成本对比

| | 熵棒（1m 管） | SEG（单环） |
|:---|:---|:---|
| SRI 报告估算 | **$220/根** | — |
| 实际工业化预估 | 远超 $220（含铑/铍等贵金属 + 极窄工艺窗口） | 数百万~数千万（取决于规模） |
| 主要成本驱动 | 超高纯原料 + 非标设备 | 稀土磁粉 + 精密加工 |
