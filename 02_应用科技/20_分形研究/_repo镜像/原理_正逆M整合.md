# ④ 正逆Mandelbrot整合对比器

**对应文件**：`分形mandelbrot整合.py`

---

## 核心功能

同时渲染标准Mandelbrot集（左）和逆Mandelbrot集（右），在一个窗口中并排对比两者的几何差异。

---

## 双面板架构

```
┌────────────────┬────────────────┐
│  标准M (左侧)   │  逆M (右侧)    │
│  z → z² + c    │ z → z² + 1/c  │
│  R=100         │  R=50         │
│  [-2, 1, -1.5, 1.5]            │
│                │ [-1.5, 4.25,   │
│                │  -2.875, 2.875]│
└────────────────┴────────────────┘
```

**默认视图**：
- 左侧：标准M的完整视图 `[-2, 1] × [-1.5, 1.5]` — 显示心形、周期2泡(-1±0i)、天线
- 右侧：逆M的完整视图 `[-1.5, 4.25] × [-2.875, 2.875]` — 显示水滴形态

---

## 核心算法

### 左侧：标准Mandelbrot

```python
@jit(nopython=True)
def mandelbrot(c, max_iter):
    z = 0j
    for n in range(max_iter):
        if abs(z) > 100:
            return n
        z = z * z + c
    return max_iter
```

### 右侧：逆Mandelbrot (c倒数法)

```python
@jit(nopython=True)
def inverse_mandelbrot(c, max_iter):
    if abs(c) > 1e-5:
        c = 1.0 / c            # ← 关键变换
        z = 0j
        for n in range(max_iter):
            if abs(z) > 50:     # 逃逸半径=50
                return n
            z = z * z + c
    return max_iter
```

### 并行计算

两个集合独立并行计算（Numba `@jit(nopython=True, parallel=True)` + `prange`），互不阻塞。

## 对比意义

将正逆M放在同一视窗中，直接观察：
1. **心形 → 水滴**：标准M的心形线如何变为逆M的水滴形外边界
2. **周期2泡(-1,0)**：标准M的(-1,0)圆对应逆M中的(-1,0)区域——这是|c|=1自对偶边界的直接体现
3. **尖点 c=0.25 → 1/c=4**：标准M的尖点在逆M中被推到无穷远附近
4. **原点 0 → ∞**：标准M的内部核心(0,0)在逆M中被翻转到无穷
