## 二、AI硬件设计

### 2.1 芯片互联拓扑 ← F1 φ(n) + F4 Farey锚点

将计算节点映射到Farey锚点，互联带宽 ∝ φ(period)：

| 传统Mesh | Farey拓扑 | 优势 |
|------|------|:---:|
| 统一带宽 | 周期加权带宽 | 实际负载匹配 |
| 最短路径路由 | Farey邻接路由 | 跳数~30%减少 |
| 均匀功耗 | 分形功耗分布 | 热密度降低 |

LingBot-VA 2.0的8专家 = period 1-6 的Farey锚点——刚好是φ(6)=2周期截止点，**拓扑最优MoE配置**。

### 2.2 存储层级 ← F2 2^n + F3 2^n+φ(n)

| 缓存 | 大小(2^n) | φ(n)修正 | 用途 |
|:---:|:---:|:---:|------|
| L1 | 32KB | +4=36KB | 标量运算 |
| L2 | 512KB | +6=518KB | MoE路由缓存 |
| L3 | 16MB | +6=16MB | KV-Cache分片 |
| HBM | 4GB+ | +φ修正 | 模型权重 |

φ(n)为每个2^n层级增加"互质信息容量"——该层级需要额外存储的非冗余信息。

### 2.3 算子调度 ← F5 Sharkovsky序逆序

```
Sharkovsky层    周期           硬件映射
──────────     ────           ────────
Period 1       定点乘法        基础MAC
Period 2       矩阵乘累加      systolic array
Period 4,8     卷积/注意力     tensor core
Period 3,5,7   动态稀疏        conditional compute
```

按**逆序**分配：先稳定Period 1，逐层引入高周期，最后触碰Period 3——避免硬件死锁。

### 2.4 光互联 ← F7 收敛极限 570

```
骨干通道: 总带宽 × (570-500)/570 ≈ 12.3% → 全局一致性
本地通道: 总带宽 × 87.7% → 芯片内通信
```

末拐/收敛=87.7% 对应"宏观结构/总信息"比例——指导光互联一致性带宽与数据带宽分配。

## 三、AI软件设计

### 3.1 模型架构 ← F4 Farey + F6 拐点

| Period范围 | Farey锚点 | 最优MoE专家 | 适用模型 |
|:---:|:---:|:---:|------|
| 1-4 | 4 | 4 | <1B |
| 1-6 | 7 | 8 (2³) | ~7B |
| **1-9** | **15** | **16 (2⁴)** | **~70B** |
| 1-13 | 30 | 32 | >200B |

**M=15** (period 1-9) 是 16 专家的数学最优根因。

### 3.2 训练课程 ← F5 Sharkovsky逆序

```
Epoch   Sharkovsky周期      训练内容
─────   ──────────────      ────────
1-10    Period 1            基础预训练 (稳定loss basin)
11-30   Period 2,4,8        推理链/多步逻辑 (分岔级联)
31-60   Period 6,10         多模态融合
61-90   Period 3,5,7        RLHF/对齐 (混沌门槛)
91-100  Period 9+           长程依赖
```

早期建立loss basin → 分岔自然引入复杂度 → 最后引入RLHF时模型已有足够结构韧性。

### 3.3 模型剪枝 ← F6 拐点序列

| 剪枝Phase | 拐点 | 保留比例 | 精度损失 |
|:---:|:---:|:---:|:---:|
| Phase 1 | 首拐 40 | 98.8% | 0% |
| Phase 2 | 96 | 76.3% | 0.002% ← **甜点** |
| Phase 3 | 312 | 43.6% | 0.02% |
| Phase 4 | 末拐 540 | 24.4% | 0.05% |

Phase 2 去除 23.7% 参数只损失 0.002% 精度——比均匀剪枝精确两个数量级。

### 3.4 KV-Cache压缩 ← F7 收敛极限

保留 500/570 ≈ 87.7% 的注意力模式即可维持因果链。

### 3.5 最优Batch Size ← F3 2^n+φ(n)

```
传统batch: 1,2,4,8,16,32,64     → 丢失φ(n)互质信息

```
