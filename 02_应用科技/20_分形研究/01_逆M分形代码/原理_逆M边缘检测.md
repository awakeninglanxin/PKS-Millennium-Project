## 文件1：高迭代点扫描法

### 核心思想

不渲染完整图像，而是扫描特定区域中"接近M集边界"的点——即逃逸迭代次数极高（> threshold）但未完全收敛的点。这些点位于 ∂M_inv 附近。

### 算法流程

```
步骤1: 将感兴趣区域分成若干子区（不同步长）
步骤2: 逐点计算逆Mandelbrot逃逸时间
步骤3: 筛选 iterations > threshold 的点
步骤4: 用红点绘制散点图
```

### 迭代公式

```python
def mandelbrot_iteration(C, max_iter=1000):
    z = 0
    for i in range(max_iter):
        z = z**2 + 1/C         # 逆M: c → 1/c
        if abs(z) > 1000:      # R=1000
            return i
    return max_iter
```

### 分区扫描策略

不同区域使用不同步长——边界密集区细扫，稀疏区粗扫：

```python
regions = [
    ((-1.4, 0), (0, 1.626), 0.05),   # 左区: 细步长 0.05
    ((0, 3.5), (0, 1.626), 0.1),      # 中区: 粗步长 0.1
    ((3.5, 4.25), (0, 1.626), 0.05)   # 右区(水滴尖): 细步长 0.05
]
```

**为什么分区**：
- 左区 (-1.4 to 0)：包含周期泡区域，边界复杂→细扫
- 中区 (0 to 3.5)：大范围稀疏区域→粗扫即可
- 右区 (3.5 to 4.25)：水滴尖端 c=4 附近→细扫捕捉尖端细节

### 边缘点判定

```python
if iterations > threshold:  # threshold=950（max_iter=1000的95%）
    red_points.append(C)
```

阈值取 max_iter 的 95%，确保选中的是最接近边界（"差一点就逃逸"或"差一点就收敛"）的参数点。

## 文件2：Y轴极值 + 牛顿法细化

### 新增功能

1. **mpmath 高精度**：`mp.dps = 30` 位精度
2. **牛顿法边界细化**：对初步筛选出的候选点，用牛顿法逼近精确边界位置
3. **Y 轴最大值计算**：找到逆M水滴上半部分边界的最高 Y 坐标

### 牛顿法细化原理

逆M集内部满足 z→z²+1/c 有界。在边界附近，可用牛顿法求解"逃逸=有界"的临界c：

```python
def newton_method(C, max_iter=500):
    z = 0
    for i in range(max_iter):
        f = z**2 + 1/C - z        # 不动点方程
        f_prime = 2*z - 1/C       # 导数
        z = z - f / f_prime       # 牛顿迭代
```

**为什么需要牛顿法？** —— 网格扫描给出的是"接近边界"的离散点，牛顿法可以从这些离散候选点出发，沿着梯度方向精确收敛到真正的边界位置。

### Y轴最大值计算

搜索上半平面 (Y>0) 的水滴边界，找到迭代次数最高且牛顿法能收敛的点的最大 Y 坐标。
