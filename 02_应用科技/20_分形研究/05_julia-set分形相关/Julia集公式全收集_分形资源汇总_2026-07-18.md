## 1. 主页面（matousstieber WordPress）全公式

### 1.1 文章摘要

该文出自 *Chimpanzee3D Fractal Blog*（计算设计 / 分形博客，作者 Matouš Stieber，含 Grasshopper、C#、Python 脚本）。核心论点：

- Julia 集由法国数学家 **Gaston Julia**（约 1915 年研究，1918 年发表名篇）定义。
- 现今 Julia 集通常与**二次多项式**关联，但 Julia 本人最初研究的是更一般的表达式。
- Julia 集的形态取决于复参数 **c**：从 Mandelbrot 集**内部**取的 c → 连通 Julia 集；从 Mandelbrot 集**外部**取的 c → 不连通（"尘埃 / dust"，无论分辨率多高都是离散点）。
- 文章展示大量**变形 Julia 集**（有理映射、Sierpinski 曲线、Cantor 集、sin(z)、Glynn、Herman 环等）。

### 1.2 全部公式（逐字收录）

#### (A) 一般形式（Julia 本人原始兴趣表达式）
$$
f_c(z) = z^{4} + \frac{z^{3}}{z-1} + \frac{z^{2}}{z^{3} + 4 z^{2} + 5} + c
$$

#### (B) 现代标准定义（任意次数的二次多项式族）
$$
f_c(z) = z^{n} + c, \qquad c \in \mathbb{C}
$$
> 文中原文："Julia sets are quadratic equation of various degrees $f_c(z)=z^n + c$; where c is a complex parameter. For almost every c, this transformation generates a fractal."

#### (C) 变形 1 — 近二次扰动（cubic with tiny quadratic perturbation）
$$
f(z) = z^{3} - z^{2.00001} + 1.0008875
$$

#### (D) 变形 2 — 带线性项的三次（λ 复参数）
$$
f_{\lambda}(z) = z^{3} + \lambda \cdot z, \qquad \lambda = 1.01 \cdot e^{0.1i}
$$

#### (E) 变形 3 — 指数 Julia 集（Exponential Julia）
$$
f(z) = \lambda e^{Z}
$$

#### (F) 有理映射一般式（Rational Maps，Sierpinski 曲线族）
$$
f_{\lambda}(z) = z^{n} + \frac{\lambda}{z^{d}}, \qquad n \ge 2,\; d \ge 1,\; \lambda \in \mathbb{C}
$$

#### (G) Sierpinski 曲线 Julia 集（特例）
$$
f(z) = \frac{2}{3}\cdot\frac{z^{3} - 2}{z}
$$
> 文中另给出 λ = −0.25、−0.0625、−0.01 的参数化 Sierpinski 曲线示例（均套用 (F) 式）。

#### (H) 带分母项的二次
$$
f(z) = z^{2} + \frac{c}{z^{6}}, \qquad c = -0.01
$$

#### (I) Cantor 集 Julia 集（有理映射，n=d=4）
$$
f_{\lambda}(z) = z^{4} + \frac{\lambda}{z^{4}}, \qquad \lambda = 0.04
$$

#### (J) 带缩放二次扰动
$$
f(z) = z^{2} - 1 - 0.005 z^{2}
$$

#### (K) 带 −i 常数与分母三次项
$$
f_{\lambda}(z) = z_{n}^{3} - i + \frac{\lambda}{z_{n}^{3}}, \qquad \lambda = 0.0001
$$

#### (L) 纯三次（无分母）
$$
f(z) = z_{n}^{3} + c, \qquad c = (-0.58384,\; -0.27022i)
$$

#### (M) 三次 + 分母三次项（λ 极小）
$$
f_{\lambda}(z) = z_{n}^{3} - 0.58384 - 0.27022i + \frac{\lambda}{z_{n}^{3}}, \qquad \lambda = 0.000001
$$

#### (N) sin(z) Julia 集 — 复正弦分解
$$
\sin(x + iy) = \sin(x)\cosh(y) + i\cos(x)\sinh(y)
$$
> 性质：沿 x 轴值指数增长至 ∞；沿 y 轴值保持有界。
> 文中示例：c = 1 + 0.3i、c = 1 + i。

#### (O) Glynn 分形（1 < n < 2，c 靠近 Mandelbrot 边界）
$$
f_c(z) = z^{1.5} - 0.2
$$

#### (P) Herman 环（无理数旋转的有理映射 Fatou 分量）
$$
z_{n+1} = \frac{e^{2\pi\Phi i}\; z_{n}^{2}\,(z_{n}-4)}{1 - 4z_{n}}, \qquad \Phi = \frac{\sqrt{5}-1}{2}
$$
> 定义：Herman 环是一个 Fatou 分量，其中有理函数共形共轭于标准环域的无理旋转。

## 2. Wikibooks — *Pictures of Julia and Mandelbrot sets*（渲染算法全套公式）

> 这是对 Julia / Mandelbrot 集**计算机绘制数学原理**最完整的开源推导。以下逐条收录。

### 2.1 有理函数与次数
$$
f(z) = \frac{p(z)}{q(z)}, \qquad \deg f = d = m - n
$$
（p、q 为多项式，m、n 分别为分子、分母次数）

### 2.2 Julia 集与 Fatou 集（定义）
- **填充 Julia 集**：$K_f = \{z : \text{轨道 } f^k(z) \text{ 有界}\}$
- **Julia 集**：$J(f)$ 是 $K_f$ 的边界 = $\mathbb{C} \setminus F(f)$，其中 $F(f)$ 为 Fatou 集（稳定域）。
$$
J(f) = \mathbb{C} \setminus F(f)
$$

### 2.3 临界点（Critical Points）
$$
f'(z) = 0
$$
> Fatou 定理：每个吸引循环都吸引至少一个临界点。

### 2.4 吸引度（Attraction）
设 $z^*$ 为周期 $r$ 的吸引循环，$f^r$ 为其 $r$ 次迭代，则吸引度：
$$
\alpha = \frac{1}{\left|(f^{r})'(z^*)\right|}
$$
- 若 $(f^{r})'(z^*) = 0$ → **超吸引（superattracting）**，$\alpha = \infty$。
- **Newton 迭代法**（求 $g(z)=0$ 的根）：
$$
f(z) = z - \frac{g(z)}{g'(z)}
$$

### 2.5 势函数（Potential Function / 着色核心）
用于给外部点按"到 Julia 集的距离感"上色。

- **非超吸引**（一般吸引）：
$$
\phi(z) = \lim_{k\to\infty} \frac{\log|z_k - z^*|}{k}
$$

- **超吸引**（$\alpha = \infty$）：
$$
\phi(z) = \lim_{k\to\infty} \frac{\log|z_k - z^*|}{s^k}, \qquad s = \text{循环阶相关常数}
$$

- **当吸引点为 ∞**（$z^* = \infty$，即标准 $z^2+c$ 外部）：
$$
\phi(z) = \lim_{k\to\infty} \frac{\log|z_k|}{d^k}
$$
（d 为映射次数）

### 2.6 实迭代次数（平滑着色 / Smooth Iteration）
为避免色带断层，用势函数反解"连续迭代次数"：
$$
k_{\text{real}} = k + \text{（由 } \phi(z) \text{ 反推的小数修正项）}
$$
> 实际实现中常以 $\mu = k + 1 - \log(\log|z_k|)/\log d$ 形式的连续逃逸时间（见 §4 bugman123 的 cet 公式）。

### 2.7 距离估计（Distance Estimation）— 外点距 Julia 集的距离
$$
\delta(z) = \lim_{k\to\infty} \frac{|z_k - z^*|}{|z_k'|}
$$
其中序列导数递推：
$$
z_{k+1}' = f'(z_k) \cdot z_k', \qquad z_0' = 1
$$
> 用于绘制**等距轮廓线**与抗锯齿边界。在 $z^*=\infty$ 情形化为经典式 $\delta = \frac{|z_k|\log|z_k|}{|z_k'|}$（见 §4）。

### 2.8 场线（Field Lines）
连接周期循环各点的"场线"，角度方程：
$$
\psi(z) = \arg(z_k - z^*) - k\cdot\theta \pmod{2\pi}
$$
（$\theta = 2\pi/m$，$m$ 为循环周期；$k$ 为迭代步）

### 2.9 填充 Julia 集（集合判定）
$$
K_f = \{z \in \mathbb{C} : \text{序列 } (f^n(z))_{n\ge 0} \text{ 有界}\}
$$

## 3. MathWorld — Julia Set（定义与维数公式）

### 3.1 有理函数定义
设 $R(z) = P(z)/Q(z)$ 为无公共因子的有理函数（P、Q 为多项式），定义域扩展到黎曼球 $\hat{\mathbb{C}} = \mathbb{C} \cup \{\infty\}$。

### 3.2 填充 Julia 集与真 Julia 集
- **填充 Julia 集** $J_R$：反复迭代 $R(z)$ 后**不趋于 ∞** 的点集（对应奇异吸引子）。
- **真 Julia 集** $J$：填充集的边界（例外点集）。
- 两类：连通集（对应 Fatou 集）或 Cantor 集（Fatou dust / 尘埃）。

### 3.3 二次 Julia 集
$$
z_{n+1} = z_n^{2} + c
$$
> 几乎对几乎所有 c 都生成分形。已知**非分形**的特例：$c = -2$（Dufner et al. 1998）与 $c = 0$。

### 3.4 著名特例（c 值命名）
| c | 名称 |
|---|------|
| $c = i$（Mandelbrot 边界） | 树枝分形（Dendrite fractal） |
| $c = -0.123 + 0.745i$ | Douady 兔（Douady's rabbit） |
| $c = -0.75$ | 圣马可分形（San Marco fractal） |
| $c = -0.391 - 0.587i$ | Siegel 盘分形（Siegel disk） |

### 3.5 共形映射性质
二次 Julia 映射是**保角映射**（保角），角度不变。若 $P \in J$，则其所有迭代像均在 $J$ 上；映射有**二值逆**。

### 3.6 容量维数（Capacity Dimension）
对 $c \ll 1$ 的小 c 二次 Julia 集：
$$
\dim_{\text{cap}}(J_c) \approx 1 + \frac{|c|\pi}{4\log 2} \quad \text{（小 c 渐近式，MathWorld 记为式 (3)）}
$$
> 小 c 时 $J_c$ 也是 Jordan 曲线，但点不可计算（computable）。

## 4. Paul Bourke — Julia Set Fractal（实践渲染）

### 4.1 迭代定义
$$
z_{n+1} = z_n^{2} + c
$$
- $z = x + iy$ 为复平面上的点；$c$ 为固定复常数（每个 c 对应一个不同 Julia 集）。
- 初始值 $z_0$ 取图像平面上**每一个像素**对应的复坐标。
- **逃逸判定**：序列模超过阈值（通常 $|z| > 2$）即判为发散（白）；否则归为集合内（黑）。彩色版按发散**速度**上色。

### 4.2 广义迭代族（非线性 f(z) 均可）
$$
z_{n+1} = f(z_n)
$$
Bourke 列举的常见有趣函数：
$$
\begin{aligned}
z_{n+1} &= c \sin(z_n) \\
z_{n+1} &= c \exp(z_n) \\
z_{n+1} &= c\, i \cos(z_n) \\
z_{n+1} &= c\, z_n (1 - z_n)
\end{aligned}
$$

### 4.3 Mandelbrot 集作为 Julia 集索引
- Julia 集要么连通、要么不连通。
- **c 取自 Mandelbrot 集内部** → 连通 Julia 集；**c 取自外部** → 不连通（尘埃）。
- 对称性：$c$ 为实数 → Julia 集关于实轴对称；含非零虚部 → 180° 旋转对称。

### 4.4 文中示例 c 值（用于复现图像）
c = 0+0.8i ｜ 0.37+0.1i ｜ 0.355+0.355i ｜ −0.54+0.54i ｜ −0.4−0.59i ｜ 0.34−0.05i ｜ 0.355534−0.337292i（Chris Thomasson 提供）等。

### 4.5 附带 sin(z) Julia 专页
- 参见 paulbourke.net/fractals/sinjulia/（复正弦 Julia 集）。

## 5. Bugman123 — Fractals（公式库，含距离估计代码式）

### 5.1 Mandelbrot 集
$$
z_{n+1} = z_n^{2} + c
$$
- 第 n 层 Mandelbrot 多项式：$F_n(z) = f^{\circ n}(z)$（n 次复合）。
- Mathematica 实现：`Mandelbrot[c_] := Module[{z=0,i=0}, While[i<100 && Abs[z]<2, z=z^2+c; i++]; i]`
- **连续逃逸时间（height field）**：
$$
\text{cet} = n + \frac{\log_2(\ln R) - \log_2(\ln|z|)}{\text{（缩放因子）}}
$$
（原文记号 `_cet = n + log2ln(R) - log2ln|z|_`）
- **距离估计**：`z = z^2 + c; dz *= 2z; ... 0.5 r Log[r]/Abs[dz]`

### 5.2 二次 Julia 集
$$
z_{n+1} = z_n^{2} + c, \qquad c = -0.63 - 0.407i
$$

### 5.3 三次 Julia 集
$$
z_{n+1} = z_n^{3} + c, \qquad c = -0.5 - 0.05i
$$
- 距离估计：`dz = 3 z^2 dz + 1; z = z^3 + c; ... 0.5 r Log[r]/Abs[dz]`

### 5.4 Glynn 集
$$
z_{n+1} = z^{1.5} + c, \qquad c = -0.2
$$
> 形似树状。

### 5.5 逆 Julia 集（IIM / MIIM，逆迭代法）
三次 Julia 的逆映射：
$$
z_{\text{next}} = (z - c)^{1/3} \cdot \{1,\; -(-1)^{1/3},\; (-1)^{2/3}\}
$$
> MIIM 在累积导数变大（映射变为压缩）时裁剪分支。

### 5.6 其他相关分形迭代式（同站收录，供扩展）
