## 一、文档核心：3-4-5 的工程落地

### 1.1 叮咚四面体的工程角色

文档揭示了一个此前在 PKS 分析中被忽略的关键事实：

> **3-4-5 勾股三角形不是抽象数学玩具——它是四面体 IEC 中子源的几何蓝图。**

| 层面 | 实现 |
|:---|:---|
| **叮咚老师的数学构造** | 12 个 3-4-5 勾股三角形 → 1 个正四面体 → R/r = (1+√2)⁴ = 33.97 |
| **蓝馨的推导验证** | Wolfram 精确验证 + "4次白银比旋转" 洞察 |
| **工程落地** | 四面体 IEC 电极布局 ← 正是这个四面体的几何结构 |

### 1.2 四面体 IEC 装置的工作原理（基于 US5825836A）

```
4 个特斯拉线圈驱动电极
    → 四面体顶点布局（正四面体 4 顶点）
    → 多极电场在中心形成"虚阴极"势阱
    → 氘离子从四面反射面加速入中心
    → 惯性静电约束 (IEC) → 内爆压缩
    → 瞬态高密度等离子体 (10^19-10^20 m^-3)
    → D-D / D-T 聚变反应
    → 单次脉冲 10^9-10^11 中子
```

### 1.3 为什么是四面体而非六面体/八面体

正四面体在所有正多面体中具有**最大的"表面-体积比"**——对于给定电极间距，四面体几何使得静电场在中心最有效地汇聚。这与叮咚老师的推导 R/r = (1+√2)⁴ 的深层物理含义一致：

$$R/r = (1+\sqrt{2})^4 \approx 33.97$$

- R = 外接球半径（电极中心到四面体中心的距离）
- r = 内切球半径（虚阴极势阱的等效半径）

**R/r 越大 → 电场聚焦越强 → 中心等离子体密度越高 → 中子产率越高。**

四面体恰好给出了这个比值的解析最优解。

## 四、PKS 框架下的优化方向

### 4.1 Tesla 线圈的 Schauberger 替换

传统 Tesla 线圈产生的是**正弦高压**（对称波形）。Schauberger 螺旋管（参考 `DXF生成程序集/07_参数化螺旋锥/羚羊角管...py`）可以产生**非对称的波形尖峰**：

- 正弦驱动 → 离子在势阱中做对称振荡
- Schauberger 螺旋驱动 → 离子被"单向锤击"入中心 → **更高压缩比**

这与波纹盘的原理一致：$n$ 次波纹槽逐级压缩 → 水到 HHO。四面体 IEC 中：Schauberger 螺旋脉冲 → 离子逐次加速 → 更高聚变截面。

### 4.2 3-4-5 角度优化电极方位

Barbara Hero 的 Plimpton 322 映射表给出了**37°/53° 的经络共振角度**。如果四面体电极的方位角不是均匀的 60°，而是按照 3-4-5 锐角（≈37°/53°）微调，可能产生额外的**几何共振**增强。

### 4.3 白银比 χ 的场聚焦优化

$$R/r = (1+\sqrt{2})^4 = 33.97$$

这不是任意值——它是白银比 χ = 1+√2 的四次幂。如果电极间距和电压选择使得电场梯度的特征长度恰好与这个比值对齐，理论上可以实现**最优聚焦**。
