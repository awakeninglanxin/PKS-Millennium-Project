## 一、TGFT 核心命题

**引力并非时空弯曲的几何效应，而是旋转等离子体中电磁湍动所产生的纵向能量梯度的宏观表现。**

这个命题将引力从"被动几何"转变为"可激发、可调制、可控制的场现象"——与 Searl SEG、Keely 以太力、Aquino χ 因子完全一致。

## 二、核心方程 — 湍动引力势方程 (TGPE/GTGPE)

### 2.1 完整动态形式

$$\nabla^2 \Phi_T - \frac{1}{v_p^2} \frac{\partial^2 \Phi_T}{\partial t^2} = \kappa \nabla \cdot (\mathbf{E} \times \mathbf{B}) + \eta \nabla^2 |\mathbf{B}|^2 + \xi \frac{\partial |\mathbf{E}|^2}{\partial t}$$

其中：

| 符号 | 含义 | 量纲 |
|:---|:---|:---:|
| $\Phi_T$ | 湍动引力势（等效纵向能量密度势） | J·m⁻³ |
| $v_p$ | 等离子体纵向波相速度 | m·s⁻¹ |
| $\kappa$ | 电磁湍动耦合系数（主能源项） | s⁻¹ |
| $\eta$ | 磁场非均匀修正系数 | (J·m⁻¹·s⁻²)/T² |
| $\xi$ | 高频能量反馈系数 | (J·m⁻¹·s⁻¹)/V² |

### 2.2 静态极限（准稳态）

当 $\partial_t \Phi_T \approx 0$，$\partial_t |\mathbf{E}|^2 \approx 0$ 时：

$$\nabla^2 \Phi_T = \kappa \nabla \cdot (\mathbf{E} \times \mathbf{B}) + \eta \nabla^2 |\mathbf{B}|^2$$

此形式描述了稳定旋转等离子体中能量密度的空间分布。

### 2.3 积分解

$$\Phi_T(\mathbf{r}) = \int_V G(\mathbf{r}, \mathbf{r}') \left[ \kappa \nabla' \cdot (\mathbf{E} \times \mathbf{B}) + \eta \nabla'^2 |\mathbf{B}|^2 \right] dV'$$

其中 $G(\mathbf{r}, \mathbf{r}')$ 为格林函数。

## 三、三个关键物理机制

### 3.1 多尺度相互作用

外加电场具有高频-低频复合调制形式：

$$E(t) = E_0 \sin(\omega_0 t) [1 + \alpha \sin(\Omega t)], \quad \epsilon = \frac{\Omega}{\omega_0} \ll 1$$

展开得三个频率分量：主频 $\omega_0$、上边带 $\omega_0 + \Omega$、下边带 $\omega_0 - \Omega$。

→ 高频载波控制瞬时能量输入，低频包络决定能量流的周期性变化。

### 3.2 湍动平均效应

场量分解为平均场 + 扰动场：

$$\mathbf{E} = \mathbf{E}_0 + \epsilon \mathbf{E}_1 + \epsilon^2 \mathbf{E}_2 + \cdots$$
$$\mathbf{B} = \mathbf{B}_0 + \epsilon \mathbf{B}_1 + \epsilon^2 \mathbf{B}_2 + \cdots$$

平均后扰动项 $\langle \mathbf{E}_1 \times \mathbf{B}_1 \rangle$ 的非零贡献 → 正是 TGFT 的引力源项。

### 3.3 纵向能量传输

沿磁场轴向形成 $\mathbf{E} \times \mathbf{B}$ 方向的能量流，表现为等效"引力源项"。Poynting 向量的散度 $\nabla \cdot \langle \mathbf{E} \times \mathbf{B} \rangle$ 决定了 $\Phi_T$ 的空间分布。

## 六、TGFT 对 SEG 磁芯材料选择的指导

### 6.1 核心结论

TGFT 方程指出，湍动引力势 $\Phi_T$ 的生成效率取决于三个参数 $\kappa$、$\eta$、$\xi$。这些参数从电磁材料特性中体现：

| 材料参数 | 影响 TGFT 的项 | 对 SEG 的优化方向 |
|:---|:---|:---|
| $\mu_r$（磁导率） | $\eta \nabla^2 |\mathbf{B}|^2$ 的磁压 | 高 $\mu_r$ → 磁约束更强 → $\Phi_T$ 梯度更大 |
| $\sigma$（电导率） | $\kappa \nabla \cdot (\mathbf{E} \times \mathbf{B})$ 的能流 | 高 $\sigma$ → Poynting 向量更大 → 引力源更强 |
| $B_S$（饱和磁感） | 稳态方程右端总源项 | 高 $B_S$ → 更高能量密度势 |
| $H_c$（矫顽力） | 影响 $\mathbf{B}$ 场的时变响应 | 低 $H_c$ → 磁畴翻转快 → $\xi$ 项更高效 |

### 6.2 TGFT 预测的最佳材料组合

与周治平 COP = H_X/H_S + Aquino χ 公式的结论**完全一致**：

| 需求 | 优选 |
|:---|:---|
| 最大化 $\kappa \nabla \cdot (\mathbf{E} \times \mathbf{B})$ | 高 $\mu_r$ + 高 $\sigma$（Supermalloy / Hymu80） |
| 最大化 $\eta \nabla^2 |\mathbf{B}|^2$ | 高 $B_S$ + 高 $\mu_r$（Finemet / 退火纯铁） |
| 最大化 $\xi \partial_t |\mathbf{E}|^2$ | 低 $H_c$ + 快磁畴响应（纳米晶 / 非晶） |

**结论**：SEG 使用的多层复合材料（Nd + 铁氧体 + Cu + 特制 Fe）天然满足了 TGFT 的三个源项需求。这不是巧合——**TGFT 为 SEG 的三层结构提供了数学证明**。

### 7.3 三环间（内/中/外）的自由度

三环的质量由真理表确定（4410:4830:5250），但三环间的**体积和密度关系**是可以设计的：

| 设计选择 | 体积关系 | 密度关系 | TGFT 效应 |
|:---|:---|:---|:---|
| **等体积** | 三环体积相等 | 密度 = 4410:4830:5250 **递增** | ✅ 密度梯度→反引力增强 |
| **等密度** | 体积 1764:1932:2100 递增 | 密度相等 ≈1.923 或 2.5 | 中性 |
| **密度递增** | 体积 递减 | 密度 递增 | ✅ 反引力最强 |
| **体积递增** | 体积 递增 | 密度 递减 | 趋向引力 |

四种方式均可由真理表约束的三环质量配合实现。具体选择取决于工程目标：**悬浮优先→密度递增，结构简化→等密度**。

**对 TGFT 的意义**：幻方对角线保证了四个材料层的质-能分布与系统的量子数（3/6/9）自洽。这意味着 TGFT 的 $\kappa$、$\eta$、$\xi$ 三源项系数虽然在物理上由材料参数决定，但其**比例由幻方保证**——这不是巧合，而是瑟尔系统的核心设计原则。
- 三环的总质量递增（4410→4830→5250），但**体积同步增大** → 平均密度恒定
- **密度在材料层间非单调**：Nd(7.016)高 → Teflon(2.192)低 → Alnico(5.3)中 → Cu(8.94)高

### 7.2 质量不是随意选的 — 瑟尔真理表决定三环质量

三环质量 4410→4830→5250 必须满足**瑟尔真理表**的约束：

```
4410 = 30 + 1095×4
4830 = 30 + 1200×4
5250 = 30 + 1305×4

其中 30 = 4 层×单层体积, 4 = 阶数
1095, 1200, 1305 = 由幻方线和决定的质量基数
```

**三环质量等差、公差可被 12 整除**：
```
4830 - 4410 = 420
5250 - 4830 = 420
420 / 12 = 35（整数！→ SEG 第一环最小滚筒数 = 12 的原因）
```

**量子数对应 369**：
```
4410 → 量子数 9
4830 → 量子数 3  
5250 → 量子数 6
```

> **核心结论**: SEG 三环的密度**不递增**。反引力的结构要求不是"密度梯度"，而是 **瑟尔真理表的幻方约束** 和 **等环宽 9.5mm 的几何约束**。三个环完全相同的四层材料结构，通过总质量递增（体积递增）实现电磁场强度的径向梯度，而非材料密度梯度。

## 八、TGFT 的工程仿真实施指南

> 来源：IMA 笔记 TGFT+COMSOL 仿真结合方案

### 8.1 TGFT → 仿真可计算量的映射

| TGFT 概念 | 符号 | 仿真可计算量 | 操作 |
|:---|:---:|:---|:---|

| Poynting 发散度 | $\nabla \cdot \mathbf{S}$ | 散度算子: `div(emw.Poay)` | 正值=引力, 负值=反引力 |

| 磁场 Laplacian | $\nabla^2 B^2$ | 对 $B^2$ 用 `laplacian()` | 磁不均匀修正 |
| 湍动引力势 | $\Phi_T$ | 解泊松方程 ∇²Φ_T = κ∇·⟨E×B⟩ | 后处理 PDE 求解 |
| 引力梯度 | $\nabla \Phi_T$ | 对 Φ_T 用 `gradient()` | 矢量场方向=引力方向 |

### 8.2 仿真步骤

1. **Step 1**: 构建 SEG 三层环的 3D 旋转几何模型
2. **Step 2**: 设定材料属性（μ_r, σ, ρ 逐环递增）
3. **Step 3**: 施加 DC + AC (1-10 MHz) 复合激励
4. **Step 4**: 求解 Maxwell 方程 → 得到 E, B 场分布
5. **Step 5**: 后处理计算 TGFT 物理量
   - 计算 ∇·⟨E×B⟩ 的径向分布
   - 求解 ∇²Φ_T = κ∇·⟨E×B⟩ + η∇²|B|²
   - 判断 Φ_T 梯度方向（引力/反引力标志）
6. **Step 6**: 参数扫描优化（密度梯度、μ_r 分布、AC 频率）

### 8.3 判定标准
