## 一、整体质量评估框架

```
生成图像 → [层级1: 基础质量] → [层级2: 人体解剖] → [层级3: 面部质量] → [层级4: 语义一致性] → 合格/重修
```

## 二、层级1：基础图像质量

### 2.1 清晰度检测

**算法**：Laplacian 方差

```python
import cv2
import numpy as np

def detect_blur(image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    variance = np.var(cv2.Laplacian(gray, cv2.CV_64F))
    # 方差越大，图像越清晰
    # 通常 < 100 被认为是模糊的
    return variance > 100
```

**原理**：拉普拉斯算子是二阶导数滤波器，检测图像的边缘强度。模糊图像的边���弱，拉普拉斯方差低。

### 2.2 噪声/伪影检测

**算法**：频谱分析（FFT）

```python
import numpy as np
from scipy.fft import fft2

def detect_artifacts(image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    spectrum = np.abs(fft2(gray))
    # 计算频谱的异常峰值
    # 正常图像频谱平滑，有伪影的频谱出现尖锐峰值
    return np.std(spectrum) / np.mean(spectrum) > threshold
```

**原理**：傅里叶变换将图像转到频域。干净的图像在频域中频谱平滑，有噪声/压缩伪影的图像会出现异常频谱分量。

## 三、层级2：人体解剖检测

### 3.1 肢体检测

**算法**：OpenPose / YOLO-Pose

```python
from mediapipe.python.solutions import pose

def detect_limb_issues(image):
    mp_pose = mp.solutions.pose
    pose_detector = mp_pose.Pose(static_image_mode=True)
    
    results = pose_detector.process(image)
    landmarks = results.pose_landmarks.landmark
    
    # 检测关键点是否存在（完整性）
    expected_keypoints = 33  # OpenPose 标准关键点数
    present_keypoints = sum(1 for kp in landmarks if kp.visibility > 0.5)
    
    completeness_ratio = present_keypoints / expected_keypoints
    
    # 检测对称性
    left_hand_visibility = landmarks[23].visibility  # 左腕
    right_hand_visibility = landmarks[25].visibility  # 右腕
    
    asymmetry = abs(left_hand_visibility - right_hand_visibility)
    
    return completeness_ratio, asymmetry
```

```python
def check_symmetry(face_landmarks):
    # 取面部关键点的左右对称轴
    mid_x = np.mean([lm.x for lm in face_landmarks])
    
    left_eye = face_landmarks[eye_left_indices]
    right_eye = face_landmarks[eye_right_indices]
    
    # 镜像翻转左边，与右边对比
    left_reflected = flip_horizontally(left_eye, mid_x)
    
    # 计算差异
```

```
└─ 眉毛位置 → 是否等高
```

### 4.2 面部扭曲检测

**算法**：Landmark 偏移 + 深度学习分类器

```python
def detect_face_distortion(landmarks):
    # 计算各部位的相对位置
    eye_distance = distance(landmarks[left_eye], landmarks[right_eye])
    nose_width = distance(landmarks[nose_left], landmarks[nose_right])
    mouth_width = distance(landmarks[mouth_left], landmarks[mouth_right])
    
    # 与人均标准比例比较（基于数百万人脸的统计）
    expected_ratio = 0.35  # 平均的眼睛间距 / 鼻子宽度
    actual_ratio = eye_distance / nose_width
    
    distortion_score = abs(actual_ratio - expected_ratio) / expected_ratio
    
    if distortion_score > 0.2:
        return "DISTORTED"
    elif distortion_score > 0.1:
        return "SUBOPTIMAL"
    else:
        return "GOOD"
```

## 六、整体决策逻辑

### 6.1 质量评分系统

```
综合评分 = Σ (权重_i × 维度_i得分)

维度及权重：
├─ 清晰度:          25%
├─ 解剖完整性:      20%
├─ 面部质量:        20%
├─ 对称性:          15%
├─ 色彩和谐:        10%
├─ Prompt一致性:    10%

合格线: ≥ 0.70
优秀线: ≥ 0.85

低于合格线 → 触发重修流程
```

### 6.2 重修策略

```
根据失败维度选择重试参数：

失败原因        调整策略
────────────────────────────────────────────
手部畸形        → 增加 "FIVE FINGERS, detailed hands"
面部不对称      → 增加 "SYMMETRICAL FACE, balanced proportions"
肢体缺失        → 增加 "COMPLETE BODY, no missing limbs"
模糊不清        → 提高 Steps, 降低 CFG, 换 Seed
颜色失真        → 调整 Color Palette prompt
不符合种族特征  → 增强 Race-specific prompt 权重
```
