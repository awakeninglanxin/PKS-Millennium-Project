## 二、AI 审美的底层逻辑

### 2.1 一句话概括

```
AI 的"审美" = 高维统计分布的匹配度

简单说：AI 见过的"美的图"多了，就会认为"长得像那些图的也是美的"
```

### 2.2 详细解释

```
训练阶段：
├─ 输入：互联网上的数百万张高质量图片 + 对应的文字描述
├─ 学习：这些图片在特征空间中的分布模式
└─ 输出：学会区分"符合分布的图（美的）" vs "偏离分布的图（丑的）"

生成阶段：
├─ 从一个随机噪声出发，逐步去噪
├─ 每一步都向"训练数据分布"靠拢
└─ 最终收敛到分布中的"典型样本"（这就是"美的"）
```

### 2.3 关键数学工具

| 工具 | 用途 | 重要性 |
|------|------|--------|
| 卷积神经网络 (CNN) | 特征提取 | ★★★★★ |
| 注意力机制 (Attention) | 关注重点区域 | ★★★★☆ |
| 傅里叶变换 (FFT) | 频域分析/清晰度 | ★★★☆☆ |
| Cosine Similarity | CLIP Score | ★★★★☆ |
| 梯度下降 | 去噪过程 | ★★★★★ |
| Wasserstein 距离 | FID 计算 | ★★★☆☆ |

## 八、IFS分形审美的补全（2026-07-19更新）

> 前七章描述的是**现有AI统计审美**的完整体系。本章描述的是IFS分形审美如何与之互补——不替代，补空白。

### 8.1 现有体系的定位：L0+L1

现有AI审美（CLIP/FID/MediaPipe/CNN拓扑）解决的是：

| 层 | 问题 | 方法 | 状态 |
|:---|:---|:---|:---:|
| L0 | 这是人还是怪形？ | CNN统计拓扑 | ✅ 完成 |
| L1 | 清晰吗？对称吗？配色好吗？ | 关键点+频域+HSV | ✅ 完成 |

现有体系止步于L0和L1。它没有L2（比例评分）和L3（动力学终审）。

### 8.2 IFS补全的L2+L3

IFS分形审美基于M集7族内生数列，补全比例层和动力学层：

| 层 | 问题 | 方法 | M集对应 |
|:---|:---|:---|:---|
| **L2** | 比例沿Fibonacci链收敛吗？ | Feigenbaum δ检验+Farey树路径 | 族1+族2+族4+族7 |
| **L3** | 不对称扭点在哪？发育质量如何？ | Misiurewicz(k,p)+逆M水滴五公理 | 族6+逆M |

### 8.3 两套体系的协作模式

```
输入：一张人体图像
  │
  ├─ L0(现有AI) ──→ 拓扑错误？ → 直接淘汰，不评分
  │     通过
  ├─ L1(现有AI) ──→ 对称/清晰/配色评分 (权重35%)
  │
  ├─ L2(IFS新增) ─→ Fibonacci链/Farey树/Kneading序列评分 (权重40%)
  │
  └─ L3(IFS新增) ─→ Misiurewicz/逆M五公理评分 (权重25%)
      │
      总分 = 0.35×L1 + 0.40×L2 + 0.25×L3
```
