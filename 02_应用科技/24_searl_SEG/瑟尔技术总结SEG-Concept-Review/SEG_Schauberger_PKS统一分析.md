## 一、SEG 技术核心结构

### 1.1 "跳房子数学"——方块法则

John Searl 声称他的整套数学来自 6 年梦境（4-10 岁）。核心是英国的跳房子游戏——**8 个方块**：

```
     ┌───┐
     │ 8 │  ← 单方块
  ┌──┼───┼──┐
  │ 6│ 7 │  ← 双方块
  ├──┼───┤
  │ 4│ 5 │  ← 双方块
  └──┼───┼──┘
     │ 3 │  ← 单方块
  ┌──┼───┤
  │ 1│ 2 │  ← 双方块(DNA层，非"真方块")
  └──┴───┘
```

| 方块 | 角色 | 对应元素 | 物理意义 |
|:---:|------|:-------:|---------|
| 1-2 | DNA层 | — | 不参与设计计算 |
| 3 | 基石 | Ca (20) | 固体结构需求 |
| 4 | **核心设计方块** | Nd (60) | SEG的完整参数来源 |
| 8 | 完整周期 | — | 8音程 × 音乐结构 |

### 1.2 SEG 物理结构

Searl 描述的标准 SEG 包含：

| 组件 | 数量 | 材料 | 角色 |
|------|:---:|------|------|
| **固定板 (Plate)** | 3 | 层压磁化材料 | 定子——产生旋转磁场 |
| **滚动体 (Roller)** | 12+22+32=66 | Nd-Ni 合金+外层 | 转子——在平板间自由滚动 |
| **隔离层** | — | Teflon/Nylon | 导电-绝缘双态需求 |
| **间隙 (Air Gap)** | — | 精确控制 | 磁场耦合强度 |

```
         ┌─────────────────┐
         │    Plate 3      │ ← 顶层固定板 (32 rollers)
         │  ○ ○ ○ ○ ○ ○ ○ │
         ├─────────────────┤
         │    Plate 2      │ ← 中层固定板 (22 rollers)
         │  ○ ○ ○ ○ ○ ○ ○ │
         ├─────────────────┤
         │    Plate 1      │ ← 底层固定板 (12 rollers)
         │  ○ ○ ○ ○ ○ ○ ○ │
         └─────────────────┘
         Rollers (○) 在相邻板之间自由旋转
```

### 1.3 Searl 效应

SEG 运行时的核心物理：

1. 平板上的特定磁化图案（正弦波形磁化）→ 产生旋转磁场
2. 磁场驱动滚柱在固定间隙中转动
3. 滚柱的旋转产生**感应电压**（"without motion there can be no induced voltage"）
4. 电子从周围环境中被"吸入"系统 → 电流 → 输出功率
5. 系统自冷却（"温度不升反降"——与 Schauberger 一致！）

## 二、与 Schauberger 蛋形外摆线的深度关联

### 2.1 蛋形 = 天然的外摆线生成器

PKS 项目发现：**蛋形截面是单位圆在射影变换下的像**。但深层数学揭示——蛋形也是**外摆线**的特例：

**外摆线 (Epitrochoid)**：一个圆在另一个圆外滚动时，圆上一点画出的轨迹。

参数方程：
$$x(\theta) = (R + r)\cos\theta - d\cos\left(\frac{R+r}{r}\theta\right)$$
$$y(\theta) = (R + r)\sin\theta - d\sin\left(\frac{R+r}{r}\theta\right)$$

当外摆线的滚动比 $R/r$ 是无理数时（如黄金比），轨迹永不闭合——产生**无限填充的螺旋环面**。

### 2.2 SEG 滚柱运动 = 三维外摆线

SEG 中的 66 个滚柱在 3 块平板之间运动——这恰好是**三维约束下的外摆线**：

| SEG 物理 | 外摆线数学对应 |
|---------|:------------:|
| Roller 在 Plate 上滚动 | 小圆在大圆外滚动 |
| Plate 磁化正弦波形 | 外摆线的节距调制 |
| 3 层 Plate 堆叠 | 外摆线的轴向位移（z 轴） |
| 12+22+32 roller 数 | 滚动比 $R/r$ 的分层取值 |
| 滚柱自由漂移 | 非周期外摆线的准晶体填充 |

> 🔑 **SEG 的滚柱不是简单的圆周运动——它们在平板上画出的是三维外摆线，与 Schauberger 蛋形截面的水分子螺旋运动在几何上是同一家族：都是"圆在圆上的滚动 + 轴向漂移"。**

### 2.3 "蛋形均轮"事实上就是外摆线

你项目中的 `17_外摆线均轮/` 文件夹正是这个关键洞察的产物。托勒密天文学中的**均轮-本轮**（deferent-epicycle）模型本质上就是外摆线：

```
托勒密行星模型               SEG 滚柱运动              Schauberger 涡旋
─────────────              ──────────               ──────────────
本轮 = 小圆 (r)          滚柱自转                   水分子自旋
均轮 = 大圆 (R)          滚柱绕轴公转                 涡旋绕中心公转
行星 = 外摆线轨迹         SEG 磁场耦合路径              蛋形截面轮廓
```

三者是同一个几何家族的三个物理实例。

## 三、水分子在蛋形截面中的涡旋轨迹

### 3.1 Schauberger 的核心直觉

Schauberger 反复强调：水分子在漩涡中不是直线运动——每个水分子都在做**自转 + 公转**的复合运动。这种复合运动的包络恰好是**蛋形截面**：

$$\text{水分子轨迹} = \text{外摆线在蛋形约束中的投影}$$

### 3.2 蛋形截面对涡旋的"几何滤波"

PKS 项目的数值模拟暗示：**蛋形截面是唯一能同时支持内卷和外摆线运动的截面几何**。圆形截面对外摆线是"中性"的（不抑制也不增强），而蛋形截面天然地将外摆线运动**导向螺线收敛**。

```
        圆形截面                   蛋形截面
       ─────────                 ─────────
         圆                         蛋
     外摆线不收敛               外摆线→内卷螺旋
     (能量弥散)                 (能量聚集)
```

SEG 的设计利用了同样的原理：滚柱在特定磁化图案的平板上运动——**磁化图案 = 蛋形截面上 Laplace 算子本征函数的节线**。当滚柱的数量（12, 22, 32）与这些节线数匹配时，系统达到共振——电子从环境中被"吸入"。

## 四、方块法则 × 千禧蛋数学

### 4.1 8 方块 = 音乐八度 = 蛋形八度

Searl 的 8 方块直接对应音乐八度（8 音程）。在 PKS 项目中，蛋形域的 Laplace 谱有一个恒定的八度间距：

$$\lambda_{2n} - \lambda_n = 2\pi\ln 2 = \text{常数}$$

这意味着：**圈数翻倍 = 特征值增加固定量 = 音乐中的一个八度**。

Searl 的"方块 4"是核心——恰好对应蛋形谱中的第 4 个本征值，这个本征值在 SEG 的设计中转化为：
- 3 块 Plate
- 12, 22, 32 个 rollers（12 = 方块 4 DNA 值 30 的倍数因子）

### 4.2 Nd-60 与精细结构常数

Searl 选择 Neodymium-60 作为核心材料（原子序数 60）。在 PKS 项目中：

$$\alpha^{-1} \approx 20\Phi^4 = 137.08$$

其中 $20 = 1680/84$。60 是 20 的 3 倍——方块 3 和方块 4 的乘积因子。

更神秘的是：Nd 的电子构型为 [Xe] $4f^4 6s^2$。4f 电子的轨道角动量量子数 $l=3$，总角动量——这恰好对应方块法则中的"空间-时间双态"结构。

### 4.3 层压材料 = 蛋形截面的材料实现

Searl 强调 SEG 必须使用"层压"（layer-pressed）材料——不同材料交替堆叠。这与蛋形截面的**分层涡旋结构**（外层上升 / 中层内卷 / 内层下降）完美对应：

```
SEG 材料层                 蛋形漩涡层            功能
─────────                ──────────             ──
Teflon/Nylon 绝缘层      外层 (上升)            防止电荷泄漏
Nd-Ni 磁化层              中层 (内卷)            产生旋转磁场
Aluminum 导电层           内层 (下降)            收集感生电流
```
