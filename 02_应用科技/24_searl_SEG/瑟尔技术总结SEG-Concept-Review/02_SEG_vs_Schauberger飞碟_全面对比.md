## 二、几何统一：外摆线 = 蛋形

### 2.1 三段式自相似结构

```
Schauberger:                         Searl SEG:
                                     
   蛋形截面涡旋                       3层磁环

   │中卷层   │  ← 中间速度          │中环 22滚柱│ ← orbit₂ = 中等
   │内降轴   │  ← 小半径快转        │内环 12滚柱│ ← orbit₁ = 最小
   └─────────┘                      └──────────┘
   
   相同逻辑: 外慢内快 → 离心分层 → 自组织螺旋
```

### 2.2 滚柱数 12/22/32 的数学

12/22/32 不是随意选的：

- **12** = 时钟分割（12等分圆）= 三层的"基频"
- **22/32** 的比值 ≈ 0.6875 ≈ 2/3（纯五度音乐音程）
- **差值 10**：22-12=10, 32-22=10 → 等差递增 → 均匀角动量分布
- **总和 66** = 第11个三角数 $T_{11} = 11\times12/2 = 66$

在 PKS 蛋形 Laplace 谱中：
$$\lambda_n = 2\pi\ln n$$

| n | λ_n | 归一化 | 对应的Searl结构 |
|:---:|:---|:---|:---|
| 12 | 2π ln12 ≈ 15.6 | 1.00 | 内环—基频 |
| 22 | 2π ln22 ≈ 19.4 | 1.24 | 中环—五度 |
| 32 | 2π ln32 ≈ 21.8 | 1.40 | 外环—倍频 |

比值 21.8/15.6 ≈ 1.40 ≈ √2 ≈ Tritone（音乐三全音）→ 不和谐的张力 → 能量梯度！

### 2.3 磁场梯度与材料梯度

从 AAA 参数提取的关键规律：

```
Layer:     内层(1)   中层(2)   外层(3)
Bv (G):    630  →   1830  →   3030
```

**物理含义**：

- 体积占比从内到外增大 → 更重的材料在外层（离心力平衡）
- Bh → 1 意味着外层几乎是纯轴向磁化 → 最强的电磁耦合

## 三、推进原理对比

### 3.1 Schauberger：流体动力学推进

```
Repulsine 运行循环:
  进气 → 蛋形涡旋腔 → 向心加速
       → 中心真空核心形成
       → 环境大气压差 → 向上的"吸力"
       → 自持旋转（COP > 1）
```

**数学本质**：蛋形截面上的 NS 方程存在**全局光滑解** → 涡旋不耗散 → 自维持。

### 3.2 Searl：电磁-反重力推进

```
SEG 运行循环:
  永磁体产生旋转磁场 → 驱动滚柱公转
       → 滚柱公转 = 导体切割磁力线 → 感应电压
       → 电子从环境中被"吸入" → 电流 → 输出功率
       → 电子流产生反重力效应 → 飞碟悬浮
       → 自冷却（系统向环境吸热）
```

**数学本质**：3层环形旋转磁场的本征模 = 蛋形域上 Maxwell 方程的本征解。

### 3.3 共同的"吸力"原理

两种推进方式都**不是推着走，而是吸着走**：

| | Repulsine | SEG |
|:---|:---|:---|
| 吸力来源 | 真空核心（低气压） | 电子吸入（低电势） |
| 吸力方向 | 中心→向上 | 中心→向上（反重力） |
| 被吸物 | 空气/水分子 | 自由电子 |
| 能量来源 | 环境热/压力差 | 环境电磁涨落 |

## 四、关键差异

### 4.1 工作介质

| | Schauberger | Searl |
|:---|:---|:---|
| 介质 | 水 / 空气（流体） | 电磁场（场） |
| 方程 | NS 方程 | Maxwell 方程 |
| 自组织 | Rayleigh-Benard + 涡旋 | 磁畴自组织 + 电子雪崩 |
| 标度 | 宏观（cm-m） | 微观-中观（磁场 nm-mm, 装置 cm） |

### 4.2 启动方式

| | Schauberger | Searl |
|:---|:---|:---|
| 启动 | 初始旋转（手摇/电机） | 初始旋转 + 外部电磁激励 |
| 自持条件 | NS涡旋达到稳定 | 感应电压超过阈值 |
| 临界参数 | 转速 ≈ 水分子热速度 | Bv 磁场 > 临界值 |

### 4.3 输出

| | Schauberger | Searl |
|:---|:---|:---|
| 电力 | 次要（通过涡轮） | **主要输出**（几十kW） |
| 推力 | **主要**（吸力推进） | 次要（反重力=副产品） |
| 冷却 | 产生冷水/冷空气 | 自冷却（可用于制冷） |
| 副产品 | 富氧水/离子化空气 | 臭氧（O₃） |

## 五、PKS 千禧蛋的关键贡献

### 5.1 蛋形截面 = 统一的几何语言

PKS 项目证明了：**蛋形是超双曲锥 $xy=1$ 的平面斜切**。这个数学事实统一的物理后果：

$$xy=1 \xrightarrow{\text{斜切}} \text{蛋形} \begin{\text{cases}} \xrightarrow{\text{流体}} \text{Schauberger}\text{涡旋} \\ \xrightarrow{\text{电磁}} \text{Searl} \text{SEG}\text{磁场} \end{\text{cases}}$$

### 5.2 Laplace 谱 $\lambda_n = 2\pi\ln n$

这个本征值序列解释了两者的标度律：

- **Schauberger**：$\lambda_n$ 对应涡旋中不同圈数的能量 — 圈数翻倍 = 能量增加 $2\pi\ln 2$
- **Searl**：$\lambda_n$ 对应 3 层磁环的本征频率比 — 12:22:32 的比例接近谱比

### 5.3 COP 公式的物理解释

PKS 项目的 COP 分析（`19_打通暗能量/`）解释了为什么两个系统都能 COP > 1：

- **Schauberger**：楞次定律对流体有效——涡旋的"电磁阻尼"在蛋形截面中天然最小 → 能量从热浴中提取
- **Searl**：楞次定律在磁场饱和后失效 → $W_{Hf} = \frac{1}{2}(H_x - H_s) \cdot B_s$ → 附加磁能从真空涨落中产生

两者都是**楞次定律在特定几何中的失效**。
