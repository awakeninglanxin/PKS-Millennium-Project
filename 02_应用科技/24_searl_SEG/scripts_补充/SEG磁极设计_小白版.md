# SEG 磁极数怎么选？—— 小白也能看懂的最优设计指南

> 3分钟看懂：为什么有些磁极组合转起来"丝滑"，有些"咔咔响"？

---

## 一、SEG 是什么？（一句话版）

SEG = 三层同心磁环，每层有固定磁极（定子）和旋转磁极（滚筒）。滚筒在定子之间旋转，磁力驱动它转。

**好设计 = 转得顺 + 寿命长**。坏设计 = 振动大 + 局部过热。

---

## 二、只需要懂两个数学概念

### ① gcd（最大公约数）

> gcd = "两个数能同时被几整除"。

- gcd(12, 8) = 4（12÷4=3, 8÷4=2）
- gcd(13, 8) = 1（13和8没有共同因子）← **这叫"互质"**

**物理直觉**：gcd=1 = 磁极从不周期性对齐 = 安静顺滑。gcd>1 = 磁极每转几圈就对齐一次 = "嘎嘣嘎嘣"振动。

### ② lcm（最小公倍数）

> lcm = "两个数一起数，多久才能再次同时数到"。

- lcm(12, 8) = 24
- lcm(13, 8) = 104 ← **大很多！**

**物理直觉**：lcm大 = 同一对磁极很久才重逢 = 磨损均匀 = 寿命长。lcm小 = 磁极频繁重逢 = 同一位置反复受最大力 = 先坏。

## 三、评分公式（简化版）

每个设计打0-100分，100=完美：

| 项目 | 满分 | 简单解释 |
|:---|:---:|:---|
| 每环磁极互质 | 66分 | 定子和滚筒磁极数不能有公约数 |
| 环之间不耦合 | 9分 | 三层定子之间不能共享公约数 |
| 寿命长 | ~10分 | lcm越大越耐久 |
| 三环滚筒各不同 | 12分 | 三个环的滚筒磁极数不一样 |

**100分设计 = 所有磁极数都是素数 + 环间互质 + 滚筒各不相同**

## 五、为什么"互质"这么重要？

想象你在一个有N个凸起的圆环里滚一个有P个齿的齿轮：

- N=12, P=12 → gcd=12 → **每转一圈所有齿同时撞凸起** → 剧烈振动 ❌
- N=12, P=13 → gcd=1 → **每次撞的凸起都不一样** → 振动分散 → 丝滑 ✅

这就是为什么SEG的定子和滚筒磁极数必须互质——本质上是**磁齿轮的消音设计**。

| 坏法 | 例子 | 问题 |

## 七、最简设计原则

如果你要自己设计SEG磁极数，记住四条铁律：

1. **全部用素数**（7, 11, 13, 17, 19, 23...）
2. **三环比例接近1.618**（黄金比，自然界最均匀的螺旋间距）
3. **三环互不相同**（不要用一样的数）
4. **越大越好**（磁极越多，lcm越大，寿命越长——但制造成本也越高）
