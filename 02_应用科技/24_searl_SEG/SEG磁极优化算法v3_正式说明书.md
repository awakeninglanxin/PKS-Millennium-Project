# SEG 三环磁极优化算法 v3.1 — 正式说明书

> **版本**：v3.1 | **日期**：2026-07-18
> **适用范围**：SEG三环定子磁极数 + 滚筒磁极数 + 滚筒数量 + 性价比的全局最优搜索
> **核心指标**：gcd(磁静音) + lcm(长寿命) + φ比例(自然间距) + 滚筒差异化(解耦) + CE(性价比)

---

## 一、问题定义

给定三层同心SEG结构（内定子/内滚筒/中定子/中滚筒/外定子/外滚筒），求：

| 变量 | 符号 | 约束 |
|:---|:---|:---|
| 内定子总磁极数 | $N_i$ | 偶数（N-S交替） |
| 中定子总磁极数 | $N_m$ | 偶数 |
| 外定子总磁极数 | $N_o$ | 偶数 |
| 内滚筒每滚筒磁极数 | $P_i$ | 偶数，≥20 |
| 中滚筒每滚筒磁极数 | $P_m$ | 偶数，≥20 |
| 外滚筒每滚筒磁极数 | $P_o$ | 偶数，≥20 |

**优化目标**：最大化综合评分 $S(N_i,N_m,N_o,P_i,P_m,P_o)$

---

## 二、评分函数 v3.0

### 2.1 极对数转换

设极对数（pole pairs）$p_i = N_i/2, p_m = N_m/2, p_o = N_o/2$，滚筒极对数 $r_i = P_i/2, r_m = P_m/2, r_o = P_o/2$

### 2.2 四项评分

$$S = S_{\text{gcd}} + S_{\text{cross}} + S_{\text{lcm}} + S_{\text{diff}}$$

#### ① 单环互质分 $S_{\text{gcd}}$（满分66）

$$S_{\text{gcd}} = 22 \times \sum_{k \in \{i,m,o\}} \mathbb{1}[\text{gcd}(p_k, r_k) = 1]$$

物理含义：每环的定子极对数和滚筒极对数互质 → 该环磁静音。

#### ② 交叉gcd分 $S_{\text{cross}}$（满分9）

$$S_{\text{cross}} = 9 \times \left( \frac{1}{\text{gcd}(p_i, p_m) + 1} + \frac{1}{\text{gcd}(p_m, p_o) + 1} \right)$$

物理含义：相邻定子环极对数交叉gcd越低 → 环间磁场耦合越小 → 不会形成共振回路。

| cg值 | 奖励 | 含义 |
|:---:|:---:|:---|
| 1 | 9.0 | 完全去耦 |
| 2 | 6.0 | 仅共享因子2 |
| 102 | 0.17 | 强耦合 |
| 882 | 0.02 | 几乎全共振 |

#### ③ LCM寿命分 $S_{\text{lcm}}$（满分≈11）

$$S_{\text{lcm}} = 0.9 \times \log_{10}\left( \text{lcm}(p_i, p_m, p_o) + 1 \right)$$

物理含义：三个定子极对数的lcm越大 → 磁极对齐周期越长 → 退磁越均匀 → 寿命越长。

- 互异素数：$\text{lcm} = p_i \times p_m \times p_o$ → 理论上限
- Fibonacci方案：lcm因共享基频而远小于乘积

#### ④ 滚筒差异化/统一性分 $S_{\text{diff}}$（满分14）

$$S_{\text{diff}} = 2 \times \left[ (r_i \neq r_m) + (r_m \neq r_o) + (r_i \neq r_o) \right] + \begin{cases} 8 & \text{若 } r_i = r_m = r_o \text{（统一极数：1套充磁电路）} \\ 0 & \text{否则} \end{cases}$$

物理含义：三环滚筒极数全异→磁耦合解耦(+6分)；三环统一→省充磁电路(+8分)。统一比差异化净赚2分——反映制造成本优势。

### 2.3 总分

$$S_{\text{total}} = \min(100, \lfloor S_{\text{gcd}} + S_{\text{cross}} + S_{\text{lcm}} + S_{\text{diff}} \rceil)$$

**满分配置**：66(gcd)+9(cross)+10.2(LCM)+14(统一)=**99分** (🏭统一34极+混合32/21/13)

### 3.2 滚筒数量优化（v3.1新增）

**定理5：素数定子下的滚筒数量无关性**

若定子极对数 $p_i,p_m,p_o$ 均为素数，则对任意滚筒数 $n < p_k$，恒有 $\text{gcd}(n, p_k)=1$。因此相位多样性 $= n$（满分），滚筒数量对磁静音无影响。

**优化准则**：在素数定子下，滚筒数量退化为纯工程决策，建议：
- 外环 30~40（Fibonacci 34 推荐）
- 中环 20~25（Fibonacci 21 推荐）
- 内环 12~17（Fibonacci 13 推荐）
- 更大数量有边际收益但制造成本上升
| v2 | 互异素数×2 | 2 | 9.8 | 94 | 共享因子2 |
| v3 | **互异奇数素数** | **1** | **10.2** | **97** | LCM非最大 |
| v3+ | 大互异奇数素数 | 1 | **10.8** | **98** | 比例偏离φ |

### 3.2 v3搜索伪代码

```
输入: 素数池 P = {p | 500 < p < 15000, p is prime}
      滚筒极对数候选 R = {8,10,11,13,...,50}

for p_i in P:
  for p_m in P, p_m > p_i:
    if p_m/p_i ∉ [1.58, 1.66]: continue  # φ约束
    for p_o in P, p_o > p_m:
      if p_o/p_m ∉ [1.58, 1.66]: continue
      if gcd(p_i,p_m)≠1 or gcd(p_m,p_o)≠1: continue
      
      N_i,N_m,N_o = 2p_i, 2p_m, 2p_o
      
      # 最优滚筒搜索
      best_r = argmax_{r_i,r_m,r_o∈R} diff(r_i,r_m,r_o)
         s.t. gcd(p_i,r_i)=gcd(p_m,r_m)=gcd(p_o,r_o)=1
      
      S = compute_score(p_i,p_m,p_o, best_r)
      if S >= threshold: output
```

### 3.3 时间复杂度

- 素数池大小 $|P| \approx 1600$（500~15000）
- 三重循环过滤后约 $10^6$ 有效组合
- 每个组合内滚筒搜索 $|R|^3 \approx 8000$ 但大部分被gcd过滤
- 总耗时约 30~120 秒（本机Python）

## 五、关键数学定理

### 定理1：素数方案的交叉gcd上界

若 $p_i,p_m,p_o$ 为互异素数，则 $\text{gcd}(p_i,p_m)=\text{gcd}(p_m,p_o)=1$，$S_{\text{cross}}=9$（满分）。

### 定理2：素数方案的LCM上界

若 $p_i,p_m,p_o$ 为互异素数，则 $\text{lcm}(p_i,p_m,p_o) = p_i \times p_m \times p_o$。这是给定大小约束下的理论上限（因任何合数方案的lcm ≤ 乘积）。

### 定理3：偶数磁极的交叉gcd下界

定子总磁极数 $N=2p$，$\text{gcd}(N_i,N_m) = 2 \times \text{gcd}(p_i,p_m)$。若 $p_i,p_m$ 为互异奇数 → $\text{gcd}(N_i,N_m)=2$。偶数磁极天然gcd≥2，必须用极对数（奇数）算交叉gcd才能降到1。

### 定理4：Fibonacci比例的交叉gcd硬伤

对Fibonacci三连比 $a:b:c$ 和基频 $B$，$p_i=Ba, p_m=Bb, p_o=Bc$。因相邻Fib互质，$\text{gcd}(p_i,p_m)=\text{gcd}(p_m,p_o)=B$。交叉gcd恒等于基频，无法降低。这是Fibonacci方案评分的根本瓶颈。
