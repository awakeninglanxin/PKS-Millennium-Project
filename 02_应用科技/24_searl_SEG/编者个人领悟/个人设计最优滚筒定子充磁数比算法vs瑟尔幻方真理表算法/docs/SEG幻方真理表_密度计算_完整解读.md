## 六、🔴 蓝馨的核心修正：磁极数只需满足有理比，不必死守幻方线

### 6.1 外摆线闭合条件的启示

水星绕地心的均轮+本轮+偏心曲柄模型中，轨迹闭合的**充要条件**是角速度比为有理数：

$$\frac{\omega_{\text{均轮}}}{\omega_{\text{本轮}}} = \frac{p}{q} \in \mathbb{Q}$$

若比值无理数 → 轨迹永远不闭合（准周期混沌）。

对 SEG 滚筒在定子环内的运动：

$$\frac{N_{\text{定子}}}{N_{\text{滚筒}}} = \frac{m}{n} \quad (\text{最简分数})$$

- $m, n$ 互质 → 轨迹在 $n$ 个完整周期后首次闭合
- $m-n$ = 外摆线的"尖点数"
- 闭合条件 = **定子与滚筒的磁极数比为有理数**

### 6.2 🔴 蓝馨修正：真正的能量来源是永不闭合的轨迹

**我的逻辑错误**：试图让磁极数比为简单有理数（如 2:1），使轨迹快速闭合达到"稳定"。但这对 SEG 是**致命的**——轨迹闭合 = 动态平衡 = 零能量提取。

**正确逻辑**：SEG 的能量来自轨迹的**永久不对称性**。磁极数比有三种层级：

$$\frac{N_{\text{定子}}}{N_{\text{滚筒}}} = \frac{p}{q}$$

| 层级 | 比值类型 | gcd(p,q) | 周期长度 | 能量输出 |
|:---|:---|:---:|:---|:---:|
| **最优** | **无理数** | — | ∞（永不闭合） | ★★★★★ 理论极限 |
| **次优** | 有理数 | **1**（互质） | 最长有理周期 | ★★★★☆ |
| **劣** | 有理数 | **>1** | 短周期 | ★★☆☆☆ |
| **最劣** | 有理数 | **=p**（整除） | 最短短周期 | ★☆☆☆☆ |

**能量输出与 gcd(p,q) 成反比**——gcd 越小（最小为 1，最理想趋近无理数的 gcd→0），周期越长，能量越大。

**物理直觉**：SEG 是"永动追逐器"——磁极之间的关系像 $\pi$ 的小数位永远不循环。无理数比 = 绝对的不对称 = 无限的能量提取潜力。

### 6.4 🔴 拉糖机的验证：gcd(p,q)=1 是遍历混合的充要条件

同样的规律早已在**太妃拉糖机**（taffy puller）中严格成立：

**拉糖机核心算法**：$v' = M v$，其中 $M$ 是 2×2 Anosov 矩阵，特征值 $\lambda = (T \pm \sqrt{T^2-4})/2$。

面团充分搅拌的条件：
$$\gcd(p, q) = 1$$

其中 $p/q$ 是拉糖机两臂的旋转周期比。如果 $\gcd \neq 1$，面团只沿同一组折痕反复折叠——产生"条纹"而非均匀混合。

**与 SEG 的精确对应**：

| 拉糖机 | SEG |
|:---|:---|
| 旋转周期比 p/q | 磁极数比 N_stator/N_roller |
| gcd(p,q)=1 → 遍历混合 | gcd(p,q)=1 → 轨迹长周期不闭合 |
| gcd(p,q)>1 → 条纹重复 | gcd(p,q)>1 → 轨迹过早闭合 |
| Fibonacci 比 → 最稠密混合 | Fibonacci 比 → 最大能量提取 |
| 判别式 Δ=T²-4 | 判别式 Δ=a²-4b (clothoid) |

**统一规律**：

$$2(x+y) = 6n + 105, \quad \frac{y}{x} \approx \phi, \quad \gcd(y, x) = 1$$

这是蓝馨在拉糖机/李萨如实验中总结的核心公式——$y/x \approx \phi$ 且互质，同时满足"趋近无理数"和"最长不重复周期"两个条件。**SEG 的磁极设计应遵循同一规律**。

### 6.5 🔴 齿轮类比：gcd 控制噪音，lcm 控制寿命

SEG 滚筒和定子的关系就是**一对磁齿轮**：

| 齿轮机械 | SEG 磁齿轮 |
|:---|:---|
| 大齿轮齿数 | 定子磁极数 $N_{\text{stator}}$ |
| 小齿轮齿数 | 滚筒磁极数 $N_{\text{roller}}$ |
| 齿与齿的物理接触 | 磁极间的磁力作用 |
| 齿轮噪音 | **磁噪音**（磁极对齐时的共振啸叫） |
| 齿轮磨损 | 磁极局部过热/退磁 |

**蓝馨的铁律**：

$$\gcd(N_{\text{stator}}, N_{\text{roller}}) \downarrow \quad \text{→ 磁噪音} \downarrow$$
$$\operatorname{lcm}(N_{\text{stator}}, N_{\text{roller}}) \uparrow \quad \text{→ 寿命} \uparrow$$

**直觉**：
- $\gcd=1$（互质）：每次磁极对齐都是**全新的组合**——没有固定的"共振拍"，磁噪音分散到极宽的频谱中 → 静音运行
- $\gcd>1$（可约分）：每隔几圈，所有磁极**同时对齐**——产生周期性的巨大磁脉冲 → 噪音 + 局部过热
- $\operatorname{lcm}$ 大：同一对磁极再次相遇需要极长时间 → 局部退磁均匀分散

**SEG 磁静音设计公式**：

$$\gcd(N_s, N_r) = 1 \quad \text{或尽可能小}$$
$$\operatorname{lcm}(N_s, N_r) \quad \text{尽可能大}$$

这两个条件自动使得 $N_s/N_r$ 趋近无理数行为——不需要刻意追求无理数，gcd↓+lcm↑ 自然导向长周期的准混沌。

**幻方真值表的新用途**：在表中找**gcd 小且 lcm 大的整数对**——这些是磁静音、长寿命的备选设计参数。
