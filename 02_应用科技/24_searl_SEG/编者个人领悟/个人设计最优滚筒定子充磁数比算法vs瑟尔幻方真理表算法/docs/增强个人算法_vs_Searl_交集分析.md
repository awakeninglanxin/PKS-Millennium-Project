# 增强个人算法(三层) — Jellium通项公式与增强设计空间

**日期**: 2026-06-07 | **Jellium**: 完整29个幻数

## 一、通项公式

### 1.1 核心种子: 5040 = 7!

$$S_0 = 5040 = 2^4 \cdot 3^2 \cdot 5 \cdot 7 = 7!$$

5040 是容纳最多小素数Jellium幻数的**最小正整数**。所有高Jellium整除的SEG候选都是5040的倍数(或其半数2520乘上新增素数)。

### 1.2 素数渐进公式

$$N(\{p_1, p_2, ..., p_n\}) = 5040 \cdot \prod_{i=1}^{n} p_i \quad \text{或} \quad 2520 \cdot \prod_{i=1}^{n} p_i$$

其中$\{p_i\}$是新增的素数(不在{2,3,5,7}中)，每种素数解锁特定Jellium幻数：

| 新增素数 | 解锁的Jellium幻数 |
|:---|:---|
| — | 2,8,18,20,28,40,90,112,126 (9个) |
| 23 | +92,138,184,322 → 13个 |
| 17 | +34,68 → 11个 |
| 11,13 | +198,234 → (需5040包含11·13) |
| 29 | +58 → 10个 |
| 37 | +148,296 → 11个 |
| 41 | +82 → 10个 |
| 53 | +106,212 → 11个 |
| 83 | +166 → 10个 |
| 67 | +268 → 10个 |
| 139 | +278 → 10个 |

### 1.3 完整通项

SEG Jellium优化候选的通用形式：

$$N = m \cdot 5040 \cdot \prod_i p_i, \quad \text{或} \quad N = m \cdot 2520 \cdot \prod_i p_i$$

约束:
1. $N$ 末位 = 0 （自动满足，因为5040含因子10）
2. Σ各位数字 ≡ 0 mod 3
3. $m$ 为整数，尽量取1使N最小化
4. 每个素数 $p_i$ 解锁特定Jellium幻数族

## 三、增强设计空间：无边界的完整通项公式

### 3.0 两个种子版本

| 版本 | 约束 | 种子 | 素因子 | Jellium数 | 适用 |
|:---:|:---|:---:|:---|:---:|:---|
| **A (含量子)** | L1+L2, N/30整数 | **5040** = 2⁴·3²·5·7 | {2,3²,5,7} | 9个 | SEG全约束 |
| **B (仅末位)** | 仅L1, N/10整数 | **1680** = 2⁴·3·5·7 | {2,3,5,7} | 6个 | SEG最小约束 |

两版本的关系：5040 = 1680 × 3。1680少一个因子3²，丢失了{18,90,126}三个Jellium幻数，换来更小的种子尺寸和更宽的适用范围（不强制被3整除）。

### 3.1 完整通项公式：N为n个Jellium整除的最小值

$$N(n) = 5040 \times \prod_{p \in P(n)} p$$

其中 $P(n)$ 是达到 $n$ 个Jellium除数所需的最小成本素数集。
