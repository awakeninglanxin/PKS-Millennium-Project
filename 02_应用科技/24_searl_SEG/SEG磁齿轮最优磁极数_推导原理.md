# SEG 磁齿轮最优磁极数推导原理

> **关联脚本**：`seg_grid_search_v2.py` | `seg_torque_smoothness_v2.png`
> **物理基础**：磁齿轮理论 + 数论（gcd/lcm）+ 幻方共振

---

## 一、核心物理：磁齿轮的"齿"

SEG 的本质是一台**三级串联永磁齿轮**。每个滚筒上的34个径向磁极 = 齿轮的"齿"。

```
定子（最外层，固定）        ← N_stator 个磁极
    ↕ 磁力耦合（第一级齿轮）
外环滚筒 × N_outer 个       ← 每个有 P_roller 个磁极
    ↕ 磁力耦合（第二级齿轮）
中环滚筒 × N_mid 个
    ↕ 磁力耦合（第三级齿轮）
内环滚筒 × N_inner 个
```

**关键认知**：这不是电磁感应（需要变化的磁通），而是**磁齿轮的直接力传递**——类似两个齿轮的齿啮合，但介质是磁场而非机械接触。

## 二、三条铁律（gcd/lcm/相位）

### 铁律1：gcd↓ → 磁静音

$$\boxed{\text{gcd}(N_{\text{定子}}, N_{\text{滚筒磁极}}) \downarrow \quad \Rightarrow \quad \text{磁噪音} \downarrow}$$

**物理直觉**：gcd=1意味着定子的磁极和滚筒的磁极**永远不会周期性对齐**。每一次滚筒公转，磁极对的相遇模式都在变化——没有重复的"脉冲"。

- gcd=1（互质）：每个磁极看到独特的定子图案 → 力波动频谱是宽带噪声 → 振动低
- gcd>1（可约）：多个磁极同步经历相同力 → 力波动同相叠加 → 振动高
- gcd=N（整除）：每圈精确同步 → 最响的磁啸叫

### 铁律2：lcm↑ → 长寿命

$$\boxed{\text{lcm}(N_{\text{定子}}, N_{\text{滚筒磁极}}) \uparrow \quad \Rightarrow \quad \text{均匀退磁} \uparrow}$$

**物理直觉**：lcm很大意味着同一对磁极需要很多圈才能再次对齐。退磁均匀分布在整个环面上，没有局部"热点"。

### 铁律3：相位多样性

$$\boxed{\text{phase\_diversity} = \frac{N_{\text{rollers}}}{\text{gcd}(N_{\text{rollers}}, N_{\text{stator}})}}$$

衡量有多少个滚筒处于**不同的相对相位**。值越大，每个滚筒经历不同的力历程 → 总振动越低。

**例子**：
- 定子34极对，滚筒数12：gcd(34,12)=2 → phase_diversity=6（只有6个独立相位）
- 定子34极对，滚筒数13：gcd(34,13)=1 → phase_diversity=13（全部独立！）

## 三、评分公式

每个配置的评分由五部分加权组成：

$$\text{score} = \underbrace{40 \cdot f_1(\text{gcd}_{\text{outer}})}_{\text{外环静音}} + \underbrace{25 \cdot f_1(\text{gcd}_{\text{mid}})}_{\text{中环静音}} + \underbrace{15 \cdot f_1(\text{gcd}_{\text{inner}})}_{\text{内环静音}} + \underbrace{10 \cdot f_2(\text{lcm})}_{\text{长寿命}} + \underbrace{10 \cdot f_3(\text{phase})}_{\text{相位多样性}}$$

其中：
- $f_1(g) = 1.0$ (当gcd=1)，否则 $f_1(g) = 0.5/g$
- $f_2(\text{lcm}) = \log_{10}(\text{lcm}+1)$
- $f_3(\text{phase}) =$ 相位多样性

**权重设计逻辑**：外环直接与定子交互 → 最高权重(40)。中环传递外环扭矩(25)。内环最终输出(15)。

### 网格搜索过程

```
对于每个候选组合 (N_stator, N_outer, N_mid, N_inner):
  1. 计算外环层: gcd(定子极对数, 滚筒极对数), lcm, 相位多样性
  2. 计算中环层: gcd(外环总极对数, 滚筒极对数), ...
     （中环的"定子"是外环滚筒的总磁极场）
  3. 计算内环层: gcd(中环总极对数, 滚筒极对数), ...
  4. 按评分公式排名
```

### 为什么 3622/2834/4410 排前三

以 **定子3622, 40/23/16** 为例：

| 层级 | 有效极对数 | gcd | 解释 |
|:---|:---|:---:|:---|
| 定子→外环 | gcd(1811, 17) | **1** ✅ | 互质 → 磁静音 |
| 外环→中环 | gcd(680, 17) | 17 | 共享因子，但此层权重较低 |
| 中环→内环 | gcd(391, 17) | 17 | 同上 |

**排名逻辑**：
- 外环 gcd=1 是必要条件（权重40），几乎所有高分配置都满足
- 外环滚筒数=40 给出 phase_diversity=40（全独立相位）→ 极低的力波动叠加
- 定子3622来自2835//2×2附近，恰好与17（滚筒极对数）互质

**为何优于原版Searl(4410, 32/22/12)**：
- 原版外环只有32个滚筒 → phase_diversity=32
- 最优版有40个 → phase_diversity=40 → 力分布更均匀
- 原版中环22、内环12 → 相位多样性更低

## 七、结论

网格搜索不是随机猜测——是对**磁齿轮数论的穷举验证**：

1. **gcd=1是磁静音的数学必要条件**（来自齿轮噪音理论）
2. **lcm最大化延长均匀退磁寿命**（来自磨损理论）
3. **相位多样性降低力波动叠加**（来自多缸发动机类比）
4. **三环全互质需要不同环不同磁极数**（新发现，原版SEG未满足）

原版Searl的4410/34极/32-22-12配置在7644个候选中的排前27.6%，说明设计者有深厚的数论直觉——但仍有显著优化空间。
