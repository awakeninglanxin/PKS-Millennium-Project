## 一、三阶幻立方的数学结构

### 1.1 基本参数

```
3×3×3 = 27 个格子
数字: 1→27 (连续自然数)
中心: 14 (14 = 9+5 = 洛书粒子 + 中心能量)
幻和: 42 (每行/每列/每条体对角线)
层次: 3 个同心球壳
类型: 4 种 essentially different
```

### 1.2 "13 线穿心" — 这个结构极其关键

以中心 14 为原点，通过 14 的线有 **13 条**，每条线两端到 14 的距离分别是：

```
距离 1: 13 ↔ 15  (|13-14| = |15-14| = 1)
距离 2: 12 ↔ 16
距离 3: 11 ↔ 17
距离 4: 10 ↔ 18
距离 5:  9 ↔ 19
距离 6:  8 ↔ 20
距离 7:  7 ↔ 21
距离 8:  6 ↔ 22
距离 9:  5 ↔ 23
距离 10: 4 ↔ 24
距离 11: 3 ↔ 25
距离 12: 2 ↔ 26
距离 13: 1 ↔ 27
```

**13 个不同的距离，恰好是 1,2,3,...,13 —— 一条不重，全部覆盖！**

### 1.3 三层同心球壳 (电子云模型)

高治源将三阶幻立方类比为电子云的 s/p/d 轨道：

| 壳层 | 包含的数字对 | 对数 | 类比 |
|:---|:---|:--:|:---|
| **内壳** | ±13, ±7, ±11 | 3对(6个数) | s轨道(球形对称) |
| **中壳** | ±1,±2,±3,±12,±11,±10 | 6对(12个数) | p轨道(哑铃形, 3取向) |
| **外壳** | ±4,±5,±6,±8 | 4对(8个数=8顶点) | d轨道(4取向) |

**这与蛋白质四面体的三级分类精确对应**:

```
内壳 (3对) → 疏水核心四面体 (紧密, 体积小)
中壳 (6对) → 界面结合四面体 (中等体积)
外壳 (4对) → 表面松散四面体 (体积大)
```

### 1.4 四款幻立方 (四套)

三阶幻立方不是只有一种——共有 **4 款 essentially different** 的类型:

```
第1套: 对角线和=42 → 标准型
第2套: 对角线和=69 → 非标准
第3套: 对角线和=15 → 非标准
第4套: 对角线和=?  → 非标准
```

对应我们方案C的模板库: 4款 × 从6个面分层 × 旋转对称 = **24种变式 ≠ 四种本质**。

## 二、mi(q) 浸染算法 — 吳硕辛的贡献

### 2.1 算法核心: "浸染" (Diffusion)

Excel 中的 `mi(q)演算程序` 展示了一种构造高阶幻方的全新方法:

```
输入: 基数=3, 浸染值=20
过程: 
  1. 从一个"种子矩阵"(3×3)开始
  2. 用 mi(q) 变换反复"浸染"——每次把当前矩阵的值扩散到更大矩阵
  3. "码置"(code placement)选择不同的起始位置 → 产生不同的幻立方变体
  4. 3次变换后 → 得到完整的 3×3×3 幻立方
```

### 2.2 mi(q) 的三个关键参数

| 参数 | 含义 | 值 |
|:---|:---|:--:|
| 基数 (base) | 幻立方的阶数 | 3 |
| 浸染值 (diffusion) | 扩散步长 | 20 |
| 码置 (code place) | 起始排列选择 | 0-18 (19种) |

**不同的码置 = 不同的幻立方类型**。这与我们 5 阶幻方中"36 essentially different × 4 族 × 25 循环位移"的分层结构异曲同工。

### 2.3 九阶幻方 → 三阶幻立方的迁移

Excel 的 `九阶幻方` sheet 展示了 mi(q) 在 9×9 平面幻方上的应用——9 阶 = 三阶幻立方每个面的展开。这说明 **mi(q) 算法可以从平面幻方推广到立体幻方，只需改变"浸染维数"**。

### 3.1 三阶 → 九阶 → 27 阶的递进

```

三者共同基因: 中心数 + 同心壳层 + 对称距离
```

### 3.2 13 个不同距离 → 四面体体积分档

三阶幻立方的 13 条穿心线给出了**13 个不同的距离等级**。如果映射到四面体:

```
距离 1-4:   核心四面体 (体积 < 10Å³)
距离 5-9:   中层四面体 (10-50Å³)
距离 10-13: 外层四面体 (> 50Å³)
```

这是**天然的体积分档方案**，不需要人工设定阈值。

### 3.3 mi(q) 浸染 = 模板库生成算法

吳硕辛的 mi(q) 浸染算法提供了一个**系统的、可编程的幻立方生成方法**:

```python
def mi_q_diffusion(seed_matrix, diffusion_value, code_place, n_iterations=3):
    """
    mi(q) 浸染算法: 从种子→完整幻立方
    
    seed_matrix: 初始 3×3 矩阵 (洛书变体)
    diffusion_value: 浸染值 (如20)
    code_place: 码置 (0-18, 选择不同的起始排列)
    n_iterations: 浸染次数 (3)
    """
    M = seed_matrix.copy()
    for _ in range(n_iterations):
        M = q_transform(M, diffusion_value, code_place)
        # 每次浸染 → 矩阵从 3×3 → 9×9 → 27×27
        # 在 3D 中: 3×3×3 → 9×9×9 → 27×27×27
    return M
```

如果这个算法被翻译为 Python，我们可以**程序化生成任何阶数的幻立方模板库**，从根本上解决方案C的模板生成问题。
