## 三、关键技术突破

### 3.1 C60坐标生成

**问题**：C60富勒烯有60个顶点，如何正确分组？

**解决方案**：
```javascript
// A组：8个五边形极点（顶部）
// B组：12个五边形顶点
// C组：24个六边形内部顶点
// D组：16个六边形极点
```

**验证**：`8 + 12 + 24 + 16 = 60` ✅

---

### 3.2 Scale缩放逻辑

**问题**：球面投影后几何体位置不对

**原则**：边界圆固定不变，只缩放几何体顶点

**公式**：
```javascript
// 球极平面投影 + 缩放
projStereoScaled(r, s) = [x*s/(1+z), y*s/(1+z)]
// 其中 s = 1/(1+z)，赤道z=0时 s=1
```

---

### 3.3 四元数旋转

**代码**：
```javascript
Quaternion.rotatePoint = function(q, point) {
  const p = [0, point[0], point[1], point[2]];
  const qInv = Quaternion.conjugate(q);
  return Quaternion.multiply(Quaternion.multiply(q, p), qInv);
};
```

## 六、使用建议

### 6.1 快速开始
1. 打开 `EEQT_ChaosGame_SelectedGeometries.html`
2. 选择几何体（如 Octahedron）
3. 调节 epsilon / points / rot-X/Y/Z
4. 点击 Generate 生成

### 6.2 最佳参数
- Octahedron：`ε=100`, points=500K
- Icosahedron：`ε=900`, points=500K
- C60 Full：`ε=800`, points=1M

### 6.3 探索方向
- 多几何体叠加 → 干涉图样
- 对齐模式切换 → 构型变化
- 公式模式切换 → 跳变行为差异
