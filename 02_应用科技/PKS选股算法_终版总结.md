## 二、最终公式（v2.4）

### 五参数计算

```
60日窗口：r₁ = max(偏上, 偏下) / MA60,  r₂ = min

h  = 2·r₁·r₂/(r₁+r₂)              市场尺度（调和平均）
γ  = arctan[(r₁−r₂)/(r₁+r₂)]      趋势烈度 ⭐ 唯一稳健因子
z₀ = (u₁²+u₂²)/(u₁+u₂), u=1/r     市场温度
α  = arctan[−u₁u₂(u₂−u₁)/(u₁+u₂)] 极化度
c  = (z₀√2/2)·tan(2α)            两力耦合（仅周期股有效）
egg_safety = z₀/2√(tan|α|)       蛋形安全裕度
```

### 评级规则

```
仅震荡市使用（沪深300: −5% < 60日Δ < +10%, price ≈ MA60±5%）:

A: γ∈[25°,30°] + h>池中位数 + (周期:c≥0)
B: γ∈[20°,25°] + h>池中位数
C: γ∈[10°,20°]
D: γ<10°
X: γ>30° 或 (周期:c<0) 或 (高波:egg_safety<1.5)

牛市/熊市：不使用 PKS，大盘方向碾压一切个股结构
```

### 3.2 h 映射反了

| 旧理解（错） | 正确理解 | 数据证据 |
