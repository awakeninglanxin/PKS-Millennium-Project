## 一、洛伦兹力 = PKS双锥体的单粒子根源

### 1.1 洛伦兹力的二分量分解

$$\mathbf{F} = q(\mathbf{E} + \mathbf{v} \times \mathbf{B})$$

| 分量 | 数学 | PKS锥体 | 物理角色 |
|:---|:---|:---|:---|
| 电力 $q\mathbf{E}$ | 线性场，无速度依赖 | **电性锥** $a=x$ | 压力场、标势、椭圆截面 |
| 磁力 $q\mathbf{v}\times\mathbf{B}$ | 非线性，依赖速度 | **磁性锥** $b=1/x$ | 速度场、矢势、蛋形截面 |

**关键**：$ab=1$ 极化原理不是凭空假设——它是洛伦兹力公式中 $\mathbf{E}$（电性）和 $\mathbf{v}\times\mathbf{B}$（磁性）永远协同出现的数学抽象。

### 1.2 从单粒子到连续介质

```
单粒子:                  连续介质:
F = q(E + v×B)    →    f = ρ_e E + J × B
                            │
                    在准中性等离子体中 ρ_e→0
                            │
                    f ≈ J × B  (MHD中的磁力项)
                            │
                    对应PKS磁性锥 b=1/x
```

**推论**：超导态的配对电子虽然形成Cooper对，但底层仍是带电粒子。PKS将配对强度映射为锥面曲率 $\kappa$，本质上是用几何语言重新表述了**磁力项 $J\times B$ 的涡旋约束强度**。

## 二、广义欧姆定律 → PKS锥面约束

### 2.1 双流体到单流体的推导链

从电子和离子的运动方程出发（各含洛伦兹力项）：

$$m_e n_e \frac{d\mathbf{v}_e}{dt} = -e n_e (\mathbf{E} + \mathbf{v}_e \times \mathbf{B}) - \nabla p_e - m_e n_e \nu_{ei}(\mathbf{v}_e - \mathbf{v}_i)$$

$$m_i n_i \frac{d\mathbf{v}_i}{dt} = e n_i (\mathbf{E} + \mathbf{v}_i \times \mathbf{B}) - \nabla p_i - m_i n_i \nu_{ie}(\mathbf{v}_i - \mathbf{v}_e)$$

定义 $\mathbf{J} = e(n_i\mathbf{v}_i - n_e\mathbf{v}_e)$，$\mathbf{v} = (m_i n_i\mathbf{v}_i + m_e n_e\mathbf{v}_e)/(m_i n_i + m_e n_e)$，消去各自速度：

$$\mathbf{E} + \mathbf{v} \times \mathbf{B} = \eta\mathbf{J} + \frac{1}{en_e}(\mathbf{J}\times\mathbf{B} - \nabla p_e) + \frac{m_e}{e^2 n_e}\frac{\partial\mathbf{J}}{\partial t}$$

### 2.2 理想MHD极限 → PKS磁场冻结

理想MHD：$\eta\to0$，所有非理想项可忽略：

$$\mathbf{E} + \mathbf{v} \times \mathbf{B} = 0$$

**PKS翻译**：流体元锁定在 $xy=1$ 锥面上——初始在锥面上的粒子**永远**不会离开。

> ⚠️ **v1.0 错误修正**：原声称"磁冻结 = Meissner 效应"。这是**方向性错误**——磁冻结是磁力线**锁入**流体，Meissner 是磁场被**排出**超导体。两者是相反的过程。正确的对应是：**锥面约束 = 等离子体磁约束**（如托卡马克中磁场将等离子体约束在环面内），而非 Meissner 效应。

### 2.3 非理想项 → 涡旋运动

当 $\eta\mathbf{J} + \frac{1}{en_e}\mathbf{J}\times\mathbf{B}$ 不可忽略 → 锥面"软化" → 流体元可穿越锥面。

**关联**：第二类超导体中磁通量子化 → 磁通涡旋格点（Abrikosov lattice）。这是**现象层面的平行**（都存在涡旋格点），但 PKS 锥面穿越等同于 flux creep 的说法未经验证。

## 三、超导配对机制的 PKS 重述（修正版）

> ⚠️ **v1.0 错误**：声称"s-wave = 椭圆，d-wave = 蛋形"。这是**对称群不匹配**的强行对应。

### 3.1 为什么形状与配对对称性无关 — 诚实分析

| | s-wave (BCS) | d-wave (自旋涨落) |
|:---|:---|:---|
| **k 空间对称性** | 完全球对称 O(3) | 四重对称 $D_{4h}$ + 符号翻转 |
| **对应的 PKS 形状** | （曾被误标为"椭圆"） | （曾被误标为"蛋形"） |
| **形状的实际对称群** | 椭圆 = $D_{2h}$（前后左右对称，非球） | 蛋形 = $C_s$（仅镜面对称，无四重对称） |
| **对称群匹配？** | ❌ 椭圆有 2 重对称 ≠ s-wave 的全对称 | ❌ 蛋形无四重对称 ≠ d-wave 的四叶结构 |

> **结论**：PKS 锥面截面（椭圆/蛋形）描述的是**实空间中涡旋管横截面的几何变形**——这是一个连续力学问题。s-wave/d-wave 描述的是**k 空间中配对波函数的对称性**——这是一个量子多体问题。两者的对称群**完全不同**，不存在映射关系。

### 3.2 PKS 超导理论中锥面的真实角色（修正版）

PKS 锥面在超导问题中的角色**不是**直接描述配对对称性，而是：

| PKS 概念 | 真正的超导对应 | 精度 |
|:---|:---|:---|
| $U/t$（Hubbard 关联强度）↔ $k_E$（蛋形度） | 关联强度越大 = 锥面截面越不对称 | 🔶 定性同向，无定量映射 |
| $n_i \leq 1$（Mott 约束）↔ $xy=1$（锥面约束） | 两者都是"不可穿越的硬边界" | 🔶 概念平行 |
| 锥面焦散线 $\kappa\to\infty$ ↔ 量子临界点 p* | 曲率发散处 = 相变边界 | 🔶 启发式 |
| **交比 CR** ↔ **费米面四点射影不变量** | 关卡2的核心——这个有具体定义 | ✅ 数学可定义，实验可测量 |

**删除的 v1.0 错误声明**：
- ~~s-wave = 椭圆，d-wave = 蛋形~~ — 对称群不匹配
- ~~$U\to0$ → 蛋形→椭圆 → $k_E\to1$ = s-wave~~ — 形状退化描述的是实空间截面，与 k 空间配对对称性无关
- ~~$U/t$ 大 → 蛋形 $k_E\gg1$ = d-wave~~ — 同上

### 3.3 PKS 与超导配对理论的真正可对话点

PKS **不取代** BCS 或自旋涨落理论。PKS 的作用是：

1. **提供一个与配对机制无关的纯几何判据**（交比 CR）来判断 $T_c$ ——这对 s-wave 和 d-wave 同样适用
2. **用 $k_E$ 统一描述关联强度**——不论配对媒介是声子还是磁振子，关联强度增加都体现为截面不对称度增加
3. **关卡1 的退化检验**保证在弱关联极限（$U\to0, k_E\to1$）自动回到 BCS——但这不意味着"$k_E\to1$ 等价于 s-wave"，只意味着"弱关联极限下 PKS 的预测与 BCS 一致"

这类似于热力学与统计力学的关系——热力学不告诉你微观机制，但它在所有微观机制下都成立。

## 五、总结：为什么这个物理基础是必要的

PKS 超导理论不是"纯几何的形式类比"。它的物理合法性根源于：

1. **洛伦兹力天然具有二分量结构**（$\mathbf{E}$ + $\mathbf{v}\times\mathbf{B}$）→ $ab=1$ 极化原理
2. **MHD 是洛伦兹力的连续介质版** → 双锥体是 MHD 的几何表示
3. **广义欧姆定律的推导链**确保了两锥之间的耦合系数 $\alpha$（精细结构常数）的自然出现
4. **磁冻结定理 = 锥面约束**提供了超导 Meissner 效应的几何等价物

这个链条使得 PKS 关卡1的退化检验、关卡2的交比不变量、关卡3的 Tc 预测——**都有具体的物理对应，而非纯粹的数学猜想**。
