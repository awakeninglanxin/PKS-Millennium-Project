# 预测2: 铁基超导的 PKS 多锥体推广 — 数学框架

> 铜氧化物是单带 Hubbard → 单锥体 $xy=1$。铁基超导（FeAs/FeSe）是多轨道系统（$d_{xz}, d_{yz}, d_{xy}, d_{x^2-y^2}, d_{z^2}$）→ 需要 **N-锥体耦合**。

---

## 一、单锥 → 多锥的必然性

### 1.1 铁基与铜氧化的根本差异

| | 铜氧化物 | 铁基 |
|:---|:---|:---|
| 活性轨道 | 1 ($d_{x^2-y^2}$) | **5** ($d_{xz},d_{yz},d_{xy},d_{x^2-y^2},d_{z^2}$) |
| 费米面 | 1个大空穴面 | **2-5个空穴+电子面** |
| 配对对称性 | d-wave | **s±** (不同费米面间反号) |
| 磁结构 | (π,π) 棋盘 | **(π,0)/(0,π) 条纹** |
| PKS锥体数 | 1 | **N(nesting vector数)** |

### 1.2 多锥体系统的超曲面

单锥体：$F(x,y)=xy-1=0$ → 截面 = 蛋形。

N-锥体：**N个锥面的纤维积**（fiber product）：

$$\mathcal{F}_{\text{Fe}} = \bigotimes_{\alpha=1}^{N} \left\{ x_\alpha y_\alpha = 1 \right\}$$

这是一个 $2N$ 维超曲面（每个轨道贡献 2 维 = 电性+磁性锥各一维）。

## 二、s± 配对对称性的 PKS 解读

### 2.1 锥面间耦合矩阵

N 个锥面之间的耦合由**锥面交比矩阵**决定：

$$\text{CR}_{\alpha\beta} = \frac{(z_\alpha - z_\beta)(w_\alpha - w_\beta)}{(z_\alpha - w_\beta)(w_\alpha - z_\beta)}$$

其中 $(z_\alpha, w_\alpha)$ 是第 $\alpha$ 个锥面的截面参数。

**s± 的条件**：$\text{CR}_{\alpha\beta} < 0$ （负交比 → 反号配对）

### 2.2 铁基能带结构的锥面分配

| 轨道 | 费米面类型 | PKS锥面 | 锥面参数 $(z_0,\alpha)$ |
|:---|:---|:---|:---|
| $d_{xz}$ | 电子面 (X点) | 锥面 A | $z_0$大, $\alpha$小 |
| $d_{yz}$ | 电子面 (Y点) | 锥面 B | $z_0$大, $\alpha$小 |
| $d_{xy}$ | 空穴面 (Γ点) | 锥面 C | $z_0$小, $\alpha$大 |
| $d_{x^2-y^2}$ | 空穴面 (Γ点) | 锥面 D | $z_0$小, $\alpha$大 |
| $d_{z^2}$ | 非费米面 | 锥面 E | 弱耦合(旁观) |

**配对机制**：电子面锥面（A,B）与空穴面锥面（C,D）之间的嵌套矢量 $\mathbf{Q}=(\pi,0)/(0,\pi)$ 产生锥面间共振 → s± 配对。

### 2.3 Tc 的多锥推广

$$T_c^{\text{multi}} = 1.14\,\omega_{\log} \exp\left(-\frac{1}{\sum_{\alpha,\beta} g_{\alpha\beta}}\right)$$

其中锥面间耦合增强因子：

$$g_{\alpha\beta} = \lambda_{\alpha\beta}^{\text{ep}} \cdot \frac{k_{E,\alpha} \cdot k_{E,\beta}}{|\text{CR}_{\alpha\beta}|}$$

**关键预测**：铁基超导的 Tc 由**锥面间嵌套质量**（nesting quality = 费米面面积的锥面截面重叠度）决定，而非单一 Hubbard U。

---

## 三、N-锥体系统的拓扑不变量

### 3.1 锥面手性矩阵

每个锥面有内在手性（左旋/右旋 = 电性/磁性占优）。N-锥系统的总手性：

$$\mathcal{H}_{\text{total}} = \sum_{\alpha} \text{sgn}(k_{E,\alpha} - 1)$$

- $\mathcal{H}=0$：全部锥面退化到椭圆（BCS极限）
- $\mathcal{H}=N$：全部强磁性（多带Mott绝缘体）
- $\mathcal{H}\approx N/2$：半锥面磁性（s± 配对最优窗口）

### 3.2 铁基"11"型（FeSe）vs "122"型（BaFe₂As₂）

| 体系 | N(锥面数) | $\mathcal{H}_{\text{total}}$ | 预测 Tc(K) | 实验 Tc(K) |
|:---|:---:|:---:|:---:|:---:|
| FeSe (11) | 3 | 1 | ~15 | 8 (块材), 65 (单层) |
| BaFe₂As₂ (122) | 5 | 3 | ~45 | 38 |
| LaFeAsO (1111) | 5 | 3 | ~60 | 55 |
| 铜氧化物 (单锥) | 1 | 1 | ~100 | 95 (Hg1223) |

> 🔶 趋势大体符合，但 FeSe/STO 单层的 65K 增强超出了当前框架——可能需要额外锥面耦合（界面声子 ↔ 锥面E 被激活）。

## 四、伪代码

```python
def predict_Tc_iron_pnictide(orbitals, fillings, U_over_t_per_orbital):
    """
    铁基超导 Tc 预测: 多锥体耦合
    
    输入:
      orbitals:            活性轨道列表 [d_xz, d_yz, ...]
      fillings:            各轨道填充数
      U_over_t_per_orbital: 各轨道关联强度比
    
    返回: Tc (K), 配对对称性, 主导嵌套矢量
    """
    N = len(orbitals)
    cones = []
    
    # 1. 每轨道 → 一个锥面
    for i in range(N):
        z0 = 1.0 / (fillings[i] - 0.5 + 1e-6)
        alpha = 0.1 + 0.3 * (fillings[i] - 0.5)
        kE = 1 + 0.1 * U_over_t_per_orbital[i]
        cones.append({'z0': z0, 'alpha': alpha, 'kE': kE})
    
    # 2. 锥面间耦合矩阵
    g_sum = 0
    CR_matrix = np.zeros((N, N))
    for i in range(N):
        for j in range(i+1, N):
            CR = abs(cones[i]['z0'] - cones[j]['z0']) / (cones[i]['kE'] * cones[j]['kE'])
            CR_matrix[i,j] = CR_matrix[j,i] = CR
            g = 0.5 * cones[i]['kE'] * cones[j]['kE'] / (CR + 0.1)
            g_sum += g
    
    # 3. Tc
    lambda_eff = 0.3 + g_sum * 0.5
    Tc = 300 * np.exp(-1.0 / lambda_eff)
    
    # 4. 配对对称性
    if np.any(CR_matrix < 1):  # 负有效交比
        pairing = 's+-' if N >= 3 else 's-wave'
    else:
        pairing = 'd-wave' if N == 1 else 's-wave'
    
    return Tc, pairing, CR_matrix
```
