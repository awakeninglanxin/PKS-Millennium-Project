## 1. 精确定义

### 1.1 B-smooth 数

给定素数集合 B = {p₁,…,p_k}，**B-smooth 数**是所有素因子均属于 B 的正整数。记为 S(B) 或 S_k。

### 1.2 C_k 生成命题

> 素数 p 被基底 B_k **可表示** ⇔ 存在 a, b ∈ S(B_k) 且 a, b ≤ D_p（部件上限），使得：
> `(p = a + b) ∧ (a + b ≤ X)`  或  `(p = |a − b|) ∧ (a, b ≤ D)`

参数说明：
- **X** = 素数范围 ≥ 2p_max
- **D** = 部件上限——由 Tijdeman(1974) 光滑数间隙定理，D 不可能远超 p·polylog(p)。本验证采用 D = max(10³X, 10¹²) 作为充足余量。

### 1.3 覆盖半径 X*(k)

> **X*(k) = 最小素数 p，使得 C_k 在 p 成立前对所有 ≤ p 的素数全覆盖，但 p 本身不可表示。**

即：X*(k) = C_k 的**首个反例尺度**。X*(k) 有限 ⇔ C_k 非普遍性。

## 3. 理论解释（为什么"零反例"必然失败）

### 3.1 易方向：C_k 能覆盖多少素数？(上界)

设 B 含 k 个素数。S(B) 在 [1, T] 内的数量为：

```
S_k(T) ∼ (log T)^k / (k! · ∏_{i=1}^k log p_i)    (Ennola/Dickman 公式)
```

- C_k 中**和式**可表示素数：a+b ≤ X ⇒ a, b 都在 [1, X] 内。最多 C(S_k(X), 2) 种组合。
- 差式：a, b ≤ D ⇒ 最多 C(S_k(D), 2) 种组合。

因此 C_k 在 [1, X] 内可表示的素数总数：

```
#representable ≤ S_k(X)^2 + S_k(D)^2 ∼ O((log X)^{2k})
```

而素数个数 π(X) ∼ X / log X。当 X → ∞ 时，`(log X)^{2k} ≪ X / log X`——**任何固定 k 的可表示素数都只有零密度**。

### 3.2 难方向：每一个特定 p 能有表示吗？(反例存在性)

**和式**：p = a + b, a,b ≤ D。如果 a,b 都在 [1, X] 内，从 3.1 的计数论证，p 获表示的概率随 p 增长而衰减为 `#pairs / ~p² ∼ (log p)^{2k−1} / p` → 泊松过程，首缺失发生在大约 e^{2k} 尺度。

**差式**：p = a − b, a,b ≤ D 且 a > b。Tijdeman(1974) 的光滑数间隙定理：相邻光滑数间距随 n_i 增长为 n_i⁰·(log n_i)^{C(k)} 形式。这意味着给定 p，方程 `x + p = y` 在光滑数集合中只有有限个解（S-unit 方程有限性，Evertse 1984 给出一致上界）。

因此在差式部分，每个 p 只有有限（由指数与光滑性参数决定）的候选表示。当 p 充分大时，这个解数趋近于零。

### 3.3 十字交叉

两个机制——和式的计数衰减 + 差式的有限性——共同决定 X*(k)。实测 X*(k) ∝ r^k, r≈5，与 Dickman 函数密度估计在 k≤11 范围内吻合。(log X*)^k / k! 项解释了指数规律。

## 4. 方法论意义：生成边界的规模共轭

这与**大数因子分解的基底选择**是同一现象：

> 在二次筛/数域筛中，基底规模 B ≈ exp(½√(log N·loglog N)) 时，"光滑关系"足够覆盖剩余类。
> 这里精准量化了**给定基底 k 的覆盖半径 X***——对任意有限 k，存在一个明确的范围 X*(k) 使其内部能被覆盖。

| | 因子分解 | 本问题 |
|---|---------|--------|
| 输入 | N（合数） | p（素数） |
| 基底 | 因子库 k(N) 个素数 | 生成基底 k 个素数 |
| 覆盖谓词 | N 是 y-smooth（光滑分解） | p 是 B-smooth 碰撞（双光滑表示） |
| 当 N→∞ 时 | k 必须增长 ~exp(√log N) | k 必须增长 ~log X*(k) （实测指数律） |

这就是为何基底规模必须随覆盖范围增长——这是**光滑性稀缺原理**的直接推论。

## 5. 深检验证：9,734,639 真的零表示吗？

```
基底: {2,3,5,7,11,13,17,19,23}
部件范围: D = 10^13  (p ≈ 1e7, 即部件可达 p 的 10^6 倍！)
23-smooth ≤ 1.0002e13: 1,550,012 个
p = 9,734,639: 0 个表示 (p+b 均非 23-smooth, p−a 同样)
邻近素数 9,734,597: 至少 1 个表示
```

→ **9,734,639 是不成立的** —— 且是一个"孤岛"式的非表示点（对光滑数集合的某种互质/偏置效应），这是 Tijdeman 类定量间距问题的典型表现。

```
同为重要的临近反例 —> 9,982,241（同样零表示 @ D=1e13）
```
