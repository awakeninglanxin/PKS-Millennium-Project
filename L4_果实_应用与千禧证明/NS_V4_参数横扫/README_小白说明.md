# NS_V4: 参数横扫 — 傻瓜说明

> **一句话**: 不再靠 AI 猜了——把 512 种参数组合全部算一遍，精确找出"最危险的点"。

---

## 这张图在说什么？

V3 用了 200 个智能采样点。V4 更狠：**512 个均匀网格点全部跑一遍**（3D 网格：Re × α × nu 各 8 个值）。暴力但精确。

---

## 每个子图的意思

```
左上: Blowup Time 热图 (nu vs Re)
    → 颜色=距离blowup还有多少时间
    → 越红=越近=越危险
    → 左上角(低nu+高Re)=最危险

右上: α 剖面 - 最佳拉伸强度
    → 对每个 α 值, 找最坏的 Re+nu 组合
    → 曲线: α 在 0.6-0.8 区间 blowup 时间最短

中左: α vs nu 热图 (@固定Re)
    → 横轴=α, 纵轴=nu(粘性)
    → 颜色=blowup时间
    → 低nu+高α = 红色 = 危险

中右: Re vs α 热图 (@固定nu)
    → 横轴=Re, 纵轴=α
    → 右上角=最危险

左下: 3D散点图
    → 三个轴=Re, α, nu
    → 每个点=一个参数组合
    → 颜色=blowup时间
    → 可以旋转着看"危险区域像什么形状"

右下: 最危险的 Top 10 参数
    → 柱状图
    → 列出哪10组参数blowup最快
```

---

## 关键结论

**(α≈0.7, 最低nu, 最高Re) 是最危险的参数组合**。V4 确认了 V3 的结论，而且更精确。

| 术语 | 大白话 |
|:---|:---|
| nu (粘性) | 水的"粘稠度"。nu小=水稀=容易blowup |
| Blowup Time | 还有多少秒漩涡会炸。越小越危险 |
| 热图 (Heatmap) | 颜色代表数值——像天气预报的红外云图 |
| 参数横扫 | 把所有可能的参数组合都试一遍 |
