# NS_V7: Sobolev 退化诊断 — 傻瓜说明

> **一句话**: 除了"漩涡无限变大"这种炸法，NS 方程还有第二种炸法——水慢慢变"碎"直到失去光滑性。

---

## 这张图在说什么？

V1-V6 都在找第一种 blowup：湍流越来越强，漩涡无限大（Leray 类型）。

V7 研究**第二种**（窦华书 2025-09 提出）：水本身在 3D 下会慢慢失去"光滑性"——像一块布被反复揉搓直到纤维断裂。这叫 **Sobolev 正则性退化**。

---

## 每个子图的意思

```
左上: 涡量场对比
    → Standard(无锥) vs PKS Cone(有锥)
    → 在 2D 下的涡量快照
    → PKS 侧的漩涡更集中

右上: 涡量最大值 + 拟涡能 + 速度散度
    → 三条曲线随时间变化
    → max vorticity: 最大漩涡强度
    → enstrophy: 全部漩涡的总能量
    → 有锥的各项指标都更高

中左: H^s Sobolev 范数
    → 最关键的图! 
    → H^1, H^2, H^3, H^4 四条线
    → H^3 (最粗线) 在退化
    → 如果 H^3 → 0 = 水失去了可微性 = "碎了"

中右: PKS-EVMP 严格化
    → EVMP = 窦华书的能量-涡量最小原理
    → 两条曲线: Standard(蓝) vs PKS(红)
    → PKS 曲线更平坦 = PKS 几何让退化更稳定
    → EVMP ratio = 0.56 (PKS比标准低了44%)

左下: 局部 Laplacian 归零检测
    → 检查哪个区域水开始变"碎"
    → 深蓝色区域 = Laplacian 接近零 = 退化区

右下: EVMP Ratio 统计
    → 柱状图
    → Standard EVMP std = 1,297,383
    → PKS EVMP std = 729,974 (降低了44%)
    → PKS 几何更稳定!
```

---

## 关键结论

**在 2D 下，H^3 范数衰减约 83%（弱退化），PKS 锥的 EVMP 比标准结构稳 44%**。但 2D 有全局正则性保证，所以 V7 的结果需要在 3D 中验证（那就是 V8 的任务）。

| 术语 | 大白话 |
|:---|:---|
| Sobolev 空间 H^s | 衡量函数"有多光滑"。H^3 = 3次可微。退化 = 从 H^3 掉到 H^{3/2} |
| Laplacian | 衡量某点周围的"曲率"。趋近于 0 = 水在那里要碎了 |
| EVMP | 窦华书的猜想：流体总能耗散遵循最小原理 |
| 正则性 | 数学上的"光滑程度"。正则性退化 = 出现尖角/断裂 |
