## 三、IIM/MIIM 算法速查

### 正向 vs 逆向

| | 正向迭代（M集） | 逆向迭代（IIM） |
|------|------|------|
| 公式 | z → z² + c | z → ±√(z - c) |
| 着色 | 逃逸时间（烟花） | 边界描点（丝缕） |
| 动力学 | ∞ = attractor, 边界 = repeller | 边界 = attractor, ∞ = repeller |
| 视觉 | 向外炸、芽苞 | 向内折、龙鳞 |
| 缺陷 | — | "冷段"走不到 → MIIM 加 hit-limit |

### MIIM（Modified IIM）核心逻辑

```python
def miim(c, n_iter=10_000_000):
    z = 2.0 + 0j  # 从外区出发
    hit = {}        # 像素命中计数
    
    for _ in range(n_iter):
        z = csqrt(z - c, branch=random.choice([+1, -1]))
        px, py = to_pixel(z)
        hit[(px, py)] = hit.get((px, py), 0) + 1
        
        if hit[(px, py)] > hit_limit:  # 冷段跳过
            z = random_boundary_point()
    
    return hit.keys()  # Julia边界点集
```
