## 先理解这条实验线的曲折（60 秒）

NS 方向想验证：**PKS 锥体几何能否放大涡量**（涡量放大链是 blowup 的前提）。

经历了三个阶段：

```
阶段1 光谱法（失败）：四种方案全部涡量衰减 → 铁律79：光谱法无真实边界层
阶段2 解析涡管（可视化）：Burgers涡管模型渲染出漂亮的螺旋结构——但不是真解
阶段3 LBM D3Q19（正路）：反弹边界=真实壁面 → 涡量放大10.3×（但显式格式易NaN）
```

**看每个动画时先问：它属于哪个阶段？** 阶段1的动画是"失败展览"，阶段2是"艺术照"，阶段3才是物理。

## 第一组：光谱法动画（阶段1 — 失败的教材）

### `PKS_Cone_Vortex_3D.gif` — 128³ 自由衰减

- **画面**：蓝色背景中一团红色云雾慢慢变淡、扩散
- **怎么看**：注意红色区域**只会变淡不会变亮**——这就是"衰减"。Growth=0.56×（涡量掉了一半）
- **为什么失败**：光谱法（Fourier 谱）根本没有"壁面"概念，涡量放大链的第一步（壁面边界层产生 ∂u/∂y 梯度）在这种离散上不存在

### `PKS_Cone_Spiral_Flow.gif` — 200³ 螺旋入流

- **画面**：加了螺旋入口力的版本，能看到一些旋转纹理
- **怎么看**：即使人为注入螺旋，Growth 也只有 0.93×——**连 1 都不到**，注入的能量在光谱离散上被耗散吃掉
- **教训**：不是入口不够猛，是数值方法根本不支持壁面物理

### `PKS_Cone_256_Re160.gif` — 256³ Re=160

- **画面**：最高分辨率版本，涡核位置随时间摆动
- **怎么看**：摆动 ≠ 放大。max|ω| 从 1.500→1.460，稳定衰减。Growth≈0.80×
- **意义**：证明了"加分辨率救不了光谱法"——96³→256³ 四种网格全部衰减

### `IBM_Cone_45deg.gif` — IBM+SDF 45° 锥体

- **画面**：用浸入边界法（IBM）+符号距离场（SDF）硬加壁面的尝试
- **怎么看**：Growth=0.86×，还是衰减——**简单 IBM 罚函数不等于真实边界层**，力的注入方式错了照样白搭
- **地位**：四连败的最后一败，直接催生了铁律79

> **四个动画共同的看点**：它们全是蓝底红云、只衰减不增长。这不是 bug，是光谱法的本质缺陷。留着它们是为了提醒：**方法选错，参数调优无意义。**

## 第二组：解析涡管渲染（阶段2 — 可视化成果）

⚠️ 这两张 PNG 视觉效果最好，但要清楚：它们是 **Burgers 解析涡管模型**（人工构造的涡量场），不是 NS 方程的真解。用途是展示"PKS 锥内螺旋涡应该长什么样"。

### `PKS_Cone_Vortex_3D_Isosurface.png` — 3D 等值面渲染 ★ 最漂亮的一张

- **画面**：锥体内三层嵌套的彩色曲面——橙色（外层，10%涡量阈值）→ 金色（涡核，22%）→ 青色（内涡核，40%）
- **数据规模**：82,182 顶点 / 164,336 面 / 锥内 437,256 网格点（见 `stats.json`）
- **怎么看**：
  1. 先看**整体轮廓**——6 圈螺旋涡管沿锥壁盘旋而下，半径随锥收缩
  2. 再看**三层颜色嵌套**——越靠内涡量越强（青色芯 = 涡量 40% 峰值区）
  3. max|ω|=9.14，均值 0.83——**涡核比背景强 11 倍**，这就是"涡量集中"的可视化
- **对应物理**：贴壁螺旋流（趋肤效应）的理想形态——16bands 导轨设计的目标流态

### `PKS_Cone_Vortex_XZ_Slice.png` — XZ 截面热力图

- **画面**：垂直剖开锥体（Y=0 截面），涡量热力图 + 锥壁线 + 标注
- **怎么看**：
  1. **锥壁附近的亮带** = 边界层涡量（光谱法缺失的正是这个！）
  2. **6 个亮斑** = 螺旋涡管被剖面切出的截面（涡管每绕一圈被切一次）
  3. **小亮点** = 80 个次级涡（涡管间相互诱导产生）
- **配套动画**：`cfd_vortex.gif` 是这个模型的动态版

## 第三组：LBM 动画（阶段3 — 正路，根目录）

### `../lbm_3d_iso.gif` — LBM 3D 等轴渲染（3.4M）★ 保留

- **来源**：`06_GPU实验脚本/C_NS涡流CFD/task_d3_iso.py`（LBM v5，128×128×192，8000步，25帧快照）
- **画面**：真实 LBM 演化的 3D 涡量等值面随时间旋转/生长
- **怎么看**：与第二组"艺术照"不同，**这是 D3Q19 反弹边界的真实流体演化**——涡结构是算出来的不是构造出来的
- **已知问题**：帧间可能有闪烁/不连贯（快照间隔320步较粗），但物理是真的

### `../lbm_v8_fixed.gif` — LBM v8 修复渲染（844K）★ 保留

- **来源**：`06_GPU实验脚本/C_NS涡流CFD/task_d6_v8.py` + 数据 `../lbm_v8.pkl`
- **画面**：v8 参数（更稳定的 τ）下的涡量演化，修复渲染版
- **怎么看**：对比 v2/v7 失败版（已删），v8 撑到了完整演化——**喉部涡量放大是 LBM 线的核心证据**（另一会话实测喉部 ω×35.8）
