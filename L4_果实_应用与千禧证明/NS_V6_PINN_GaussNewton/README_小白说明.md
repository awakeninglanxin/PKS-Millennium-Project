# NS_V6: PINN + Gauss-Newton 高精度 — 傻瓜说明

> **一句话**: 用 Google DeepMind 2025 年最强的数学优化方法，把搜索精度推到小数点后 6 位。

---

## 这张图在说什么？

V3-V5 搜索的精度还比较粗糙（靠随机采样或网格）。V6 用了**Gauss-Newton 二阶优化**——这是 DeepMind 2025-09 论文里用来看 3D Euler 方程奇点的同款方法。

简单说：V3 是"在沙漠里骑自行车找水"，V6 是"卫星定位+钻井机直达水源"。

---

## 每个子图的意思

```
左上: PKS 自相似 Profile
    → PKS 双曲锥下的速度场形状
    → U(η) = α·exp(-β·η²)/(η+0.5)
    → 曲线=速度, 横轴=离锥喉的距离

右上: Gauss-Newton 收敛曲线
    → 横轴=迭代次数, 纵轴=误差
    → 曲线指数下降→每步都在变准
    → 最终到 round-off 精度(10^-15)

中左: Blowup Rate vs Instability Order
    → 横轴=不稳定度阶数, 纵轴=blowup速度
    → 🔥 这是 DeepMind 的核心发现!
    → 点沿一条直线分布=两者有线性关联
    → Pearson r = -0.71 (强相关)

中右: 相位图 α vs β
    → 最优参数在哪里
    → 横轴=α, 纵轴=β
    → 每个点是一次 Gauss-Newton 迭代的位置
    → 点汇聚到一处=找到了最优解

左下: 残差分布
    → PINN 在每个格点的预测误差
    → 越趋近于0越好
    → 直方图

右下: 不同T时刻的 Profile 扫描
    → 多条曲线重叠, 每条是不同时刻的自相似解
    → 找哪个时刻 blowup 最快
```

---

## 关键结论

**最优参数: α=0.144, β=0.351。Blowup rate 与不稳定度阶线性相关 (r=-0.71)**——重现了 DeepMind 2025 的"线性模式"发现。在 PKS 双曲坐标下，这个"线性模式"实际上对应 $1/x$ 反比例——是 PKS 几何的自然结果。

| 术语 | 大白话 |
|:---|:---|
| Gauss-Newton | 一种"二阶"优化——不光看坡度，还看曲率，走得更准 |
| Pearson r | 相关性。r=-0.71 表示很强的负相关 |
| Round-off 精度 | 计算机能算到的最精确程度 (~10^-15) |
| 不稳定度阶数 | 漩涡"疯狂程度"的一个指标 |
