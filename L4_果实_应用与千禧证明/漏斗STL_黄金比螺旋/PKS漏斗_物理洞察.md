## 一、三大物理效应排序

| 效应 | 机制 | 量级 |
|:---|:---|:---:|
| **体积效应** | $V\uparrow \Rightarrow$ 更多水分子加速 $\Rightarrow$ 涡量$\uparrow$ | **主导**（等体积下形状差<4%） |
| **路径效应** | 长加速路径 $\Rightarrow$ 温和持续加速 $\Rightarrow$ 少耗散 | **二阶**（~15%差距） |
| **曲率效应** | 双曲壁面 $\Rightarrow$ 流线弯曲 $\Rightarrow$ Dean二次涡 | **三阶**（~2-3%差距） |

## 二、为什么缓坡比极端收缩好

```
极端收缩(z=-1/x, 64:1):
  ├─ 喉部猛烈加速 ⋯但路径极短（H=0.85）
  └─ 水来不及"享受"加速就离开了

缓坡(30°直锥, 6.5:1):
  ├─ 温和但持续的加速（H=2.03）
  └─ 长路径意味着每滴水经历了更多加速长度

类比: 短跑冲刺 vs 长跑匀速。
```

## 三、为什么对数螺线最弱

黄金比螺旋 $z = -\log x/\ln\phi$ 的特点是"渐近逼近但永不触及"轴线：

```
Z: 1.3 → -0.2 → -1.6 → ... → -5.6
R: 1.9 → 0.98 → 0.48 → ... → 0.07
```

前面收缩很快，但最后**永远在逼近而不触及**。这意味着：
- 前半段：半径从 1.9 到 0.3，够快 ✓
- 后半段：从 0.3 到 0.07 用了总长度的 60%——效率极低 ✗

**"优雅"的数学性质恰好是"低效"的流体力学性质。**

## 四、表面积的双重角色 — 这是两种完全不同的物理

### 模式一：均匀轴向流入（已用 CFD 验证）

```
水从上方竖直落入 → 充满整个截面 → 中心涡核+壁面边界层

壁面: 摩擦 = 能量损耗
  → 表面积大 = 边界层阻力大 = 涡量被"磨掉"
  → 排名: 30° > 超双曲 > 16bands > 8bands > 45° > 黄金比
  → 标志: 无量纲效率 = Vort / SA（越小越好用铜）
```

### 模式二：贴壁螺旋流入（物理推定，CFD 待验证）

```
水沿壁面斜径向螺旋向下 → 贴壁环状薄层 → 中心空气柱+杨柳辫

壁面: 摩擦 → 角动量注入 → 加速跑道
  → 表面积大 = 更长的螺旋加速路径
  → 螺旋导轨 = 每圈一次"踢"水注入角动量
  → 排名推定: 16bands > 8bands > 超双曲 > 黄金比 > 30° > 45°
  → 标志: 无量纲效率 = Vort / SA（衡量单位铜材的涡量产率）
```

### 关键认知

**同一物理量（表面积）在两种流入模式下有相反的作用。**
**两种模式下的 CFD 数据不能互用——各自的排名基于各自物理假设。**

## 五、趋肤效应与杨柳辫

高频电流沿电线表面走（趋肤效应）。水流类似：
- 重力 + 螺旋壁 → 水被压在壁上 → **贴壁环状流**
- 中心区域 → 气流低压 → **空气柱**
- 气水交界面 → 涡丝失稳 → **多股螺旋辫（杨柳辫）**

这解释了为什么：
1. 壁面面积至关重要（水不走中心，走壁面）
2. **Bands 不是回转体+纹理** — 是双轨扫掠面，截面沿 Z 轴+对数螺线从顶到底扭转 8 或 16 圈
3. **16bands vs 8bands**: 截面多扭转 8 圈 = 水多经历 8 次完整的角动量方向变换
   → 之前用"间距 22.5° vs 45°"分析是错误的——那是 STL 三角网格化 artifact
4. 我的轴对称模型从根本上忽略了这个物理

## 六、空化链

```
漩涡加速 → 核心压力↓ → P<2300Pa → 冷水沸腾
→ 气泡生成 → 流到高压区 → 气泡坍塌
→ 局部5000K + 冲击波 + OH自由基 → 杀菌/降解
```

PKS锥的核心价值：**让空化在更低流速/更小体积下发生。**
