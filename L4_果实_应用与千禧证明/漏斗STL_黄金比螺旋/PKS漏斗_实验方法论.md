# PKS 漏斗 CFD — 实验方法论总结

> **9 轮实验, 6 种漏斗, 1 个下午**

---

## 实验链演进

```
v1: 原始尺寸, 同顶半径归一化
    → 问题: Smooth 7m vs Bands 400m, 不公平
    → 收获: 发现体积差主导结果

v2: Smooth×100, 保留长径比
    → 问题: 体积仍差 17 倍
    → 收获: 长径比的二阶效应

v3: Smooth×118.3, 等体积 (vS 16bands)
    → 🔥 突破: 等体积下差异 <4%
    → 收获: 体积是第一决定因素

v4: 4 种曲线, 同顶半径 (新 STL 加入)
    → 收获: 黄金比≠超双曲, 曲线形状差异确认

v5: 两组逐对等体积 PK
    → 收获: 超双曲>黄金比, 30°>45°

v6: 4 种全等体积对决
    → 收获: 双曲直锥 30° 冠军

v7: 4 种同入口面积对决
    → 收获: 与 v6 一致排名, 验证稳定性

v8: 8bands vs 16bands 同入口
    → 收获: 波段数影响 < 0.5%

v9: 6 种全等体积对决
    → 收获: 完整排名, 双曲直锥 30° 双冠王
```

---

## 学到了什么

### 方法论

| 教训 | 细节 |
|:---|:---|
| **控制变量 > 暴力对比** | v1 没控制体积得出的"Smooth 最优"是假象 |
| **三组对照铁律** | 基准→隔离面积→隔离形状→隔离曲率 |
| **STL 数据不撒谎** | 设计的 1:12 径比实际是 6.5:1 — 相信数据 |
| **渐进式推理** | v1→v3 花了 3 轮才找到"体积主导"的真相 |

### 物理直觉

| 发现 | 解释 |
|:---|:---|
| 缓坡 > 陡坡 | 长加速路径 > 短路径, 即使收窄比更小 |
| 对数螺线最弱 | "渐近逼近" → 长距离 → 粘性耗散压倒了加速 |
| 波段数不重要 | 8 vs 16 bands < 0.5% — 制造选便宜的 |
| 体积是王者 | 差距 < 4% 时形状才显作用 |

### 数值方法

| 修复 | 问题 |
|:---|:---|
| IBM 力 ∝ 位置, 不 ∝ 速度 | v8 的正反馈 bug 修复 |
| dt=0.001, nu=0.01, clamp[-50,50] | 陡壁面溢出防护 |
| 谱法 dealias 2/3 rule | 避免混叠 |
| CJK 字体 SimHei + ASCII 后备 | matplotlib 乱码 |
