# PKS 漏斗 — 工程应用建议

> **基于 9 轮 CFD 实验, 6 种漏斗, 2 种公平对比条件**

---

## 一、核心建议

| 场景 | 推荐 | 理由 |
|:---|:---|:---|
| **最大涡量** | 双曲直锥 30° | 双冠王, 两种条件下都第一 |
| **最简单制造** | 双曲直锥 30° | 无螺旋带, 单一旋转体, CNC 直接加工 |
| **最低成本** | 光滑墙壁 (黄金比) | 虽然涡量弱 23%, 但制造最简单 |
| **大规模建筑（贴壁流）** | 16bands | 下口径多8圈导轨=加速区多8次助推 |

---

## 二、为什么双曲直锥 30° 赢了？

```
缓坡长路径 = 温和加速 = 粘性耗散最小
                    vs
极端收缩 = 猛烈加速但路径短 = 喉部能量耗散大
                    vs
对数螺线 = 渐近逼近但路径最长 = 粘性吃掉所有加速收益
```

**类比**: 滑冰运动员收手臂——不是收得越猛越好，是收得**平稳且持续**才转得最快。

---

## 三、设计指南

### 如果要造一个 PKS 水处理器：

```
1. 选双曲直锥 30° 形状 (涡量冠军)
2. 上半段: 双曲线 (z = -1/x) — 提供初始涡量注入
3. 下半段: 30° 直线锥 — 提供温和长加速路径
4. 体积: 越大越好 (涡量 ∝ 体积)
5. 材料: 光滑壁面 (不需要螺旋带 —— 省成本)
```

### 如果要最大化涡量/材料比：

```
等体积 + 均匀轴向流:
  双曲直锥 30° > 超双曲 > bands > 黄金比 > 45°直锥
  
等体积 + 贴壁螺旋流（推测）:
  16bands > 8bands > 超双曲 > 黄金比 > 30° > 45°
  → 导轨圈数决定角动量注入频率
```

---

## 四、不要做的事

| ❌ 误区 | ✅ 正确 |
|:---|:---|
| 追求极端收窄比 | 缓坡比极端收缩好 |
| 加螺旋带增加涡量 | 均匀轴向流下影响<0.5%（轴对称模型），贴壁流下16bands多8圈导轨=优势 |
| 用对数螺线（黄金比） | 优雅但不实用, 涡量最弱 |
| 不控制体积直接对比形状 | 体积效应掩盖形状效应 |
