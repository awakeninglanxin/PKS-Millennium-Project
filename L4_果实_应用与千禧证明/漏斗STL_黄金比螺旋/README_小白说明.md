# 漏斗STL CFD实验 — 小白说明

> **一句话**: 拿犀牛建模的三个真实漏斗（光滑/8条螺旋带/16条螺旋带），放进 CFD 仿真看哪种形状让水转得最有力。

---

## 四张图的分别说明

### 1. 漏斗STL_几何对比.png — 三种漏斗长什么样

```
左图: 归一化形状对比
    → 横轴: Z(从底到顶), 纵轴: 半径
    → 蓝线=光滑墙壁, 绿线=8bands, 红线=16bands
    → 归一化后三条曲线形状不同:
       - 光滑的最细长(长径比1.88)
       - bands的比较矮胖(长径比~0.7)

右图: 原始尺寸对比
    → 注意: 光滑漏斗只有7m高！
    → 8/16bands有400m高
    → 尺寸差60倍——这就是为什么要做v1→v2→v3对比
```

### 2. 漏斗STL_IBM_三方对比.png — v1: 原始尺寸归一化

```
上排: 时间序列
    最大涡量:  红线(16bands)最高 → 16bands涡量最强
    拟涡能:    三条线纠缠
    H3范数:   类似趋势

下排: 终态截面(Z=0切片)
    → 三个热力图的颜色代表漩涡强度
    → 白色虚线=漏斗壁的位置
    → 圆圈内=水流区域, 圆圈外=固体壁
    → 颜色越红/黄=漩涡越强
```

**结论: 16bands微弱领先(Smooth 25:1 收窄比 → 但此处 Smooth 被严重缩小了)**

### 3. 漏斗STL_v2_Smooth100x对比.png — v2: 光滑放大100倍

```
上排: 时间序列(Smooth明显最高)
    → 蓝线(Smooth×100)全程领先
    → 绿线(8bands)和红线(16bands)在下方

下排: 终态截面
    → Smooth 的截面涡量明显更集中
    → 壁面轮廓更大(因为高瘦形状)

总结: Smooth×100 后长径比1.88, 收窄比29:1 → 大胜
```

### 4. 漏斗STL_v3_等体积对比.png — v3: 决定性实验

```
上排: 时间序列(三者几乎完全重叠!)
    → Smooth(蓝)/8bands(绿)/16bands(红)三条线分不开
    → 差值 <4%！

下排: 终态截面(几乎一样)
    → 三个涡量热力图看起来几乎相同
    → 这是最关键的发现！

总结: 等体积下形状不影响涡量(<4%).
     v1/v2 看似 Smooth 赢是因为体积差, 不是形状好.
```

---

## 为什么做了四轮？

| 轮次 | 问的问题 | 答案 |
|:---|:---|:---|
| v1 | 原始尺寸谁最强？ | 16bands 微弱领先 |
| v2 | 如果 Smooth 和 bands 一样大呢？ | Smooth 赢 |
| v3 | **如果体积也完全一样呢？** | **几乎平手 (<4%)** |

v1/v2 的结论是**错的**——不是因为计算错了，而是因为没控制体积。v3 才是公平对比。

---

## 工程的教训

> **想让水转得更快→做大漏斗(体积)，不是纠结加几条螺旋带。（仅限均匀轴向来水）**
> **水如果贴壁螺旋流下→螺旋带=导轨→每圈=一次角动量注入→16bands(多8圈导轨)推定更好。**

| 术语 | 大白话 |
|:---|:---|
| 拟涡能 (Enstrophy) | 全部漩涡的总能量 |
| H3 Sobolev 范数 | 漩涡"碎片化程度"，越高=越乱 |
| 长径比 | 高度÷直径。>1=高瘦, <1=矮胖 |
| 归一化 | 把不同大小的东西缩放到同一个尺寸再比 |
