# 漏斗 STL CFD 实验 — 小白说明

> **一句话**: 6 种犀牛建模漏斗放进 CFD 仿真。4 种是回转体（绕 Z 轴旋转），2 种 bands 是异形截面沿 Z 轴+对数螺线双轨扫掠（8bands=8圈, 16bands=16圈）。

## 逐图详解

### 1. 漏斗STL_几何对比.png — 三种漏斗长什么样

```
左图: 归一化形状对比 (同入口半径)
    → 蓝线=Smooth(高瘦, 长径比1.88)
    → 绿线=8bands(矮胖, ~0.7), 红线=16bands(矮胖, ~0.68)

右图: 原始尺寸对比
    → Smooth 只有 7m 高, bands 有 400m → 差 60 倍！
```

### 2. 漏斗STL_IBM_三方对比.png — v1: 原始尺寸归一化到同入口

```
上排: max|ω|/Enstrophy/H³ 时间序列
    → 红线(16bands)全程最高

下排: 终态涡量截面 (Z=0, 白虚线=壁面)
    → 颜色越红=漩涡越强
    → 16bands 微弱领先 (Smooth 被严重缩小了)
```

### 3. 漏斗STL_v2_Smooth100x对比.png — v2: Smooth 放大 100 倍

```
上排: 蓝线(Smooth)全程碾压 → 涡量/Enstrophy 都大幅领先
下排: Smooth 截面涡量最集中

结论: 这不是形状优势, 是体积优势
```

### 4. 漏斗STL_v3_等体积对比.png — v3: 等体积对决 🔥

```
上排: 三条线几乎完全重叠 (差 <4%)
下排: 三个截面看起来几乎一样

★ 决定性结论: 等体积下形状不影响涡量
  v1/v2 的"Smooth赢"是因为体积大, 不是形状好
```

### 5. 漏斗STL_v4_新漏斗对比.png — v4: 四种母线对比

```
加入超双曲/30°/45° 三种新 STL
同入口面积下对比四种形状:
  → 蓝=黄金比(对数螺线), 橙=超双曲(z=-1/x)
  → 绿=30°直锥, 红=45°直锥

上排: 时间序列 (30°最高, 黄金比最低)
下排: 终态截面
```

### 6. 漏斗STL_v5_两组等体积对比.png — v5: 逐对等体积 PK

```
左: 超双曲 vs 黄金比 (等体积)
    → 超双曲 > 黄金比 (+27%)

右: 30° vs 45° (等体积)
    → 30° > 45° (+8%)

验证: 超双曲和 30° 在等体积下保持优势
```

### 7. 漏斗STL_v6_全等体积对决.png — v6: 4 种全 V=100

```
左: max|ω| 时间序列 (4条线)
右: 涡量增长排名柱状图

排名: 30°(2.35x) > 超双曲(2.32x) > 45°(2.18x) > 黄金比(1.82x)
```

### 8. 漏斗STL_v7_同入口面积.png — v7: 4 种同入口

```
与 v6 一致排名, 验证实验可重复性
30° > 超双曲 > 45° > 黄金比
```

### 9. 漏斗STL_v8_bands对比.png — v8: 8bands vs 16bands

```
同入口面积下对比两个 bands:
  → 两根线几乎重叠 (差 <0.5%)

★ 轴对称模型把螺旋纹理平滑了 → 无法区分
  真实差异(双轨扫掠 8圈 vs 16圈)需全 3D CFD 验证
```

### 10. 漏斗STL_v9_6种等体积.png — v9: 完整排名

```
全部 6 种漏斗等体积 V=100:
  → 30°(2.35x) 夺冠
  → 16bands(2.31x) 第二
  → 黄金比(1.82x) 垫底

右上: 完整排名柱状图
```

### 11. 漏斗STL_最终双排名.png — 两种物理的排名

```
左图: 均匀轴向流 (CFD已验证) — 30°夺冠
右图: 贴壁螺旋流 (物理推断) — 16bands推定夺冠

核心信息: 两种流态排名完全反转
  → 轴对称模型数据不能套用到贴壁螺旋流
  → 最终答案需全 3D CFD 验证
```

### 12. 漏斗STL_表面积x涡量_综合排名.png — 省铜效率

```
铜材性价比 = 涡量 / 表面积
两种条件下各自排名

同入口: 超双曲效率最高 (SA最小但涡量几乎最高)
等体积: 45° 效率最高 (SA最小)
```

## 实验设计逻辑

```
v1: 原始尺寸 → 体积差掩盖一切
v2: Smooth 放大 → 验证体积效应
v3: 等体积 → 体积控制后形状差<4% ← 决定性
v4: 加入新STL → 四种母线形状对比
v5: 逐对PK → 验证等体积下的形状排序
v6: 全部等体积 → 完整排名
v7: 同入口验证 → 可重复性
v8: bands 对比 → 轴对称模型局限
v9: 6种完整体积 → 最终均匀流排名
```

## 两种流入模式的根本差异 ⚠️

| | 均匀轴向 (CFD已验证) | 贴壁螺旋 (物理推断) |
|:---|:---|:---|
| 水流路径 | 竖直落入, 充满截面 | 沿壁螺旋, 贴壁环流 |
| 中心区域 | 高速涡核 | 空气柱 + 杨柳辫 |
| 壁面作用 | 摩擦 = 损耗 | 摩擦 → 角动量 |
| 最优形状 | **30° 直锥** | **16bands** (推定) |
| CFD状态 | ✅ 9轮验证 | ❌ 需全3D |

---

| 术语 | 大白话 |
|:---|:---|
| 拟涡能 (Enstrophy) | 全部漩涡的总能量 |
| H³ Sobolev 范数 | 漩涡碎片化程度, 越高越乱 |
| 收窄比 | 入口半径 ÷ 出口半径 |
| 长径比 | 高度 ÷ 直径, >1=高瘦, <1=矮胖 |
| IBM | 沉浸边界法 — 把 STL 变成 CFD 里的墙 |
| SDF | 符号距离函数 — 判断水在墙内还是墙外 |
| 归一化 | 把不同大小的漏斗缩到同尺寸再比 |
