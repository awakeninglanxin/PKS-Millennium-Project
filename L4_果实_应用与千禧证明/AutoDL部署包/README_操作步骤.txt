========================================
AutoDL 一键部署 — PKS千禧难题
========================================

前提：你已登录 AutoDL，账户已充值

第1步：创建实例
  1. 打开 https://www.autodl.com/console/instance/create
  2. 配置如下：
     - 镜像：PyTorch → 2.1.0 + CUDA 12.1
     - GPU：RTX 4090 (24GB) 或 RTX 3090 (24GB)
     - 硬盘：默认 50GB
     - 计费：按量计费
  3. 点「立即创建」，等30秒变绿

第2步：上传代码
  1. 点实例右侧「JupyterLab」进入
  2. 左侧文件栏点↑上传按钮
  3. 选中这个目录里的所有 .py 文件上传
  4. 在 JupyterLab 里打开 Terminal

第3步：运行
  复制粘贴这一行：
  
  pip install cupy-cuda12x matplotlib -q && python BSD_v2_DerivativeKernel.py

  等3-5分钟，出结果！
