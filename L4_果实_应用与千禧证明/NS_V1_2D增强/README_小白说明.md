# NS_V1: 增强2D涡量分析 — 傻瓜说明

> **一句话**: 对比"有双曲锥"和"无双曲锥"时，水流中的漩涡强度有什么不同。

---

## 这张图在说什么？

想象你把一个漏斗放进水里。水从大口流进，从小口流出。水在流过锥喉的时候会转得更快（像滑冰运动员收手臂）。

这张图比较了**两种情况**：
- **无锥（Standard）**: 普通水流，自由流动
- **有锥（PKS Cone）**: 水里放了一个双曲形状的漏斗

---

## 每个子图的意思

```
左上: 涡量场 (Vorticity Field)
    → 红色=水在顺时针转, 蓝色=水在逆时针转
    → 两列对比: 左边无锥, 右边有锥
    → 有锥的那列漩涡更大更集中

右上: 涡量随时间变化
    → 横轴=时间, 纵轴=漩涡强度
    → 蓝线=无锥, 红线=有锥
    → 红线更高 = 有锥的漩涡始终更强

中左: 通量分析
    → 水流过锥喉的速度分布
    → 有锥时中部速度明显更快

中右: 谱能量 (Spectral Energy)
    → 横轴=尺度(大漩涡↔小漩涡), 纵轴=能量
    → 双对数坐标
    → 有锥时小尺度能量更丰富

左下: SGS尺度分析
    → 亚格子应力, 代表未解析的小漩涡有多活跃
    → 有锥时亚格子活动更剧烈

右下: 统计对比
    → 柱状图对比 max涡量 / 平均涡量 / 涡量放大倍数
    → PKS Cone 的柱子总是更高
```

---

## 关键结论

**有双曲锥时涡量约为无锥时的 1.2 倍**。V1 只是第一步验证——确认 PKS 锥确实能让水转得更有力。

### 如果你看不太懂某个术语

| 术语 | 大白话 |
|:---|:---|
| 涡量 (Vorticity) | 水旋转的强度。越大=转得越猛 |
| 谱能量 (Spectral Energy) | 不同大小漩涡各有多少能量 |
| 亚格子 (SGS) | 太小的漩涡看不清，用数学"猜"它们有多活跃 |
