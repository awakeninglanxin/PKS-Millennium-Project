## 这张图在说什么？

V7 在 2D 中发现 PKS 锥让退化更稳定。但 2D 和 3D 有一个**决定性的差异**：

**3D 有"涡量拉伸项" $(\omega\cdot\nabla)u$，2D 没有。**

就像滑冰运动员收手臂——在 2D 的纸上你收不了手臂，但在 3D 的真实世界里你可以。这就是为什么 V8 的结果和 V7 完全相反。
