## "漩涡炸掉"到底是什么物理现象？

数学上的"blowup" = 涡量在有限时间内变成无穷大。在现实物理中，漩涡不可能真的变到无穷大——它会被一个更早发生的物理过程"截停"：**空化（cavitation）**。

```
漩涡核心越转越快
    ↓
伯努利原理: 流速↑ → 压力↓
    ↓
压力跌破 2300 Pa (水的蒸汽压)
    ↓
20°C 的水自己沸腾 → 气泡产生 → 空化!
    ↓
气泡被周围高压挤爆
    ↓
坍塌瞬间: 局部 5000K + 冲击波 + OH·自由基
```

**PKS 锥的作用**：它不是让水分子内爆，而是**让漩涡更快到达空化压力的那个点**。

## 具体现实应用价值

| 应用 | 原理 | PKS锥优势 |
|:---|:---|:---|
| **水处理/杀菌** | 空化气泡坍塌→冲击波撕碎细菌细胞壁 | 锥让空化在更低流速下发生(节能) |
| **乳化/混合** | 空化产生剧烈微射流→油水强制混合 | 更均匀, 更节能 |
| **声化学** | 空化产生OH·自由基→降解有机污染物 | 不需要化学药剂 |
| **海水淡化** | 空化辅助蒸发/膜过滤 | 降低能耗 |
| **水力发电** | 涡量放大→更多动能可提取 | 小型化微水电 |

**本质**: PKS 双曲锥 = 空化的**几何催化剂**。普通锥需要流速 ~14 m/s 才能触发空化；PKS 锥因为双曲曲率诱导的额外涡量拉伸，可能在更低流速下就触发——**同样的泵，更早产生空化效应。**

## 和 V3 漏斗实验的关系

V3 等体积实验证明：**体积是涡量放大的第一决定因素（>96%），形状是次要的（<4%）。**

这意味着实际造设备时：
- 先做大体积 → 涡量放大(免费)
- 再用 PKS 双曲锥形状 → 额外 ~4% 曲率贡献(在已有体积基础上的"锦上添花")
- 等体积下选光滑墙壁 → 制造最简单
