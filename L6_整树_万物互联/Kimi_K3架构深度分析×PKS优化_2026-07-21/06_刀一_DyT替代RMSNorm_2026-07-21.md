# 刀一：DyT 替代 RMSNorm — 训练时扔掉逐 token 统计

> **动什么**: Layer Normalization (2016) → RMSNorm (2019) → **DyT (Dynamic Tanh, 2025)**  
> **复杂度收益**: 每层省两次逐 token 的 mean/var 计算 → 深模型累积可观  
> **风险**: 🟢 极低 — 已有独立论文验证，直接替换

---

## 一、Norm 层的进化史

```
2016  LayerNorm: y = (x-μ)/σ * γ + β     ← 逐 token 算 mean 和 var
2019  RMSNorm:   y = x / RMS(x) * γ       ← 只算 RMS，省了 mean
2025  DyT:       y = tanh(α·x) * γ        ← 完全不用统计量
```

**每层两次 Norm** (PreNorm + PostNorm 或 PreNorm × 2)。K3 估计 90+ 层 → 180 次逐 token 统计 / 每个 token。在 1M 上下文的推理中，这 180 次 `mean()` + `std()` 的累积开销是**数量可观的常数因子**，不是复杂度瓶颈但是实打实的墙。

## 二、DyT 的数学原理

**核心公式**:
```
RMSNorm(x) = x / sqrt(mean(x²) + ε) * γ
DyT(x)     = tanh(α · x) * γ
```

**为什么可以替换**: RMSNorm 的本质是对激活值做一个**数据依赖的缩放**（除以 RMS）。RMS 在不同 token/层之间波动——但波动的范围在 tanh 的饱和区内其实等价于一个**可学习的常数缩放**。DyT 用一个 α 参数学会了这个缩放，不再需要逐 token 计算 RMS。

**数学等价性**:
```
RMS(x) 的作用 = 把 x 压到统一尺度 → 防止梯度爆炸/消失
tanh(αx) 的作用 = 用门控函数压到统一尺度 → 同样防止梯度问题
```

核心洞见是：**RMS 的"逐 token 自适应"在训练后期几乎恒定了**——大部分 token 的 RMS 收敛到一个窄带内。此时 RMSNorm 和 DyT 的行为等价，但 RMSNorm 还在白白计算统计量。

## 三、K3 上的注入方案

```
当前 K3 (推测): PreNorm RMSNorm + PostNorm RMSNorm (每层 2 次)
优化 K4:        DyT 替代全部 RMSNorm
                + 训练初期混用 (warmup 阶段保留 RMSNorm, 稳定后切 DyT)
                + α 从 RMS 统计量初始化 (使切换时 loss 不跳)
```

**初始化技巧**: 不是随机初始化 α——而是在训练的最后 1000 步收集每层的 RMS 均值，用 `α = 1/RMS_mean` 初始化 DyT 的 α。这样切换到 DyT 时 loss 零跳变。

**预期收益**:
| 维度 | RMSNorm | DyT |
|:---|:--:|:--:|
| 每 token 计算 | mean + sqrt | tanh (一次数学函数) |
| batch 依赖性 | 需要大 batch 稳定统计 | 无 |
| 训练→推理一致性 | 推理可吸收进权重 | 同样可吸收 |
| 深模型累积 (90 层) | ~1-2% 总 FLOPs | ~0.1% |

数值不大但**白捡的**——类似 FlashAttention 把 attention 从 O(n²) 内存压到 O(n)，单看一次操作省的不多，但 90 层 × 1M token 乘起来就是显著差异。

## 四、与 PKS 方法论的关系

DyT 是 **"算数学 > 学数学"** 的教科书级案例:

- RMSNorm 需要**学**（统计量是数据依赖的）
- DyT 直接**算**（tanh 是闭式函数，α 收敛后是常数）
- α 还可以从 RMS 历史值直接推导——不需要另训

这恰好对应我们 7/20 方法论文档中的 FFT/Kalman/CfC 那些案例——用数学结构替代逐点统计。

## 五、实施路径

```
第 1 步: K3 checkpoint 上跑 1000 步, 收集每层 RMS 均值
第 2 步: 用 α = 1/RMS_mean 初始化 DyT
第 3 步: 替换所有 RMSNorm → DyT, 继续训练
第 4 步: 验证 loss 不跳、下游指标不变
第 5 步: 推理时直接 tanh(αx) 走, 不用算 RMS
```

**如果 K3 开源权重 7/27 放出，这是第一个值得做的实验**——不需要重新训练，只需在 checkpoint 上做替换 + 验证。
