## 一、KDA 注意力 O(n) → O(log n)？幻方稀疏化注意力

### 当前状态
KDA 把 QK^T 从 O(n²) 压到 O(n) (线性注意力)，但仍有 n 次 Delta 函数计算。

### PKS 注入方案: 幻方行列正交 → 稀疏化 Delta 计算

**数学基础**: 5 阶幻方 24 条线 (5 行+5 列+2 对角线+12 折断对角线) 和 = 65。行列正交意味着某些线独立于其他线——可以跳过。

**优化思路**:
```
当前 KDA: 对每个 token 计算 Delta(Q_i, K_j) → O(n)
优化 KDA-PKS:
  1. 用幻方行列约束筛选"关键 token 对" → 只对行/列/对角线上的注意力做精确计算
  2. 非关键 token 对 → 用幻和 65 的一步近似 (O(1))
  3. 残差补偿只对"折断对角线"上的 token 做
  
复杂度: O(n) → O(n / 5) (每 step 只处理 5 行/5 列/对角线 token)
       ↓ 幻方排列 → 每层循环取不同的行列集合
       → 等效 O(log n) 如果幻方阶数随层增加
```

**为什么可行**: KDA 的 Delta 压缩本质是"稀疏化"——幻方的行列结构提供了一种**数学可证的非冗余覆盖方案**。5 阶幻方恰好保证 25 个 token 可以通过 10 条线 (5 行+5 列) 全覆盖，且折断对角线 (12 条) 做残差补偿。

| 指标 | 标准 KDA | KDA-PKS (5阶幻方) |
|:---|:--:|:--:|
| 每 token 计算 | O(n) | O(n/5) |
| 覆盖完备性 | Delta 近似 | 行列正交全覆盖 |
| 残差信息 | 随机丢失 | 折断对角线精确捕获 |
| 额外参数 | 0 | 0 (幻方是静态常量) |

### GPU 验证可行性: 🟢 可直接跑
只需在 KDA 的 Delta 函数计算前插一个幻方筛选层——对输入 token 序列按 5×5 幻方做 block 分组，只计算行列上的注意力。改一行代码。

## 二、AttnRes → Fantasy AttnRes (幻方因子化残差)

### 当前状态
AttnRes 对 depth 轴做 softmax 加权，本质是"每层独立学权重"。

### PKS 注入方案: 幻方因子化 = 三个正交分量的 depth attention

**灵感来源**: Super k-Shape 幻方因子化 (ARI +74%) 证明了 `M = 5·A + B + 1` 可以分解时间序列为趋势/季节/残差。

**优化思路**:
```
当前 AttnRes: x_layer = Σ softmax(w_i) · x_i (对所有历史层)
Fantasy AttnRes:
  1. 把每层表示分解为三个正交分量:
     - Trend 分量 (趋势轴): 长程记忆 (5 行为一组)
     - Season 分量 (季节轴): 中程记忆 (5 列为一组)  
     - Residual 分量 (残差轴): 短程局部
  2. 三个分量各自做轻量 AttnRes (每个只用 5 个权重而非全层数)
  3. 三分量结果按幻方矩阵 M = 5·A + B 重组

参数: 15 个权重 (vs 标准 L 个), 压缩比 L/15
```

| 版本 | 层权重数 | Block 复杂度 |
|:---|:--:|:--:|
| Full AttnRes | L² | O(L²·d) |
| Block AttnRes (陈+张) | L·b (b=块数) | O(Nd) |
| **Fantasy AttnRes** | **15** | **O(5d)** |

## 三、MoE Routing → Farey 分数路由

### 当前状态
K3 用 Quantile Balancing 替代 aux-loss——根据 router-score 分位数直接派专家。

### PKS 注入方案: Farey 序列 → 互素专家分配

**数学基础**: Farey 分数 a/b (gcd(a,b)=1) 天然保证相位永不重叠。

**优化思路**:
```
当前 Quantile Balancing: 按 score 分位数均匀切 → 可能多个高互相关的专家同时激活
Farey Routing:
  1. 用 Farey 序列预先排列 896 个专家 → 互素分数的专家天然"去相关"
  2. Top-K 选择时加入"互素距离"惩罚:
     - 两个专家的 Farey 分数差↑ → 激活重叠↓ → 权重可加
     - 两个专家的 Farey 分数差↓ → 可能互相关 → 权重减半
  3. 公式: router_score_mod = router_score × (1-cost · |FareyDist(i,j)|)
```

**与现有 MoE routing 对比**:

| 路由方法 | 平衡机制 | 专家多样性 | 超参 |
|:---|:---|:--:|:--:|
| Aux-loss (传统) | loss 约束 | 无保证 | 需调系数 |
| Bias 动态 (DeepSeek-V3) | bias 调 | 无保证 | γ 步长 |
| Quantile (K3) | 分位数切 | 无保证 | 0 |
| **Farey Routing** | 分位数 + Farey 去重 | **互素保证** | cost 系数 |

> 📌 **【来自 3600 幻方算法的修订：互素用法的合法性边界】**
> 幻方里「步长 2 与阶 5 互素 → 平移遍历所有位置（环上群作用）」是严格成立的，因为它解决的是
> **位置覆盖**问题。Farey 插入同理——用互素分数决定 Mamba-2 插在哪几层（间距互素 → 每层看到
> 不同历史窗口），这是合法的位置覆盖，与幻方步长**同构**。
>
> ❌ **但 `05` 已诚实指出**：把 Farey τ 当成「多尺度时间常数天然最优」是**未证的**——互素只保证
> **不重叠**，不保证**分布最优**。本条 Farey Routing 属于「位置覆盖」合法用法；而下方 #5 的
> Farey τ「时间常数分布最优」属于**未证**用法，两者必须**分家**（见 #5 修订框）。

### 🟡 升级方向（原创猜想，来自幻方等价类思维）：144 专家原型 × 变换路由

幻方 3600 = 144 基方 × 25 循环移位，本质是「少数本质不同的对象通过对称群作用膨胀成全集」。
K3 的 896 专家目前是**扁平**的——所有专家同层被 router 平等看待，Farey 去相关仍是扁平操作。

**等价类压缩路由（猜想）**：
```
1. 用聚类/谱方法从 896 专家中找出 ~144 个「本质专家原型」
   （类比 144 基方；原型 = 功能正交、不可互约的专家族）
2. 其余 752 专家视为原型的「变换副本」
   （类比循环移位；如同一原型的不同量化版本 / 不同稀疏度版本）
3. 路由先在 144 原型上跑闭式匹配（刀二费用流只需在 144 上跑，O(144) ≪ O(896)）
4. 匹配到原型后，再派发到其具体副本（按激活预算选副本）
```

**价值**：把「互素去相关」升级为「等价类压缩」——和幻方从 3600 压到 144 是同一个操作；
闭式路由的计算量进一步降一个数量级。

**风险与证伪条件（必须配套，否则只是猜想）**：
- 若专家是**连续流形**而非离散轨道 → 聚类不出清晰 144 原型（轮廓系数 < 0.3）→ 本猜想证伪，回退扁平路由。
- 若 144 原型路由的下游指标（专精度 / 困惑度）**≤** 直接 896 扁平路由 → 等价类压缩在 MoE 上不成立，证伪。
- 🔬 最小实验：在 K3 规模的 proxy（如 32 专家小 MoE）上先验证「原型聚类 + 副本派发」是否
  不损精度；通过后再放大到 896。详见 `09` 第四节 + 配套 `10_144专家原型路由_可实验设计`。

## 四、B-smooth X*(k) 指数定律 → K3 训练稳定性预测

### PKS 注入
锚点 2 的 `X*(k) ≈ 1500 × 5.5^k` ——给定 MoE 专家数 k，能训练不崩的最大 token 数可预测。

**应用于 K3**:
- K3 896 专家 → 等效 k ≈ 10 (素数基底) → X*(10) ≈ 45M token
- K3 训练 2.8T token 远超 X*(10) → 必须用 Per-Head Muon + QK-Clip 防崩
- **预言**: 如果 K4 继续加专家 (k=12, X*(12)≈600M), 训练稳定边界会上移但不会消失

> ⚠️ 这是**类比**非精确对应。B-smooth 碰撞覆盖的数学结构和 MoE 训练稳定性**共享"有限基底→渐近失效"的动力学**，但不是数学等价。

## 五、液态 NN τ 多尺度 → KDA 时间常数分层

### PKS 注入
我们 7/20 的液态 NN × 幻方分析 (映射 1): LTC 时间常数 τ ←→ Farey 互素分相。

**应用于 KDA**:
```
当前 KDA: 3 个 KDA block 用相同 Delta 函数 → 所有层"看同一时间尺度"
优化: 每个 KDA block 用不同的 τ (从 Farey 序列初始化)
  - Block 0: τ = Farey(1/2) → 快速衰减 → 抓近期依赖
  - Block 1: τ = Farey(1/3) → 中速衰减 → 抓中期模式
  - Block 2: τ = Farey(1/5) → 慢速衰减 → 抓长程记忆

效果: 三档 τ 互素 → 时间常数永不重叠 → 多尺度信息自动分层
      → 对应 M 集思维 4 "多尺度时间常数"
```

**这与 K3 已有架构的关系**: Gated MLA block 本身就做"精确回看"——把 KDA 的 τ 也做成 Farey 多尺度，可以让 KDA 侧更加结构化。不需要额外的训练参数，只需在 Delta 函数的衰减参数上做 Farey 初始化。

> ⚠️ **【来自 3600 幻方算法的修订：Farey 的两种用法必须分家】**
> - ✅ **位置覆盖（合法，同构幻方步长）**：Farey 分数决定**插入哪几层 / 哪个位置**——互素保证插入位
>   间距不重复（无冗余覆盖）。这是 `08` 的 Farey 插入层位置策略，站得住。
> - ❌ **时间常数分布（未证）**：把 Farey τ 当「多尺度时间常数天然最优」——互素只保证 τ **不重叠**，
>   **不保证** τ 的**分布最优**（如均匀 τ 间隔可能长程衰减更平滑）。`05` 已诚实点出，此处用环上
>   轨道闭合的语言说透：互素 = 无冗余，≠ 最优化。
> - 🔬 **证伪条件**：若 Farey τ 初始化 ≤ 均匀 τ 初始化（在相同训练步数下困惑度），则「Farey 时间分布
>   最优」假设证伪，但 Farey **位置覆盖**用法仍独立有效（变量隔离）。
