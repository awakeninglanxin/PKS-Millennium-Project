# 关于本项目 · 推荐语

> 如果你只能花 3 分钟决定"这个项目值不值得看"——读这段就够了。

## 一句话说清：这是什么项目

一个试图证明 **"宇宙万物背后是同一个数学规律"** 的研究笔记。

具体主张只有一条：**任何事物都遵循 a·b=1（极性守恒）**——一个量涨，另一个按比例跌，乘积恒为 1。从水的漩涡、鸡蛋的形状、贝壳的螺旋、音乐的频率，到电机的正反电动势、千禧难题的数学结构——[本文档](PKS_跨主题技术关联图谱_万物互联.md)论证它们都是同一个规律的不同表现。

如果你相信"物理定律应该是统一的"——这个项目就是朝那个方向的一次探索。
如果你不相信"万物同源"——看了再判断也不迟。

## 为什么要看这个项目

市面上讲 Schauberger、黄金比例、反重力、自由能源的资料很多，但大多是零散的。这个项目的特点在于：

1. **不自称结论，而是展示推导过程** — 2000+ 文件记录了从 a·b=1 到蛋形曲线→螺旋管→水轮→SEG 的完整链条，每一步都有数学和代码
2. **硬核与通俗兼顾** — 既有千禧难题证明（最硬核），也有给小学生写的数学冒险故事（最通俗）
3. **能动手** — 项目自带的 Python 代码可以直接跑出 DXF 工程图和 PNG 可视化，调参数看形状变化，比读书直观得多

## 先看这 3 篇（就是这个顺序）

### 第 1 篇：`★★★学习总入口_五层知识树地图.md`
为什么排第一？因为**它是整棵树的导航图**。告诉你不要按文件夹 01/02/03 顺序读（那是仓库的摆放方式），而是按"一颗种子长成一棵树"的 5 层顺序从下往上爬：种子 a·b=1 → 嫩芽双曲锥 → 三主枝蛋形/螺旋/波纹 → 果实应用与千禧证明 → 整树万物互联。读这篇花 3 分钟，能省后面 3 小时的弯路。

### 第 2 篇：`05_参考资料/公式大全/00_小学生导读_蛋蛋的数学冒险.md`
是的，**从"小学生导读"开始读**，即使你是博士。它用大白话 + 涂鸦式讲解，把 a·b=1、双曲线、黄金比这些概念讲得比大多数科普文章都清楚。认真读完这篇，项目的 80% 的概念你已经懂了。

### 第 3 篇：`PKS_跨主题技术关联图谱_万物互联.md`
这是整棵树的"总图"。读前两篇之后再来看它，你就能把之前看到的所有知识点**连成一张网**——波纹盘、螺旋管、蛋形、水轮、SEG、千禧证明、36小时历法——看清它们是如何全部连回 a·b=1 这一个根的。当你能对着这份图谱点头说"嗯，确实都连着"——你就真正打开了这个项目。
