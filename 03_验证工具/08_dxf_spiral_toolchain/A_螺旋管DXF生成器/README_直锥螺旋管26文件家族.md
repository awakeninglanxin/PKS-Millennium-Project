### 3.2 核心参数调参指南

所有参数的入口在类构造函数中（`__init__` 方法）：

```python
def __init__(self, spline_data_path, 
             t_min=2*np.pi, t_max=7*np.pi,   # 螺旋线t参数范围
             d=15,                             # 螺旋线半径
             z_v=5,                            # 轴向速率
             amp=1.5,                          # 断面首端大小
             sign_spiral_dir=1,                # 旋转方向
             sign=1,                           # 断面方向
             twist=1):                         # 扭转方向
```

| 参数 | 默认值 | 含义 | 调参效果 |
|:---|:---:|:---|:---|
| `t_min, t_max` | 2π, 7π | 螺旋线t范围 | t_max↑→更多圈数; t_max↓→更短锥 |
| `d` | 15 | 螺旋线基础半径 | ↑→更宽的锥底 |
| `z_v` | 5 | 轴向推进速率 | ↑→更高的锥; ↓→更扁的盘 |
| `amp` | 1.5 | 断面首端大小 | ↑→更粗的管 |
| `sign_spiral_dir` | ±1 | 旋转方向 | `1`=逆时针; `-1`=顺时针（俯视） |
| `sign` | ±1 | 断面初始方向 | `-1`→断面旋转180°→管看似旋转180° |
| `twist` | ±1 | 断面扭转方向 | `1`=顺时针扭转; `-1`=逆时针扭转 |

**断面首尾大小渐变**：
```python
self.a = np.linspace(self.amp（首）, self.amp / 2.5（尾）, self.num_t)
```
改成 `self.a = np.linspace(2.0, 0.3, self.num_t)` → 首粗尾细的锥管。

### 3.3 断面镜像（左右翻转控制）

管道断面可关于 **X轴或Y轴镜像**（影响最终管道的"扭向感"）：

| 要镜像哪个轴 | 改哪行代码 | 操作 |
|:---|:---|:---|
| 关于 **Y轴镜像** | `N_term1 = self.sign * y(j) * N_unit[i]` | 删除 `self.sign *`，只留 `y(j) * N_unit[i]` |
| 关于 **X轴镜像** | `B_term1 = self.sign * x(j) * B_unit[i]` | 删除 `self.sign *`，只留 `x(j) * B_unit[i]` |

### 3.4 散点密度与整除规则

| 参数 | 位置 | 规则 | 为什么 |
|:---|:---|:---|:---|
| `num_t` | 构造函数 | 默认 **350**（可被7整除） | 确保螺旋线上7个等距位置各有一个断面 |
| 整除判断 | 第201行 | `if i == 0 or (i + 1) % 7 == 0:` | 首尾必有一个断面 + 每隔(num_t/7)=50个散点一个断面 |

改 `num_t` 必须保持被7整除（如 70, 140, 210, 280, 350, 420…）。

### 3.5 扭转剧烈程度控制

扭转计算（`update_psi` 方法）：
```python
self.psi = np.linspace(0, self._twist * self.t_length / 2, self.num_t)
```

| 除数 | 扭转程度 | 视觉效果 |
|:---|:---|:---|
| `/2` | **默认** — 扭转度=螺旋周期的一半 | 平滑柔和 |
| `/1` | 扭转为整个螺旋周期 | 较剧烈 |
| `*2` | **极度扭转** — 4倍于默认 | 🔥 火龙卷风效果 |

螺旋周期：`t_length = t_max - t_min = 7π - 2π = 5π`，默认扭转 = 5π/2 = 2.5π。

### 3.6 螺旋线公式 — 换公式指南

这是整个程序的核心——**螺旋线的(x,y,z)坐标方程**：

```python
def calculate_spiral_coords(self):
    x = np.exp(self.b * self.t_spiral) * self.d * np.sin(self.t_spiral) * self.r
    y = np.exp(self.b * self.t_spiral) * self.d * np.cos(self.t_spiral)
    z = np.exp(self.b * self.t_spiral) * (self.t_spiral - self.t_min) * self.z_v
    return x, y, z
```

**替换方法**：
1. 复制此函数到 ChatGPT/DeepSeek
2. 描述你想要的新螺旋形状（如"等距螺旋"、"阿基米德螺旋"、"对数螺旋改param"）
3. 让AI给你 `np.xxx` 格式的新公式
4. 替换 `x, y, z` 三行即可

> 注意：必须保持返回 `(x, y, z)` 三个 numpy 数组，长度一致。
