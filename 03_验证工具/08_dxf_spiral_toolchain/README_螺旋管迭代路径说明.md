## 四、核心技术架构

### SpiralSweepGenerator 类

所有螺旋管文件共享的基类（在不同版本中逐步演化）：

```python
class SpiralSweepGenerator:
    def __init__(self, t_min=2π, t_max=7π, d=15, z_v=5, 
                 amp=1.5, sign_spiral_dir=1, sign=1, twist=1):
        ...
    
    # 核心公式（指数螺旋）:
    #   x = exp(b·t) · d · sin(t) · r
    #   y = exp(b·t) · d · cos(t)
    #   z = exp(b·t) · (t-t_min) · z_v
```

### 关键可调参数

| 参数 | 含义 | 典型范围 |
|------|------|---------|
| b | 螺距增长率 | 0.05~0.15 |
| t_min/t_max | 参数范围 | 2π~12π |
| d | 管径 | 10~30 |
| z_v | 锥度 | 0~10 |
| amp | 断面振幅 | 1~3 |
| twist | 扭转方向 | ±1 |
| psi | 扭转角度 | 0~2π |

### 技术栈演化

| 阶段 | 核心技术 | 新能力 |
|------|---------|--------|
| Phase 1 | numpy + ezdxf + scipy | 基础螺旋管生成 |
| Phase 2 | scipy.linalg | 多断面双轨扫掠 |
| Phase 3 | scipy.splprep/splev | 样条平滑 |
| Phase 4 | numba + multiprocessing + mpmath | JIT 加速 + 高精度 |

## 七、文件命名约定

```
[产品类型] + [几何特征] + [扫掠方式] + [参数特征] + [特殊标记]

示例:
  舒式螺旋斜流泵 + 多断面 - dxf 样条线 - 阴阳-t平方 - 顶上不干涉 - 去尾
  ├── 产品 ──┤ ├─扫掠─┤ ├────── 参数特征 ──────┤ ├─特殊标记─┤

特殊标记含义:
  简化    = 代码精简版，去除冗余分支
  优化    = 曲线平滑度或参数优化
  最终版  = NURBS 阶段成熟版
  去尾    = 去除尾端多余螺旋
  丝滑曲线 = 改用 scipy 样条替代线性插值
```
