## 一、万花尺原理（一图看懂）

```
传统万花尺（塑料玩具）
  ↓ 数字化
Python + ezdxf: 读取任意DXF闭合曲线 → 齿轮滚动 → 笔尖画花
  ↓ 输出
.DXF 花瓣曲线 → 导入 Rhino → 3D几何加工
```

不是简单的圆形齿轮——**任何闭合的 DXF 样条线都可以作为"轨道"**。

## 三、④ → ③ → ② → ① 迭代详解

### ③ 单条曲线生成（入门版）

```
用户操作:
  ① 选择齿轮形状 → 读入 DXF 闭合曲线
  ② 移动滑块调整参数: 齿轮半径 / 笔距 / 颜色
  ③ 预览 → 满意 → 导出 .DXF

核心类:
  DXFCurveReader    — 读取 DXF 中 SPLINE → 提取控制点
  SpirographEngine  — 齿轮滚动模拟 → 笔尖轨迹计算
  SpirographGUI     — tkinter GUI (滑块+Canvas预览)
```

### ② 多条曲线生成版（升级）

相比③新增：
- `self.original_curves` — 存储原始曲线的副本，缩放后可恢复
- `多曲线叠加` — 多个齿轮同时在不同轨道上滚动
- `self._length_cache` — 曲线长度缓存加速重复计算

### ① 完整版（终极）

相比②新增的**3大功能**：

| 新增项 | 技术 | 效果 |
|:---|:---|:---|
| `@jit, @njit` | **Numba JIT 编译** | 轨迹计算加速 10~100× |
| `auto_save: True` | 自动存档历史 | 每次生成自动保存PNG |
| `saturation/value/pen_size` | **高级渲染** | 笔触粗细/透明度/饱和度可调 |
| `tempfile + shutil` | 临时文件管理 | 大图不撑爆内存 |

### ④ 尖峰点平滑（后处理）

独立工具——不画花，只修复画的太尖的花：

```python
class SpirographGenerator:
    load_dxf()              → 加载已有DXF
    extract_contour_curves() → 提取SPLINE → 点列表
    smooth_peaks()          → 检测角度突变点 → 局部样条平滑重采样
    save_dxf()              → 输出修复后的DXF
```

## 六、与 PKS 项目的类比（非直接对应）

| 万花尺概念 | PKS 类比 | 说明 |
|:---|:---|:---|
| 任意闭合曲线作为轨道 | SEG 滚柱在环内的外摆线轨道 | 都是闭合轨道上的周期性运动 |
| 齿轮半径决定花瓣数 | SEG 滚柱数 12/22/32 决定磁极图案密度 | 都是半径→周期的几何关系 |
| 笔距（pen distance） | Bv 磁场强度（离轨道中心越远，磁场越强） | 都是"离中心距离→强度"的映射 |
| 尖峰点 = 角度突变 | 磁饱和点 → H > H_S → COP 跳跃 | 都是临界状态的类比 |
| 多条曲线叠加 | SEG 三层环的磁场复合 | 都是多层叠加的类比 |
