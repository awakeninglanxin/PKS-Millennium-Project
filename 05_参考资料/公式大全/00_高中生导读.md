## 一、超双曲锥的参数化

### 1.1 从 $xy = 1$ 到空间锥面

反比例函数 $xy = 1$ 在平面直角坐标系中是双曲线。将其绕 z 轴旋转一圈，得到**超双曲锥面**（hyperbolic cone）：

锥面上任一点可表示为螺旋线上的一点在绕 z 轴旋转 $2\pi$ 时扫过的轨迹：

$$P(n, \theta) = \left(\frac{1}{n}\cos\theta,\; \frac{1}{n}\sin\theta,\; n\right)$$

其中 $n > 0$ 是"谐波编号"（沿 z 轴），$\theta \in [0, 2\pi)$ 是旋转角。

**关键性质**：$n$ 同时充当两个角色——

| 角色 | 含义 | 对应 |
|------|------|------|
| z 坐标 | 锥面上的高度 | $z = n$ |
| 到 z 轴的距离 | $r = \frac{1}{n}$ | 离轴越远，高度越低 |

正因为 $r = \frac{1}{n}$，这个锥面是**反比例螺旋**的旋转体——这就是 Schauberger 超双曲锥的核心结构。

### 1.2 斜切操作：截面方程

选择一个倾斜平面切割锥面：

$$z = z_0 - \bar{y}\sin\alpha$$

其中：
- $z_0$：截面中心在 z 轴上的截距
- $\alpha$：截面与水平面的夹角
- $\bar{y}$：截面纵坐标

联立锥面方程 $n = z$ 和截面方程：

$$n = z_0 - \bar{y}\sin\alpha$$

代入 $r = \frac{1}{n}$，投影到截面：

$$\bar{x}^2 = \frac{1}{(z_0 - \bar{y}\sin\alpha)^2} - (\bar{y}\cos\alpha)^2$$

这就是蛋形曲线的**显式形式**。

### 1.3 参数决定方式

两个交点在锥面上的 n 值（$z_1$ 与 $z_2$）决定了一切：

$$z_0 = \frac{z_1^2 + z_2^2}{z_1 + z_2}$$

$$\alpha = \arctan\left(\frac{z_1z_2(z_2 - z_1)}{z_1 + z_2}\right)$$

**这就好比**：给定圆的方程只需要半径 r 一个参数，蛋形只需要 $z_1$ 和 $z_2$ 两个参数。固定 $z_1 = 1$，则蛋形方程只有一个自由度 $k_E = z_2$。

## 二、六种蛋形参数方程与曲率分析

### 2.1 算法家族

同一底层几何 $xy=1$ 斜切产出 6 种代数表达，各有适用场景：

| 算法 | 类型 | 公式核心 | 适用场景 |
|------|------|---------|---------|
| **A1** | 显式 | $\bar{x} = \pm\sqrt{\frac{1}{(z_0-\bar{y}\sin\alpha)^2} - (\bar{y}\cos\alpha)^2}$ | 快速原型 |
| **A2** | 极坐标参数 | $r = \frac{z_0\pm\sqrt{z_0^2-4\cos\phi\sin\alpha}}{2\cos\phi\sin\alpha}$ | 曲率分析 |
| **A3** | t-参数 | $x = \sqrt{\frac{1}{(z_0+t\sin\alpha)^2} - (t\cos\alpha)^2},\; y=-t$ | 边界精确计算 |
| **B** | Rhino 参数 | $x(t) = \frac{2\sin t}{b+\sqrt{b^2-4k\cos t}}$ | 实时渲染、CAD |
| **C** | r⁻ 分支 | 与 A2 同源，仅用负分支 | 科研插图 |
| **D** | 八度族 | 同 B，递推 $k=\frac{4^n}{6},\;b=\frac{5\cdot2^n}{6}$ | 蛋形族参数扫描 |

### 2.2 曲率分析：为什么蛋形流体阻力最小？

给定蛋形参数方程 $(x(t), y(t))$，曲率公式：

$$\kappa(t) = \frac{|x'y'' - y'x''|}{(x'^2 + y'^2)^{3/2}}$$

对八度蛋（$k_E=2.0$）计算曲率分布：

```text
位置 t     曲率 κ    特点
─────────────────────────────
蛋尖   →    高（尖锐）  流体在此处必须分离——但没有！
蛋中   →    低（平坦）  边界层平滑发展
蛋圆端 →    中（圆润）  尾涡推迟分离
```

**曲率单调递减**（从尖头到圆头）是蛋形区别于椭圆的根本特征。椭圆在长轴两端曲率相同，而蛋形曲率**不对称**——正是这种不对称让边界层能"粘"在表面上更久，延迟了流动分离。

> 🔑 **核心物理直觉**：流动分离发生在曲率的陡变处。蛋形的曲率变化率（$\frac{d\kappa}{ds}$）在整条曲线上连续且单调——不给流体任何"分离借口"。

### 3.1 参数空间

将 $z_1=1$ 固定，$k_E = z_2$ 从 1 到 ∞ 扫描，得到 $\alpha$-$\kappa$ 相图：

| 1.414 | 16.1° | 圆蛋 | 离散 | 鸽蛋 |
| **1.618** | **25.1°** | **黄金蛋** | **离散→连续过渡** | **向日葵、海螺** |

### 3.2 $k_E \approx 1.937$：黄金极限的特殊性

当 $k_E = \varphi^{-1} + \varepsilon$（其中 $\varphi = \frac{1+\sqrt{5}}{2}$），蛋形曲线的特征值谱呈现**连续谱聚集**（spectral crowding）现象——无限多本征值逼近同一极限值。这在数学物理学中对应 **KAM 稳定性**（Kolmogorov-Arnold-Moser 理论）：在正则区域与混沌区域的边界上，连续谱与离散谱交织。

这正是八度蛋（$k_E=2.0$，纯离散）做"最优涡旋截体"而黄金蛋（$k_E\approx1.937$，连续聚集）做"普适变形核"的几何根源。

## 四、射影几何的形式化框架

### 4.1 齐次坐标

在射影几何中，平面上的点 $(x, y)$ 被提升为三维齐次坐标 $(X:Y:Z)$，其中 $x = X/Z$，$y = Y/Z$（$Z \neq 0$）。

超双曲锥方程 $xy=1$ 的齐次化：

$$X \cdot Y = Z^2$$

当 $Z=1$（仿射卡）：$xy=1$，经典双曲线。

当 $Z=0$（无穷远）：$XY=0$，即 $X=0$ 或 $Y=0$——**两条互相垂直的无穷远直线**。这就是双曲线有两条渐近线（坐标轴）的射影几何本质——它们在无穷远处"合拢"。

### 4.2 射影变换群

射影变换 $\mathbb{P}^2 \to \mathbb{P}^2$ 由可逆 $3\times3$ 矩阵表示（差一个标量因子）：

$$(X', Y', Z')^T = T \cdot (X, Y, Z)^T, \quad T = (a_{ij})_{3\times 3}$$

**核心结论**：蛋形曲线 $C_{egg}$ 与圆 $C_{circle}$ 在射影变换 $T$ 下等价：

$$C_{egg} = T(C_{circle})$$

### 4.3 交比不变量

共线四点 A, B, C, D 的交比：

$$(A,B;C,D) = \frac{\overline{AC}}{\overline{BC}} : \frac{\overline{AD}}{\overline{BD}}$$

在 Schauberger 谐波序列上取 $P(1)$、$P(\frac{1}{2})$、$P(\frac{1}{3})$、$P(\frac{1}{4})$：

$$(P_1,P_2;P_3,P_4) = \frac{4}{3}$$

这正是纯四度音程的频率比。**射影几何给了我们：音乐 = 交比不变量的谐波级联。**

### 4.4 极点-极线对偶


在射影几何中，每一点有一条极线，每一条线有一个极点。Schauberger 极性原理 $p \cdot q = const$ 就是这个定理的物理实例：

| 射影对偶 | Schauberger 对偶 | NS 方程对偶 |
|---------|----------------|-----------|
| 点 ↔ 极线 | 弦长 ↔ 频率 | 速度 $u$ ↔ 压力 $p$ |
| 共线点 ↔ 共点线 | x 轴 ↔ y 轴 | 涡量 $\omega$ ↔ 流函数 $\psi$ |
| 二次曲线 ↔ 对偶曲线 | 蛋形 ↔ 极曲线 | 流线 ↔ 等势线 |

## 五、蛋咬一口公式

### 5.1 数学构造：C∞ dip 函数

"蛋咬一口"是在完整的蛋形曲线上叠加一个 **C∞ 光滑凹陷**，同时严格保持曲线周长不变。

核心 dip 函数：

$$d(t; t_0, \sigma) = e^{-\sigma(1-\cos(t-t_0))}$$

**为什么选 $e^{\cos}$？**

| 条件 | $e^{\cos}$ 满足吗？ | Gaussian 满足吗？ |
|------|-------------------|-----------------|
| 在 $t=0$ 和 $t=2\pi$ 处 C∞ 光滑闭合 | ✅ 天然周期 | ❌ 需截断→C0 接缝 |
| 任意阶导数连续 | ✅ 全阶可导 | ❌ 接缝有锯齿 |
| 周期边界自动满足 | ✅ $\cos$ 周期 $2\pi$ | ❌ 需人工处理 |

### 5.2 双波调制

主凹陷波（咬口）和对侧微调波（蛋尖补偿）相乘：

$$w_1(t) = 1 - k_1 \cdot d(t, t_{01}, \sigma_1) \cdot \left[1 + \alpha_1 \sin(t-t_{01}) \cdot d(t, t_{01}, \sigma_1)\right]$$

$$w_2(t) = 1 - k_2 \cdot d(t, 2\pi, \sigma_2) \cdot \left[1 + \alpha_2 \sin(t-2\pi) \cdot d(t, 2\pi, \sigma_2)\right]$$

$$g(t) = w_1(t) \cdot w_2(t)$$

### 5.3 周长保持

$$c = \frac{\int_0^{2\pi} (1-g(t)) \cdot |r'(t)| \, dt}{\int_0^{2\pi} |r'(t)| \, dt}$$

$$g_p(t) = g(t) + c$$

最终曲线：$(x_g, y_g) = g_p(t) \cdot (x(t), y(t))$

周长相对误差控制在 $|P/P_0 - 1| < 0.001$。

### 5.4 七参数调节表

| 参数 | 默认 | 范围 | 控制什么？ |
|------|------|------|----------|
| $k_1$ | 0.265 | 0~0.5 | 凹陷深度 |
| $t_{01}$ | 0.72 rad (41°) | 0~$\pi/4$ | 凹陷在蛋壳上的位置 |
| $\sigma_1$ | 4.6 | 1~15 | 凹陷宽度（值越小越尖锐） |
| $\alpha_1$ | 0 | -0.5~1.5 | 凹陷不对称偏斜 |
| $k_2$ | 0.05 | 0~0.25 | 蛋尖微调深度 |
| $\sigma_2$ | 7.2 | 1~15 | 蛋尖微调宽度 |
| $\alpha_2$ | 0.10 | -0.5~1.5 | 蛋尖微调不对称 |

> 💡 推荐调节顺序：$t_{01}$（定位）→ $\sigma_1$（定形）→ $k_1$（定深）→ 其余微调

## 六、NS 方程与双锥体统一理论

### 6.1 纳维-斯托克斯方程

不可压缩流体的 NS 方程：

$$\frac{\partial u}{\partial t} + (u\cdot\nabla)u = -\frac{1}{\rho}\nabla p + \nu\nabla^2u$$

$$\nabla\cdotu = 0$$

### 6.2 双锥体对应

PKS 理论的核心发现：$ab=1$ 的极化原理产生两个对偶锥体，恰好对应 NS 方程的两部分：

| $ab=1$ 分支 | 函数 | 锥体类型 | 截面 | NS 对应项 |
|-------------|------|---------|------|---------|
| **电性分支** | $a = x$（线性） | 直圆锥 | **椭圆** | **$-\frac{1}{\rho}\nabla p$**（压力梯度） |
| **磁性分支** | $b = \frac{1}{x}$（反比例） | 超双曲锥 | **蛋形** | **$(u\cdot\nabla)u$**（对流项） |

**物理直觉**：
- 压力（p）是标量 → 直圆锥（各向同性）→ 椭圆截面
- 速度（u）是矢量 → 超双曲锥（各向异性）→ 蛋形截面

NS 方程的不可解性，源于这两个锥体的数学结构被混在了一起——PKS 双锥体统一理论将它们分开，分别处理，再通过对偶原理耦合回去。

### 6.3 12 步证明路径（简化版）

```
Step 1: 蛋形截面 = 圆的射影变换       （二次曲线等价）
Step 2: Laplace 算子射影不变性        （算子保持形式）

Step 5: NS 解的蛋形谐波级数展开
Step 6: Burgers 涡 → 蛋形涡映射      （射影变换）
Step 7: 射影不变性验证                （残差分析）
Step 8: 涡量拉伸算子本征值推导
Step 9: k_E=2.0 → 离散谱 → Fibonacci 模态
Step 10: k_E≈1.937 → 连续谱聚集 → KAM 稳定
Step 11: 残差 + 能量估计
```

## 七、工程应用

### 7.1 航空工程：机翼与扩压器截面

当 $\alpha > 50°$（即 $k_E > 3$），截面呈现细长弧形——正是航空扩压器叶片所需形状。

| 应用 | $k_E$ 范围 | 原理 |
|------|----------|------|
| 亚音速机翼 | 3.0–4.0 | 前缘蛋形曲率渐变→层流保持更长 |
| 超音速机翼 | 5.0+ | 细长蛋形→激波面更靠后 |
| 扩压器叶片 | 2.5–3.5 | 蛋形扩散段→压力恢复更平滑 |

### 7.2 水利工程：蛋形截面管道

蛋形截面管道在低流量时仍能保持自清洁流速。这源于蛋形曲率的不对称性：底部窄（高剪应力）→ 上部宽（大通量）。

$$Q = \frac{1}{n} A R_h^{2/3} S^{1/2} \quad (Manning 公式)$$

蛋形截面比圆形截面的 $R_h/A$ 比在高水位和低水位下变化更小，意味着流量对水深的依赖性更低——这就是自调节能力。

### 7.3 生物工程：蛋形发酵罐

负压真空发酵罐采用蛋形几何的理论优势：

1. **零死区**：$C^\infty$ 光滑内壁使流体全域参与环流
2. **热对流传导**：蛋形曲率自然引导 Rayleigh-Bénard 对流环
3. **自清洁**：曲面连续→无附壁沉积
4. **压力分布**：蛋形$\Leftrightarrow$椭圆的对偶性使内压自平衡

### 7.4 农业工程：铜制蛋形犁

蛋形剖面犁铧的流体力学原理：

- 土壤被看作非牛顿流体
- 蛋形剖面的曲率单调递减→剪切速率均匀→土壤卷曲翻转而不产生裂隙
- 铜的化学抑菌性 + 蛋形的物理流畅性 = 双重优化

### 7.5 前沿方向：HHO 涡旋约束腔

超双曲锥 $xy=1$ 的几何本质是**收缩-扩张通道**。在涡旋约束腔中，蛋形截面可以作为"流体二极管"：

```text
  入口（尖头）→ 加速段 → 喷道（蛋腰）→ 扩散段 → 出口（圆头）
      ↑                              ↑
    压力高                          压力低
      ↑            ← 压力梯度方向 →   ↑
```

Schauberger 提出的"内爆技术"（implosion technology）正是利用蛋形截面在收缩段加速、在扩散段产生负压梯度——"流体自己吸着自己走"。

> ⚠️ HHO 飞碟方向仍属理论假说，但其数学骨架——$xy=1$ 涡旋约束腔 + 蛋形截面 + 射影对偶——是严谨的。

## 八、实践练习

### 练习 1：用 Python 画蛋

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

def egg_explicit(k_E, n_points=1000):
    """用显式公式画蛋形"""
    z1, z2 = 1, k_E
    z0 = (z1**2 + z2**2) / (z1 + z2)
    alpha = np.arctan(z1*z2*(z2-z1) / (z1+z2))
    
    # 求解 y 的范围（当 x=0 时）
    from numpy import sin, cos, sqrt
    
    # A3 t-参数法求边界
    a_coeff = sin(alpha) * cos(alpha)
    b_coeff = z0 * cos(alpha)
    discriminant = b_coeff**2 + 4*a_coeff  # 内根方程
    
    y_min = (-b_coeff + sqrt(discriminant)) / (2*a_coeff)  # 尖头（内根）
    y_max = (1 - z0) / sin(alpha)  # 钝头（x=0 另一解）
    
    y = np.linspace(y_min, y_max, n_points)
    inner = 1/(z0 - y*sin(alpha))**2 - (y*cos(alpha))**2
    
    x_pos = np.sqrt(np.maximum(inner, 0))
    x_neg = -x_pos
    
    # 拼接左右两半
    x_full = np.concatenate([x_pos, x_neg[::-1]])
    y_full = np.concatenate([y, y[::-1]])
    
    return x_full, y_full

# 画不同 k_E 的蛋形
fig, axes = plt.subplots(1, 3, figsize=(15, 5))
for ax, k_E, title in zip(axes, 
    [1.5, 2.0, 3.0],
    ['k_E=1.5（圆蛋）', 'k_E=2.0（八度蛋）', 'k_E=3.0（长蛋）']):
    x, y = egg_explicit(k_E)
    ax.plot(x, y, 'b-', linewidth=2)
    ax.set_aspect('equal')
    ax.set_title(title)
    ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('egg_family.png', dpi=150)
plt.show()
```
