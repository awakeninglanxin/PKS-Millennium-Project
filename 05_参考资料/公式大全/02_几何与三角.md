## 一、平面几何

### 1.1 三角形

**面积**：
- $A = \frac{1}{2}bh$（b=底边，h=高）
- Heron 公式：$A = \sqrt{s(s-a)(s-b)(s-c)}$，其中 $s = \frac{a+b+c}{2}$

**勾股定理**：$a^2 + b^2 = c^2$（c 是斜边）

### 1.2 常见三角比（3-4-5, 5-12-13）
- 3,4,5 → $3^2+4^2=5^2$ ✓
- 5,12,13 → $5^2+12^2=13^2$ ✓

### 1.3 四边形
- 正方形：$A = a^2$
- 矩形：$A = ab$
- 菱形：$A = \frac{1}{2}d_1 d_2$

### 1.4 圆
- 周长：$C = 2\pi r$
- 面积：$A = \pi r^2$
- 弧长：$L = r\theta$（θ 用弧度）

### 1.5 扇形
- 面积：$A = \frac{1}{2}r^2\theta$

## 二、立体几何

### 2.1 柱体/锥体/球

| 立体 | 体积 V | 表面积 S |
|:---|:---|:---|
| 长方体 | $\text{abc}$ | $2(ab+bc+ca)$ |
| 圆柱 | $\pi r^2 h$ | $2\pi r(r+h)$ |
| 圆锥 | $\frac{1}{3}\pi r^2 h$ | $\pi r(r+\sqrt{r^2+h^2})$ |
| 球 | $\frac{4}{3}\pi r^3$ | $4\pi r^2$ |

### 2.2 环体公式

SEG 中最重要的几何体是**同心圆环**：

- 环面积：$A_{\text{环}} = \pi(R^2 - r^2)$（R 外径，r 内径）
- 环体积（厚为 t）：$V_{\text{环}} = \pi(R^2 - r^2) \times t$

### 2.3 球冠/球带
- 球冠面积：$S = 2\pi R h$

## 三、三角函数

### 3.1 基本定义

```
在直角三角形中（角为 θ）：

  斜边 c /| 对边 a
        /θ|
       /__|
     邻边 b

sin θ = a/c    cos θ = b/c    tan θ = a/b
```

### 3.2 特殊角的值

| θ (度) | θ (弧度) | sin θ | cos θ | tan θ |
|:---:|:---:|:---:|:---:|:---:|
| 0° | 0 | 0 | 1 | 0 |
| 30° | π/6 | 1/2 | √3/2 | 1/√3 |
| 45° | π/4 | 1/√2 | 1/√2 | 1 |
| 60° | π/3 | √3/2 | 1/2 | √3 |
| 90° | π/2 | 1 | 0 | — |
| 120° | 2π/3 | √3/2 | -1/2 | -√3 |
| 180° | π | 0 | -1 | 0 |

### 3.3 基本恒等式

- $\sin^2\theta + \cos^2\theta = 1$（勾股定理！）
- $\tan\theta = \frac{\sin\theta}{\cos\theta}$
- $\cot\theta = \frac{1}{\tan\theta} = \frac{\cos\theta}{\sin\theta}$

### 3.4 和差公式
- $\sin(A \pm B) = \sin A\cos B \pm \cos A\sin B$
- $\cos(A \pm B) = \cos A\cos B \mp \sin A\sin B$
- $\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A\tan B}$

### 3.5 倍角公式
- $\sin(2\theta) = 2\sin\theta\cos\theta$
- $\cos(2\theta) = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta$
- $\tan(2\theta) = \frac{2\tan\theta}{1-\tan^2\theta}$

### 3.6 半角公式
- $\sin^2(\theta/2) = \frac{1-\cos\theta}{2}$
- $\cos^2(\theta/2) = \frac{1+\cos\theta}{2}$

### 3.7 乘积与和差互化
- $\sin A\sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$
- $\cos A\cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]$
- $\sin A\cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$

### 3.8 弧度
- $1 \text{rad} = \frac{180^\circ}{\pi} \approx 57.3^\circ$
- $\pi \text{rad} = 180^\circ$
- $2\pi \text{rad} = 360^\circ$

### 3.9 反正弦/反余弦/反正切
- $\arcsin(x)$ → 满足 $\sin(\arcsin x) = x$
- $\arccos(x)$ → 满足 $\cos(\arccos x) = x$
- $\arctan(x)$ → 满足 $\tan(\arctan x) = x$

## 四、常用坐标

### 4.1 平面坐标
- 笛卡儿坐标 (x,y)
- 极坐标 $(r,\theta)$：$x = r\cos\theta, y = r\sin\theta$

### 4.2 球坐标
- $(r,\theta,\phi)$
- $x = r\sin\theta\cos\phi$
- $y = r\sin\theta\sin\phi$
- $z = r\cos\theta$

### 4.3 柱坐标
- $(r,\theta,z)$ → $x = r\cos\theta, y = r\sin\theta, z = z$

## 六、外摆线

### 6.1 定义
外摆线 = 一个小圆在大圆外部无滑动地滚动时，小圆上一点的轨迹。

### 6.2 参数方程
令大圆半径为 R，小圆半径为 r，比值为 $k = R/r$：

$$
x(\theta) = (R+r)\cos\theta - r\cos\left(\frac{R+r}{r}\theta\right)
$$
$$
y(\theta) = (R+r)\sin\theta - r\sin\left(\frac{R+r}{r}\theta\right)
$$

### 6.3 SEG 三层环外摆线
- 内环轨道：外环半径 $R_1=21$, 内环 $r_2=29$ → orbit = $(R_1+r_2)/2 = 25$mm
- 中环轨道：$R_2=38.5$, $r_3=46.5$ → orbit = 42.5mm

### 6.4 全曲率公式
- 曲率 $K = (R+2r) / (R(R+r))^{1/2}$
- 拐点个数取决于 $k = R/r$ 是否是有理数

## 七、SEG 环几何中的面积与体积

### 7.1 环截面积
- $A_{\text{环}} = \pi(BR^2 - Br^2)$（外半径 BR，内半径 Br）

### 7.2 体积材料填充率
- 若环厚 $t_n$ 和宽度 $w_n$ 已知：
- $V_n = \pi(BR_n^2 - Br_n^2) \times w_n$

### 7.3 Type4 等环宽
- BR₁-Br₁ = BR₂-Br₂ = BR₃-Br₃ = 9.5mm
- 这意味着**三环使用相同的滚柱直径**
