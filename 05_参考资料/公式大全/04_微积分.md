# 04 微积分入门
> 难度 ★★★★☆ | 前置：03 数列与对数 | PKS 用到：Hausdorff维数、磁场能量体积积分

## 一、极限

### 1.1 定义
$$\lim_{x \to a} f(x) = L$$

当 x 无限逼近 a（但 ≠ a）时，f(x) 无限逼近 L。

### 1.2 常用极限
- $\lim_{x \to 0} \frac{\sin x}{x} = 1$
- $\lim_{x \to 0} \frac{1-\cos x}{x} = 0$
- $\lim_{x \to 0} \frac{e^x-1}{x} = 1$
- $\lim_{x \to 0} \frac{\ln(1+x)}{x} = 1$

### 1.3 无穷与 0 极限
- $\lim_{x \to \infty} \frac{1}{x^n} = 0$（n>0）
- $\lim_{x \to 0^+} \frac{1}{x} = +\infty$

## 二、导数 — 变化率

### 2.1 定义
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

导数是函数在每一点的**瞬时变化率**（斜率）。

### 2.2 常用导数

| f(x) | f'(x) |
|:---|:---|
| c（常数）| 0 |
| xⁿ | nxⁿ⁻¹ |
| 1/x | -1/x² |
| √x | 1/(2√x) |
| eˣ | eˣ |
| aˣ | aˣ ln a |
| ln x | 1/x |
| sin x | cos x |
| cos x | -sin x |
| tan x | sec² x |

### 2.3 运算法则

- 加法：$(f+g)' = f' + g'$
- 数乘：$(cf)' = cf'$
- 乘积：$(fg)' = f'g + fg'$
- 商：$(f/g)' = (f'g - fg')/g^2$
- 链式：$h(g(x))$ 求导：$h'(g(x)) \cdot g'(x)$

### 2.4 高阶导数
- $f''(x)$ = 二阶导数 = 导数的导数
- $f'''(x)$ = 三阶导数
- $f^{(n)}(x)$ = n 阶导数

---

## 三、积分 — 总和与面积

### 3.1 不定积分（反导数）
$$\int f(x) dx = F(x) + C$$

C = 积分常数（"待定"）

### 3.2 定积分（总面积）
$$\int_a^b f(x) dx = \text{从} a \text{到} b \text{的曲线下面积}$$

### 3.3 常用积分

| f(x) | ∫f(x)dx |
|:---|:---|
| xⁿ | xⁿ⁺¹/(n+1) + C |
| 1/x | ln\|x\| + C |
| eˣ | eˣ + C |
| aˣ | aˣ/ln a + C |
| sin x | -cos x + C |
| cos x | sin x + C |
| tan x | ln\|sec x\| + C |

### 3.4 积分方法

**分部积分**：
$$\int u dv = uv - \int v du$$

**换元积分**（令 $u = g(x)$）：
$$\int f(g(x))g'(x)dx = \int f(u)du$$

## 四、基本函数导数

### 4.1 反三角
- d/dx arcsin x = $1/\sqrt{1-x^2}$
- d/dx arccos x = $-1/\sqrt{1-x^2}$
- d/dx arctan x = $1/(1+x^2)$

### 4.2 双曲函数
- $\sinh x = (e^x - e^{-x})/2$
- $\cosh x = (e^x + e^{-x})/2$
- $\tanh x = \sinh x/\cosh x$
- d/dx sinh x = cosh x
- d/dx cosh x = sinh x

---

## 五、泰勒展开

### 5.1 泰勒公式
$$f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \frac{f'''(a)}{3!}(x-a)^3 + \dots$$

### 5.2 麦克劳林公式（a=0）
$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(0)}{n!} x^n$$

### 5.3 常用展开式

| f(x) | 展开 |
|:---|:---|
| eˣ | $1 + x + x^2/2! + x^3/3! + \dots$ |
| sin x | $x - x^3/3! + x^5/5! - \dots$ |
| cos x | $1 - x^2/2! + x^4/4! - \dots$ |
| ln(1+x) | $x - x^2/2 + x^3/3 - \dots$ |
| 1/(1-x) | $1 + x + x^2 + x^3 + \dots$ |

---

## 六、偏微分 — 多维微积分

### 6.1 偏导数
$$\frac{\partial f}{\partial x} = \text{保持} y \text{不变时} f \text{对} x \text{的变化率}$$

### 6.2 梯度
$$\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

### 6.3 散度
$$\nabla \cdot \mathbf{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}$$

- >0 = 流源（流出）; <0 = 流汇（流入）

### 6.4 旋度
$$\nabla \times \mathbf{F} = \left(\frac{\partial F_z}{\partial y} - \frac{\partial F_y}{\partial z},\,
\frac{\partial F_x}{\partial z} - \frac{\partial F_z}{\partial x},\,
\frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y}\right)$$

### 6.5 Laplacian
$$\nabla^2 f = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} + \frac{\partial^2 f}{\partial z^2}$$

---

## 七、微分方程

### 7.1 一阶 ODE
$$\frac{dy}{dx} + P(x)y = Q(x)$$

解：$y = e^{-\int \text{Pdx}}\left(\int Q e^{\int \text{Pdx}} dx + C\right)$

### 7.2 Poisson 方程
$$\nabla^2 \Phi = -\rho$$

### 7.3 波动方程
$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$

### 7.4 扩散/热方程
$$\frac{\partial u}{\partial t} = \alpha \nabla^2 u$$

---

## 八、应用在 PKS 中

### 8.1 立体体积积分
磁体能量密度：$W = \frac{1}{2}\int_V \mu_0 H^2 dV$

### 8.2 Hausdorff 维数（box-counting）
用导数估算体积增量，再代入对数公式。

### 8.3 TGFT 方程
$$\nabla^2 \Phi_T = \kappa \nabla\cdot\langle\mathbf{E}\times\mathbf{B}\rangle + \eta \nabla^2|\mathbf{B}|^2 + \xi \frac{\partial|\mathbf{E}|^2}{\partial t}$$

用到了散度、Laplacian、时间偏导——**全是多变量微积分**。
