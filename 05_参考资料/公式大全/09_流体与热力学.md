# 09 流体力学与热力学
> 难度 ★★★★☆ | 前置：04 微积分、08 电磁学 | PKS：NS/Schauberger涡旋/SEG热

---

## 一、连续性方程

$$\frac{\partial\rho}{\partial t} + \nabla\cdot(\rho\mathbf{v}) = 0$$

- 非压缩性流体：∇·v = 0

## 二、Navier-Stokes方程

$$\rho\left(\frac{\partial\mathbf{v}}{\partial t} + \mathbf{v}\cdot\nabla\mathbf{v}\right) = -\nabla p + \mu\nabla^2\mathbf{v} + \mathbf{f}$$

| 项 | 意义 |
|:---|:---|
| ρ∂v/∂t | 非稳态加速 |
| ρ(v·∇)v | 对流加速 |
| -∇p | 压力驱动 |
| μ∇²v | 黏滞阻力 |
| f | 外力（重力/电磁力） |

## 三、雷诺数

$$Re = \frac{\rho v L}{\mu}$$

- Re < 2000：层流 | Re > 4000：湍流

## 四、涡旋

涡量：$\boldsymbol{\omega} = \nabla \times \mathbf{v}$

Schauberger 双曲锥体涡旋：半径指数收缩 → 流速增加 → 温降（内爆原理）

## 五、热力学基本方程

$$PV = \text{nRT}$$

- R ≈ 8.314 J/(mol·K)
- ΔU = Q - W（热力学第一定律）
- ΔS ≥ Q/T（热力学第二定律）

## 六、Carnot 效率

$$\eta_C = 1 - \frac{T_L}{T_H}$$
