## 一、特殊相对论

### 1.1 质能等价
$$E = mc^2$$

### 1.2 运动质量
$$m = \frac{m_0}{\sqrt{1 - v^2/c^2}}$$

### 1.3 Lorentz 变换
$$x' = \gamma(x - vt), \quad t' = \gamma(t - vx/c^2)$$

其中 $\gamma = 1/\sqrt{1 - v^2/c^2}$

### 1.4 Aquino χ 因子公式
$$\chi = \frac{m_g}{m_i} = \left\{1 - 2\left[\sqrt{1+\left(\frac{U}{m_i c^2}\right)^2}-1\right]\right\}^{-1/2}$$

- U = 辐射/磁能吸收
- χ = 引力质量 / 惯性质量
- χ < 1 → 有效质量降低

### 2.1 能量量子化
$$E = h\nu = \frac{hc}{\lambda}$$

## 三、量子力学基本方程

### 3.1 Schrödinger 方程

$$i\hbar\frac{\partial\Psi}{\partial t} = -\frac{\hbar^2}{2m}\nabla^2\Psi + V\Psi$$

### 3.2 波函数
$$|\Psi(x,t)|^2 = \text{在位置} x \text{找到粒子的概率密度}$$

### 3.3 Bloch 球体表示
任意二态量子系统（如自旋 ½）：
$$|\psi\rangle = \cos\frac{\theta}{2}|0\rangle + e^{i\phi}\sin\frac{\theta}{2}|1\rangle$$

EEQT 在球面上做混沌博弈 → 自旋态在 Bloch 球面上游走

## 四、量子跳跃（EEQT的核心）

### 4.1 投影振幅
$$\langle\phi_n^k|\psi(t)\rangle = \text{自旋态在检测器} k \text{方向的投影}$$

### 4.2 跳跃概率
$$p_k = \frac{|\langle\phi_n^k|\psi(t)\rangle|^2}{\sum_j |\langle\phi_n^j|\psi(t)\rangle|^2}$$

### 4.3 连续演化 vs 跳跃

- 跳跃之间：Schrödinger 方程控制的连续演化
- 跳跃时刻：自旋态突然跳到检测器 j
- 跳跃的时机由 ε 控制

**EEQT 的分形就是经典自旋测量序列的几何痕迹**
