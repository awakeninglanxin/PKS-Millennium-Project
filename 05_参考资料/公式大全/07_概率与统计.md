# 07 概率与统计

## 一、基本概率

### 1.1 概率公理
- P(A) ∈ [0,1]
- P(必然事件) = 1
- 互斥事件 A, B：P(A∪B) = P(A) + P(B)
- 独立事件 A, B：P(A∩B) = P(A)×P(B)

### 1.2 条件概率
$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

### 1.3 全概率公式
若 $B_1, B_2, \dots, B_n$ 划分样本空间：
$$P(A) = \sum_{i=1}^n P(A|B_i)P(B_i)$$

### 1.4 贝叶斯公式
$$P(B|A) = \frac{P(A|B)P(B)}{P(A)}$$

## 二、离散分布

### 2.1 伯努利分布（抛硬币）
- P(success) = p, P(failure) = 1-p
- E[X] = p, Var(X) = p(1-p)

### 2.2 二项分布
n 次伯努利试验中成功 k 次的概率：
$$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$

### 2.3 泊松分布
$$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$$

---

## 三、连续分布

### 3.1 均匀分布
$$P(x) = \frac{1}{b-a}, \quad a \le x \le b$$

### 3.2 正态分布（高斯分布）
$$P(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

- μ：均值，σ：标准差
- 68% 数据在 μ±σ 内
- 95% 数据在 μ±2σ 内

### 3.3 指数分布
$$P(x) = \lambda e^{-\lambda x}, \quad x \ge 0$$

---

## 四、统计量

### 4.1 均值
$$\mu = \frac{1}{n}\sum_{i=1}^n x_i$$

### 4.2 方差
$$\sigma^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \mu)^2$$

### 4.3 协方差
$$\text{Cov}(X,Y) = E[(X-\mu_X)(Y-\mu_Y)]$$

### 4.4 相关系数
$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$$

## 五、PKS 中的应用

### 5.1 EEQT 概率跳跃
- 量子跳跃的方向由**概率分布**决定
- 概率与检测器方向的**投影振幅平方**成正比
- 多次迭代形成分形吸引子

### 5.2 混沌与随机
- EEQT 的迭代是**确定性的**（IFS 方程规则确定）
- 但是初始条件随机（随机选择初始检测器）
- 这就是"确定性混沌"的核心特征
