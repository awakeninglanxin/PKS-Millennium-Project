# PJK 深度解读 ① — Robert Adams 脉冲电机 vs 周治平 COP

> PJK Ch2 "Moving Pulsed Systems" | Adams 声称 800% 效率
> PKS 对应: 周治平 "正磁矩" 概念的电机实现版本

---

## 一、Adams 电机工作原理

### 1.1 与传统电机的根本区别

```
传统电机：线圈通电 → 产生磁极 → 推/拉转子磁铁
         └─ 线圈持续通电 → 磁通持续 → 反电动势持续消耗电能

Adams 电机：线圈**短暂脉冲通电** → 产生强磁极 → 转子磁铁被推离
           └─ 脉冲结束 → 线圈断电 → 磁通湮灭
           └─ 湮灭期间：转子继续靠惯性/永磁偏置力运动
           └─ 永磁体提供持续的偏置吸引力
```

### 1.2 脉冲时序

```
      ┌── T_ON ──┐┌─── T_OFF (长) ───┐
      │          ││                    │
线圈   ┤ 通电      └────────────────────
      │                                  
转子   ├──加速──┤├───── 惯性滑行 ──────┤
      │        ││                      │
磁力   ├ 排斥力 │├ 永磁体偏置吸引力     │
```

**关键**: T_ON << T_OFF。脉冲仅占整个周期的一小部分。Adams 说线圈**不产生转矩**——线圈的作用只是**消除永磁体的反向拉力**，让转子在大部分时间里靠永磁吸引力前进。

## 二、周治平视角的 Adams 分析

### 2.1 "不产生转矩" = 周治平的 "正磁矩"

Adams 声称线圈的通电**不产生转矩**——只"解除"永磁体在当前位置的反向吸引力。这对应周治平的**正磁矩**概念：

```
周治平：诱导磁路 → 感应电流同时形成正磁矩和反磁矩
Adams:    脉冲线圈 → 消除反磁矩, 让永磁吸引力成为唯一驱动力
```

**这个"消除反磁矩"就是 COP>1 的基础**——转子在大部分时间不受反磁矩的阻碍，永磁吸引力做净正功。

### 2.2 COP 计算

$$W_{in} = \text{脉冲电能} \times T_{ON}/T_{\text{cycle}}$$
$$W_{\text{out}} = \text{永磁吸引力做的总功} \times \text{转子旋转角度}$$

因为脉冲极短（T_ON << T_OFF），W_in 非常小，但同时永磁吸引力在全程做功。

**Adams 声称 800%** 意味着：输出轴机械功 = 8 × 输入电脉冲能量

## 三、Adams 与 Ufopolitics 的对比

| 维度 | Adams | Ufopolitics | 周治平 |
|:---|:---|:---|:---|
| 定子磁场 | 永磁体 (被动) | 永磁体 (被动) | **无(仅电激磁)** |
| 脉冲作用 | 消除反磁矩 | 产生排斥/吸引力 | **产生磁饱和** |
| 铁心饱和 | 不需要 | 不需要 | **必需** |
| COP 机制 | 脉冲"解禁"永磁做功 | 磁场崩解同相回收 | 楞次失效+暗能量 |

| Adams | SEG |
