# 重力轮 / Cahill 热泵 — 环境梯度引擎

> 来源：PJKbook Ch4 (p.355-357), Ch8 (p.671-673)
> 关联：Schauberger 涡旋、SEG 反重力、永动机避坑

---

## 一、两类"伪永动机"的真面目

### 1.1 为什么它们不是永动机

PJKbook Ch4 和 Ch8 的装置都**不违反热力学第二定律**，因为它们的能量来源是**环境梯度**而非"凭空产生"。

| 装置类别 | 能量来源 | 违反第二定律？ |
|:---|:---|:---|
| **重力轮**（弹簧舌+偏转轨） | 旋转惯性的改变 | ❌ 能量来自初始动能转换 |
| **Cahill 热泵** | **日-夜温差 / 季节温差** | ❌ 标准热泵原理 |
| **Veljko Milkovic 摆** | 重力势能 ↔ 动能的反复转换 | ❌ 标准机械能守恒 |

### 1.2 重要提醒：这些小装置 ≠ COP>1

**PJKbook 将这些装置归入"自由能源"是误导性的**。真正的"自由能源"（COP>1）必须从环境背景场（电磁以太/中微子凝聚态）中提取能量。这些重力/热力装置只是**梯度的巧妙利用**。

## 二、重力轮（Ch4）

### 2.1 弹簧舌设计

```
轮子 + 8 根辐条
每根辐条：滑动重块 + 弹簧 + 舌扣
重力 → 重块滑出 → 轮子一侧偏重 → 旋转
旋转到顶部 → 弹簧被压缩 → 舌扣锁定
下落到底部 → 舌扣释放 → 弹簧弹出 → 重复
```

### 2.2 受力分析

顺时针力矩 vs 逆时针力矩：

$$M_{\text{net}} = M_{\text{clockwise}} - M_{\text{counter}}$$

设弹簧压紧位置的力臂为 $r_{in}$，释放位置的力臂为 $r_{\text{out}}$：

$$M_{\text{net}} \propto mg \cdot (r_{\text{out}} - r_{in})$$

如果 $M_{\text{net}} > 0$ 持续，轮子自转——但这需要**弹簧释放的力克服摩擦力 + 压缩弹簧的力**。

### 2.3 实际结果

PJK 诚实地说：
> "The only way of determining if this design will work correctly is to build one and test it."

这等于承认：**重力轮在理论上没有自持的保证**。多数构建者报告失败。这与 Stevin 原理（永动机避坑指南的核心武器）一致。

## 三、偏转轨重力引擎（Veljko Milkovic 类）

### 3.1 工作机制

```
外缘滚轮轴承 + 偏转轨
    ↓
旋转时滚轮遇到固定的偏转轨
    ↓
滚轮被推向内侧 → 力臂突然减小
    ↓
另一侧的滚轮仍在外缘 → 力臂大 → 净旋转力矩
```

### 3.2 能量分析

这不是 COP>1。能量来源是**初始旋转动能**（手推或启动马达）。偏转轨改变了力臂，但不增加系统总能量。

**工程价值**：作为机械**飞轮储能**装置有价值（类似 KERS 动能回收），但不是"自由能源"。

## 四、Cahill & Scott 太阳能热泵（1970s）

### 4.1 核心设计

| 组件 | 描述 |
|:---|:---|
| 工作液 | **专有液体**（低压即可汽化） |
| 热源 | **任何温差**：日照/多云/雨天/雪天/昼-夜/季节 |
| 膨胀 | 液体 → 气体（巨大体积膨胀） |
| 输出 | 驱动涡轮或引擎 |
| 燃料 | **零** |

### 4.2 物理原理

标准热泵逆向卡诺循环：

$$\text{COP}_{\text{heat} \text{pump}} = \frac{T_{\text{hot}}}{T_{\text{hot}} - T_{\text{cold}}}$$

Cahill 系统的关键是**工作液配方**——它在极低温差下即可汽化。普通制冷剂需要几十度的温差，Cahill 液只需要几度。

### 4.3 与 PKS 的关联

| Cahill 热泵 | PKS |
|:---|:---|
| 环境温差 → 机械功 | 环境背景场（电磁以太）→ 电能 |
| 专有工作液降低阈值 | 超坡莫合金降低 $H_S$ |
| 热泵 COP = 3~5 标准 | 电磁泵 COP >> 1（理论） |

两者都是**降低"启动阈值"来提取环境能量**。Cahill 降低的是温差阈值，周治平降低的是磁饱和阈值。

## 五、关系网：重力/热力引擎在 PJKbook 中的真实地位

```
PJKbook 能量提取装置
        │
  ┌─────┴──────────────────────┐
  │                            │
  A 类：真正 COP>1              B 类：梯度利用 COP=1
  │                            │
  ├─ Bedini SSG              ├─ 重力轮
  ├─ Adams 脉冲电机          ├─ Milkovic 摆
  ├─ Floyd Sweet VTA         ├─ Cahill 热泵
  ├─ Moray 天线              └─ Wimshurst 起电机
  ├─ Don Smith
  ├─ Colman 管
  └─ Imris 光学
```

**B 类装置**的教学价值：它们揭示了"能量可以从环境提取"的基本原理——即使不是 COP>1，但**环境梯度 → 可用功**的概念是正确的。

## 六、与永动机避坑指南的衔接

| 避坑指南（Stevin 原理） | PJKbook B 类装置 |
|:---|:---|
| "转一圈后状态不变 → 不会自转" | 重力轮失败的根本原因 |
| "必须有外部能量源" | 重力/温差 = 外部能量源 |
| "真正开放系统" | Cahill 热泵是真正的开放系统 |
| "伪永动机" | 重力轮多数是伪永动机（Ch13 已归类为"questionable"） |

## 七、方法论教训

PJKbook 的伟大之处在于：它既展示了真正突破的装置（A 类），也展示了伪装的梯度装置（B 类），还诚实标注了"questionable"装置（Ch13）。

**对 PKS 的启示**：
1. 任何声称 COP>1 的装置，必须明确**能量来源**（环境电磁场？中微子凝聚态？温差？）
2. B 类装置不是"错"——它们是正确的工程原理（热泵、飞轮），只是不应该被误标为"自由能源"
3. **区分 A 类和 B 类的能力 = 区分真科学和伪科学的能力**
