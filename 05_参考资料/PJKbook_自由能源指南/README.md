## 三、核心洞察：PJK 全书的统一物理原理

Patrick Kelly 本人从未提出统一理论，但他收集的**数百个装置共享同一个底层原理**：

```
输入一个小功率 → 建立一个与环境的耦合接口 → 环境能量通过接口流入 → 输出功率 >> 输入
    ↑                                                                              ↑
这个"接口"就是 PKS 项目的全部研究对象
```

| 装置 | "接口"的物理形式 | PKS 框架中的位置 |
|:---|:---|:---|
| Adams 电机 | 永磁体偏置 + ON/OFF 脉冲 | 周治平 COP (H>H_S) |
| Floyd Sweet VTA | 钡铁氧体的自旋排列 | TGFT η∇²|B|² |
| Bedini SSG | 电感储能 + 脉冲回收 | Ufopolitics ON→OFF |
| Don Smith | Tesla 线圈的高压位移电流 | Aquino χ (U/mc²) |
| Hans Coler | 6永磁体的非对称排列 | 瑟尔真理表 |
| Bob Boyce | 101 板电解 + 3相 PWM | 周治平 COP×电化学 |
