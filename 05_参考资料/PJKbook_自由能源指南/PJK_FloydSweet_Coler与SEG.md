## 一、Floyd Sweet VTA — 磁化钡铁氧体的自激振荡

### 1.1 基本原理

Sweet 使用**特殊磁化**的钡铁氧体磁体（尺寸: 6×4×1英寸）。这些磁体在特定条件下表现出**自激振荡**——不需要外部电源，磁体本身产生振荡磁场。

### 1.2 TGFT 视角

VTA 的磁体是 TGFT 三项的一个**物理实例**：

| TGFT 项 | VTA 实现 |
|:---|:---|
| κ∇·⟨E×B⟩ | 磁化图案的不均匀性 → Poynting 流自发产生 |
| η∇²|B|² | 磁化密度的空间梯度 → 稳定涡流 |
| ξ∂|E|²/∂t | 自激振荡 → 时变电场泵浦 |

**关键**: Sweet 对磁体的特殊磁化过程（"conditioning"）——用强磁场 + 特定频率的交流叠加充磁——这**完全等同于** SEG 的 DC+AC 复合充磁。两者的目标都是"印刻"不均匀的磁化图案到材料中，故意破坏磁化均匀性，创造 Poynting 流的自发散度。

### 1.3 实测数据 (PJK p.254)

- 输出功率可达 **>1kW**
- 常规仪器在 >1kW 时停止工作（显示零或随机读数）
- 装置可以被短暂中断供电来停止
- 9V 脉冲可启动自持运行

### 1.4 PKS 关联

Floyd Sweet 的 "conditioning" = SEG 磁化层的 DC+AC 复合充磁。两者都在材料中**印刻非均匀磁化图案**。

## 二、Hans Coler 被动永磁装置 — SEG 的前身

### 2.1 结构

Coler 使用 **6 块永磁体**排列在特定的几何结构上，配合线圈绕组。**没有移动部件，没有外部电源，纯被动输出 60W**。

### 2.2 与瑟尔真理表的对比

| Coler | SEG |
|:---|:---|
| 6 块永磁体 | 12/22/32 滚柱(永磁/电磁) |
| 静态几何排列 | 动态旋转排列 |
| 60W 输出 | 理论 kW 级输出 |
| 无外部电源 | 极低外部电源(W_in≈0) |
| **发明于 1920s** | **发明于 1960s** |

Coler 是 SEG 的**静态版**——如果 Coler 的 6 块磁体是 SEG 的 12 个滚柱，那 Coler 就是 SEG 在时间冻结状态下的表现。

### 2.3 为什么 Coler 能工作？

Coler 的 6 磁体排列 + 线圈相当于**在材料中建立了一个非零的空间平均 Poynting 流**:

⟨E × B⟩ ≠ 0 （在设备体积内的空间平均）

这意味着 TGFT 的 κ 项**静不为零** → 即使没有运动部件，也存在净的反引力梯度。
