# 第一层：小学算术（★★☆☆☆）

```

## 1.2 分数

```
1/2  = 0.5       一半

## 1.3 估算

```

算术总结：

| 符号 | 读法 | 意思 |
|:---|:---|:---|
| `+`  | 加 | 合并 |
| `×`  | 乘 | 重复加 |
| `÷`  | 除以 | 平均分 |
| `=`  | 等于 | 两边一样 |
| `≈`  | 约等于 | 差不多 |
| `>`  | 大于 | 左边更大 |
| `<`  | 小于 | 右边更大 |

# 第二层：小学几何（★★☆☆☆）

## 2.1 圆形

```
直径 = 2 × 半径               d = 2r
周长 = π × 直径               C = πd
面积 = π × 半径²              A = πr²
```

想象你在画一个圆：半径是你从圆心到边缘的距离，直径是穿过圆心从一边到另一边的距离。圆的周长总是直径的 π 倍——无论圆有多大，π 都是≈3.14。

## 2.2 三角形

```
周长 = a + b + c
面积 = (底边 × 高) ÷ 2
```

## 2.3 长方体 / 立方体

```
体积 = 长 × 宽 × 高
表面积 = 2 × (长×宽 + 宽×高 + 高×长)
```

**项目连接**：蛋形 = 圆形被拉长。圆形周长 `2πr → 2π ln n`，这就是 PKS
蛋形 Laplace 谱的来源。

# 第三层：比例与百分比（★★☆☆☆）

## 3.1 比例

```
a : b = c : d    读作：a 比 b 等于 c 比 d
3 : 6 = 1 : 2    3 和 6 的比例等于 1 和 2 的比例
a/b = c/d         同样的意思，用分数线写
```

## 3.2 百分比

```
10% = 10/100 = 0.1   百分之十
50% = 50/100 = 0.5   一半
100% = 1              全部
```

## 3.3 黄金比例 φ

```
φ = (1 + √5) / 2 ≈ 1.618

两条线段 a > b，
如果满足  a/b = (a+b)/a，
比值就是黄金比例 φ。
```

**计算过程**：
```
a/b = (a+b)/a = 1 + b/a

令 x = a/b，则：
x = 1 + 1/x
x² - x - 1 = 0
x = (1 + √5) / 2 ≈ 1.618033988749895
```

**项目连接**：
- SEG 三环质量比与斐波那契数列
- 地球驻波图中全球遗址间距的比例
- 正二十面体和十二面体顶点坐标中都出现 φ

# 第六层：对数（★★★☆☆）

> **前置**：会指数运算

## 6.1 对数的基本概念

**对数是指数的反运算**：

```
因为 2³ = 8
所以 log₂(8) = 3      ("以2为底8的对数等于3")

因为 10² = 100
所以 log₁₀(100) = 2    ("以10为底100的对数等于2")
```

**一句话**：`logₐ(b) = c` 就是问"a 的多少次方等于 b？"

## 6.2 自然对数 ln

```
ln(x) = logₑ(x)

e ≈ 2.718281828459045   (另一个重要常数)
ln(e) = 1
ln(1) = 0
ln(e²) = 2
```

## 6.3 对数运算规则

```
log(a×b) = loga + logb
log(a÷b) = loga - logb
log(aⁿ)  = n × loga
```

**项目连接**——这是 PKS 蛋形谱的核心！

```
λₙ = 2π × ln(n)

n=2:  λ₂ = 2π × ln2 ≈ 4.36
n=3:  λ₃ = 2π × ln3 ≈ 7.15
n=7:  λ₇ = 2π × ln7 ≈ 12.89

# 第八层：代数（★★★★☆）

## 8.1 一元二次方程

```
ax² + bx + c = 0

求根公式：
x = [-b ± √(b² - 4ac)] / (2a)

判别式 Δ = b² - 4ac：
  Δ > 0 → 两个不同的实数根
  Δ = 0 → 一个根（重根）
  Δ < 0 → 两个复数根
```

## 8.2 因式分解

```
(a+b)(a-b) = a² - b²        平方差公式
(a+b)²     = a² + 2ab + b²  完全平方公式
(a-b)²     = a² - 2ab + b²
```

## 8.3 多项式

```
一次函数：y = ax + b            直线
二次函数：y = ax² + bx + c      抛物线
三次函数：y = ax³ + bx² + cx + d
```

# 第十五层：分形与迭代（★★★★★）

> **前置**：会复数

## 15.1 什么是分形？

**分形 = 无限重复的数学图案，无论放大多少倍看都有结构**

```
一条线的长度 = 1 → Hausdorff 维数 = 1
一个面的面积 = 1 → Hausdorff 维数 = 2
分形介于线与面之间 → 维数不取整数，如 1.26
```

## 15.2 迭代函数系统

```
Zₙ₊₁ = Zₙ²

从 Z₀ = 0 开始：
Z₁ = 0² = 0      （什么都没发生）
Z₂ = 0           仍然为0

从 Z₀ = 2 开始：
Z₁ = 2² = 4 → Z₂ = 16 → Z₃ = 256 → 无限大！(不收敛)

从 Z₀ = 0.5 开始：
Z₁ = 0.25 → Z₂ = 0.0625 → 趋近于0（收敛）

介于收敛和发散之间的边界 → 分形图案！
```

**这就是著名的 Mandelbrot 集！**

## 15.3 Hausdorff 维数

```
维数 D = log(N) / log(1/r)

Koch 雪花：每次细分变4份，每份是原来的1/3
D = log4 / log3 ≈ 1.26

Sierpinski 三角：每次细分变3份，每份是原来的1/2
D = log3 / log2 ≈ 1.585
```

**项目连接**：
- EEQT 混沌博弈 = 在球面上运行 IFS
- Putney 的全球驻波图用同一迭代函数
- Hausdorff 维数测量分形复杂度

正方形的对称群：
