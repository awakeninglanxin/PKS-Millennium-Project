# 08 将非对称电机升级为 COP>1 发电机 — 工程设计方案

> 融合：Ufopolitics 非对称绕线 + 周治平磁饱和 + Bedini 脉冲回收

---

## 一、设计目标

| 参数 | 当前(Ufopolitics) | 目标(COP>1 发电机) |
|:---|:---:|:---:|
| 输出功率 | ~数W (回收) | ~100W+ |
| COP | <1 (实测) | >2 (目标) |
| 铁心材料 | 硅钢片 (H_S~50-100 A/m) | 超坡莫合金 (H_S~0.15 A/m) |
| 换相方式 | 光耦 + MOSFET | 光耦 + MOSFET (不变) |
| 回收路径 | 同一线圈的"发电机端子" | 独立回收绕组 + 反激拓扑 |

---

## 二、核心设计改动

### 2.1 铁心材料升级 — 最关键的一步

```
硅钢片: H_S ≈ 50-100 A/m, μ_r ≈ 5,000
    → 需要较大电流才能饱和
    → Ufopolitics 的 PWM 电压通常不足

超坡莫合金: H_S ≈ 0.15-0.3 A/m, μ_r ≈ 200,000
    → 极小电流即可饱和
    → 12V 电池 + 几百mA 即可触发周治平机制
```

### 2.2 回收绕组设计

```
每个极上绕两组线圈(代替 Ufopolitics 的单组):

主线圈: 25 匝 AWG30 (电动激励)
回收线圈: 100 匝 AWG36 (发电回收)
    → 匝比 = 100:25 = 4:1 → 理论回收电压 = 4×主电压

回收线圈通过 UF4007 全桥整流到充电电容
```

### 2.3 时序优化

```
修改光耦盘的开孔角度:

原 Ufopolitics: ON 窗口 ~120° (约 33% 占空比)
COP 发电机版: ON 窗口缩小到 ~30° (约 8% 占空比)

目的：ON 时间足够让铁心进入饱和(H>H_S)，
      但不够让楞次效应起显著作用。
      OFF 时间长 → 充分的磁场崩解时间 → 更完整的能量回收。
```

## 三、预期 COP 计算

### 3.1 输入能量

$$W_{\text{in}} = V_{\text{batt}} \times I_{\text{avg}} \times T_{\text{ON}}$$

$$W_{\text{in}} = 12\text{V} \times 0.3\text{A} \times T_{\text{ON}}$$

### 3.2 饱和后附加能量 (周治平机制)

$$W_{Hf} = \frac{1}{2} \int_V (H_X - H_S) B_S dv$$

### 3.3 回收能量

$$W_{\text{rec}} = \text{电感储能} \times \text{耦合效率} + W_{Hf}$$

### 3.4 估算 COP

| 阶段 | 能量 (mJ/周期) | 来源 |
|:---|:---:|:---|
| 输入电能 | ~5 | 电池 |
| 电感储能 (磁场崩解) | ~3.5 | 电感储能回收 |
| 附加磁能 (周治平) | ~2~10 | 暗能量 |
| 总回收 | ~5.5~13.5 | |
| COP | **1.1~2.7** | |

## 四、验证步骤

### Phase 1: 基准测量 (硅钢片铁心, 不修改任何东西)
- 测量输入功率 vs 输出功率 → 计算基准 COP_Baseline

### Phase 2: 改用坡莫合金铁心
- 保持所有电气参数不变
- 测量饱和电流是否大幅下降 (确认 H_S 更低)

### Phase 3: 添加回收绕组 + 全桥整流
- 测量回收功率

### Phase 4: 调光耦盘 ON 窗口
- 在 10° ~ 120° 间搜索最优 COP
