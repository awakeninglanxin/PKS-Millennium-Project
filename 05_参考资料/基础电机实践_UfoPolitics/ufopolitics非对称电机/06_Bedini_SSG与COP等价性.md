# 06 能量回收脉冲电路 — Bedini SSG 与 Ufopolitics 的 COP 等价性

> Bedini SSG (Simplified School Girl) = 最简洁的脉冲电机+能量回收电路
> 核心原理与 Ufopolitics + 周治平的反激 PWM 是完全相同的物理

---

## 一、Bedini SSG 电路原理

### 1.1 最简电路

```
          +12V
           │
    ┌──────┤
    │   ┌──┴──┐
    │   │ 线圈  │ (主绕组: ~800匝 AWG23)
    │   └──┬──┘
    │      │
    │   ┌──┴──┐ D
    │   │  NPN ├─┤  2N3055
    │   └──┬──┘ G
    │      │    S
    │      │    │
    │   ┌──┴────┴──┐
    │   │  触发绕组  │ (~6匝 AWG26, 绕在主绕组同铁心上)
    │   └─────┬────┘
    │         │
    │    10kΩ │
    │    ┌──┴──┐ (基极限流)
    │    │     │
    │    └─────┘
    │         │
    └─────────┴─── GND

输出 (回收):
    线圈上另绕一个次级绕组 → 全桥整流 → 充电电池
```

### 1.2 工作循环

```
Phase 1: 永磁体接近铁心 → 触发绕组感应 →
         NPN导通 → 主绕组有电流 → 铁心磁化 → 吸引磁铁

Phase 2: 磁铁到达最近点 → 触发绕组感应反转 →
         NPN关断 → 主绕组电流截断 →
         磁场崩解 → 在主绕组上产生高压脉冲 (50-200V)

Phase 3: 高压脉冲 → 经次级绕组耦合 → 
         全桥整流 → 充电电池
         此脉冲能量 = 最初输入磁化能量的回收 + 机械动能转换
```

### 1.3 COP "声称"

Bedini 声称他的 SSG 电路可以同时充电 4 块电池，而只消耗 1 块电池的电量。但从物理上看，被充电池接受的是**转子动能转换回的电能**，而不是"凭空多出来的能量"。

**但是**：如果铁心能在 Phase 1 达到磁饱和（H > H_S），则 Phase 2 的磁场崩解会引入**暗能量**——与周治平 COP 的机制完全一致。

## 二、Bedini ⊗ Ufopolitics ⊗ 周治平 — 同一电路的不同实现

### 2.1 电路等价表

| 功能块 | Bedini SSG | Ufopolitics | 周治平 PWM |
|:---|:---|:---|:---|
| 开关器件 | NPN 2N3055 (自激振荡) | MOSFET IRFZ44N (光耦驱动) | MOSFET/IGBT |
| 能量储存 | 主绕组电感 | 电动线圈电感 | 变压器原边 |
| 关断触发 | 触发绕组 (6匝, 被动) | 光耦盘 (主动) | PWM 控制器 |
| 回收路径 | 次级绕组 + 桥式整流 | 同一线圈的"发电机端子" | 反激绕组 |
| COP 机制 | 磁场崩解脉冲回收 | ON→OFF 磁场崩解回收 | H>H_S 暗能量注入 |
| 饱和控制 | 无 (取决于电源电压+匝数) | 无 | **核心特征: 必须有** |

### 2.2 核心差异：谁触发了磁饱和？

```
Bedini:  偶发 (取决于电池电压, 可能偶尔饱和)
Ufo:     从未 (永磁体偏置, PWM 电压不足以让铁心饱和)
周治平:  故意且必然 (核心设计目标)
```

### 2.3 隐含的共同前提

三条路线都在说同一件事：**电感储能的 ON→OFF 切换过程中，磁场崩解产生的高压脉冲可以高效回收。** 但是只有周治平明确指出了 COP>1 的必要条件——必须让铁心磁饱和。

## 三、可实验验证的升级路线

```
Step 1: 搭建最简 Bedini SSG
    验证：磁场崩解脉冲在示波器上的波形
    测量：输入功率 vs 回收功率

Step 2: 改用光耦主动控制(像 Ufopolitics)
    验证：可调 ON/OFF 占空比的效率改善
    寻找：甜点(电流谷+转速稳定)

Step 3: 改用超坡莫合金铁心(H_S ~0.15 A/m)
    验证：极低的饱和所需电流
    测量：是否出现 COP>1

Step 4: 对比三种铁心的 COP 曲线
    硅钢片(H_S ~100 A/m), 铁氧体(H_S ~20), 超坡莫(H_S ~0.15)
```
