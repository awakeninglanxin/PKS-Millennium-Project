## 一、绕组规格的精确数据（Page 4 提取）

### 1.1 5 极电枢绕线步骤

**25 匝/极，AWG 对应规格**：

```
标记步骤：
  转子上端面 — 用记号笔从 1 到 5 编号

绕线 (以极 1→3 为例，共 25 次)：
  P1: 极1的上端面
      线从 P1 上端经过极1铁心到 P1 下端
      从 P1 下端跨到 P3 上端
      = 这是 "1 turn"（一匝）
  重复 25 次 → 得到 25 Left Windings（左旋绕组）

电阻规格：单线圈组电阻应为 1.2 ~ 1.6 Ω
```

### 1.2 V 型串联连接

> "In total you make five V-shaped (two-pair serial) connections across the commutator."

```
V型连接 = 2个极串联 = 物理上在电枢上形成 "V" 字形
5个V型连接 = 总共10个极通过换向器成对串联

电路等效：每个V型组 = 2×25匝 = 50匝/组
```

### 1.3 为什么是 1.2~1.6 Ω？

```
AWG30 铜线的电阻 ≈ 0.338 Ω/m (@20°C)

每匝长度 ≈ 极高 × 2 + 跨接线 ≈ 0.1m × 2 + 0.05m = 0.25m
25匝/极 × 2极/组 = 50匝 × 0.25m/匝 = 12.5m
12.5m × 0.338 Ω/m ≈ 4.2 Ω

但实际测量为 1.2~1.6 Ω → 说明使用了更粗的线
(AWG28 或 AWG26, 电阻更低)
或绕线长度比估计的更短(极更矮)
```

| 5 | "Start-Up Winding of shorter poles for greater torque" | 启动力矩绕组——电磁"变速箱" |
| 6 | "Electromagnetic Transmission within just one embodiment" | 电机=电磁变速箱 |

### 2.2 "磁力抛出角度"的物理分析

**传统电机**：磁力线从 N 到 S 直线穿过气隙 → 力 = BIL → 纯切线方向力

**非对称电机**（推测）：
- 单向磁通的 N 极投影在特定角度
- 此投影角与转子运动方向不垂直
- 磁力有一个**径向分力**（排斥）+ 一个**切线分力**（旋转）
- 在"抛出角度"最优时，排斥分力最大的同时切线分力也足够
- 等效于磁力线的方向被"倾斜"，转矩不是靠纯 BIL 而是靠磁极的倾斜投影产生

**与周治平"正磁矩"的相似性**：周治平的诱导磁路让感生电流的磁力线有一个"倾斜"——产生既有反作用也有同作用的力矩组成。Ufopolitics 的"抛出角度"也在本质上产生了力矩的方向性不对称。

### 2.3 "电磁变速箱"概念

> "a kind of Electromagnetic Transmission within just one embodiment"

这个声明相当于说 ：通过调节线圈组的 ON/OFF 时序，电机可以实现类似**机械变速器**的效果——不同占空比 = 不同"档位"：
- 高占空比 = 低档（高转矩，低转速）
- 低占空比 = 高档（低转矩，高转速）
- 甜点 = 最佳效率档（输入电流最低，转速稳定）

这与现代**直接转矩控制(DTC)**的理念完全一致——只是 Ufopolitics 用机械光耦代替了 DSP 数字信号处理器。

## 三、darkoni 实测数据（Page 11 提取）

### 3.1 关键测量（原文）

> "While powering the motor through the input connection, any load I connect to the output generator connections causes an increase in current draw from my source 12v battery."

> "Btw, I'm getting around 6v on the generator output and 18 volts when using the..."

> "Your output voltage is only 8 volts..."

> "COP is a measure of input power versus output power. It is NOT a measure of input voltage versus output voltage. Power is voltage times amperage."

### 3.2 数据重构

| 测量项 | 空载 | 带发电机负载 |
|:---|:---:|:---:|
| 输入电压 | 12V | 12V |
| 输入电流 | I₀（基线） | I₀ + ΔI（ΔI > 0） |
| 发电机输出电压 | 18V（空载峰值） | 6~8V（带载） |
| 发电机输出电流 | 0 | 取决于负载 |

**核心结论**：空载发电机输出 18V（高），但一旦接负载，输出电压掉到 6~8V，同时输入电流增加。这说明**发电机线圈和电动线圈之间有磁耦合**——负载电流的磁场反射回电源路径。

### 3.3 为什么两个线圈间不是独立隔离？

```
电感耦合 = 变压器的基本物理：

线圈 A (电动)  ↔ 铁心 ↔ 线圈 B (发电)
      ↑                    ↑
   电源电流           负载电流在铁心产生磁通
                      |
                这个磁通与线圈A的磁通相互干扰
                      |
                线圈A感受到额外"负载"
                      |
                需要更多电池电流来克服
```

**要打破这个耦合，需要什么？**
- 周治平的答案：磁饱和 → 楞次失效时的铁心行为与电感耦合时完全不同
- Ufopolitics 没有做到这一点

## 五、Figuera 发电机关联（Page 10 提及）

### 5.1 Figuera 发电机的基本原理

Clemente Figuera (1902)：用**旋转鼓式换向器**产生多相交流，通过感应耦合驱动多个次级线圈。Figuera 声称他的发电机产生的功率比输入多——因为次级线圈的感应电流来自空间磁场变化，不是直接由原动机提供。

### 5.2 Figuera → Ufopolitics → SEG 的演变

```
Figuera (1902)：
    鼓式换向器 → 多相脉冲 → 多个次级线圈感应
         ↓
Ufopolitics (2012)：
    光电换相 → 多组独立线圈 → 每组兼作电动/发电
         ↓
SEG (1960s)：
    滚柱机械换相 → 12/22/32 独立通道 → 同组线圈兼作电动/发电
```

三者都用了**多通道脉冲 + 独立线圈组**的同一基本架构。Figuera 最早提出这个想法，Ufopolitics 用现代 MOSFET+光耦重新实现，SEG 用机械滚柱在更高能量的版本上实现。

### 5.3 Figuera 的数学结构

Figuera 鼓式换向器产生近似正弦的脉冲序列：
$$V_n(t) = V_0 \sin(\omega t + n\phi)$$
n=1..N 个通道，φ = 2π/N 的相移

如果 N=5（五边形），这就是 Ufopolitics 的 5 极拓扑的数学基础。
