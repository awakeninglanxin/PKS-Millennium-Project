# 非对称电机实操指南
## 从零搭建 + 测量 + 调谐

---

## 一、Dual Pentagon 10极绕线步骤（推荐入门）

### 第1步：拆除原绕组
- 拆开电机，记下原绕线方向（顺时针/逆时针）
- 拆除所有旧线，清洁铁心槽
- 测量每个极的槽尺寸

### 第2步：标记极编号
```
按顺时针标注 1..10 在电枢铁心上：

  第1极 → North 定子方向（正上方）
  第2极 → 右侧
  ...
  第10极 → 左侧
```

### 第3步：绕线

每组 2 个极，用 AWG30 线，每个极绕 150~200 匝：

| 线圈组 | 包含的极 | 匝数/极 | 绕向 |
|:---|:---|:---|:---|
| 组 A | 1 + 6 | 180 | 顺时针 |
| 组 B | 3 + 8 | 180 | 顺时针 |
| 组 C | 5 + 10 | 180 | 顺时针 |
| 组 D | 7 + 2 | 180 | 逆时针 |
| 组 E | 9 + 4 | 180 | 逆时针 |

**关键**：组 D/E 的绕向与 A/B/C **相反**！这建立了磁路的不对称。

### 第4步：连接换向器
- 组 A 的 2 根引线接到换向片 1-2（将 2 片短接在一起）
- 组 B 接到 3-4
- 组 C 接到 5-6
- 组 D 接到 7-8
- 组 E 接到 9-10

### 第5步：安装光耦传感器

```
定子上安装 5 个光电传感器，与 5 组线圈一一对应：

传感器 A 对准线圈组 A 的激励位置
传感器 B 对准线圈组 B （错开 72°）
...
```

## 二、MOSFET 驱动电路

### 2.1 每路电路图（一路一个线圈组）

```
12V ──┬─────────── 线圈组 A ──────┐
      │                           │ D(漏极)
      │                    IRFZ44N │─┤  MOSFET
      │                          G │─┤  Nch
      │                           │ S(源极)
      │      10kΩ                  │
      ├─────┤¯├─────┤  ├────────┤
      │              │          │
      │ 红外光耦      │          │
      │ (接收端)      │          │
      │              │          │
      └──────────────┴──────────┴── GND
```

### 2.2 光耦型号

TCRT5000 或类似反射式红外传感器：
- 发射极：接 12V 经 220Ω 限流电阻
- 接收极：集电极接 10kΩ 上拉电阻到 12V，发射极接 GND
- 输出：反射表面（盘上贴锡箔纸） → 低电平；黑色区域 → 高电平

## 三、甜点调谐步骤

### 3.1 初始测试
1. 用 12V 供电，电机空载运行
2. 测量输入电流（数字万用表）
3. 记录 RPM（光电转速计或手机 RPM 应用）

### 3.2 调光耦位置
- 每次旋转光耦座 5° 变化
- 记录输入电流对应转速

### 3.3 绘制效率曲线

```
横坐标：光耦 ON 窗口角度 (°)
纵坐标：输入电流 (A) / 转速 (RPM)

甜点判断标准：
  - 输入电流出现局部最低谷
  - 同时转速不降或略升
  - 电流谷内还有无自激励波动
```

## 四、测量清单

| 测量项 | 仪器 | 频率 |
|:---|:---|:---|
| 输入电压 DC | 万用表 | 每次调节后 |
| 输入电流 DC | 万用表/电流探头 | 每次调节后 |
| 转速 RPM | 光电转速计 | 每次调节后 |
| 发电机输出 AC | 示波器 | 找到甜点后 |
| 线圈两端电压波形 | 示波器 (×10探头) | 验证甜点时 |
| 磁铁温度 | 红外测温枪 | 运行 5 分钟后 |
| 时间 | 秒表 | 记录热稳定性 |

## 五、常见问题排查

| 问题 | 可能原因 | 修复 |
|:---|:---|:---|
| 电机不转 | 光耦方向反/遮挡盘开孔位置不对 | 用万用表查 MOSFET G 极有无 ON 脉冲 |
| 转矩太弱 | 匝数不够或气隙过大 | 增加匝数或减小磁隙 |
| 声音异常（嘎嘎响） | MOSFET 导通时序错误，多路同时 ON | 用示波器依次测每路射随信号 |
| 输入电流过大但不转 | 线圈短路/换向片焊点粘连 | 万用表测各线圈组电阻是否 ~2-5Ω |
| 甜点找不到 | ON 窗口范围不足 | 扩大光耦触发弧角，或换更大直径的遮光盘 |
