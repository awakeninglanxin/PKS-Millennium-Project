## 一、反电动势规避的核心原理

### 1.1 传统电机的反电动势问题

```
传统有刷直流电机：

电刷接触到换向片 A → 线圈 A 通电 → 产生 N 极
    ↓ 转子转过一定角度
电刷断开换向片 A，接触换向片 B → 线圈 A 极性反转 → N 变 S
    ↓
线圈 A 中产生的反电动势与输入电压**反向叠加**
    ↓
有效电压 = V_supply - V_backEMF → 输入电流随转速升高而降低
    ↓
转矩下降，效率天花板（理论极限 ~85%）
```

**一句话**：反电动势永远**抵消**电源电压，转速越高越厉害。这是所有对称电机无法打破的物理定律。

### 1.2 非对称绕线的解决方案

```
Ufopolitics 非对称电机：

换向系统不反转线圈极性 → 电流方向始终同一
    ↓
线圈工作时：电动模式（产生转矩）
线圈断开时：发电模式（磁场崩解 → 感应脉冲电压）
    ↓
关键：这个感应脉冲不与输入电压反向！
    ↓
感应脉冲可以被**回收**或**辅助下一个线圈**做功
    ↓
理论效率 >> 100%（有待验证）
```

| 对比 | 对称电机 | 非对称电机 |
|:---|:---|:---|
| 线圈极性 | 旋转中 N→S→N→S 翻转 | 每个线圈固定 N 或 S |
| 反电动势 | 抵消电源电压 | **不抵消——可回收** |
| 电流波形 | 方波(换向跳变) | ON/OFF 脉冲 |
| 能效极限 | ~85% | 理论可 >100% |

## 二、六种绕线拓扑详解

### 2.1 拓扑 A：Dual Pentagon（双五边形）— 10 极电枢

```
线圈分布（10 个极，每 2 个换向片短接为 1 组）：

     ①
  ⑩   ②
 ⑨  ⊕  ③     ⊕ = 电枢中心
  ⑧   ④
   ⑦ ⑥ ⑤

换向片连接：
  [1+2] → 线圈组 A
  [3+4] → 线圈组 B
  [5+6] → 线圈组 C
  [7+8] → 线圈组 D
  [9+10] → 线圈组 E

每组 2 个换向片短接 = 空间上不相邻的 2 个极同时激励
→ 每一瞬间有 2 个极在电动做功，3 个极在待机/回收
```

**关键**：换向片**不是**顺序连接（不是 1→2→3→...），而是**跳跃**着的分组。这打破了传统电机的均匀激励节奏，让不同线圈承担不同的时间角色。

### 2.2 拓扑 B：Dual Star (David Star / 双三角星) — 12 极电枢

```
12 极（4 组 × 3 极）：

每组 3 个换向片短接 → 每组形成一个等效线圈
4 组线圈 = 4 个等效磁极对

线圈分布在 360° 上不是均匀的 30°/pole，
而是通过不相等间距产生**非均匀的三磁激励**。
```

**效果**：4 组线圈在空间上不均匀分布 → 旋转一周中有 4 次"空档" → 在空档期间线圈进入发电模式。

### 2.3 拓扑 C：5 Pole / 2 Stator — 最简单演示机

```
电枢: 5 个极（奇数！传统电机必须是偶数）
定子: 2 个永磁体（南北对置）

奇数极的巧妙处：
当每个极转到 N 定子前时被 ON → 排斥 → 转离
当下一个极到达前 → 中间有空隙（因为极数是奇数！）
→ 这个空隙 = 天然的同时段承载回收窗口
```

### 2.4 多股并绕技术（Trifilar / Quadfilar）

```
Trifilar 并绕（3 股线）：

AWG30 × 3 股 → 同时缠在同一个极上
3 股线的功能分配：
  ├─ 线 1: 电动激励（接主电源）
  ├─ 线 2: 回收/触发（感应回收）
  └─ 线 3: 级联到下一个线圈组

Quadfilar（4 股）：加一路为门控触发
```

### 2.5 非对称时序的本质

```
对称电机：一个线圈 = 360° 每圈 1 个角色切换

非对称电机：同一线圈在 360° 一圈中承担多个角色
├─ 0°~120°: 电动（接收电源）
├─ 120°~180°: 断开（磁场崩解 → 发电）
├─ 180°~300°: 闲置（等待/级联）
└─ 300°~360°: 辅助（回收的感应输出耦合到下一个线圈）
```

**这个时序控制是反电动势规避的关键**：线圈断开后，磁场崩解产生的脉冲与电源电压**不是反向关系**，因为当它崩解时它的输入已经断开了——崩解方向与源方向相同（同相）。

## 三、光电换相系统详解

### 3.1 为什么不用电刷？

传统电刷的致命缺陷：**摩擦 + 自动极性反转**。
- 电刷 = 强制机械换相 → 每次换相就是一次极性反转 → 反电动势波形与电源波形在电刷上相遇 → 产生电弧/火花 → 能量损失

### 3.2 光电换相设计

```
传感器：红外 LED + 光敏三极管对管
触发盘：电机轴上的打孔圆盘（或反光条）

┌──────────────────┐
│ 光耦传感器阵列     │
│  ├─ 传感器 1 → 开关管 1 → 线圈组 A
│  ├─ 传感器 2 → 开关管 2 → 线圈组 B
│  ├─ 传感器 3 → 开关管 3 → 线圈组 C
│  └─ 传感器 4 → 开关管 4 → 线圈组 D
└──────────────────┘

触发逻辑：光束穿过盘上孔洞 → 光敏管导通 → MOSFET开通 → 线圈激励
光束被盘挡住 → 光敏管截止 → MOSFET关断 → 线圈断开（回收阶段开始）
```

### 3.3 转换时序图

```
180° ON 窗口：线圈持续激励
  ←──────→
├─────────┬─────────┬─────────┬─────────┤
  线圈A     线圈B     线圈C     线圈D
  ON        OFF       OFF       ON
(回收A)    (发电B)   (等待C)   (激励D)
```

## 四、"甜点" (Sweet Spot) 物理原理

### 4.1 现象

当光电传感器的 ON/OFF 窗口被调节到某**特定角度**时，观察到：
- **输入电流突然下降**
- **转速保持不变**（甚至略升）
- 从电池取电减少 → 效率显著提高

### 4.2 物理解释（非 overunity）

甜点 = $d\Phi/dt$ 与机械相位的最佳匹配点：

电功率：$P_{\text{in}} = V_{\text{supply}} \times I_{\text{avg}}$

平均电流 $I_{\text{avg}}$ 随占空比 D 增大而增大

但当 D 让磁通变化的速率 $d\Phi/dt$ 与**下一个线圈正在经过定子磁场的相位**同相时：
- 磁场崩解脉冲的感应电压与下一个线圈的源电压**同方向叠加**
- "回收能量"帮助了下一个线圈做功
- 等效于部分驱动能量由回收提供 → 不再完全依赖电池

**这不违反能量守恒**——回收能量本身来自之前的输入电机械能转换，并非凭空产生。但**通过精细的时序控制**，这个回收能量可以高效地重新用于做功，而不形成传统反电动势的"对抗"。

## 五、从 Ufopolitics 到周治平 COP

### 5.1 两者的电路等价性

| Ufopolitics 电机 | 周治平 PWM 电路 | 等价关系 |
|:---|:---|:---|
| 机械旋转 + 定子磁铁 | 电子开关( MOSFET/IGBT ) | 同：由外部信号控制 ON/OFF |
| 光电传感器 | PWM 控制器 | 同：决定开关时序 |
| 线圈电感 | 变压器原边电感 | 同：感性负载 |
| 磁场崩解脉冲 | 反激能量 | 同：电感储能的回收 |
| 甜点的 dΦ/dt 同相 | H > H_S 暗能量激活 | 不同：**Ufopolitics 没到饱和** |

### 5.2 关键差异

```
Ufopolitics：铁芯从未磁饱和（只有永磁体的偏置）
周治平：  铁芯必须被 Push 到 H > H_S

Ufopolitics 甜点 → 效率优化（仍然 COP 可能 >1 但幅度有限）
周治平楞次失效 → 能量放大（COP >> 1 理论可行）
```

### 5.3 将 Ufopolitics 升级到 COP>>1

如果在一个 Ufopolitics 型电机中：
1. 将永磁定子换成**电激磁铁心**（可 PWM 控制）
2. 让激磁电流大到足以让铁心进入饱和 (H > H_S)
3. 在这种状态下脉冲驱动电枢 → 磁饱和区间触发楞次失效
4. 从电枢线圈同时收集附加磁能

→ 电机就从"效率优化"升级为"暗能量放大"

## 八、实操清单：搭建第一台非对称电机

### 所需材料

| 组件 | 规格 | 来源 |
|:---|:---|:---|
| 直流电机 | 5极或10极，可拆卸换向器 | 旧电动工具/模型电机 |
| 磁线 | AWG30 或 AWG28, 200g | 电子市场 |
| 红外光耦 | TCRT5000 或类似 | 电子模块 |
| MOSFET | IRFZ44N × 4（每组线圈1个） | 电子模块 |
| 12V 电池 | 铅酸 7Ah 以上 | 旧 UPS 电池 |
| 示波器 | 至少 2 通道 | 即使最便宜的 DS0120 |

### 步骤

1. 拆开电机，拆除原电刷和换向器
2. 在转轴上安装打孔遮光盘
3. 移除原绕组，重绕为新拓扑（从 Dual Pentagon 10 极开始最简单）
4. 安装光耦传感器在定子上
5. 焊接 MOSFET 驱动板
6. 连接电池 + 示波器
7. 调光耦位置找甜点
