# 07 PWM+MOSFET 实践电路设计指南
## Ufopolitics 非对称电机的最简可搭建电路

---

## 一、主功率电路

### 1.1 MOSFET 选择

| 型号 | Vds_max | Id_max | Rds_on | 适用场景 |
|:---|:---:|:---:|:---:|:---|
| **IRFZ44N** | 55V | 49A | 17.5mΩ | 12V 电池, 大电流 |
| IRLZ44N | 55V | 47A | 22mΩ | 逻辑电平驱动(5V Gate) |
| IRF3205 | 55V | 110A | 8mΩ | 大功率(超低导通电阻) |

**推荐：IRFZ44N** — 便宜(¥2/颗), 12V 电池电压内工作良好, 49A 足够

### 1.2 栅极驱动

```
                +12V
                 │
              ┌──┴──┐
              │ 10kΩ │ (上拉电阻, 确保默认为低)
              └──┬──┘
                 │
  光耦输出 ──────┼─── MOSFET Gate
                 │
              ┌──┴──┐
              │ 10kΩ │ (下拉电阻)
              └──┬──┘
                 │
                GND
```

**重要：MOSFET 是电压驱动器件 — 不要直接接光耦输出到 Gate，需要上下拉电阻！**

### 1.3 自由续流二极管 (Flyback Diode)

```
               +12V ────┬─────── 线圈
                        │
                     ┌──┴──┐
                     │ 线圈  │  
                     └──┬──┘
                        │
                    ┌───┴─── D
                    │ MOSFET
                    └───┬─── S
                        │
          ┌─────────────┼───────────┐
          │             │           │
    续流二极管       ┌──┴──┐      │
    (UF4007)          │ 电流 │      │
    反并联在          │ 传感器│      │
    线圈两端          └──┬──┘      │
          │             │           │
          └─────────────┴───────────┘
                        │
                       GND
```

**续流二极管的作用**：当 MOSFET 关断时, 线圈电感试图维持电流 → 高压尖峰 → 通过续流二极管泄放到电源 → 保护 MOSFET 不被击穿。

**但如果要回收这个脉冲(Bedini 方式)**：续流二极管不接地, 而是接到充电电池或电容。

## 二、光耦传感器电路

### 2.1 TCRT5000 接线

```
    TCRT5000
    ┌──────┐
    │  LED │────── 220Ω ─── +5V    (发射极)
    │      │────── GND
    │      │
    │  PT  │────── 10kΩ ─── +5V    (接收极-集电极)
    │      │────── GND              (接收极-发射极)
    └──────┘

输出(从 PT 集电极引出): 光反射时 = 低电平 (~0.2V),
                      无反射时 = 高电平 (~5V)
```

### 2.2 多个传感器并联

```
5 个 TCRT5000 共用同一个 5V 电源：
  5V ───┬─── 220Ω ─── LED1 ─── GND
        ├─── 220Ω ─── LED2 ─── GND
        ├─── 220Ω ─── LED3 ─── GND
        ├─── 220Ω ─── LED4 ─── GND
        └─── 220Ω ─── LED5 ─── GND

每个 PT 输出独立驱动一路 MOSFET
```

## 三、电池与保险

### 3.1 推荐电池

| 类型 | 容量 | 优势 | 局限 |
|:---|:---|:---|:---|
| 铅酸 12V/7Ah | 适合初测 | 耐过充, 便宜 | 重, 老化快 |
| LiFePO4 12V/10Ah | 长期用 | 轻, 循环寿命长 | 贵, 需要 BMS |
| 锂电池包 3S | 最轻 | 高功率密度 | 需要平衡充电 |

### 3.2 保护电路

```
  +12V ───┬─── 5A 保险丝 ─── 主电路
          │
       ┌──┴──┐
       │ 100μF│ (电解, 滤波)
       │  25V │
       └──┬──┘
          │
       ┌──┴──┐
       │ 0.1μF│ (陶瓷, 高频噪声)
       └──┬──┘
          │
         GND
```

## 四、测量电路

### 4.1 电流传感器

使用 ACS712 (5A 版本) 电流传感器在电池正极：

```
  +12V ─── ACS712 ─── 主电路
              │
         Vout ─── 示波器/Arduino ADC
              │
         0.185V/A (灵敏度)
```

### 4.2 电压分压器（测量发电机输出）

```
  发电机输出端子 ────┬─── 10kΩ ───┬─── 示波器通道2
                     │            │
                     │         ┌──┴──┐
                     │         │ 1kΩ │ (分压 = 10:1)
                     │         └──┬──┘
                     │            │
                     └────────────┴─── GND

V_示波器 = V_发电机 × 1/(10+1) = V_发电机/11
```
