# CalcPlot3D URL Generator

## 触发条件

当用户输入代数曲面公式并想让它在 CalcPlot3D 里可视化时使用。

触发词：CalcPlot3D、3D 蛋形、3D 曲面链接、可视化这个方程、生成链接、把这个公式放到网上看 3D 图、帮我生成 CalcPlot3D 链接

## 工作流程

1. 接收用户输入的代数方程（如 `x^2+y^2+z^2=1-z^3`）
2. 调用 `scripts/generate.py` 生成正确 URL
3. 自动打开浏览器预览
4. 告知用户关键信息：方程形式、域范围、可用滑块

## 核心知识点（从踩坑中提炼）

### URL 语法
- `~` = 等号（因为 `=` 是 URL 的 key=value 分隔符）
- `^` → `%5E`
- `type=implicit` → 隐式曲面，`type=slider` → 滑块，`type=window` → 必须放最后

### 相机参数
CalcPlot3D 的 center/focus/up 是**四元数编码的齐次坐标**，不能随便填。
经验证可用的默认相机：
```
center=0.6,-39.4,6.75,1
focus=0,0,0,1
up=0.03,0.26,0.96,1
```

### 域推断规则
- 常数 RHS：x,y,z ∈ [-√|C|, √|C|]
- 含 z³：z ∈ [-1.5, 2]（不对称）
- 含指数：z ∈ [-1.5, 1.5]
- 默认：[-2, 2]

### 参数检测
方程中出现的 a,b,c,d 自动加滑块（value=1, range=[0,3]）
可用参数名：仅 a,b,c,d,t

## 已知限制
- 暂不支持柱坐标 / 球坐标
- 不支持空间曲线 / 参数曲面 / 向量场
- 相机参数为固定默认值（如需精细调整需先在 CalcPlot3D 界面里摆好位置再 File→Encode View in URL）
