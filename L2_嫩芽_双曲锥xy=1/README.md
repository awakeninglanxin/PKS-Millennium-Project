# 🌱 第 2 层 · 嫩芽 — 双曲线 xy=1 → 双曲锥

> **种子发的第一根芽。** a·b=1 画在坐标纸上就是双曲线 xy=1，转一圈就成了双曲锥（沙漏/漏斗形）。
> 项目里所有立体形状，都是这个沙漏的变体。
> 预计停留：1 周

## 这一层要搞懂什么

**目标**：看到"沙漏形/漏斗形"就想到 xy=1，理解"水锥、螺旋管、波纹盘都是这个沙漏的变形"。
