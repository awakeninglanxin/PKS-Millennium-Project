## 〇、公理：pq=1 极化原理

Schauberger/PKS 体系的第一性原理：

$$p \cdot q = 1$$

**物理来源**：毕达哥拉斯单弦琴 $l \cdot f = 1$（弦长×频率=常数）
**数学等价**：开普勒第二定律 $r \cdot v = \text{const}$、角动量守恒

**关键在于**：将这条公理中的两个变量分别赋予不同的函数形式，产生两个不同的锥体。

## 第一篇：直圆锥（电性分支）—— $p = x$

### 1.1 锥面方程

令 $p = x$，则自公理得 $q = 1/x$，但**这里只取 p=x 的分支**。

将 $p = x$ 直接解释为三维中的半径约束：

$$z = \sqrt{x^2 + y^2} = r$$

即**直圆锥**（right circular cone），锥半角 $= 45^\circ$（因为 $\tan(45^\circ) = 1 = r/z$）。

### 1.2 斜切面的参数

用一个平面斜切该锥。切面由两个参数决定：

- $h$：切面与锥轴（z 轴）的交点高度
- $\gamma$：切面与锥轴的夹角（$\gamma = 0$ → 水平切面）

### 1.3 截面类型判别

圆锥曲线的经典结论：截面类型取决于切面与锥面母线的相对角度。

锥面母线与轴的夹角 $= 45^\circ$（因为 $r = z$，即 $\tan\theta = 1$，$\theta = 45^\circ$）。

切面与轴的夹角 $= \gamma$。比较两角：

| 条件 | 几何关系 | 截面 | 数学形式 |
|------|----------|------|----------|
| $\gamma < 45^\circ$ | 切面比母线更平 | **椭圆** | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ |
| $\gamma = 45^\circ$ | 切面平行于母线 | **抛物线** | $y = ax^2 + bx + c$ |
| $\gamma > 45^\circ$ | 切面比母线更陡 | **双曲线** | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ |

特例：$\gamma = 0$（水平切面）→ **圆**（$a = b$，椭圆的退化）。

### 1.4 椭圆参数

对于 $\gamma < 45^\circ$（正常市场），椭圆的长短轴：

$$a = \frac{h}{\cos\gamma}, \quad b = h$$

$$e = \sin\gamma \quad (\text{离心率})$$

- $\gamma = 0$ → $a = b = h$ → 圆（$e=0$）
- $\gamma \to 45^\circ$ → $a \to \infty$ → 抛物线极限（$e \to 1$）

### 1.5 股市映射：直圆锥参数

| 几何参数 | 股市提取 | 物理含义 |
|----------|----------|----------|
| $h$（切面高度） | $\text{MA}_{60}$（60日均价） | 市场价格基线 |
| $\gamma$（切面角） | $\arctan(\Delta p_{\text{range}} / \text{MA}_{60})$ | 价格波动烈度 |

$\Delta p_{\text{range}}$ 可取 $\max(60日最高 - \text{MA}_{60}, \text{MA}_{60} - 60日最低)$ 以捕获最大偏离方向。

## 第二篇：超双曲锥（磁性分支）—— $q = 1/x$

### 2.1 锥面方程

令 $q = 1/x$（即保持 $pq=1$ 中的 $q$ 为 $1/x$ 形式）。

在三维中绕 $y$ 轴旋转，得**超双曲锥**：

$$z = \frac{1}{\sqrt{x^2 + y^2}} = \frac{1}{r}$$

这是 Schauberger 体系的核心锥体，代表非线性能量场（涡旋、湍流）。

### 2.2 斜切面的参数

切面由两个参数决定：

- $z_0$：切面与 $z$ 轴交点的纵坐标（"切割高度"）
- $\alpha$：切面与水平面的夹角

### 2.3 蛋形曲线的数学推导

#### 步骤 1：切面方程

切面通过点 $(0, z_0)$，与水平面夹角 $\alpha$。在 $xz$ 平面内的直线方程：

$$z = z_0 - x \cdot \tan\alpha$$

#### 步骤 2：代入锥面求交线

将锥面 $r = 1/z$（即 $x^2+z^2 = 1/z^2$... 注意此处 $z$ 是高度而非 $z$ 坐标）代入切面方程。

**纠正坐标系**：在标准的 PKS 表示中，锥面方程在 $(x, y, z)$ 空间为 $r = 1/y$，切面用 $y = z_0 - z \cdot \tan\alpha$ 表示（这里 $y$ 是原双曲线的纵坐标，在三维中成为高度轴）。

交点条件：

$$\frac{1}{\sqrt{x^2+z^2}} = z_0 - z \cdot \sin\alpha$$

令 $R = \sqrt{x^2+z^2}$ 为旋转半径，则：

$$R = \frac{1}{z_0 - z \cdot \sin\alpha}$$

又因旋转关系 $x^2+z^2 = R^2$，且 $z$ 方向坐标在切面内的投影为 $z \cos\alpha$...

#### 步骤 3：蛋形截面的显式半径公式

经完整推导（详见原著 §3），蛋形截面在切面极坐标 $(\rho, \varphi)$ 下的显式形式：

$$\boxed{ \rho(\varphi) = \frac{1}{2\cos\varphi \cdot \sin\alpha} \left[ z_0 \pm \sqrt{ z_0^2 - \frac{4\cos\varphi \cdot \sin\alpha}{\sqrt{\sin^2\varphi + \cos^2\varphi \cdot \cos^2\alpha}} } \right] }$$

**符号选择**：$+$ 得锥管截面的"大圈"，$-$ 得扁平"山脉"截面。

### 2.4 蛋形存在性判别

根号内的表达式必须非负。最严约束出现在 $\varphi = 0$（极角零点方向）：

$$\rho \text{ 存在} \iff z_0^2 - \frac{4\cos 0 \cdot \sin\alpha}{\sqrt{\sin^2 0 + \cos^2 0 \cdot \cos^2\alpha}} \geq 0$$

化简：$\cos 0 = 1$, $\sin 0 = 0$, $\sqrt{\cos^2\alpha} = \cos\alpha$

$$z_0^2 - \frac{4\sin\alpha}{\cos\alpha} = z_0^2 - 4\tan\alpha \geq 0$$

$$\boxed{z_0 > 2\sqrt{\tan\alpha}} \quad (\text{蛋形存在的几何必要条件})$$

### 2.5 两点形式（原著 §4）：从顶点反推切割参数

这是 Schauberger 原著中最实用的公式。已知蛋形的两个主顶点 $z_1$（尖端）和 $z_2$（钝端）——两者都在双曲线 $y = 1/z$ 的正半支上。

切割参数由下式精确给出：

$$\boxed{ z_0 = \frac{z_1^2 + z_2^2}{z_1 + z_2} }$$

$$\boxed{ \tan\alpha = -\frac{z_1 z_2 (z_2 - z_1)}{z_1 + z_2} }$$

**推导**：利用 $z_1$ 和 $z_2$ 在双曲线 $y = 1/z$ 上的坐标 $(z_1, 1/z_1)$ 和 $(z_2, 1/z_2)$，代入切面方程 $y = z_0 - z \cdot \tan\alpha$，解联立方程组。

**符号约束**：$z_1, z_2 > 0$（都在正半支），且 $z_2 > z_1$（钝端 > 尖端），故 $\tan\alpha < 0$（$\alpha$ 为负角）。

### 2.6 特例：$\alpha = 0$

水平切面 $\alpha = 0$ → $\tan\alpha = 0$, $\sin\alpha = 0$, $\cos\alpha = 1$。

蛋形半径公式退化为：

$$\rho(\varphi) = \frac{1}{0} \left[ z_0 \pm \sqrt{z_0^2 - 0} \right] \quad (\text{需要取极限})$$

极限结果：**圆**。水平切超双曲锥 = 圆形截面。这和直圆锥的 $\gamma = 0$ → 圆完全相同——两个锥在对称极限处趋同。

### 2.7 股市映射：超双曲锥参数

| 几何参数 | 股市提取（原著两点公式） | 物理含义 |
|----------|--------------------------|----------|
| $z_1$ | $60日最低价 / \text{MA}_{60}$ | 尖端：价格下界 |
| $z_2$ | $60日最高价 / \text{MA}_{60}$ | 钝端：价格上界 |
| $z_0$ | $(z_1^2+z_2^2)/(z_1+z_2)$（精确） | 市场中心化程度 |
| $\alpha$ | $\arctan[-z_1 z_2(z_2-z_1)/(z_1+z_2)]$（精确） | 市场极化不对称度 |

### 3.2 截面相变总表

| $\gamma$（直锥角） | 直圆锥截面 | $\alpha$（超双曲锥角） | 超双曲锥截面 | 市场状态 |
|:---:|------|:---:|------|------|
| $0°$ | **圆** | $0°$ | **圆** | 🟢 完全对称，窄幅盘整 |

### 3.3 金融决策规则

```
对每只股票，提取两组截面信息：

IF γ > 45°        → ⚫ 双曲线：截面不闭合，跳过（崩溃或暴涨失控）
ELIF γ → 45°      → 🔴 抛物线临界：密切关注，即将突破
ELSE               → 🟢 椭圆：价格被约束，均值回归有效

IF 蛋形不存在      → ⬚ 无动量信号，无方向
ELIF |α| 小        → 🟢 弱蛋形：温和趋势
ELSE               → 🟡 强蛋形：明确动量，高 z₀+大|α| = 最优
```
