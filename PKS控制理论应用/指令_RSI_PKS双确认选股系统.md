## 二、RSI 50 穿越公式

```
RSI = 100 − 100/(1 + RS)
RS  = avg_gain(14) / avg_loss(14)

穿越条件:
  前一日 RSI ∈ [30, 45]  — 空方占优
  当日   RSI > 50       — 多方夺回控制权
```

数学本质：多空力量比 RS 从 <1 变成 >1。RSI=50 是唯一的中性平衡点。

## 三、PKS G 级公式

```
k_E = r₁/r₂  (蛋形不对称度)
搜索: |k_E − 1.937| / 1.937 < 0.15  (黄金比附近)
且:   z₀ ≥ 10  (趋势空间充裕)

k_E* = 1.937 来自黄金比极限方程: Φ·k² − Φ²·k − 1 = 0
对应 α* = 31.72°, γ ≈ 17.7°
```

几何含义：超双曲锥的蛋形截面恰好达到黄金比例不对称度——自然均衡中即将突破。
