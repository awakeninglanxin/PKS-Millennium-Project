## 二、核心公式（v2.4 最终版）

```python
# 从60日K线提取
r₁ = max((高-MA60)/MA60, (MA60-低)/MA60)   # 大端径向偏差
r₂ = min((高-MA60)/MA60, (MA60-低)/MA60)   # 小端径向偏差

# 直圆锥（椭圆截面）
h = 2·r₁·r₂ / (r₁+r₂)          # 调和平均 → 市场尺度
γ = arctan((r₁-r₂)/(r₁+r₂))    # 归一化不对称度 → 趋势烈度

# 超双曲锥（蛋形截面）
u₁=1/r₁, u₂=1/r₂
z₀ = (u₁²+u₂²) / (u₁+u₂)      # 原著两点公式 → 市场温度
α  = arctan[-u₁·u₂·(u₂-u₁)/(u₁+u₂)]  # 极化度
c  = (z₀·√2/2)·tan(2α)         # 两锥耦合

egg_safety = z₀ / 2·√(tan|α|)  # 蛋形安全裕度
```

### PKS v2.4 评级规则

```
A: γ≥20° + h>池中位数（趋势扎实） + 周期股:c≥0
B: γ≥20° + h≤池中位数
C: γ<20° + egg_safety≥2
D: γ<20° + egg_safety<2
X: γ>30° 或 c<-1 或 c>3（极端排除）
```
