## 三、核心公式（v2.4 最终版，别再改了）

```python
# 从60日K线提取
r₁ = max((高-MA60)/MA60, (MA60-低)/MA60)   # 大端径向偏差
r₂ = min((高-MA60)/MA60, (MA60-低)/MA60)   # 小端径向偏差

# 直圆锥（椭圆截面）
h = 2·r₁·r₂ / (r₁+r₂)          # 调和平均 → 市场尺度
γ = arctan((r₁-r₂)/(r₁+r₂))    # 归一化不对称度 → 趋势烈度 ⭐最重要

# 超双曲锥（蛋形截面）
u₁=1/r₁, u₂=1/r₂
z₀ = (u₁²+u₂²)/(u₁+u₂)         # 市场温度
α  = arctan[-u₁·u₂·(u₂-u₁)/(u₁+u₂)]  # 极化度
c  = (z₀·√2/2)·tan(2α)         # 两锥耦合（仅周期股有效）
egg_safety = z₀/2·√(tan|α|)    # 蛋形安全裕度
```

### 评级规则
```
A: γ≥20° + h>池中位数（趋势扎实）+ 周期股c≥0
B: γ≥20° + h≤池中位数
C: γ<20° + egg_safety≥2
D: γ<20° + egg_safety<2
X: γ>30° 或 c<-1 或 c>3（极端排除）
```

## 四、⭐ 最重要的实战铁律（三轮验证得出）

### 铁律 1：γ 25-30° 是"铁律中的铁律"
- **3/3 轮跨池验证**：始终 +2.2%~+7.5%，胜率 52%~60%
- v3成长池 [25,30) 区间：20日收益 **+7.86%，胜率 59.6%**（全场最高）
- **这是唯一跨所有池型都稳健的单一因子**

### 铁律 2：PKS 只在"纯成长/动量池"里有效
| 池型 | PKS vs 全量 | 结论 |
|------|-----------|------|
| v1 96fen成长(游戏/芯片/医药) | +3.36% > +2.38% | ✅ PKS赢 |
| v2 消费/新能源/医药**混合** | +0.69% < +0.93% | ❌ PKS输 |
| v3 半导体/AI/军工成长 | +3.02% > +2.07% | ✅ PKS赢 |

**→ 混合池里别用 PKS，会跑输全量。**

### 铁律 3：不需要智能路由切换
- 早期想按"牛/熊/震荡"大盘环境切换算法，**证伪了**
- 正确做法：**始终 96fen→PKS 固定流水线**，在成长池全环境都赢

### 铁律 4：PKS 是 96fen 的天然搭档
- 96fen 选出来的天然是高波动量股，**恰好落在 PKS 的甜区**
