## 一、核心命题

**PKS（Pythagoras-Kepler-Schule）几何体系不是一套脱离现实的数学游戏。它提供了一族天然的非对称 Lyapunov 函数，可以直接嵌入现代非线性控制的最前沿框架（CLF + CBF + QP），实现经典椭圆 CLF 做不到的事情。**

## 三、蛋形曲线的数学

### 3.1 显式形式

$$y = f(x) = z_0 \cdot \sin\alpha \pm \sqrt{\frac{1}{x^2 + (z_0 \cos\alpha)^2} - (z_0 \cos\alpha)^2}$$

### 3.2 参数含义

| 参数 | 几何意义 | 控制意义 |
|------|----------|----------|
| z₀ | 切割高度（双曲锥顶点到切割平面的垂直距离） | 控制蛋形整体大小 |
| α | 切割平面与水平面的夹角 | 控制蛋形不对称程度 |
| z₁ | 蛋形尖端顶点（离切割面中心最近的点） | "高灵敏度方向"的极限位置 |
| z₂ | 蛋形钝端顶点（离切割面中心最远的点） | "低灵敏度方向"的极限位置 |

### 3.3 谐波音程与蛋形的对应

Walter Schauberger 发现不同音程比例产生不同蛋形：

| 音程 | 比例 | 蛋形特征 | 工程用途 |
|------|------|----------|----------|
| 八度 | 1:2 | 对称性最强 | 通用安全走廊 |
| 五度 | 2:3 | 中等不对称 | 机械臂精细操作 |
| 四度 | 3:4 | 较强不对称 | 火箭着陆偏航约束 |
| 大三度 | 4:5 | 强不对称 | 手术机器人亚毫米精度 |
| 小三度 | 5:6 | 极强不对称 | 无人机贴线飞行 |

## 四、PKS-CLF（蛋形控制 Lyapunov 函数）

### 4.1 构造

$$V(x) = \frac{1}{(z_0 - x_2 \sin\alpha)^2} + (x_2 \cos\alpha)^2 + x_1^2$$

核心性质：
- V 的等值面就是蛋形曲线（天然非对称）
- 在原点附近正定（满足 CLF 条件）
- 尖头方向梯度陡 → 大控制量 → 高精度
- 钝头方向梯度缓 → 小控制量 → 快速运动

### 4.2 控制律

对系统 ẋ = f(x) + g(x)u，要求：

$$\dot{V}(x) = L_f V(x) + L_g V(x) \cdot u \leq -\gamma V(x)$$

反解出 u，即得让系统沿蛋形螺旋收敛的控制律。

### 4.3 与经典椭圆的量化对比

| 维度 | 经典椭圆 CLF (V = xᵀPx) | PKS 蛋形 CLF |
|------|-------------------------|-------------|
| 等值面 | 对称椭圆 | **非对称蛋形** |
| 收敛模式 | 只能渐近收敛到原点 | 可选：原点 或 极限环 |
| 调参方式 | 试凑 P 矩阵元素 | **按音程比例选 (z₁, z₂)** |
| 避障 | CBF 需额外设计 | 蛋形天然非对称，尖头方向自动高灵敏 |
| 参数物理意义 | P 矩阵无直观几何 | 直接对应蛋形顶点几何 |

## 五、PKS-CBF（蛋形安全屏障函数）

### 5.1 构造

$$h(x) = \frac{1}{(z_0 - x_2 \sin\alpha)^2} - x_1^2 - (x_2 \cos\alpha)^2 \geq 0$$

安全约束：`ḣ(x) + η · h(x) ≥ 0`

### 5.2 非对称安全走廊

蛋形 CBF 的核心优势：**不需要"一刀切"的保护圈**。

```
传统球形 CBF：         PKS 蛋形 CBF：
    ●──────●              ╭──────╮
   /        \            ╱  安 全  ╲
  │  安全区  │          │  走  廊   │
   \        /            ╲  区 域  ╱
    ●──────●              ╰──────╯
                          ↑       ↑
                        尖头     宽腹
                      （高精度）（快速）
```

## 六、PKS-CLF-CBF-QP 完整架构

每个控制周期求解：

$$\begin{aligned}
\min_u \quad & \frac{1}{2} u^2 \\
\text{s.t.} \quad & \dot{V}(x) + \gamma V(x) \leq 0 \quad \text{(CLF：保收敛)} \\
& \dot{h}(x) + \eta h(x) \geq 0 \quad \text{(CBF：保安全)}
\end{aligned}$$

**性能指标**：每周期 ~10-50 微秒，可嵌入式实时运行。

## 七、羚羊角管：极限环控制

### 7.1 3D 参数方程

$$\begin{cases}
x(t, \phi) = e^{-\lambda t} \cdot r_{\text{egg}}(\phi) \cdot \cos(\omega t + \kappa t + \phi) \\
y(t, \phi) = e^{-\lambda t} \cdot r_{\text{egg}}(\phi) \cdot \sin(\omega t + \kappa t + \phi) \\
z(t, \phi) = t \cdot \cos\theta
\end{cases}$$

### 7.2 κ 参数的魔性

| κ 值 | 动力学行为 | 工程对应 |
|------|-----------|----------|
| κ = 0 | 收敛到中心点（点吸引子） | 火箭着陆最终位置 |
| κ > 0 | 绕中心做极限环（周期吸引子） | 卫星轨道保持、编队飞行 |
| κ ≫ 0 | 快速螺旋转动 | 无人机群编队相位差 |

## 九、与 Schauberger 体系的深层连接

PKS 不是凭空创造的数学——它根植于 Viktor Schauberger 的自然观察：

| Schauberger 观察 | PKS 数学化 | 工程对应 |
|-----------------|-----------|----------|
| 溪流中的鳟鱼在激流中静止不动 | 羚羊角管 κ > 0 的极限环平衡 | 卫星在引力涡旋中的轨道保持 |
| 河流自然形成螺旋涡旋 | 双曲锥斜切→蛋形截面 | 管道流体力学优化 |
| 抛入水中的石块以螺旋轨道沉降 | 蛋形螺旋收敛 | 火箭软着陆轨迹 |
| 森林土壤的温跃层 | (k,b) 时变参数蛋形 | 自适应控制器 |
| 直管输水效率低、弯管效率高 | 螺旋扭转 vs 直线衰减 | AGV 通道规划 |
