## 案例 1：🚀 火箭垂直着陆控制

### 问题
SpaceX Falcon 9 的可回收火箭需要在着陆时精确避开发射架、人员区和障碍物，同时保证软着陆。

### PKS 方案
- **蛋形方向性**：尖头对准发射架方向（高灵敏度、零容忍）
- **宽腹方向**：留给安全沙地（有一定容错空间）
- **z1:z2 = 3:4（四度音程）**：非对称性适中

### 控制目标
1. 位置误差 < 1m（着陆平台半径）
2. 速度 < 0.5 m/s（软着陆）
3. 攻角 < 5°

### 蛋形 CBF 安全约束
```
h(x) >= 0 → 火箭始终在蛋形安全走廊内
尖头方向（发射架方向）：距边界 < 2m 即触发紧急制动
宽腹方向（沙地方向）：距边界 < 10m 才触发规避
```

### 核心优势 vs 传统方案
| 传统 | PKS 蛋形 |
|------|----------|
| 球形安全区（半径=50m）→ 浪费开阔空间 | 蛋形安全区 → 危险方向紧、安全方向松 |
| 障碍物列表中每个需单独建模 | 蛋形 CBF 一次性编码方向性安全约束 |
| 凸优化避障在密集障碍物中无解 | QP 每步 10μs，总能找到可行解（或接近可行解） |

传统：地面计算最优轨迹 → 上传指令序列 → 星上开环执行

## 案例 3：🚗 自动驾驶车道保持

### 问题
自动驾驶需要在车道内安全行驶。左侧实线不可越过，右侧虚线在紧急时可越过。

### PKS 方案
- **蛋形方向性**：尖头对准左侧实线（高灵敏度、不可越）
- **宽腹对准右侧虚线**（允许偏右）
- **z1:z2 = 5:6（小三度）**：极强方向不对称

### 数学映射
```
蛋形安全走廊 CBF:
  h(x) >= 0 ← 车辆在车道内
  ḣ + η·h >= 0 ← 保证不越线

尖头方向（左实线）：h 对左偏极其敏感 → d(h)/d(左偏) 很大
宽腹方向（右虚线）：h 对右偏容忍 → d(h)/d(右偏) 较小
```

### 对比传统
| 传统 LQR/LTV-MPC | PKS 蛋形 CLF-CBF-QP |
|------------------|---------------------|
| 对左右约束等同对待 | 方向性安全约束 |
| 车道变窄需要重新整定 | 调 z1:z2 比例即时适配 |
| 计算量随预测时域增加 | 固定每步 QP 求解时间 |

## 案例 4：🤖 工业机械臂精密装配

### 问题
六轴机械臂在装配时需要：靠近工件时高精度（亚毫米）、远离工件时快速运动。

### PKS 方案
- **蛋形方向性**：尖头对准工件（趋近时精度极高）
- **宽腹远离工件**（退回时速度快）
- **z1:z2 = 2:3（五度音程）**：中等不对称

### 两种模式切换
```
模式 A（趋近）：
  尖头方向 = 工件方向
  控制精度：位置误差 < 0.1mm
  速度上限：0.05 m/s

模式 B（退回）：
  宽腹方向 = 远离工件
  速度上限：2 m/s（快速回到起点）
  精度要求：位置误差 < 5mm
```

### PKS 的优势
- 同一个蛋形 CLF 同时处理两种模式（只需翻转蛋形方向）
- 不需要"模式切换"的离散逻辑，避免切换时的冲击

## 案例 9：🐟 Schauberger 鳟鱼跳跃——自然界中的蛋形控制

### 观察
Viktor Schauberger 观察到鳟鱼在激流中可以"静止不动"——不消耗能量地停留在同一个位置，即便水流很猛。

### PKS 解释
鳟鱼利用的是**羚羊角管 κ > 0 的极限环动力学**：
- 鱼体周围的涡旋形成一个蛋形截面的漏斗
- κ > 0 使水流形成螺旋扭转 → 鱼在涡旋中心处于稳定平衡
- 鱼不需要主动游动，只需要微调鳍的角度来修正 κ

### 工程启示
这个自然现象的工程对应是：
- 卫星在行星引力涡旋中的**无燃料轨道保持**
- 飞行器在旋涡气流中的**被动稳定**
- MWO 线圈中电磁场的**自持谐振**
