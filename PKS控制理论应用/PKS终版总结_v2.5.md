## 核心公式

```python
# 60日窗口
r₁ = max((H-MA60)/MA60, (MA60-L)/MA60)
r₂ = min((H-MA60)/MA60, (MA60-L)/MA60)
u₁, u₂ = 1/r₁, 1/r₂

γ  = atan((r₁-r₂)/(r₁+r₂))          # 趋势方向 [0,45°]
h  = 2·r₁·r₂/(r₁+r₂)                # 趋势结构
z₀ = (u₁²+u₂²)/(u₁+u₂)              # 趋势空间
c  = (z₀·√2/2)·tan(2α)              # 两锥耦合
```
