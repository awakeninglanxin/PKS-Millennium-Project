## 三、PKS v2.4 公式（生产版）

### 参数计算

```
60日窗口：r₁=上偏MA60, r₂=下偏MA60

γ  = arctan[(r₁−r₂)/(r₁+r₂)]  趋势烈度（唯一核心）
h  = 2r₁r₂/(r₁+r₂)           市场尺度（调和平均）
c  = (z₀√2/2)·tan(2α)        两力耦合（仅周期股用）
z₀ = (u₁²+u₂²)/(u₁+u₂)       市场温度
egg_safety = z₀/2√(tan|α|)   安全裕度（高波池有效）
```

### 评级规则

```
A: γ≥20° + h>池中位数 + (周期:c≥0)
B: γ≥20° + h≤池中位数
C: γ≥10° + h>池中位数
D: γ<10°
X: γ>30° 或 c<−1 或 c>3 或 (高波:egg_safety<1.5)
```
