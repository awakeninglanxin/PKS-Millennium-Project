香农信息熵公式：H = −Σ p(i) log p(i)。010（二进制）H=1 → 012（三进制）H≈1.58 → 015（十六进制）H=4.00，恰好是二进制的 4 倍整数。螺旋上升回到 2 的幂，但站在更高的维度。

为什么 015 = 十六进制（0~15）？

不是线性增长，是指数跳变：2 态 → 3 态 → 16 态。这正是"光子信号"相对于"电子/量子宏"应有的代际差距。

🔢 命名逻辑

「015」= 0 到 15 的闭区间，共 16 个值。如同「010」= 0 到 1（2 个值），「012」= 0 到 2（3 个值）。人类用范围表达进制。
