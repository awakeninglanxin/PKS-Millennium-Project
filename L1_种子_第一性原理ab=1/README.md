## 这一层要搞懂什么

**唯一目标**：能用大白话解释——"为什么 a·b=1 是万物的根？"

一句话理解：**a·b=1 = 极性平衡**。像跷跷板，一头下去多少，另一头就上来多少，乘积永远守恒。项目主张大自然处处是这种平衡。
