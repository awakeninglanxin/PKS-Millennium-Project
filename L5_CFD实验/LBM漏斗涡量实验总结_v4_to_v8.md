# LBM D3Q19 PKS 锥体 CFD — 实验总结

> 2026-07-17 · RTX 4090 · 5 轮迭代 (v4→v8) · 累计 GPU ~1.5h

---

## 一、做了什么

用 **LBM (Lattice Boltzmann Method) D3Q19** 在 GPU 上仿真水流通过 PKS 双曲锥体漏斗，观测涡量是否被几何收缩放大。

这是光谱法全线失败后的正确路线——LBM 的**反弹边界 (bounce-back)** 天然处理复杂壁面，不需要 STL→SDF→IBM 的复杂管线。

## 三、v8 的核心技术决策（可复现）

### 3.1 EQ 重置入口（而非体加力）

```python
# ❌ 体加力 → 发散
f[inlet, k] += 3.0 * w[k] * (cx*Fx + cy*Fy + cz*Fz)

# ✅ EQ 重置 → 稳定
feq = w[k] * (1 + 3c·u + 0.5(3c·u)² - 1.5u²)
f[inlet, k] = feq[inlet]  # 直接写到平衡态
```

### 3.2 入口跳过碰撞

```python
collide_zone = (~mask) & (~inlet_mask)  # 碰撞算符不碰入口区
f[collide_zone, k] -= omega * (f - feq)[collide_zone]
```

### 3.3 切向速度剖面

```python
scale = (r / inner_r)²  # 0 在中心, 1 在壁面
ux = -u_tan * (y-cy)/r * scale   # 顺时针
uy = +u_tan * (x-cx)/r * scale
uz = -u_down * scale              # 向下
```

### 3.4 重力全域施加

```python
g_grav = 8e-5  # 5× 前期版本的背景力
f[~mask, k] += 3.0 * w[k] * (-cz) * g_grav  # -Z 方向
```

## 四、为什么 4400 步后崩溃

| 可能原因 | 可能性 |
|------|:---:|
| 切向速度 0.10 在喉部收缩区形成过冲 | 高 |
| 128³ 网格的喉部仅 8 格点 → 梯度数值噪声 | 中 |
| EQ 重置不守恒 → 质量/动量累积 | 低（rho≈1 始终稳定） |

**下一版修复**：`tau=0.58`（增粘性）+ `u_tan=0.08` + `u_down=0.05`，或换基于熵的 MRT 碰撞模型。

---

## 五、与光谱法的根本区别

| | 光谱法（v1-v3 全败） | LBM（v4-v8） |
|------|:---:|:---:|
| 壁面 | 傅里叶模糊层 → 无边界层 | 反弹边界 → 真实无滑移 |
| GPU 模式 | CuPy FFT（全局通信） | 纯邻域（完美 GPU 并行） |
| 复杂几何 | 几乎不可能 | mask 布尔数组 |
| 稳定性 | 恒炸 | 4400步 + 可延长 |
