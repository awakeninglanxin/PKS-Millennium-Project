## 一、羚羊角管物理本质

### 1.1 它不是一根"管子"

| 几何特征 | 物理效应 |
|----------|---------|
| 空间扭转（羚羊角螺旋） | 强制旋流发生器 |
| 蛋形咬口截面（一侧穹顶、一侧凹陷） | 偏心蓄流+二次流诱导 |
| 双轨扫掠渐缩（入口大→出口小） | 变截面文丘里效应→负压泵 |
| 涡旋入流 | 初始角动量注入 |
| 出口骤缩 | 速度↑→压力↓→空化触发 |

### 1.2 物理链

```
涡旋入流 → 螺旋管内角动量守恒 → 切向速度v_θ随半径缩小而增大
    → 离心力将水压向管壁 → 中心形成低压气核
    → 气核气泡沿管壁滑行（受向心力约束不立即溃灭）
    → 管截面最窄处 → 压力骤降 → 气核失稳
    → Rayleigh-Plesset collapse → 脉冲释能
    → 表观负摩擦（压力恢复 > 壁面摩擦 = 净推力）
```

## 三、步骤1：参数化几何生成

### 3.1 几何参数

| 参数 | 符号 | 典型值 | 说明 |
|------|------|--------|------|
| 入口半径 | R_in | 50 mm | 入口截面等效半径 |
| 出口半径 | R_out | 20 mm | 出口截面等效半径 |
| 管长 | L | 300 mm | 沿管轴线的总长度 |
| 扭转率 | τ | 2π/L | 单位长度的扭转角 |
| 蛋形参数 | a | 0.7 | 咬口凹陷深度(0~1) |
| 穹顶参数 | b | 1.3 | 穹顶凸起高度 |

### 3.2 截面曲线参数方程（已升级至 C∞）

蛋形咬口截面原始定义为两段 C0 拼接，**已升级为 C∞ 连续**——实现见 [`蛋咬一口公式生成法——数学原理.md`](蛋咬一口公式生成法——数学原理.md)，含 7 滑块+DXF 导出的交互 Python 脚本。

```
原始分段描述（C0 拼接，ϕ=π 处仅位置连续）：
 穹顶段 (φ ∈ [0, π]):  r(φ) = R₀ · (1 + (b-1)·sin²(φ/2))
 凹陷段 (φ ∈ [π, 2π]): r(φ) = R₀ · (1 - a·sin²(φ/2))

C∞ 升级版（蛋咬一口公式生成法.py）：
  d(t,t0,σ) = exp(-σ·(1-cos(t-t0)))   ← C∞ 周期 dip
  g(t) = [1-k1·d1(t)·asym1]·[1-k2·d2(t)·asym2]   ← 双波乘积，天然 C∞
  (x_g, y_g) = g(t)·(x(t), y(t))

升级原因：C0 拼接在 ϕ=π 处切线不连续→CFD 产生伪湍流→淹没负摩擦力。
         C∞ 确保 ∇P = -∇²u 全场光滑→Schauberger 效应可稳定观测。
```

### 3.3 双轨扫掠

管轴线为螺旋线：
```
x(t) = R_axis · cos(τ·L·t)
y(t) = R_axis · sin(τ·L·t)
z(t) = L · t          (t ∈ [0, 1])
```

管半径沿轴线线性缩减：
```
R₀(t) = R_in + (R_out - R_in)·t
```

## 四、步骤2：微元控制体ODE快速评估

### 4.1 降维假设

将3D螺旋管沿流线降为1D问题，取N个截面，每个截面的流动参数由5个ODE描述。

### 4.2 五个微元方程

**质量守恒**：
$$\frac{d}{dz}(\rho v_z A) = 0$$

**轴向动量**：
$$\rho v_z \frac{dv_z}{dz} = -\frac{dp}{dz} - \frac{2\tau_w}{R_{eq}} + F_{\text{cent},z}$$

**切向动量**（角动量）：
$$\rho v_z \frac{d(rv_\theta)}{dz} = -\frac{2\tau_{w,\theta}}{R_{eq}}$$

**能量方程**：
$$\rho v_z \frac{d}{dz}\left(h + \frac{v^2}{2}\right) = \dot{q}$$

**空化质量传递**：
$$\frac{d\alpha}{dz} = \frac{\dot{m}}{\rho v_z A}$$

### 4.3 封闭关系

| 量 | 公式 |
|----|------|
| 壁面剪切 | τ_w = (f/8)·ρv², f = 0.316/Re⁰·²⁵ (Blasius) |
| 等效半径 | R_eq = √(A/π)，A由蛋形截面几何积分得到 |
| 离心力轴向分量 | F_cent,z = ρv_θ²/r · dr/dz |

### 4.4 后处理：滑行因子G与脉冲强度I

**滑行因子**：
$$G(z) = \frac{\rho v_\theta^2 / r}{|\partial p/\partial z|}$$

**脉冲强度**：
$$I = \frac{1}{\Delta z_{\text{collapse}}} \int_{\text{collapse}} \frac{dE_{\text{bubble}}}{dz} dz$$

### 4.5 性能特征

| 指标 | 值 |
|------|-----|
| 单次评估时间 | < 1 ms |
| 精度 vs CFD | ±15~25%（趋势正确） |
| 适用场景 | 参数扫描、AI在线评估 |
| 不适用场景 | 需要精确3D流场分布时 |

## 五、步骤3：OpenFOAM CFD验证

### 5.1 求解器选择

| 求解器 | 适用 |
|--------|------|
| interPhaseChangeFoam | 两相瞬态空化（VOF+相变模型） |
| compressibleInterFoam | 可压缩两相流 |
| reactingTwoPhaseEulerFoam | 带反应的两相 |

### 5.2 关键设置

```
求解器: interPhaseChangeFoam
湍流模型: k-ω SST（带螺旋度修正）
空化模型: Schnerr-Sauer
时间步: adaptive (maxCo=0.3)
网格: snappyHexMesh, y+≈1, 500万~2000万cells
入口: 螺旋速度剖面 (v_z + v_θ)
出口: 固定压力 p=101325 Pa
```

### 5.3 监控量

| 量 | 用途 |
|----|------|
| 体积平均空化率 | 空化强度 |
| 出口流量 | 验证质量守恒 |
| 壁面压力分布 | 找C_pr转折点 |
| 截面涡量分布 | 验证螺旋度是否保持 |

### 5.4 后处理对比

| ODE预估 | CFD实测 | 用途 |

## 六、步骤4：AI拓扑优化闭环

### 6.1 优化目标

最大化综合性能指标：
$$J = w_1 \cdot \frac{C_{pr}}{C_{pr,\text{max}}} + w_2 \cdot \frac{I}{I_{\text{max}}} - w_3 \cdot \frac{\Delta p_{\text{loss}}}{\Delta p_{\text{ref}}}$$

其中 $w_1, w_2, w_3$ 为权重系数。

### 6.2 参数搜索空间

| 参数 | 范围 |
|------|------|
| R_out/R_in | 0.2 ~ 0.8 |
| τ·L | π ~ 8π (扭转总角) |
| a (凹陷深度) | 0.3 ~ 0.9 |
| b (穹顶高度) | 1.1 ~ 2.0 |

### 6.3 优化循环

```
遗传算法(GA) 或 贝叶斯优化
    │
    ├─→ 生成候选参数 → 步骤1(PythonOCC) → 步骤2(ODE评估)
    │        │
    │    J > J_best ? ──是──→ 步骤3(OpenFOAM验证)
    │        │                      │
    │        └──否──→ 继续搜索      J_CFD ≈ J_ODE ? ──是──→ 存入精英库
    │                                        │
    └── 下一代 ──────────────────────────────┘
```

### 6.4 多保真优化

| 保真度 | 评估方法 | 单次时间 | 使用频率 |
|--------|---------|---------|---------|
| 低 | ODE | <1ms | 每代1000次 |

## 七、算法效率数学说明

### 7.1 为什么ODE比CFD快4~6个数量级

| 维度 | CFD | ODE |
|------|-----|-----|
| 空间维度 | 3D | 1D（沿流线） |
| 未知数 | 10⁷个格点×5变量 | 500截面×5变量 |
| 求解方式 | 迭代求解(Newton-Raphson) | 前向积分(RK4) |
| 计算量 | O(N³·logN) | O(N) |

### 7.2 解析解可能性

**轴向动量+质量守恒**耦合方程本质是非线性的（二次速度项+摩擦项+变截面项），除极简几何外**无法求解析解**。但ODE的前向积分（RK4）是**半解析**——每步内的导数精确计算，步间误差O(h⁵)。

**空化方程**(Rayleigh-Plesset)在忽略表面张力和黏性时有**解析解**（能量积分），但在有外加压力场时需数值求解。
