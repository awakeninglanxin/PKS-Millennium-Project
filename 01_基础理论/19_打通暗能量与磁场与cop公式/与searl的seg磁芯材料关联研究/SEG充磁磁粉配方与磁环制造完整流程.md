## 一、磁粉化合物配方

### 1.1 元素组成与质量配比

| 元素 | 质量比 | 功能 | 分类 |
|------|--------|------|------|
| **铁 Fe** | **余量(~76%)** | 主要磁性材料，提供B_S | 阳性 |
| 铝 Al | 15% | 改善粘结性和机械强度 | 中性 |
| 硅 Si | 5% | 提高电阻→减少涡流损耗 | 中性 |
| 硫 S | 5% | 改善磁导率μ | 阴性 |
| 钛 Ti | 1% | 提高居里温度→高温稳定性 | 阳性 |
| 钕 Nd | 1~2% | 提高磁导率和矫顽力 | 阴性 |

### 1.2 阴阳比例——黄金分割1.618

```
阳性元素(Fe+Ti) / 阴性元素(Nd+S) ≈ 1.618

Fe+Ti = 76%+1% = 77%（阳性）
Nd+S = 2%+5% = 7%（阴性）
Al+Si = 20%（中性基质）

注：Fel属性+钛Ti阳，钕Nd+硫S阴→阴阳比符合黄金分割
```

### 1.3 理论磁性能

| 参数 | 理论值 | 实际估计(70~80%) | 对应COP参数 |
|------|--------|-----------------|------------|
| 饱和磁化强度 B_S | 17,320 G ≈ **1.73 T** | 12,000~14,000 G ≈ **1.2~1.4 T** | **B_S** |
| 铝硅基质B_S贡献 | ~0 | — | 降低有效B_S但提升电阻 |
| 矫顽力 Hc | 中等（同性材料） | 取决于烧结工艺 | **H_S** |

**关键**：配方追求的是**同性磁铁（各向同性）**——磁化易轴方向无方向依赖性，适合三方向立体辐射充磁。

### 1.4 粉末规格

| 参数 | 要求 |
|------|------|
| 颗粒尺寸 | 20~90微米 |
| 目数 | 300目标准 |
| 剩余部分 | 可用铁粉填充 |

## 二、磁环制造十步流程

### 第一阶段：材料选择与配料

| 步骤 | 关键参数 | COP关联 |
|------|---------|---------|
| 1. 选料 | 四层结构：Nd电子库→尼龙66控制门→**Fe-Al-Si-S-Ti-Nd磁芯**→Cu发射器 | 磁芯层=COP的"铁磁芯" |
| 2. 称重 | **公差0.05克/滚筒组**，质量比例由幻方数字决定 | COP可重复性保证 |
| 3. 混合 | 惰性气体保护，粉末均匀分散，分层或均质混合 | 电子流连续性 |

### 第二阶段：成型与加工

| 步骤 | 参数 | 物理目的 |
|------|------|---------|
| 4. 模压成型 | **200~400 bar压力，150°C，≥20分钟**，冷却后脱模 | 获得正确晶体结构 |
| 5. 烧结 | 氮气/氢气保护，1000~1400°C，升温5~10°C/min，保温2~4h | 粒子融合→多晶结构 |
| 6. 机加工 | **外径微米级精度**，表面抛光 | 决定气隙(≈1mm)稳定性 |

### 第三阶段：充磁——核心机密

| 步骤 | 参数 | COP等效 |
|------|------|---------|
| 7. 充磁 | **三明治结构**：两套三层扁平线圈夹住磁环 | 上下同时充磁→磁化深度覆盖全厚度 |
| 7a. 三层同轴 | **AC轴向(内)→Phasing环向横行(中)→DC轴向(外)** | 每层扁平煎饼状，三层物理叠套 |
| 7b. Phasing层 | 环向横行绕线，**层内双线并绕+异向电流** | 异向电流→径向N-S极对→34极正弦波磁畴 |
| 7c. 两套线圈 | 上套(定子)Phasing=0°；下套(滚筒)Phasing=**90°** | 90°相位差→自转驱动力 |
| 7d. 同步关断 | AC+Phasing+DC**同时开关**，占空比<2秒 | 磁畴"冻结"在写入状态 |
| 7e. 冷却 | **水冷系统**——线圈浸入水中 | 150A DC大电流散热 |

### 第四阶段：检验与组装

| 步骤 | 内容 |
|------|------|
| 8. 磁化后检验 | 验证波形偏斜/扭曲正确，N-S极互锁（滚子N对定子S） |
| 9. 组装 | 8个磁化段**纯磁力自吸附**成一个滚筒（无胶水），滚子-环间隙≈1mm涡流悬浮 |
| 10. 最终控制 | 系统自启动→共振→温度下降→发射负健康离子→ZPE共振标志 |

## 四、粉末冶金烧结工艺（同性磁铁专用）

### 4.1 为什么必须用同性磁铁

| 同性磁铁 | 异性磁铁 |
|---------|---------|
| 粉末烧结→多晶结构→打破晶界效应 | 铸造/压延→单域粒子→晶轴对齐 |
| 各向磁化曲线重合→**无方向依赖性** | 仅易磁化轴方向磁性强 |
| 适合**三方向立体辐射充磁** | 只适合轴向充磁 |
| SEG磁环需要径向+轴向+切向三向充磁 | ❌ 无法满足 |

### 4.2 烧结关键控制点

```
升温: 5-10°C/min → 峰温1000-1400°C → 保温2-4h
  → 冷却: 5-15°C/min（全程惰性气体保护）
  → 目标：磁性粒子间距接近超交换长度(2-10nm)
  → 结果：多层同性磁体，各向同性BH曲线
```

## 五、GMR巨磁阻效应与高频性能

铝硅硫钛钕铁磁环具有GMR（巨磁阻）效应：

| 频率 | 磁阻行为 | 对SEG的意义 |
|------|---------|------------|
| 几Hz~几kHz | 磁阻恒定 | 启动阶段 |
| 几十kHz~几百kHz | **磁阻开始下降** | 运行阶段→等效H_S自动降低 |
| 几MHz+ | **磁阻急剧衰减→趋于零** | H_S→极小→COP急剧增大 |

**关键互验**：SEG充磁AC频率=1.5MHz恰好处于GMR急剧衰减区→材料在运行频率下**自动降低等效H_S**→COP自动增大！
