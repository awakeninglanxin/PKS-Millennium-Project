## 一、瑟尔充磁的真实结构——三明治式三层扁平线圈

### 1.1 三明治夹层结构

瑟尔充磁不是一组线圈环绕磁环，而是**两套完全相同的三层扁平线圈**像三明治一样夹住待充磁的空心磁环：

```
                上充磁线圈组（定子）
        ┌─────────────────────────────┐
        │ 外层DC / 中层Phasing / 内层AC │ ← 三层扁平线圈
        │ 做得非常扁，像多层煎饼         │
        ├─────────────────────────────┤
        │                             │
        │     待充磁的空心磁环          │ ← 磁环放在两套线圈中间夹层
        │     （定子环或滚筒）           │
        │                             │
        ├─────────────────────────────┤
        │ 外层DC / 中层Phasing / 内层AC │ ← 三层扁平线圈
        │ 做得非常扁，像多层煎饼         │
        └─────────────────────────────┘
                下充磁线圈组（滚筒）

  关键：
  ├─ 两套线圈完全相同（三层结构一致）
  ├─ 每套线圈都是扁平煎饼形状（轴向很薄，径向宽）
  ├─ 磁环被夹在两套线圈中间
  ├─ 上套 → 充磁定子环；下套 → 充磁滚筒
  └─ 两套的Phasing线圈相位相差90° → 产生自转驱动力
```

### 1.2 每套线圈的三层扁平结构（内→外）

每套线圈从内到外有三层，做得非常扁（轴向薄，径向宽）：

```
一套三层扁平线圈的径向截面（从内→外）：

┌──────────────────────────────────────────────────┐
│  第3层（最外层）：DC粗线线圈                       │
│  ═══════════════════════════════════════════════  │
│  绕线方向：轴向（沿环的轴向绕制）                   │
│  匝数：~2000匝 粗铜线，电流：DC 100~150A           │
│  功能：提供直流饱和偏置场 → 将磁芯推到 B_S          │
│  ┌────────────────────────────────────────────┐   │
│  │  第2层（中间层）：Phasing 环形多极充磁线圈    │   │
│  │  ── 环向横行绕组 ── 环向横行绕组 ──        │   │
│  │  绕线方向：环向横行（沿环的圆周方向绕制）     │   │
│  │  ⚡层内绕线方式：双线并绕 + 充磁电流异向     │   │
│  │  两根线并行缠绕 → 一正一反电流              │   │
│  │  → 每个绕线周期产生一对N-S极                │   │
│  │  结构：四段独立控流 → 可雕刻34极磁化图案    │   │
│  │  ┌──────────────────────────────────────┐ │   │
│  │  │  第1层（最内层）：AC细线线圈           │ │   │
│  │  │  ──────────────────────────────      │ │   │
│  │  │  绕线方向：轴向（沿环的轴向绕制）       │ │   │
│  │  │  匝数：~1000匝 细铜线                  │ │   │
│  │  │  电流：AC 100~200mA @ 1.5MHz           │ │   │
│  │  │  功能：高频振荡 → 降低H_S               │ │   │
│  │  └──────────────────────────────────────┘ │   │
│  └────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────┘
```

### 1.3 Phasing层内的双线并绕——关键细节

整体线圈结构是三层同轴（非双线并绕），但**Phasing层内部**的绕组是双线并绕的：

```
Phasing层的环向横行绕线（展开为长条视图）：

  环面圆周方向 →
  ┌────────────────────────────────────────────┐
  │  线A：电流方向 → → → → → → → → →          │
  │  线B：电流方向 ← ← ← ← ← ← ← ← ←  ← 双线并行│
  │                                            │
  │  ┌──N──┐ ┌──S──┐ ┌──N──┐ ┌──S──┐         │
  │  │ ↑A  │ │ ↓B  │ │ ↑A  │ │ ↓B  │         │
  │  │ ↓B  │ │ ↑A  │ │ ↓B  │ │ ↑A  │  ← 异向电流│
  │  └─────┘ └─────┘ └─────┘ └─────┘         │
  │    N极     S极     N极     S极              │
  └────────────────────────────────────────────┘

  线A和线B并行缠绕，但通电方向相反：
  ├─ 线A向左→线B向右 → 两线间磁场叠加=加强→形成N极
  ├─ 线A向右→线B向左 → 两线间磁场叠加=加强→形成S极
  └─ 每个绕线周期产生一对N-S极 → 34个周期=34对磁极

  这种双线并绕在Phasing层内部
  → 利用Ampere定律：异向电流在相邻导线间产生最强磁场
  → 每对N-S极的磁场方向径向辐射（垂直环面向外）
```

### 1.4 三层线圈的功能拆解

| 层 | 绕线方向 | 层内绕法 | 电流 | 功能 | COP等效 |
|----|---------|---------|------|------|---------|
| 内层AC | 轴向 | 单线 | AC 100mA @ 1.5MHz | 高频振荡→"振动"磁畴 | 降低H_S |
| 中层Phasing | **环向横行** | **双线并绕+异向电流** | DC脉冲四段控流 | 径向多极(34极)正弦波磁化 | 写入H_X分布 |
| 外层DC | 轴向 | 单线 | DC 100A | 推到B_S饱和 | 设定B_S基准 |

## 二、核心感悟一：三明治结构 = 上下同时充磁

### 2.1 为什么用两套线圈？

```
单套线圈充磁：          三明治充磁（瑟尔方案）：

  ┌──线圈──┐              ┌──上线圈──┐
  │        │              │          │
  │  磁环  │              │  磁环    │  ← 上下同时施加磁场
  │        │              │          │
  └────────┘              └──下线圈──┘

  磁场从一面进入磁环        磁场从上+下两面同时进入
  → 磁化深度有限            → 磁化深度覆盖整个磁环厚度
  → 背面磁化弱              → 两面磁化均匀
  → 磁畴图案不对称          → 磁畴图案上下对称
```

### 2.2 扁平煎饼线圈的物理意义

每套线圈做得非常扁（轴向很薄），意味着：
- 线圈与磁环的距离非常近 → 磁场衰减最小
- 扁线圈的漏磁通小 → 磁化效率高
- 多层煎饼叠加 → 每一层薄但总匝数够 → 安匝数大

## 三、核心感悟二：Phasing层内双线并绕 = 径向多极的来源

### 3.1 Phasing异向双线并绕的磁场

```
Phasing层展开视图——异向双线并绕产生径向多极磁场：

  线A电流: → → → → → → → → → → → → → → → → → →
  线B电流: ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←

  在A→B←交叉处：同向电流区域 → 磁场向外辐射 → N极
  在B→A←交叉处：反向电流区域 → 磁场向内辐射 → S极

  磁环表面：
    N  S  N  S  N  S  N  S  N  S  ... (共34对)
    ↑  ↓  ↑  ↓  ↑  ↓  ↑  ↓  ↑  ↓
   径向磁场方向垂直于环面（像车轮辐条向外/向内交替）
```

### 3.2 异向电流的COP意义

| 特性 | 常规单线绕制 | Phasing异向双线并绕 |
|------|------------|-------------------|
| 每个绕线周期 | 产生1个磁极 | 产生1对N-S极 |
| 磁场方向 | 主要沿轴向 | **径向辐射垂直环面** |
| 磁场强度 | 单线A/m | 双线叠加→2×单线强度 |
| 磁极间距控制 | 等于绕线间距 | 由线A/B的交叉点精确决定 |

## 四、核心感悟三：定子与滚筒Phasing的90°相位差

### 4.1 上下两套线圈的差异

上下两套线圈在结构上完全相同，唯一区别是**Phasing层的相位**：

```
上套线圈（充磁定子环）：
  Phasing层四段：A(0°), B(90°), C(180°), D(270°) 
  绕线起始点相位=0°

下套线圈（充磁滚筒）：
  Phasing层四段：A(90°), B(180°), C(270°), D(0°)  
  绕线起始点相位=90°  ← 与上套相差90°
```

### 4.2 90°相位差 → 自动旋转

定子环和滚筒的磁极图案在空间上错开90°，当滚筒放在定子环之间时：

```
定子磁极:  N  S  N  S  N  S  N  S  ...
滚筒磁极:    N  S  N  S  N  S  N  S  ...
           ↑
          90°相位偏移

定子N极吸引滚筒S极 + 定子S极排斥滚筒S极
→ 产生切向力 → 滚筒沿圆周方向自转
→ 不需要外部驱动 → SEG自持运行
```

## 五、与COP仿真参数的映射

| COP仿真参数 | 充磁实现 | 说明 |
|------------|---------|------|
| H_S | AC内层(1.5MHz) → 振动磁畴 → 降低翻转势垒 | 调节AC幅值/频率 |
| B_S | DC外层(100A) → 推到饱和 | 调节DC电流 |
| H_X | Phasing中层(异向双线并绕) → 径向34极磁场 | 调节各段脉冲电流 |
| 34极 | Phasing四段控流 + 异向双线交叉 | 由幻方线34决定 |
| 90°相位差 | 上套(定子) vs 下套(滚筒) Phasing相差90° | 自转驱动力 |

## 六、错误修正汇总

| 之前 | 修正后 |
|------|--------|
| "双线并绕是整体结构" | **三层同轴**是整体结构；**Phasing层内**才是双线并绕 |
| "DC+AC双线并绕" | AC(内层轴向单线) + Phasing(中层环向异向双线) + DC(外层轴向单线) |
| "一组线圈环绕磁环" | **两套三层扁平线圈**三明治夹住磁环（上下同时充磁） |
| "单线绕制Phasing" | Phasing层内**双线并绕+异向电流**→径向多对N-S极 |
| 未提两套线圈 | 上套定子Phasing=0°；下套滚筒Phasing=90°→自动旋转 |
