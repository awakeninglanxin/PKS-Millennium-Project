# 🥚 蛋形管涡旋CFD仿真项目（Clay NS方程探索路径）

## 研究目标

基于 Schauberger 超双曲锥蛋形曲线，构造非轴对称截面涡管的 CFD 仿真，
探索三维 Navier-Stokes 方程在蛋形截面上的解析解候选形式。

---

## 核心研究思路

```
超双曲锥 xy=1 斜切 → 蛋形参数 k_E
    ↓
蛋形自然坐标 (s,η,z) → Laplacian 展开 → NS 算子变换
    ↓
k_E ∈ (1.9371, 2.0] → 涡量拉伸本征值谱
    ↓ 离散谱（k_E=2.0）    ↓ 连续谱聚集点（k_E=1.9371）
  Fibonacci 离散解     →    黄金比极限蛋 → KAM稳定流形
    ↓
NS 残差验证 + 弱解能量估计 → Clay 论文路径
```

## 关键参数体系

### 八度蛋（k_E = 2.0）

| 参数 | 符号 | 值 | 物理意义 |
|------|------|----|---------|
| 尖顶高度 | z₁ | 1 | 基音 |
| 钝顶高度 | z₂ | 2 | 高八度 |
| 截面中心高度 | z₀ | 5/3 | = (z₁²+z₂²)/(z₁+z₂) |
| 截面倾角 | α | 33.69° | arctan(2/3) |
| 蛋形度 | **k_E** | **2.0** | R/r = 八度比 |
| 谱性质 | — | 离散点谱 | Fibonacci 可分离模态 |

### 黄金极限蛋（k_E ≈ 1.9371）

| 参数 | 值 | 来源 |
|------|----|------|
| k_E | ≈ 1.9371 | Fibonacci n→∞ 极限 = 1/φ |
| 截面倾角 | 31.72° | |
| 谱性质 | 连续谱聚集点 | 最无理数 φ → KAM 环面稳定 |
| 流体意义 | 最抗扰动截面 | Couette-Taylor 流最高阈值 |

## k_E — α 映射关系

**核心公式**（取 z₁=1, z₂=k_E）:
```
α = arctan(z₁·z₂·(z₂-z₁)/(z₁+z₂)) = arctan(k_E·(k_E-1)/(k_E+1))
```

| k_E | α (°) | 音乐谐波 | 谱性质 |
|------|-------|---------|-------|
| **1.9371** | **31.72** | — | **连续谱聚集点（黄金极限）** |
| **2.0** | **33.69** | **八度(Octave)** | **离散谱第一谐波（最优）** |
| 2.5 | 46.97 | — | 混合谱 |
| 4.0 | 67.38 | 双八度 | 离散谱 |

### 1. 蛋形摆线与弹簧运动

从双曲锥切面蛋形的标准参数方程出发，可以叠加摆线/锯齿\sin波运动：

```
x = term_x * (1 - arcsin(sin(c*t)) / (1.5*c))    ← 锯齿摆线（阴在内）
x = term_x * (1 + sin(c*t) / (1.5*c))            ← sin波（阳在外）
y = term_y * 同上因子
```

其中 `term_x = 2*sin(t) / (b + sqrt(b² - 4k*cos(t)))` 是简化后的蛋形参数方程。

**首尾错位180度俯视蛋形弹簧**：调节参数 `a=1.5`，使蛋形弹簧在俯视图中首尾恰好错位180°，产生类似羚羊角的双向螺旋压缩效果。
