## 🔑 三大核心技术原理速览

### 1. Keely素数共振与齿轮优化

```
核心发现：610Hz（基频）+ 42800Hz（石英共振）
→ 42800/610 = 4280/61（61是素数，107是素数）
→ gcd(4280,61)=1，lcm=261080
→ 齿轮定律：gcd→0 噪音最小，lcm→∞ 寿命最长
→ 本质：频率对的素数性决定共振效率
```

💡 **工程应用**：设计齿轮齿数对时，选互素的齿数→噪音最低、寿命最长

### 2. 舒伯格圆盘飞碟原理

```
超导线圈 → 洛伦兹力始终向内（与电流方向无关）
→ 1万rpm = 57吨升力
→ 2万rpm = 228吨升力（v²定律）
→ 负电阻效应：旋转越快→等离子水压入越难→电阻越大→能量越大
→ 涡环侧缝喷出高温等离子体→斥力场
→ 材料：铜汞合金（常温超导优先）
```

📖 详见 `PKS几何_NS方程映射.md` §10 和 `Schauberger数学人生_三维交叉解读.md`

### 3. HHO水燃料"呼吸"技术

```
Stanley Meyer + Puharich 技术融合：
1. 低频AC(20-200Hz) 调制高频载波(200-100000Hz)
2. 腔共振（腔径、电极间距决定共振频率）
3. HHO气体在高温高压下→水→HHO→水（"呼吸"循环）
4. 惰性气体混入→控制燃烧温度
5. 316L不锈钢半球反应腔效率最高
```

## 🏗️ 工程实践速查

### 3D打印参数

| 材料 | 壁厚 | 缩水补偿 | 备注 |
|------|------|----------|------|
| 光固化白色树脂 | ≥1.5mm | 1.005× | 表面光滑 |
| 纯铜 | ≥0.6mm | 1.05× | 表面粗糙，需车床攻丝 |
| 316L不锈钢 | ≥1.5mm | 1.05× | 管内可能有支撑 |

### 螺纹处理
- **禁用金属打印螺纹**：表面粗糙→需车床二次攻丝
- **推荐方案**：淘宝买双四分外丝→车床截半焊接
