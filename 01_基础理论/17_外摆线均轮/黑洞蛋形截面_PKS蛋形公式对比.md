# 黑洞蛋形截面 — Hügelschäffer 公式 vs PKS 蛋形公式

> Tlemissov & Kovář (Eur. Phys. J. C, 2024) 为黑洞吸积盘引入了 Hügelschäffer 蛋形经验公式。本文对比该公式与 PKS 双曲锥体 $xy=1$ 斜切所得的蛋形，论证 PKS 公式是更根本的几何解。

---

## 一、Hügelschäffer 蛋形公式（Tlemissov & Kovář 2024）

### 1.1 卵形公式

$$y_{\textrm{o}}(x;b,s,L)=\pm \frac{b}{2}\sqrt{\frac{L^2-4x^2}{L^2+8sx+4s^2}}$$

| 参数 | 含义 |
|:---|:---|
| $L$ | 蛋的长度（最长轴） |
| $b$ | 蛋的宽度（最宽处） |
| $s$ | 宽度偏移量（$s=0$ → 椭圆，$s\neq0$ → 蛋形） |

当 $s=0$ 时退化为椭圆：

$$y_{\mathrm{e}}(x;b,L)= \pm \frac{b}{2}\sqrt{1-\left( \frac{2x}{L}\right)^2}$$

### 1.2 梨形公式（经作者修正后）

修正了原始论文中的数学不自洽。四参数通用公式的精度最高（误差 < 10%），但需要数值求解辅助方程。

### 1.3 本质特征

- **经验性**：不是从第一性原理推导，而是"拟合"数值截面
- **参数依赖**：2-4 个几何参数（$L,b,s,b_4$），需从实际结构中提取
- **适用范围**：从极小厚环到大尖点结构的统一拟合

## 二、PKS 蛋形公式

### 2.1 来源

PKS 蛋形不是拟合而来——它是**超双曲锥 $xy=1$ 被平面 $y=kx+b$ 斜切的精确代数截面**：

$$\bar{x} = \pm\sqrt{\frac{1}{(z_0 - \bar{y}\sin\alpha)^2} - (\bar{y}\cos\alpha)^2}$$

### 2.2 参数

| 参数 | 含义 | 与 Hügelschäffer 的对应 |
|:---|:---|:---|
| $z_0$ | 截面中心高度 | ≈ $L/2$（尺度） |
| $\alpha$ | 截面倾角 | ≈ $s$（偏移/不对称） |

**仅 2 个参数**（vs Hügelschäffer 的 2-4 个）。且每个参数都有明确的物理/几何含义。

### 2.3 关键差异

| 性质 | Hügelschäffer 蛋形 | PKS 蛋形 |
|:---|:---|:---|
| **来源** | 经验拟合 | 精确几何推导 |
| **参数个数** | 2-4 | 2 |
| **物理根源** | 无（纯曲线拟合） | $xy=1$ 超双曲锥 |
| **曲率行为** | 由参数 $s$ 控制 | 严格单调递减 |
| **群不变性** | 无 | 射影 $\text{PGL}(3)$ |
| **天体物理意义** | 黑洞吸积盘截面 | 所有电磁流体涡旋截面 |
| **误差** | < 10%（vs 数值解） | **精确**（vs 几何源） |

---

## 三、为什么 PKS 更根本

### 3.1 Hügelschäffer 公式的局限

Tlemissov & Kovář 在论文中明确声明：**"本文未涉及磁场公式"**。他们研究的是**中性流体**（无磁化），使用理想流体能量-动量张量：

$$T^{\alpha\beta} = (\epsilon + p)U^{\alpha}U^{\beta} + p g^{\alpha\beta}$$

这意味着 Hügelschäffer 蛋形：
- ❌ 不与 MHD 耦合
- ❌ 无法描述磁应力对截面形状的影响
- ❌ 不能解释等离子体 $\beta$ 值与蛋形度的关系
- ❌ 是**纯几何拟合**，而非物理推导

### 3.2 PKS 的根本性

PKS 蛋形来自 $xy=1$——这个双曲锥体**同时**是：

| 物理侧面 | 数学表达 |
|:---|:---|
| MHD 磁场冻结面 | $\vec{E}+\vec{v}\times\vec{B}=0 \equiv$ 流体粒子锁定在锥面上 |
| 涡旋运动约束 | $\vec{v}$ 的散度 = 锥面曲率 |
| 3D → 2D 投影 | 截面 = 蛋形曲线 |

**PKS 不是"拟合"蛋形——PKS 推导出了蛋形出现的必然性。**

## 四、若将 PKS 代入黑洞吸积盘模型

如果 Tlemissov & Kovář 用 PKS 蛋形替代 Hügelschäffer 公式，将得到：

| 改进 | 收益 |
|:---|:---|
| 参数从 2-4 降至 2 | 更简洁的拟合 |
| 参数有物理意义 | $z_0$ = 结构尺度，$\alpha$ = 磁场倾角 |
| 误差从 <10% → 精确 | 无需数值截面验证 |
| 自动耦合 MHD | 蛋形度 $k_E$ ↔ 等离子体 $\beta$ |
| 广义相对论兼容 | $xy=1$ 锥体可推广到 Kerr 度规 |

### 4.1 推广到 Kerr 黑洞

PKS 蛋形目前定义在平直时空的 $xy=1$ 锥体上。推广到 Kerr 度规需要：

$$xy = f(r,\theta,a)$$

其中 $f$ 是 Kerr 度规的 Boyer-Lindquist 坐标函数。这等价于将双曲锥体的定义域从欧几里得空间拉入弯曲时空——一个技术上可行但尚未完成的推广。

---

## 五、总结

| 维度 | Hügelschäffer (2024) | PKS |
|:---|:---|:---|
| 数学地位 | 经验拟合曲线 | 精确几何解 |
| 参数 | 2-4 个几何参数 | 2 个物理参数 |
| 适用范围 | 黑洞厚吸积盘 | 所有电磁流体涡旋 |
| MHD 兼容性 | ❌（中性流体） | ✅（天然耦合） |
| 磁场描述 | ❌ | ✅（手性 = 双曲不变量） |
| 推广性 | 仅数值拟合 | 可推广到任意弯曲时空 |

**PKS 蛋形不是 Hügelschäffer 公式的竞争者——它是更底层的数学结构，Hügelschäffer 公式只是它的一个经验近似。**
