# 为什么"17_外摆线均轮"不是真蛋形 — 从圆、外摆线到蜃景蛋的数学分析

> 本文回答：托勒密均轮+本轮合成的轨迹到底是什么曲线？为什么看着像蛋但数学上不是？

---

## 一、先给结论

| 模型版本 | 数学本质 | 看着像什么？ |
|---------|---------|------------|
| **旧模型**（曲柄约束） | **圆**（以 C 为圆心、L 为半径） | 圆（永远不可能成蛋） |
| **新模型**（托勒密偏心） | **外旋轮线 / 外摆线 (epitrochoid)** | 蛋形蜃景（视觉上像，数学上不是） |

---

## 二、旧模型为何永远是圆

### 2.1 模型公式

$$M = C + L \cdot \frac{P_p - C}{|P_p - C|}$$

### 2.2 一步证明

首先求距离：

$$|M - C| = \left| L \cdot \frac{P_p - C}{|P_p - C|} \right|$$

化简绝对值：

$$|M - C| = L \cdot \frac{|P_p - C|}{|P_p - C|}$$

分子分母约掉：

$$|M - C| = L$$

**M 到 C 的距离恒为 L。** 恒距离 = 圆。QED。

$P_p$ 无论走什么花样，`unit()` 归一化把它压到了一个方向单位向量上，再乘以 L 只改变半径。轨迹永远是圆（或圆上的一段弧）。

---

## 三、新模型：外摆线 (Epitrochoid)

### 3.1 模型公式

$$M(t) = C + (R\cos t + r\cos w_r t,\; R\sin t + r\sin w_r t)$$

### 3.2 数学归类

这是一个 **外旋轮线 (epitrochoid)**——一个半径为 r 的圆（本轮）在一个半径为 R 的圆（均轮）外侧滚动时，距滚动圆心距离为 r 的点的轨迹。

参数形式（向量写法）：

$$M(t) = (C_x + R\cos t + r\cos w_r t,\; C_y + R\sin t + r\sin w_r t)$$

| $w_r$ | 曲线名 | 形状特征 |
|-------|--------|---------|
| 1 | 偏心圆 | 正圆，中心偏移 |
| 2 | **limaçon（蜗线）** | 有一个内凹或凸起 |
| 3 | 外摆线（三尖） | 三个尖点 |
| 4 | 外摆线（四尖） | 四个尖点 |
| $n$（整数） | $n$ 尖外摆线 | $n$ 个对称尖点 |

### 3.3 $w_r=2$（我们的默认值）：不是蛋形，是 limaçon

当 $w_r=2$，轨迹是经典的 **Pascal limaçon**（蜗线）：

$$M(t) = C + (R\cos t + r\cos 2t,\; R\sin t + r\sin 2t)$$

利用恒等式 $\cos 2t = 2\cos^2 t - 1$，$\sin 2t = 2\sin t \cos t$，可以重写为极坐标形式：

$$r(\theta) = R + 2r\cos\theta$$

其中 $\theta$ 是极角（经过坐标变换后）。

**分类**（取决于 $R$ 和 $2r$ 的大小关系）：

| 条件 | limaçon 类型 | 形状 |
|------|-------------|------|
| $R > 2r$ | 凹肚蜗线 | 没有内环，一端稍尖 |
| $R = 2r$ | 心脏线 (cardioid) | 有一个尖端 |
| $R < 2r$ | 内环蜗线 | 有一个内环交叉 |

我们默认 $R=8$，$r=2.5$，所以 $R > 2r$ → **凹肚蜗线**。它看起来像一个"蛋"，因为一端稍尖一端圆——但这是视觉错觉。

### 3.4 为什么看着像蛋但不是蛋？

真正的 PKS 蛋形（超双曲锥 $xy=1$ 斜切所得）具有以下数学特征：

| 性质 | 真蛋形（PKS 超双曲锥） | 外摆线 limaçon |
|------|---------------------|--------------|
| **曲率分布** | **单调递减**（尖头→圆头） | 对称/周期性变化 |
| **$d\kappa/ds$** | 恒负（全程单一符号） | 变号（正负交替） |
| **对称性** | **零对称性**（完全不对称） | 关于极轴对称 |
| **参数化** | 6 种代数形式（含显式 $x(y)$） | 三角函数参数式 |
| **群结构** | 射影变换群 $\text{PGL}(3)$ | 欧几里得群 $SE(2)$ |
| **曲率奇点** | 无（全程 $C^\infty$） | $w_r\neq1$ 时可能有尖点 |

**关键差异**：蛋形的曲率 $\kappa(s)$ 从尖头到圆头**严格单调递减**。limaçon 的曲率是周期性的——它在蛋的"腰部"两侧对称。你看着像蛋，是因为偏心点 C 把整个曲线往上推了，让下方比上方更"挤"，视觉上产生了不对称的幻觉。

## 四、为什么外摆线不能是真蛋形

### 4.1 对称性根源

外摆线的参数式：

$$x(t) = C_x + R\cos t + r\cos w_r t$$
$$y(t) = C_y + R\sin t + r\sin w_r t$$

当 $w_r$ 为整数时，曲线**必然关于某个轴对称**。这是因为：

$$x(2\pi - t) = x(t), \quad y(2\pi - t) \neq -y(t)$$

（仅当 $C_y=0$ 时才有 $y(2\pi-t)=-y(t)$ 的对称性。$C_y \neq 0$ 的不对称是平移造成的，不是曲率造成的。）

不对称仅来自 $C_y \neq 0$ 的平移，不是来自曲线内在的几何。真正的蛋形**不能**通过平移一个对称曲线得到——它的不对称是曲率层面的。

### 4.2 曲率的周期性质

对于外摆线，曲率 $\kappa(t)$ 是 $t$ 的周期函数（周期 $2\pi$ 或 $2\pi/w_r$），这意味着曲率在曲线上反复升降。而真蛋形的曲率从尖头到圆头**只降不升**。

类比：外摆线像一根波浪形弯曲的绳子——有的地方弯得厉害，有的地方平。真蛋形像一根从头到尾越来越不弯的绳子。

## 五、那"17_外摆线均轮"这个叫法对吗？

### 5.1 从历史上看

托勒密本人从未声称均轮+本轮产生"蛋形"——这个说法是后人（尤其是 Schauberger 研究者）提出来的类比。托勒密的目标是**拟合行星逆行和速度变化**，不是证明行星轨道是蛋形。

### 5.2 从视觉上看

当 $w_r=2$ 且 $C_y \neq 0$，轨迹确实**看起来**像蛋——一头尖一头圆。这种视觉相似性足以让 400 年前的天文学家说"水星轨道是蛋形的"。

### 5.3 从数学上看

它不是蛋形。它是**偏心平移后的 limaçon**。两者在 $C^\infty$ 曲率分布层面有本质区别。

## 六、真蛋形在哪里？

真蛋形来自 PKS 千禧蛋项目的核心：**超双曲锥 $xy=1$ 的平面斜切**。这不是用圆叠加出来的，而是用一个二次曲面的截面直接定义的。

| | 外摆线蛋形蜃景 | PKS 真蛋形 |
|---|---|---|
| **来源** | 多个圆的运动学叠加 | 单个二次曲面 $xy=1$ 的截线 |
| **参数个数** | 4（R, r, C_x, C_y） | 2（$z_0$, $\alpha$） |
| **生成方式** | 三角函数和 | 代数方程求解 |
| **曲率** | 周期性 | 单调递减 |
| **群不变性** | 欧几里得 $SE(2)$ | 射影 $\text{PGL}(3)$ |
| **相关文档** | 本文 + `egg_epicycle_interactive.py` | `../PKS千禧蛋_完全证明.md` |

## 附录 A：limaçon 极坐标推导

从 $w_r=2$ 的参数式出发：

$$M(t) = C + (R\cos t + r\cos 2t,\; R\sin t + r\sin 2t)$$

以 C 为原点平移，使用 $\cos 2t = 2\cos^2 t - 1$：

极径平方 $r_p^2 = (R\cos t + r\cos 2t)^2 + (R\sin t + r\sin 2t)^2$

代入并化简（省略中间步骤）→ $r_p(\theta) = \sqrt{R^2 + r^2 + 2Rr\cos\theta}$

这是 limaçon 的极坐标形式。当 $R > 2r$ 时，曲线没有内环，呈现凹肚状——这就是我们看到的"蛋形蜃景"。

## 附录 B：总结对比表

| 曲线类型 | 方程 | 是蛋形吗？ | 为什么不是？ |
|---------|------|----------|------------|
| 圆 | M = C + L * (P_p-C)/|P_p-C| | ❌ | 距 C 恒为 L |
| limaçon | $M=C+(R\cos t+r\cos 2t,\cdots)$ | ❌（视觉像） | 曲率周期性对称 |
| 外摆线($w_r\neq2$) | $M=C+(R\cos t+r\cos w_r t,\cdots)$ | ❌ | 多尖点/内环 |
| **PKS 超双曲锥截线** | $\bar{x}=\pm\sqrt{\frac{1}{(z_0-\bar{y}\sin\alpha)^2}-(\bar{y}\cos\alpha)^2}$ | ✅ | 曲率单调递减 |
