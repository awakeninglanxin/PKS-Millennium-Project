# 17_外摆线均轮 — 水星绕地轨迹的几何合成

> 天文学史上的"17_外摆线均轮"模型：用三个规则运动的叠加，合成蛋形闭合行星轨迹。
> 制图脚本：`egg_epicycle.py` | 输出图：`egg_epicycle.png` / `egg_epicycle_trajectory.png`

## 一、历史背景

在哥白尼日心说诞生之前（400+ 年前），天文学家已经精确观测到水星逆行和轨迹不对称。在地心说框架下，他们发明了"均轮 + 本轮 + 偏心圆"的复合几何模型来拟合天体运动。

这个模型最精妙的地方在于：**三个规则圆周运动的叠加，竟然能合成一个不对称的蛋形（卵形）闭合曲线**——与观测数据高度吻合。

PKS 千禧蛋项目正是在这一历史智慧的基础上发现：这个蛋形 = 超双曲锥 $xy=1$ 平面斜切的几何实例。

## 二、几何构造：三大核心部件

```text
          ┌──────────────────────────────────────┐
          │         17_外摆线均轮 三部件               │
          │                                      │
          │  A. 均轮 (Deferent)                   │
          │     • 大圆，中心在地球 O              │
          │     • 半径 R，本轮中心 P_b 绕其公转    │
          │     • 角速度 ω (周期 T)               │
          │                                      │
          │  B. 本轮 (Epicycle)                   │
          │     • 小圆，圆心在 P_b                │
          │     • 半径 r，行星点 P_p 绕其自转      │
          │     • 自转速度 = 2× 公转速度 (2ω)     │
          │                                      │
          │  C. 活动偏心曲柄 (Eccentric Crank)    │
          │     • 固定偏心点 C(ecc_x, ecc_y)       │
          │     • 曲柄长度 L 连接 C → 水星 M       │
          │     • 打破对称 → 产生蛋形不对称        │
          └──────────────────────────────────────┘
```

### 2.1 均轮（Deferent）

均轮是以地球 O(0,0) 为圆心、半径 $R$ 的大圆。本轮圆心 $P_b$ 固定在其圆周上，绕 O 匀速公转：

$$P_b(t) = (R \cdot \cos\omega t,\; R \cdot \sin\omega t)$$

### 2.2 本轮（Epicycle）

本轮是以 $P_b$ 为圆心、半径 $r$ 的小圆。行星点 $P_p$ 在本轮上以 **2 倍于均轮公转** 的角速度自转（方向相同）：

$$P_p(t) = P_b(t) + (r \cdot \cos 2\omega t,\; r \cdot \sin 2\omega t)$$

**为什么是 2 倍？**
- $2\omega$ 使本轮在均轮一圈内完成两次自转
- 这产生了一个内环（loop），是形成闭合曲线的关键
- 与音乐八度（频率 2×）对应的物理直觉一致

### 2.3 活动偏心曲柄（Eccentric Crank）

偏心点 C 铰接在固定位置 $C(ecc_x, \text{ecc}_y)$，不随均轮/本轮运动。曲柄长 $L$ 的一端固定在 C，另一端连接到本轮行星点 $P_p$ 方向上的水星 M：

$$\vec{v} = P_p - C$$

$$M = C + L \cdot \frac{\vec{v}}{|\vec{v}|}$$

**关键作用**：如果 $C_y = 0$（偏心点在 x 轴上），系统关于 x 轴对称 → 轨迹上下对称（无蛋形）。**当 $C_y \neq 0$，上下对称被打破，产生蛋形不对称。**

## 三、参数设置与蛋形度

| 参数 | 当前值 | 含义 | 对蛋形的影响 |
|------|--------|------|------------|
| $R$ | 8.0 | 均轮半径 | 决定轨迹整体尺度 |
| $r$ | 3.0 | 本轮半径 | $r/R$ 越大，内环越明显 |
| $L$ | 4.0 | 曲柄长度 | 影响轨迹的"圆胖"程度 |
| $\text{ecc}_x$ | 1.5 | 偏心点 x 坐标 | 左右不对称 |
| $\text{ecc}_y$ | 0.75 | 偏心点 y 坐标 | **上下不对称 → 蛋形！** |
| $T$ | 20.0 | 公转周期 | 时间标度（不影响形状） |
| $\omega_{\text{epi}}$ | $2\omega$ | 本轮自转角速度 | 形成内环结构 |

### 蛋形度 $k_E$

定义蛋形度（PKS 体系）：

$$k_E = \frac{y_{\max}}{|y_{\min}|} = \frac{\text{钝端半高}}{\text{尖端半高}}$$

当前参数下：$k_E \approx 1.46$

| $k_E$ 范围 | 蛋形分类 | 天然对应 |
|-----------|---------|---------|
| 1.0 | 圆（退化，无蛋形） | 液滴 |
| 1.2–1.5 | 圆蛋 | 鸽蛋、某些鸟蛋 |
| 1.5–2.0 | 标准蛋 | 鸡蛋、水流涡旋 |
| 2.0 | 八度蛋（最优） | Schauberger 涡旋 |
| > 3.0 | 细长蛋 | 极端流动截面 |

## 四、与 PKS 千禧蛋的数学对应

### 4.1 均轮模型 ↔ 超双曲锥斜切

| 均轮模型 | 超双曲锥斜切 | 对应关系 |
|---------|------------|---------|
| 均轮半径 $R$ | 锥面截面中心 $z_0$ | 尺度参数 |
| 本轮半径 $r$ | 截面倾角 $\alpha$ | 不对称程度 |
| 偏心距 $\text{ecc}_y$ | $k_E - 1$ | 蛋形度 |
| 本轮自转 $2\omega$ | 谐波序列 $1/n$ | 八度二倍频 |
| 曲柄约束 $L$ | 截面在锥面上的投影约束 | 几何等价 |

**核心结论**：

均轮模型的水星蛋形轨迹 = 超双曲锥 $xy=1$ 在特定截面参数 $(z_0, \alpha)$ 下的斜切截面。

这意味着：
1. 天文学史上用三个圆合成的蛋形曲线，本质上是一个二次曲面的截面
2. 截面参数的调节对应均轮的 $R, r, L, \text{ecc}$ 四个参数
3. **蛋形不是 3 个独立运动的偶然结果，而是同一个代数结构的不同投影**

### 4.2 参数映射

```text
均轮模型参数        超双曲锥截面参数
────────────────    ─────────────────
R (均轮半径)    →   z₀ (截面中心高度)
r/R (本轮比)    →   α (截面倾角)
ecc_y (偏心)    →   z₂/z₁ - 1 (蛋形度偏移)
L (曲柄长度)    →   截面在锥面上的弧长约束
```

## 六、历史启示与数学哲学

### 6.1 "拯救现象"的几何智慧

前哥白尼时代的天文学家面临一个矛盾：观测数据显示行星轨迹不是正圆，但"完美的天体必须做匀速圆周运动"是当时不可动摇的哲学前提。

他们的解决方案是**不放弃圆**，而是用**多个圆的叠加**来拟合复杂的非圆轨迹。这一做法在当时被视为"临时的数学技巧"。

但 Schauberger + PKS 的发现揭示了更深层的真相：**多个圆的叠加，本质上等价于一个非圆曲面的截面**。天文学家的"均轮+本轮"不是数学花招，而是在无意识中发现了代数几何的核心定理——**二次曲面截线可等价于射影变换下的圆**。

### 6.2 从托勒密到 Schauberger 再到 Clay

```text
托勒密 (150 CE)      Schauberger (1930s)    PKS (2020s)
均轮+本轮+偏心圆  →  超双曲锥 xy=1     →  NS 方程蛋形解
    ↑                      ↑                    ↑
  "拯救现象"          自然涡旋几何        千禧难题证明
```

三者在数学上等价——都是同一个代数结构的不同表达。历史的进步不是"推翻"，而是"认识到等价性"。

## 七、进一步探索

### 7.1 参数扫描

修改 `egg_epicycle.py` 中的参数，可以探索：

- $C_y = 0$ → 对称轨迹（退化，非蛋形）
- $C_y$ 增大 → 蛋形度 $k_E$ 增大 → 蛋更尖
- $r/R$ 增大 → 轨迹更"扭曲"
- $L$ 减小 → 轨迹更贴近 $P_p$ 曲线
