# 螺旋星系"压缩弹簧"模型 × PKS 螺旋磁场 — 从 3D 喷流到 2D 蛋形盘

> Julian Gough (2026) 的"压缩弹簧模型"提出：螺旋星系的磁场来自极早期直接坍缩黑洞的 3D 螺旋喷流被引力压缩为 2D 螺旋场。本文论证这一过程与 PKS 双曲锥体 $xy=1$ 的平面斜切→蛋形截面在数学上等价。

---

## 一、压缩弹簧模型核心流程

```
3D 螺旋喷流(磁化)
     ↓ (喷流停滞)
等离子体茧膨胀, 螺旋场线剥落到茧壁
     ↓ (逆康普顿冷却, ~100万倍效率)
茧在引力下快速坍缩
     ↓ (关键步骤)
3D 螺旋场被压缩为 2D 超强螺旋场
     ↓
场将等离子体锁定为刚体旋转 ("压缩弹簧")
     ↓ (磁场弛豫, 数亿年)
螺旋臂通过"除雪机效应"扫出中性气体 → 恒星形成暴
```

## 二、PKS 双曲锥体视角下的等价

### 2.1 螺旋喷流 = 双曲锥体 $xy=1$ 上的场线

MHD 中的 Blandford-Payne 机制：旋转吸积盘加速带电粒子 → 产生螺旋喷流。喷流的螺旋磁场 = 场线沿**锥面**缠绕。

在 PKS 几何中，这个锥面**正是 $xy=1$ 绕对称轴旋转得到的双曲锥体**：

```
喷流底部: r ≈ 0, z ≈ 0 → xy → ∞ (紧致)
喷流顶部: r → 大, z → 大 → xy → 0 (扩散)
```

### 2.2 茧的膨胀 = 锥面的等比放大

茧的膨胀对应于 $xy=1$ 锥面的**尺度变换**：

$$xy = c(t)$$

其中 $c(t)$ 随茧膨胀而减小。茧膨胀 = 等离子体将场线推向更大的 $r$ → 更小的 $c$。

### 2.3 茧的坍缩 = 截面参数 $z_0 \to 0$

茧的引力坍缩将 3D 球泡压缩为 2D 薄盘。在 PKS 术语中，这等价于：

$$z_0 \to 0 \quad (\text{截面中心趋近锥面顶点})$$

此时截面从蛋形过渡到正态分布型（银河系侧视图截面）：

```
3D 茧(球泡)     →     2D 盘(薄盘)
z₀ 大, α 任意    →     z₀ → 0, α → π/2
蛋形截面         →     正态分布型截面
```

### 2.4 压缩增强的磁场 = 锥面曲率增大

"3D 场被压入更薄的 2D 盘 = 场强增强 3 个数量级"（Gough 2026）。PKS 解释：

$$\kappa_{\text{锥面}} \propto \frac{1}{z_0}$$

当 $z_0 \to 0$，锥面曲率 $\kappa \to \infty$ → 场强暴涨。**场强的 1000× 增强 = 曲率发散的物理后果**。

## 三、螺旋臂的 PKS 解释：锥面截线的旋转

### 3.1 场线从 3D 到 2D 的投影

压缩弹簧模型的关键一步是将 3D 螺旋场 "压平" 为 2D 螺旋场。在 PKS 中，这对应：

```
3D 螺旋场线 = xy=1 锥面上的螺旋线
     ↓ (截面 z = z₀, z₀ → 0)
2D 螺旋场 = 锥面在 z→0 截面上的投影
```

### 3.2 旋臂的形成 = 场线在 2D 的截线

当 $z_0 \to 0$ 且 $\alpha \to \pi/2$（截面趋近于沿锥面切线方向），截线呈现为**对数螺旋**——这正是观测到的银河系旋臂形状：

$$r(\theta) = r_0 e^{b\theta}$$

| 参数 | 压缩弹簧模型 | PKS 几何 |
|:---|:---|:---|
| 旋臂数 | 由初始喷流模式决定 | 截面参数 $\alpha$ |
| 旋臂缠绕紧度 | 场强 / 盘质量 | $z_0/\|\ln\phi\|$ |
| 旋臂持续时间 | 磁场弛豫 ~ 亿年 | $z_0$ 演化的时间尺度 |

### 3.3 "除雪机"效应 = 场线扫过物质

Gough 描述："剩余螺旋磁场臂通过碰撞将中性气体扫到螺旋密度波中"。

这在 PKS 中是：**场线约束面扫过空间 → $xy=1$ 锥面的时间演化 → 物质被"推"入截线轨迹**。不是磁力扫雪，而是**几何约束的时间演化**。

## 四、高质量失败模式 = PKS 的曲率上限

Gough 发现：在 $M_{\text{BH}} > 10^{10} M_\odot$ 时，压缩弹簧模型失效 → 椭圆星系。

PKS 解释：

当黑洞质量过大 → 茧的初始 $z_0$ 过小 → 锥面曲率在坍缩前就极陡 → 场线无法形成稳定的 2D 螺旋 → 锥面拓扑破裂 → **场线纠缠、断裂** → 混沌恒星形成（椭圆星系）。

这对应于 $xy=1$ 锥体的**曲率奇点**：

$$\kappa_{\text{max}} = \frac{1}{z_{0,\min}}$$

当 $z_0$ 小于临界值时，截面从"连续曲线"退化为"自交曲线"—— 这是 PKS 体系对椭圆星系形成条件的一个**可定量化**的预测。
