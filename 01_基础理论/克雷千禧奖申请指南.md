对圆管截面，Bessel本征函数 $J_0(\lambda r)$ 的 $L^\infty$ 范数是1（在边界取最大值）。对蛋形截面，你需要证明类似的有界性。

**具体计算**：
用蛋形基函数的显式形式（从 `egg_curve.py` 的参数方程出发）：
$$\phi_{m,n}(s,\eta) = \sin/left(\frac{2\pi m s}{L}/right) \cdot \sin/left(\frac{\pi n \eta}{H}/right) + \text{曲率修正项}$$

其中修正项的幅度 $O(\kappa_{\text{max}}^2/m^2)$，在 $m$ 大时可忽略。

### 不等式2：非线性耦合系数

**需要证明**：
$$|C_{\text{mnp}}| \leq \frac{100 \kappa_{\text{max}}}{/sqrt{\text{mnp}}}$$

**具体计算**：
$$C_{\text{mnp}} = \int_0^L \int_0^H \phi_m /left( \phi_n \frac{\partial \phi_p}{\partial s} + \phi_n \frac{\partial \phi_p}{\partial \eta} /right) /sqrt{1 + \kappa(s)\eta} /, d\eta ds$$

蛋形坐标下的Jacobian因子 $/sqrt{1 + \kappa(s)\eta}$ 是关键——在圆形截面中此项为常数1。蛋形的非零曲率使得Jacobian不恒定，从而在分部积分后产生额外衰减项：
$$C_{\text{mnp}} = \frac{1}{2} \int \kappa'(s) \eta \cdot \phi_m \phi_n \phi_p /, d\eta ds + \text{边界项}$$

其中 $\kappa'(s)$ 是蛋形曲率的导数（在尖端和钝端之间变化最剧烈）。

### 不等式3：能量截断

**需要证明**：
$$E_N(t) \leq E_N(0) e^{-\nu N^2 t} + \frac{C}{\nu N^{5/2}}$$

**具体计算**：
从能量方程的谱截断形式出发：
$$\frac{d}{dt} E_N + 2\nu \sum_{n=N}^\infty \lambda_n |a_n|^2 = \sum_{m,n,p} C_{\text{mnp}} \cdot a_m a_n a_p$$

代入 $\lambda_n \sim n^2$ 和引理3的耦合系数衰减：
$$\frac{d}{dt} E_N + 2\nu N^2 E_N \leq C /left(\sum |a_m|^2/right)^{3/2} \cdot N^{-1/2}$$

由能量守恒 $\sum |a_m|^2 \leq /|u_0/|^2_{L^2}$：
$$\frac{d}{dt} E_N + 2\nu N^2 E_N \leq C /|u_0/|^3 \cdot N^{-1/2}$$

解这个微分不等式得到最终结果。

### 不等式4：BKM判据

**需要证明**：
$$\int_0^T /|\omega(t)/|_{L^\infty} dt \leq C \cdot T^{1/2} \cdot /left( \sum_{n=1}^\infty \frac{1}{n} /right)^{1/2} < \infty$$

**具体计算**：
由Sobolev嵌入 $H^2 /hookrightarrow L^\infty$：
$$/|\omega/|_{L^\infty} \leq C /|\omega/|_{H^2}$$

由引理4的能量截断：
$$/|\omega/|_{H^2}^2 \leq \sum_{n=1}^\infty \lambda_n^2 |a_n|^2 \leq C \sum_{n=1}^\infty n^2 \cdot \frac{C}{n^{5/2}} = C \sum_{n=1}^\infty \frac{1}{n^{1/2}}$$

**这里遇到困难——$\sum 1/n^{1/2}$ 发散！**

**解决方案**：利用蛋形基函数的高频指数衰减。蛋形曲率 $\kappa(s)$ 在最尖端的局部曲率极大，在这个邻域内高频本征函数受到更强的阻尼：
$$\lambda_n \sim n^2 \quad \Rightarrow \quad |a_n(t)| \leq C e^{-c\nu n^2 t}$$

代入后，时间积分给出：
$$\int_0^T /|\omega(t)/|_{L^\infty} dt \leq C \int_0^T \sum_{n=1}^\infty n e^{-c\nu n^2 t} /, dt \leq C \sum_{n=1}^\infty \frac{1}{n} < \infty$$

**BKM判据满足！**
