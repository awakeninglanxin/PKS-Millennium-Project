## 二、分学科名相对照

### 2.1 电磁学

| Schauberger | 传统电磁学 | 差异 |
|------------|----------|------|
| 电性力 | 电流 $I = dQ/dt$ | 电性力**不产生焦耳热**，不需要闭合回路 |
| 磁性力 | 磁场 $B = \mu_0(H+M)$ | 磁性力是**永久的自组织**，不依赖外部 H 场 |
| 生物电容器 | 电容器 $C = Q/V$ | 生物电容器的电介质**是活的**（水/树/大气） |
| 地球充电 | 全球电路 (Wilson) | Wilson 电路是**线性直流**；Schauberger 的是**脉冲振荡** |

### 2.2 热力学

| Schauberger | 传统热力学 | 差异 |
|------------|----------|------|
| 向心冷却 | 焦耳-汤姆逊效应（膨胀冷却） | 焦-汤效应是膨胀→冷却；向心是**压缩→冷却**（违反直觉） |
| 内爆 | 负熵 (Negentropy) | 负熵是**统计**概念；内爆是**动力学**过程 |
| 冷火 | 低温等离子体 | 普通等离子需要**电离**；冷火是**自发电离** |
| 温度异常点 | 相变点 | 传统只视为物性；Schauberger 视为**生命的关键** |

### 2.3 流体力学

| Schauberger | 传统流体力学 | 差异 |
|------------|-----------|------|
| 摆线螺旋 | 位势涡 $v_\theta = \Gamma/(2\pi r)$ | 位势涡角速度恒定；摆线螺旋**加速** |
| 向心加速 | 埃克曼层（边界层效应） | 埃克曼层是**边界**现象；向心加速是**体相**现象 |
| 自净河流 | 湍流混合 | 湍流是耗散的；螺旋自净是**有序的** |
| 螺旋管减阻 | 层流→湍流转捩 | 传统管流阻力单调增；螺旋管在**共振点下降** |

### 2.4 生物学

| Schauberger | 传统生物学 | 差异 |
|------------|----------|------|
| 蛋形 = 生命原形 | 卵细胞 = 球体 | 球体是各向同性的；蛋形是**各向异性储能** |
| 树木 = 生物电容器 | 树木 = 光合作用器 | 光合是化学的；生物电容是**电磁的** |
| +4°C 生命零点 | 最适温度 | 最适温度因物种而异；+4°C 是**普适的** |
| 微生物 = 能量转换器 | 微生物 = 分解者 | 分解是化学的；Schauberger 认为是**能量转换的** |

### 2.5 天文学/宇宙学

| Schauberger | 传统天文学 | 差异 |
|------------|----------|------|
| 蛋形轨道 | 开普勒椭圆 | 椭圆 = 近似；蛋形 = **真实轨道**（德语 p65, p80） |
| 行星 = 呼吸者 | 行星 = 引力束缚体 | 引力束缚是**死的**；呼吸者是**活的** |
| 太阳不是热核反应堆 | 太阳 = 核聚变 | 核聚变需要极高温度；Schauberger 认为太阳是**冷的** |
| 银河涡旋 | 银河 = 引力束缚 | 银河旋臂 = 引力+旋转；Schauberger = **向心内爆** |

### 2.6 材料科学

| Schauberger | 传统材料科学 | 差异 |
|------------|-----------|------|
| 铜 = 电荷累积器 | 铜 = 良导体 | 导体**传导**电荷；累积器**储存**电荷 |
| 铁 = 生锈 = 失去能量 | 铁锈 = 氧化 | 氧化是化学的；Schauberger 赋予**能量损失**含义 |
| 木材 = 活的绝缘体 | 木材 = 纤维素 | 纤维素是死的结构；活的木材有**电容特性** |
| 金属接触水 → 电荷 | 接触带电（摩擦电） | 摩擦电是表面效应；金属/水是**体积效应** |

## 四、名相壁垒的根源分析

### 4.1 为什么传统科学不接受这些术语

| 壁垒 | 根源 | 如何突破 |
|------|------|---------|
| **术语壁垒** | Schauberger 用日常德语造词，不遵循学术命名规则 | 建立精确的名相对照表（本文档） |
| **方法论壁垒** | Schauberger 是定性观察者，不给定量公式 | Walter 提供了数学框架→本项目实现了定量化 |
| **范式壁垒** | Schauberger 的世界观是"活的地球"，传统科学是"死的机器" | 用 COP 公式桥接：$H_X/H_S$ 同时有电磁和生物解释 |
| **实验壁垒** | Schauberger 的实验难以复现（细节未公开） | 用本项目的交互仿真+Stanford 数据验证 |
| **学术壁垒** | Schauberger 没有同行评审论文 | 本项目可以提供出版级的技术文档 |

### 4.2 已突破的名相壁垒

| 壁垒 | 突破方式 |
|------|---------|
| "内爆" = 冷聚变？ | ≠。内爆 = 向心涡流中的能量集中，不涉及核反应 |
| "生物真空" = 伪科学？ | = 有序量子真空抽取，可量化为 $W_{Hf} = \frac{1}{2}B_S(H_X-H_S)$ |
| "蛋形轨道" = 反牛顿？ | ≠。蛋形轨道 = 非闭合椭圆 = $dE/dt \neq 0$，与牛顿力学兼容 |
| "水记忆" = 安慰剂？ | = 涡流极化后的介电弛豫，可用阻抗谱测量 |
