## 论文核心发现

> "We demonstrate for pipe flow that appropriate distortions to the velocity profile lead to a complete collapse of turbulence and subsequently **friction losses are reduced by as much as 90%**."

**方法**：通过管道壁面 25 个小孔（0.5mm 直径）注入射流，扰动湍流速度剖面。主动实时反馈控制。

**关键数字**：
- 阻降幅度：最高 90%
- 实验雷诺数：Re = 3,100-5,000
- 控制孔数：25
- 孔的分布：沿管周排列

## 与 Popel 1952 的对应

| 参数 | Hof 2018 | Popel 1952 |
|:---|:---|:---|
| 方法 | 主动（25孔射流 + 反馈控制） | 被动（螺旋管几何 + 蛋咬一口截面） |
| 机制 | 打破湍流条纹自维持循环 | Dean涡旋扭曲速度剖面 + 非对称截面阻断条纹再生成 |
| 阻降 | 最高 90% | 指数 2.00→1.51（等效~25-32% 流量增幅） |
| 持续性 | 射流持续工作 | 永久几何，无需能耗 |
| 可缩放性 | 实验室尺度 | 任意尺度（管可以做大） |

**Hof 证明了"阻降在物理上可能"。Popel 证明了"钝几何就能做到"。两者互补。**
