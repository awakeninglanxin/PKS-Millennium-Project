# 双波纹盘的 CFD 建模 — 源项修正与几何正源

> **日期**: 2026-06-15
> **前提**: 已阅读 IMA 笔记的 UDF 源项建模内容 + 双波纹盘 Python 代码
> **核心修正**: IMA 笔记中的 CFD 源项模型使用了错误的基础几何（蛋形公式 → 应改为正弦波纹公式）

## 一、IMA 笔记 UDF 的原始源项（引用自 note_7472207856932302）

```c
// IMA 笔记中提出的源项（精简）
S_radial = -ρ · v_θ²/r · (1 + β_egg)   // 强制向心加速度
         - μ_eff · (v - v_res)/δ_gap    // 负粘性修正
S_axial  = ξ · ρ · (∂gap/∂θ) · f_res   // 几何谐振抽吸

// 其中 β_egg 来自蛋形方程 —— ⚠️ 这是错误的来源
```

### 1.1 问题诊断

| 源项分量 | IMA 笔记推导 | 实际几何来源 | 影响 |
|:---|:---|:---|:---|
| 强制向心项 $S_r$ | β_egg ∝ k_E (蛋形不对称度) | 锥面曲率 κ ∝ 1/z₀² ——与蛋形无关 | 量级错误 |
| 负粘性项 $S_μ$ | v_res = 0.26 m/s (Popel 实验) | n = 1.51 指数变化的局部等效，不是单一速度 | 适用范围错误 |
| 轴向抽吸 $S_z$ | ∂gap/∂θ (来自蛋形沿圆周扫描) | ∂gap/∂θ = 2A·n·|cos(nt)| (来自正弦波纹) | **函数形式错误** |

## 二、修正后的源项模型

### 2.1 几何正源：间隙函数的正确形式

上下波纹盘由反相正弦波纹定义：

$$r_{\text{上}}(t) = R_0 + A\sin(nt), \quad r_{\text{下}}(t) = R_0 - A\sin(nt)$$

实际三维间隙（锥面投影后）：

$$\text{gap}(r,\theta) = 2A\cdot|\sin(n\theta)| \cdot \frac{z_0}{z(r)}$$

其中 $z(r) = C_1 + C_2\ln r$ 是锥面高度（`get_cone_profile`）。

$$\frac{\partial\text{gap}}{\partial\theta} = 2A n \cdot \text{sgn}(\sin(n\theta))\cdot\cos(n\theta) \cdot \frac{z_0}{z(r)}$$

**这不是蛋形函数的导数**——是正弦波乘以锥面缩放因子。

### 2.2 修正后的源项

```c
// ===== 修正后的 UDF 源项 =====

real A = 5.0e-3;          // 波纹振幅 [m] (来自 Python代码)
real n_bands = 8;         // 波纹数 (默认)
real R0 = 0.05;           // 基准半径 [m]
real C1 = log(0.5)/log(phi);  // 锥面常数C₁
real C2 = user_high / (log(turns_num/2) - log(0.5));  // 锥面常数C₂

// 1. 计算锥面高度
real r_xy = sqrt(x*x + y*y);
real z_cone = C1 + C2 * log(R0/r_xy);

// 2. 计算周向角度
real theta = atan2(y, x);

// 3. 间隙函数 ─── 正弦，不是蛋形！
real gap = 2.0 * A * fabs(sin(n_bands * theta)) * (0.1 / z_cone);

// 4. 强制向心项 ─── 来源：锥面曲率约束，不是蛋形
real kappa = 1.0 / (z_cone * z_cone + 0.01);  // 锥面曲率
real S_radial = -rho * v_theta * v_theta / r_xy * (1.0 + kappa);

// 5. 间隙泵浦项 ─── 来源：正弦波纹的压缩-释放
real dgap_dtheta = 2.0 * A * n_bands * cos(n_bands * theta) * (0.1 / z_cone);
real v_gap = (v_theta / r_xy);  // 通过波纹的频率
real S_axial = dgap_dtheta * v_gap * rho;

// 6. 负粘性修正 ─── 仅在压缩区间生效
real v_compression = (dgap_dtheta < 0) ? fabs(dgap_dtheta * v_gap) : 0.0;
real S_mu = -mu_t * (v_mag - v_compression) / fmax(gap, 1e-4);
```

### 2.3 与 IMA 版本的差异总结

| 差异点 | IMA 版本 | 修正版 |
|:---|:---|:---|
| 间隙函数 | 隐式来自蛋形扫描 | 显式正弦 $2A\|\sin(nt)\|$ |
| 向心项系数 | β_egg（无关） | κ（锥面曲率） |
| 轴力来源 | 蛋形 ∂gap/∂θ | 正弦 ∂gap/∂θ = 2An·cos(nt) |
| 谐振速度 | v_res = 0.26 (单一值) | v_compression 局部动态 |
| 几何参数 | z₀, α（管道参数） | R₀, A, n（波纹参数） |

## 三、Popel 数据在 CFD 中的正确位置

IMA 笔记将 Popel 实验的指数 1.51 用作 CFD 源项中的"谐振速度阈值"。但 Popel 的 n=1.51 是**管道截面不对称**的全局效应，不是波纹盘间隙的局部效应。

| Popel 的适用域 | 波纹盘的适用域 |
|:---|:---|
| 管道截面几何 → 全局阻力指数 | 盘面波纹几何 → 局部间隙泵浦 |
| 稳态管流 → 截面形状影响全部长度 | 瞬态压缩释放 → 仅在波峰/波谷切换时生效 |
| 实验验证了 70 年 | 尚未有现代 CFD 验证 |

> **在波纹盘 CFD 中引用 Popel 的数据应限制为**：当计算域局部流速超过 Popel 的阈值反转点（v≈0.17 m/s）时，可以用缩减的等效局部摩擦系数——但这需要从 Popel 的整体指数外推到局部雷诺数，此步骤未经实验或 DNS 验证。

## 四、五个可验证的 CFD 预测

基于修正后的几何和物理模型，给出可数值验证的预测：

| # | 预测 | 验证方法 |
|:---|:---|:---|
| 1 | 间隙最窄处的局部流速应呈现周期性的峰-谷交替，与正弦波纹同频 | 提取 gap(r, θ) 最窄半径处的 v_θ 时程曲线 |
| 2 | 波纹数 n 从 5 增到 16，轴向泵浦力应增加 ~3.2 倍 | 固定网格跑两组 n，对比 F_z 积分值 |
| 3 | 下盘反相运动使间隙压缩率加倍，轴向力应比单盘波纹大 2-3 倍 | 对比单盘/双盘两种边界条件 |
| 4 | 锥面投影（z∝ln r）使外侧波纹间隙大于内侧——外侧是主要泵浦区 | 沿径向提取 ∂gap/∂θ 空间分布 |
| 5 | 在 v≈0.17 m/s 附近应出现轴向力局部峰值（对应 Popel h=10.5cm 的流体力学同构点） | 扫描 inflow 速度，监测 F_z 最大值位置 |

## 五、IMA 笔记中仍需保留的正确洞见

| 洞见 | 保留原因 |
|:---|:---|
| "负粘性"作为动态 μ_eff 的构思 | NS 框架内唯一可将 Popel 数据融入 CFD 的路径 |
| 源项 S_i 在共振态注入能量的基本思想 | 无论几何是蛋形还是正弦，能量注入的方向是对的 |
| 谐振频率与波纹数耦合 | 这是 n 在 CFD 中最自然的角色——决定泵浦频率 |
