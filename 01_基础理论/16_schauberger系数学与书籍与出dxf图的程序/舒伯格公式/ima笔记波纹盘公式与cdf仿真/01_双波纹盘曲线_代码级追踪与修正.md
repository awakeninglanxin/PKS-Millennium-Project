# 双波纹盘曲线的真实几何 — 代码级追踪与 IMA 笔记修正

> **日期**: 2026-06-15
> **来源**: 15 篇 IMA 笔记 × PKS 项目 Python 代码 × Energy Evolution PDF
> **核心修正**: IMA 笔记混淆了"蛋形截面曲线"与"波纹盘正弦波纹曲线"——两者是**完全不同的几何对象**

---

## 一、两个被混淆的几何对象

### 1.1 蛋形截面曲线（来自 Berechnung_Ei-Kurve.pdf）

这是 Schauberger 管道（如 Popel 实验 No.6/7/8）的**横截面形状**。

$$x = \pm \sqrt{\frac{1}{(z_0 - y\sin\alpha)^2} - (y\cos\alpha)^2}$$

- 决定的是管道内流动截面的不对称度
- 对应 Popel 实验中 n 从 2.00→1.51 的核心几何原因
- **这个公式与波纹盘的波纹曲面无关**

### 1.2 波纹盘正弦波纹曲线（来自 `平面环绕sin波.py`）

这是 Repulsine 飞碟**盘面的径向波纹**——上下两盘咬合形成的周期性间隙变化。

```python
# 平面环绕sin波.py 第27行
ripple = math.sin(t * self.bands) * 5
x += ripple * math.cos(t)
y += ripple * math.sin(t)
```

- 不是蛋形，是**纯正弦波**
- 参数 `bands` = 波纹数（常用 5, 8, 16, 32）
- 振幅 = 5mm（径向摆动量）
- 叠加在螺旋线 $r=R_0$ 的圆周上

### 1.3 两者在 IMA 笔记中被混为一谈

| IMA 笔记声称 | 实际 |
|:---|:---|
| "波纹盘间隙由蛋形方程决定" | ❌ 间隙 = 上下正弦波的咬合差 |
| "蛋形曲线沿圆周扫掠 = 波纹盘" | ❌ 正弦波沿圆周径向振动 = 波纹盘 |
| "z₀, α 是波纹盘的关键参数" | ❌ z₀, α 是管道截面参数，与盘面波纹无关 |

## 二、双波纹盘的真实生成过程（三条代码链）

### 2.1 链1：平面正弦波纹线

**文件**: `平面环绕sin波.py` → `RippleSpiralPipe.generate_spiral_points()`

```
步骤1: 生成螺旋线 r=R₀, θ=t, z∝t
步骤2: 叠加正弦径向振动 r(t) = R₀ + A·sin(n·t)
步骤3: 投影到锥面 z ∝ ln(r)
```

结果：一条在圆锥面上振荡的正弦螺旋线——这是**单盘波纹槽的中线**。

### 2.2 链2：锥面投影（二元锥体）

**文件**: `舒式螺旋羚羊角管圆盘.py` → `get_cone_profile(x, y)`

```python
r_xy = sqrt(x² + y²)
z_axis = C₁ + C₂·log(r_xy)    # z ∝ ln(r) ↔ xy=1
```

**这是 $xy=1$ 双曲锥在代码中的精确实现**。$z \propto \ln r$ 等价于 $xy = \text{常数}$ —— 因为 $\ln(xy) = \ln x + \ln y = \text{常数} + z$。

> 这个 `get_cone_profile` 是**唯一真正对应 PKS 双锥体的代码函数**。蛋形截面曲线（Berechnung_Ei-Kurve.pdf）对应的代码是独立于锥面投影的。

### 2.3 链3：双盘咬合 → 周期性间隙

```
上盘: r(t) = R₀ + A·sin(n·t)    (波纹数 n, 振幅 A)
下盘: r(t) = R₀ - A·sin(n·t)    (反相, 振幅相同)
间隙: gap(t) = 2A·|sin(n·t)|    (纯正弦间隙变化!)
```

- **间隙不是蛋形**——是正弦的绝对值
- 波纹数 n 决定了每圈的压缩-释放循环次数
- 双盘反相运动产生"咀嚼"效应——在波峰处挤压，波谷处释放

## 三、IMA 笔记中需要修正的具体观点

| # | IMA 笔记错误声明 | 修正 |
|:---|:---|:---|
| 1 | "波纹盘截面 = 蛋形曲线 x = ±√(1/(z₀-y·sinα)² - (y·cosα)²)" | 波纹盘截面 = $r(t) = R₀ + A·\sin(n·t)$，是**正弦曲线**，不是蛋形 |
| 2 | "z₀ 和 α 决定波纹盘的间隙" | z₀ 和 α 决定的是**管道截面**（Popel No.6/7/8 的蛋咬一口形状），与波纹盘间隙无关 |
| 3 | "蛋形曲线沿圆周扫掠 = 三维波纹盘" | 正确表述：正弦波纹曲线沿锥面螺旋缠绕 = 三维波纹盘 |
| 4 | "负电阻源自蛋形几何" | 负电阻（如果存在）源自**锥面约束 + 正弦间隙泵浦**的联合效应，不是单独的蛋形 |
| 5 | "波纹盘 = HORIZONT 2: α~45°" | 这是 8图PKS 中的特定切割角对应黄金漏斗，有独立的物理含义（过度延伸） |

### 3.1 为什么会产生这个混淆

Schauberger 的完整系统有两条独立的设计线：

```
线A（管道）: 蛋形截面 (x,y) → Popel 实验 → n 从 2.00→1.51
线B（盘面）: 正弦波纹 (r,θ) → Repulsine → 压缩-释放循环

两者都涉及不对称性，但是：
- 线A的不对称: 截面的钝端/尖端 + 凹缺 (蛋咬一口)
- 线B的不对称: 径向的正弦振动 (波峰/波谷)
```

IMA 笔记把线A的蛋形公式直接套用到线B的波纹描述上——这是两块**不同物理组件**的公式交叉污染。

## 四、正确理解：双波纹盘的完整物理链条

```
上盘波纹: r(t) = R₀ + A·sin(n·t)          ← 纯正弦
下盘波纹: r(t) = R₀ - A·sin(n·t)          ← 反相正弦
         ↓
间隙变化: gap(t) = 2A·|sin(n·t)|           ← 正弦绝对值
         ↓
锥面投影: z ∝ ln(r) ↔ xy=1               ← get_cone_profile
         ↓
压缩释放: n 次/圈的径向挤压-释放循环
         ↓
物理效应: 水分子在波峰处被压缩→HHO分解
         在波谷处释放→冷等离子体膨胀
         ↓
推动机制: 螺旋离心甩出 + 锥面焦散线喷射
```

## 五、代码 → 物理的逐行对应表

| 代码行 | 数学 | 物理含义 |
|:---|:---|:---|
| `t = i/points_per_turn * 2π * turns` | $t \in [0, 2\pi N]$ | 旋转角变量 |
| `ripple = sin(t * bands) * 5` | $\Delta r = A\sin(nt)$ | 正弦波纹，A=5mm, n=bands |
| `x += ripple * cos(t)` | $x + \Delta r\cos t$ | 径向振动分解到 X 分量 |
| `y += ripple * sin(t)` | $y + \Delta r\sin t$ | 径向振动分解到 Y 分量 |
| `get_cone_profile(x,y)` | $z = C_1+C_2\ln r$ | 双曲锥面投影 ($xy=1$) |
| `bottom_r * φ^(t/a)` | $r=r_0\cdot\phi^{t/a}$ | 黄金比对数螺旋收缩 |
