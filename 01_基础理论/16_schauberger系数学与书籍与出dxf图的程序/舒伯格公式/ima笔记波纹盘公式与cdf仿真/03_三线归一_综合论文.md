# 双波纹盘 × PKS 双锥体 × Popel 实验 — 三线归一

> **日期**: 2026-06-15
> **类型**: 综合性学术论文（含修正）
> **涉及文献**: 15 篇 IMA 笔记、Popel Report 1952、Hof 2018 Nature Physics、项目 Python 代码 3 链
> **核心论题**: Schauberger 体系中有三条独立的设计线——管道截面（蛋咬一口）、盘面波纹（正弦）、锥面投影（双曲锥）。IMA 笔记混淆了前两者。本文给出三线的精确区分和统一框架。

## §零、问题陈述

2026年6月15日下午，蓝馨在 IMA 笔记中与 AI 进行了关于双波纹盘 CFD 仿真的深入讨论。AI 在笔记中使用了 Berechnung_Ei-Kurve.pdf 的蛋形截面公式来推导波纹盘的几何源项。然而，PKS 项目中实际生成波纹盘曲线的 Python 代码（`平面环绕sin波.py`、`舒式螺旋羚羊角管圆盘.py`）使用的是**纯正弦波纹**叠加在**双曲锥面**上——与蛋形公式所描述的管道截面是**两个完全不同的几何对象**。

本文旨在：
1. 精确区分这三条设计线
2. 给出每条线的数学公式、对应代码、物理角色和 IMA 笔记中的正确定位
3. 提供一个统一的三线框架——三条线在 Popel 实验中同时出现但分别起作用

## §一、三线精确区分

### 1.1 线A：管道截面曲线（蛋咬一口）

| 属性 | 值 |
|:---|:---|
| **数学** | $x = \pm \sqrt{\frac{1}{(z_0 - y\sin\alpha)^2} - (y\cos\alpha)^2}$ |
| **代码文件** | `Berechnung_Ei-Kurve.pdf`（解析公式）、`cone_geometry_constructor.py`（截面构造） |
| **物理角色** | 决定螺旋管中流体截面的不对称度 |
| **Popel 实验对应** | 管 No.6/7/8 的截面形状——直接决定摩擦指数 $n$ 从 2.00 降到 1.51 |
| **IMA 笔记误用** | 被当作波纹盘间隙函数的来源（❌） |

> 线A是**管道截面内的 2D 几何**。它不参与盘面的径向振动。

### 1.2 线B：盘面波纹曲线（正弦）

| 属性 | 值 |
|:---|:---|
| **数学** | $r(\theta) = R_0 + A\sin(n\theta)$ |
| **代码文件** | `平面环绕sin波.py` → `ripple = sin(t * bands) * 5` |
| **物理角色** | 决定波纹盘间隙的周期性压缩-释放 |
| **Repulsine 对应** | 盘面上 $n$ 个波纹——每圈的压缩次数 |
| **IMA 笔记漏用** | IMA 笔记完全未引用 `平面环绕sin波.py`（❌） |

> 线B是**盘面上的 1D 径向振动**叠加在圆周上。每个波纹周期对应一次局部压缩——水分子在此被"锤击"。

### 1.3 线C：锥面投影（双曲锥）

| 属性 | 值 |
|:---|:---|
| **数学** | $z = C_1 + C_2\ln r \;\leftrightarrow\; xy = \text{常数}$ |
| **代码文件** | `舒式螺旋羚羊角管圆盘.py` → `get_cone_profile(x, y)` |
| **物理角色** | 决定波纹盘的整体三维形状——垂直于圆周的轴向位移 |
| **PKS 对应** | 这是 PKS 双锥体中 $xy=1$ 的唯一代码实现 |
| **IMA 笔记处理** | 正确地在 8图PKS 语境中提到，但未与代码挂钩 |

> 线C是三线中**最根本的**——它是 Schauberger 双曲锥的纯数学表达。没有线C，波纹盘只是一个平面的正弦波——有了线C，波纹盘才是三维的涡旋约束面。

## §二、三线在 Popel 实验中的联合运作

Popel 1952 年的 8 管实验无意中为三线提供了一个完整的交叉验证表：

| 管号 | A(截面) | B(螺旋) | 指数 | 线组合 |
|:---:|:---|:---|:---:|:---|
| 3 | 圆形 | 无 | 2.00 | 基线 |
| 5 | 渐变圆形 | 无 | 2.00 | A(对称) → 无效果 |
| 2 | 圆形 | **有** | 1.67 | **B(螺旋)** → -16.5% |
| 6 | 🔴 蛋咬一口 | **有** | 1.57 | **A+B** → -21.5% |
| 8 | 🔴 蛋咬一口(小) | **有** | **1.51** | **A+B+截面缩放** → -24.5% |

> **管 2 vs 管 3**：B（螺旋）单独贡献 16.5% 的指数降低。对应 Repulsine 的波纹盘螺旋效应——每个波纹周期引入 Dean 涡旋。
>
> **管 6 vs 管 2**：A（蛋咬一口截面）在 B 的基础上额外贡献 5%。对应 Popel 实验中波纹盘间隙如果使用蛋咬一口而非圆形截面会额外增加收益。
>
> **管 8 vs 管 6**：截面缩放（更小的蛋咬一口）再贡献 2%。对应波纹盘外缘 vs 内缘的间隙差异——锥面投影（线C）使外侧间隙 > 内侧间隙。

### 2.1 波纹盘应如何利用 Popel 数据

波纹盘的间隙是**线B**（正弦波纹）主导的，不是**线A**（蛋形截面）主导的。因此：

- Popel 的指数降低主要来自**线B（螺旋）**而不是**线A（截面不对称）**
- 线B 单独贡献约 16.5% 的指数降幅（管2 vs 管3）
- 线A 在管6/7/8 中额外贡献约 5-8%
- **如果波纹盘使用圆形截面间隙（仅有正弦波纹而无蛋咬一口截面），预期指数从 2.00→~1.67**
- **如果波纹盘使用蛋咬一口截面间隙（正弦波纹 + 不对称截面），预期指数从 2.00→~1.51**

> 这在工程上意味着：制造蛋咬一口截面的波纹间隙在加工难度上远大于制造简单的正弦波纹间隙——但能带来额外 5-8% 的性能提升。

## §四、三线的统一框架

```
PKS 双锥体 ab=1
    │
    ├── 线A: 管道截面曲线 (Berechnung_Ei-Kurve.pdf)
    │     x = ±√(1/(z₀-y·sinα)² - (y·cosα)²)
    │     代码: cone_geometry_constructor.py
    │     物理: 决定流体截面的不对称度
    │     Popel: n = 1.51 (管 No.8)
    │     IMA: 正确使用 ✅
    │
    ├── 线B: 盘面正弦波纹 (平面环绕sin波.py)
    │     r(θ) = R₀ + A·sin(nθ)
    │     代码: ripple = sin(t * bands) * 5
    │     物理: 决定压缩-释放的周期数
    │     Popel: 螺旋线 = 1D 波纹 (n=1 长波长)
    │     IMA: 完全遗漏 ❌
    │
    └── 线C: 锥面投影 (get_cone_profile)
          z = C₁ + C₂·ln(r) ↔ xy=1
          代码: z_axis = C₁ + C₂·log(r_xy)
          物理: 三维涡旋约束面
          PKS: 双锥体的唯一代码实现
          IMA: 定性提到但未与代码挂钩 🔶
```

### 4.1 Popel 实验中三线的权重

在 Popel 实验中，三线的相对贡献（以指数偏离 2.00 为度量）：

| 线 | 独立贡献 | 联合贡献 |
|:---|:---:|:---|
| B (螺旋) 单独 | 16.5% | — |
| A (截面) 单独 | 0%（管5, 锥形圆形 → n=2.00） | — |
| B+A 联合 | — | 21.5%（管6） |
| B+A+缩放 | — | 24.5%（管8） |

> **线A 不能独立产生阻降**——它必须与线B联合工作。这解释了为什么单有螺旋（n=1.67，管2）或单有不对称截面（n=2.00，管5）的效果都不如两者的组合——且小截面蛋咬一口比大截面更有效。

## §六、最终陈述

Schauberger 的数学体系中有三个独立的几何构造——管道截面（蛋咬一口）、盘面波纹（正弦）、锥面投影（双曲锥）——三者共同构成完整的内爆流体力学的几何框架。其中：

- **线B（正弦波纹）是 Repulsine 波纹盘的专属几何**——它在 `平面环绕sin波.py` 中已精确实现，但未被 IMA 笔记引用
- **线A（蛋形截面）是 Popel 管道的专属几何**——它在 `Berechnung_Ei-Kurve.pdf` 中已精确实现，但被 IMA 笔记错误地扩展到了波纹盘
- **线C（双曲锥投影）是两条线的共同基础**——它在 `get_cone_profile` 中已精确实现，是整个 PKS 体系的几何根

修正之后，三条线各归其位——如同 Schauberger 本人设计的原始系统中就有分别的管道截面（圆→蛋咬一口）、盘面波形（正弦）、和整体锥面（双曲锥）——三者从未混淆。
