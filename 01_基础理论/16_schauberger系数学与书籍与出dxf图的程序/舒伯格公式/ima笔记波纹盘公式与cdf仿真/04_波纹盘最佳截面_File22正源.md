## §一、现有 IMA 笔记的问题

### 问题 1：01/02/03 将 `平面环绕sin波.py` 作为 Line B 的唯一代表

`平面环绕sin波.py` 的核心公式：

```python
ripple = sin(t * bands) * 5   # 等幅正弦波纹，叠加到螺旋管上
```

这是**等幅正交径向调制**——不适合拉伐尔喷管建模，因为：
- 无喉部收缩→扩张几何
- 无对数衰减补偿边界层
- 无蛋形不对称产生定向冲量

### 问题 2：02_CFD源项修正 将蛋形截面（Line A）错误用作波纹盘间隙

Line A 公式（`Berechnung_Ei-Kurve.pdf`）描述的是**管道固定截面**，不适合描述**随角度变化的盘面间隙**。正确的间隙函数应来自 Line B（正弦波纹）的新定义——即 File 22 的上下曲线差。

### 问题 3：03_三线归一 中 Line B 的数学形式过于简单

当前 Line B 写为 $r(\theta) = R_0 + A\sin(n\theta)$，但 File 22 表明实际使用的是：

$$\begin{cases}
\text{upper}(\theta) = \text{amp}(\theta) \cdot \left[\dfrac{2\sin\theta}{b+\sqrt{b^2-4k\cos\theta}} - t_{\min}\right] \\[8pt]
\text{lower}(\theta) = \text{amp}(\theta) \cdot \left[m\cdot\text{term}_1 + \text{term}_2 + t_{\min}\right]
\end{cases}$$

这是一个**蛋形调制 + 对数衰减**的复合波形，远比 $R_0 + A\sin(n\theta)$ 复杂。

## §二、修正后的三线定义

### Line A（不变）：管道固定截面

$$x(t) = \frac{2\sin t}{b + \sqrt{b^2 - 4k\cos t}}, \quad (k,b) = \left(\frac{2}{3}, \frac{5}{3}\right)$$

对应：Popel No.6/7/8 管截面 · `舒伯格蛋我的参数方程.py`

### Line B（修正 ✏️）：盘面波纹间隙函数

**旧定义**（IMA 笔记 01-03）：
$$r(\theta) = R_0 + A\sin(n\theta)$$

**新定义**（基于 File 22）：
$$\boxed{h(\theta) = \text{amp}(\theta) \cdot \left[y_{\text{add}}(\theta) - x_{\text{minus}}(\theta)\right]}$$

其中：

$$\text{amp}(\theta) = \frac{1}{\left(1 + \frac{\theta}{2\pi}\right) \cdot \ln(n+1)}$$

$$x_{\text{minus}}(\theta) = a\left[\frac{2\sin\theta}{b + \sqrt{b^2 - 4k\cos\theta}} - t_{\min}\right]$$

$$y_{\text{add}}(\theta) = a\left[m\cdot\text{term}_1 + \text{term}_2 + t_{\min}\right]$$

对应：`22_双波纹盘最佳截面曲线.py` · File 22 拉伐尔喷管论证

**关键差异**：

| 属性 | 旧 Line B（平面环绕sin波） | 新 Line B（File 22） |
|:---|:---|:---|
| 波形 | 纯正弦 $A\sin(n\theta)$ | 蛋形调制正弦 $x_{\text{egg}}(\theta)$ + 非对称 $y_{\text{egg}}(\theta)$ |
| 衰减 | 无 | 对数衰减 $\propto 1/(1+\theta/(2\pi))$ |
| 喉部 | 无（等幅无壅塞） | 有（上下曲线差自然形成 $h_{\min}$） |
| 多脉冲 | 叠加在螺旋管上 | 明确的 $n=5$ 个压缩-膨胀周期 |
| 非对称冲量 | 无 | 蛋形天然不对称 |

### Line C（不变）：锥面投影

$$z \propto \ln r \quad \leftrightarrow \quad xy = 1$$

对应：双曲线 $y=1/t$ · 衰减包络 $\text{amp}(t)$ 的数学来源

### 3.2 新源项（基于 File 22）

```python
# ✅ 新：用 File 22 的上下曲线差作为间隙
import math

# 舒伯格蛋参数
k, b = 2.0/3.0, 5.0/3.0
a, m = 2*math.pi, 2.0/3.0
n_cycles = 5
t_min = 2*math.pi / (n_cycles + 1)

def amp(theta):
    """对数积分衰减包络"""
    return 1.0 / ((1.0 + theta/(2*math.pi)) * math.log(n_cycles + 1))

def x_minus(theta):
    """上截面 — 蛋的 x 分量"""
    denom = b + math.sqrt(b*b - 4*k*math.cos(theta))
    return a * (2*math.sin(theta)/denom - t_min)

def y_add(theta):
    """下截面 — 蛋的 y 分量"""
    term1 = -(math.sqrt(1 + k*k) * (-math.sqrt(b*b - 4*k) +
              math.sqrt(b*b - 4*k*math.cos(math.pi)))) / (2*k)
    term2 = ((1/(2*math.sqrt(1 + k*k))) *
             (((k*k - 1)/k)*b + ((k*k + 1)/k)*
              math.sqrt(b*b - 4*k*math.cos(theta)))) - \
            ((b*(-1 + k*k) + math.sqrt(b*b - 4*k)*(1 + k*k)) /
             (2*k*math.sqrt(1 + k*k)))
    return a * (m*term1 + term2 + t_min)

def gap(theta):
    """波纹盘间隙函数 — CFD UDF 源项"""
    return amp(theta) * (y_add(theta) - x_minus(theta))

def gap_derivative(theta):
    """间隙导数 — 用于判断收敛/扩张段"""
    delta = 1e-6
    return (gap(theta + delta) - gap(theta - delta)) / (2*delta)
```

### 3.3 源项物理含义

| CFD 变量 | UDF 宏 | 源项表达式 | 说明 |
|:---|:---|:---|:---|
| 间隙高度 | `DEFINE_PROFILE` | `h = gap(theta)` | 随角度变化的通道高度 |
| 壁面运动 | `DEFINE_GRID_MOTION` | 上下曲面在 $z$ 方向移动 | $z_{\text{upper}} = h/2$, $z_{\text{lower}} = -h/2$ |
| 质量源 | `DEFINE_SOURCE` | $S_m = \rho A \cdot dh/dt$ | 间隙变化引起的压缩/膨胀 |
