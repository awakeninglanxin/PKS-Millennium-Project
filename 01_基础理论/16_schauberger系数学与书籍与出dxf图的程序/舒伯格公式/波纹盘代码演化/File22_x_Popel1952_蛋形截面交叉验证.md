# File 22 × Popel 1952 · 蛋形截面交叉验证

**波纹盘截面参数 $(k,b)=(\frac{2}{3},\frac{5}{3})$ 与 Popel 实验 $n=1.51$ 的定量对应**

> 交叉参考：`Popel_Report_1952_实验数据表.md` · `File22_拉伐尔喷管流体力学论证.md`  
> 分析日期：2026-06-15

## 1. 核心假设

> 如果 Popel 1952 的 No.6/7/8 管（"蛋咬一口"截面）的减阻效应（$n: 2.00 \to 1.51$）可以**从截面形状角度**被 File 22 的 $(k,b)=(\frac{2}{3},\frac{5}{3})$ 蛋形参数化所覆盖，那么 Popel 数据就是 File 22 波纹盘截面的**独立实验验证**——蛋形截面减阻不仅在直管中有效，在旋转波纹盘中也应当有效。

## 2. 截面参数对应

### 2.1 File 22 的蛋形参数化

舒伯格蛋公式：

$$x(t) = \frac{2\sin t}{b + \sqrt{b^2 - 4k\cos t}}$$

基准参数：$k = 2/3$，$b = 5/3$。

这两个值来自 **3-4-5 勾股三角形**的边长比：

$$\frac{k}{1} = \frac{2}{3}, \quad \frac{b}{1} = \frac{5}{3}$$

完整边长比：$2:5:3\sqrt{3} \approx 2:5:5.196$（归一化到斜边 $=1$：$k \approx 0.333, b \approx 0.833$）。

### 2.2 截面形状的几何性质

舒伯格蛋的截面不对称性来自分母 $\sqrt{b^2 - 4k\cos t}$：
- $t=0$（cos=1）：分母 $= \sqrt{25/9 - 8/3} = \sqrt{1/9} = 1/3$，$x \approx 0$（钝端中心）
- $t=\pi$（cos=−1）：分母 $= \sqrt{25/9 + 8/3} = \sqrt{49/9} = 7/3$，$x \approx 0$（尖端中心）
- $t=\pi/2$（cos=0）：分母 $= \sqrt{25/9} = 5/3$，$x = 2/(5/3 + 5/3) = 3/5$（最大宽度）

截面**宽高比** $\approx 3:1$（宽度/尖端到钝端长度 ≈ 0.6 : 约2.0）。

### 2.3 Popel 的"蛋咬一口"截面

Popel 原文描述："蛋咬一口"形状，类比为 Apple 公司 logo——圆形轮廓上有一个**凹缺（notch）**。

普通蛋形（只有钝端+尖端的不对称）→ 对应 $x(t) = 2\sin t/(b+\sqrt{b^2-4k\cos t})$，单参数 $(k,b)$ 控制的平滑曲线。

蛋咬一口形状 = 在普通蛋形截面上额外叠加一个**局部凹入**。从数学上，这相当于在蛋形公式上叠加一个局部负扰动：

$$x_{\text{bitten}}(t) = x_{\text{egg}}(t) - d \cdot \exp\left(-\frac{(t-t_{\text{notch}})^2}{2\sigma^2}\right)$$

其中 $d$ 是凹缺深度，$t_{\text{notch}}$ 是凹缺位置，$\sigma$ 是凹缺宽度。

**File 22 的 $(k,b)$ 参数化覆盖了"普通蛋形"部分，但尚未显式建模凹缺。**

## 3. Popel 数据 → 波纹盘的推论

### 3.1 核心对应

| Popel 1952 发现 | File 22 对应 | 推论 |
|:---|:---|:---|
| 圆形截面 $n=2.00$（标准湍流） | — | 若波纹盘截面为圆形，间隙 $h(t)$ 为纯正弦 → 对称压缩膨胀，净功为零 |
| 蛋形截面（No.6）$n=1.57$ | 舒伯格蛋 $(k,b)=(2/3, 5/3)$，非对称正弦波 | 蛋形截面减阻 → 波纹盘中间隙流体在压缩段遇到的阻力小于圆形截面 |
| 蛋咬一口截面（No.8）$n=1.51$ | 蛋 + 凹缺（未建模） | 凹缺进一步减阻 → 可能需要优化 $(k,b)$ 或增加凹缺参数 |

### 3.2 摩擦指数与拉伐尔效率

Popel 的摩擦指数 $n=1.51$ 意味着水头损失 $h \propto q^{1.51}$（而非标准湍流的 $q^{2.00}$）。在波纹盘语境中：

- $h$（水头损失）↔ 间隙中的压力降 $\Delta p$
- $q$（流量）↔ 间隙质量流量 $\dot{m}$

标准拉伐尔喷管的**总压损失**来自激波和边界层摩擦。若蛋形截面将湍流摩擦指数从 2.0 降到 1.51，则**总压恢复系数**提高：

$$\frac{p_{02}}{p_{01}} \approx \left(\frac{\dot{m}_2}{\dot{m}_1}\right)^{1.51/2.00} = (\dot{m}\text{ ratio})^{0.755}$$

对于相同的喉部尺寸，蛋形截面的总压损失仅为圆形截面的 $(\Delta p_{\text{egg}}/\Delta p_{\text{circ}}) \propto q^{1.51-2.00} = q^{-0.49}$。在高流量（$q$ 大）时，蛋形的优势更显著。

**对应到波纹盘**：$q$ 在喉部达到最大 → 蛋形截面在喉部的减阻最显著 → 壅塞质量流量 $\dot{m}$ 更高 → 真空效果更强。

### 3.3 阈值反转与波纹盘启动条件

Popel 发现了水头 $h = 10.5$ cm 的阈值反转：低于此值，直管流量 > 螺旋管；高于此值，螺旋管永远优于直管。

在波纹盘语境中：

$$h \leftrightarrow \omega^2\text{（转速的平方）}$$

存在一个**临界转速** $\omega_c$：低于此值，蛋形截面减阻不足以克服粘性耗散；高于此值，$\propto q^{-0.49}$ 的优势被放大，真空效应启动。

$$\omega_c \propto \sqrt{h_c} = \sqrt{10.5\text{ cm}} \propto \text{约 } 1000\text{–}2000\text{ RPM}$$

这与 Schauberger 描述的 Repulsine "需要一定转速才开始产生升力" 一致。
