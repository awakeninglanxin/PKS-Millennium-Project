## 一、蛋形曲线 — 三种形式

### 1.1 显式形式 (Explizite Form)

$$\bar{x} = \pm\sqrt{\frac{1}{(z_0 - \bar{y}\sin\alpha)^2} - (\bar{y}\cos\alpha)^2}$$

| 符号 | 含义 | 项目对应 |
|:---|:---|:---|
| $\bar{x}, \bar{y}$ | 切割平面上的新坐标 | `舒伯格蛋形公式.txt` 中的 (x,y) |
| $z_0$ | 切割高度 (Schnitthöhe) | `b=1.667` 参数 |
| $\alpha$ | 切割角度 (Schnittwinkel) | `a=0.588` rad = 33.69° |

**项目实现**：`舒伯格公式 — 导读.md` §1.3(B) — 曲面方程直接使用此形式的旋转推广。

### 1.2 参数形式 (Parameter-Form)

$$r = \frac{1}{2\cos\phi\cdot\sin\alpha}\left[z_0 \pm \sqrt{z_0^2 - \frac{4\cos\phi\cdot\sin\alpha}{\sqrt{\sin^2\phi + \cos^2\phi\cdot\cos^2\alpha}}}\right]$$
$$x = r\sin\phi, \quad y = r\cos\phi$$

| 根号符号 | 切割区域 | 视觉效果 |
|:---|:---|:---|
| $+$ | 锥管部分 (Kegelrohr) | 蛋形外轮廓 |
| $-$ | 扁平山脉部分 (Gebirge) | 蛋形内凹陷 |

**项目实现**：`舒伯格蛋形公式.txt` — Rhino 参数方程中的 `sqrt(b² - 4cost·sina/√(...))` 项精确对应此公式（b=z₀, a=α）。

### 1.3 两点形式 (Zwei-Punkte-Form)

给定尖锐主顶点 P₁(y₁,z₁) 和钝主顶点 P₂(y₂,z₂)，$z=1/y$ (锥面约束)：

$$z_0 = \frac{z_1^2 + z_2^2}{z_1 + z_2}$$

$$\tan\alpha = -\frac{z_1 z_2(z_2 - z_1)}{z_1 + z_2}$$

**八度蛋示例** (Oktav-Ei): $z_1=1, z_2=2$

$$z_0 = \frac{1+4}{1+2} = \frac{5}{3} \approx 1.667$$
$$\tan\alpha = -\frac{1\cdot2(2-1)}{1+2} = -\frac{2}{3} \approx -0.667$$
$$\alpha = \arctan(2/3) \approx 0.588\text{ rad} \approx 33.69°$$

> 🔴 **这正是项目中的默认参数 $a=0.588, b=1.667$！** — 八度蛋不是任意选，是 $z_1:z_2 = 1:2$（纯八度）的精确几何解。

### 1.4 点-斜率形式

A: 已知 z₁ 和 α → $z_0 = z_1 - \frac{1}{z_1}\tan\alpha$
B: 已知 z₂ 和 α → $z_0 = z_2 + \frac{1}{z_2}\tan\alpha$

## 二、体积公式

$$V = \frac{\pi}{\cos\alpha}\left[\frac{z_1+z_2}{(z_1 z_0 - \tan\alpha)(z_2 z_0 + \tan\alpha)} - \frac{1}{3}\left(\frac{1}{z_1^3} + \frac{1}{z_2^3}\right)\right]$$

## 三、从蛋形到椭圆 — 圆锥替代推导

### 3.1 直线替代双曲线

若将 $z=1/y$（双曲锥）替换为 $z=y$（直圆锥 → 即 PKS双锥体理论中的 **电性锥**！）：

$$x^2 + z^2 = y^2 \quad (\text{圆锥取代双曲锥})$$

相同切割参数 $(z_0, \alpha)$ 下，截面退化为**椭圆**：

### 3.2 椭圆方程

$$\bar{x}^2 + \bar{y}^2 \cdot \cos 2\alpha + 2z_0 \bar{y} \sin\alpha - z_0^2 = 0$$

标准形式：

$$\frac{\bar{x}^2}{a^2} + \frac{(\bar{y} + z_0\sin\alpha/\cos2\alpha)^2}{b^2} = 1$$

半轴：

$$a = \frac{z_0\cos\alpha}{\sqrt{\cos 2\alpha}}, \quad b = \frac{z_0\cos\alpha}{\cos 2\alpha}$$

### 3.3 椭圆半焦距 c（🔴 本项目关键公式）

$$c = \sqrt{b^2 - a^2} = \frac{z_0\sqrt{2}}{2} \cdot \tan(2\alpha)$$

| 输入 | 值 |
|:---|:---|
| $z_0$ | 5/3 = 1.667 |
| $\alpha$ | arctan(2/3) ≈ 0.588 rad |
| $2\alpha$ | 1.176 rad |
| $\tan(2\alpha)$ | ≈ 1.966 |
| **c** | **≈ 2.316** |

> 🔴 **这是 PKS 项目白银比体系中的核心参数！** $c = \frac{z_0\sqrt{2}}{2}\tan(2\alpha)$ 出现在 `白银比_sqrt2_在项目中的全面体现.md` 中——它是**蛋形截面与椭圆截面之间的"耦合尺度"**，即双曲锥（磁性）和直锥（电性）的半焦距差。

## 五、与 PKS 双锥体统一的深层连接

### 5.1 蛋形 ↔ 椭圆 = 双曲锥 ↔ 直锥

```
PDF §三: 圆锥替代推导

直圆锥 (电性)               超双曲锥 (磁性) 
z = y                       z = 1/y
截面: 椭圆                  截面: 蛋形
                                |
半焦距: c = (z₀√2/2)·tan(2α)   |
                                |
        └──────────┬──────────┘
                   ↓
          PKS双锥体统一理论
          ab=1 极化原理
          α = R₀/R_c = 1/137
```

### 5.2 八度蛋 = 两锥耦合的基准点

$$z_1:z_2 = 1:2 \quad\Rightarrow\quad z_0=\frac{5}{3},\ \alpha=\arctan\frac{2}{3}$$

这个比例不是随意选的——它是**纯八度（频率比 2:1）**在三锥几何中的实现。$z_0=5/3$ 对应 3-4-5 三角形的弦边比（5/3 = 大六度音程），$α=33.69°$ 接近 Pythagorean 大三度（5:4 → arctan(3/4)≈36.87°）。
