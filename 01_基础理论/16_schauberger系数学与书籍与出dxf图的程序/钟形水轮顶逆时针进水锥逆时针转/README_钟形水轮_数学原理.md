## 二、核心数学

### 2.1 螺旋中心线方程

三种程序共享同一个参数化曲线族：

$$x_{\text{spiral}} = \pm\sin(3ft) \cdot \sqrt{\frac{1}{(b - t\sin\alpha)^2} - (t\cos\alpha)^2}$$

$$y_{\text{spiral}} = \pm\cos(3ft) \cdot \sqrt{\frac{1}{(b - t\sin\alpha)^2} - (t\cos\alpha)^2}$$

$$z_{\text{spiral}} = -t \cdot z_v$$

| 参数 | 含义 | 典型值 | PKS对应 |
|:---|:---|:---:|:---|
| $\alpha$ (angle) | **螺锥半角** | 0.588 rad (33.7°) | 控制锥体张角的可变参数 |
| $b$ | **螺旋膨胀控制** | 1.6 / 1.618(φ) | 黄金比例φ = 蛋形的自然缩放因子 |
| $f$ | **螺旋频率** | 1.5 / 3 | $3f$ = 周期=3的谐波 |
| $z_v$ | **z轴线性速率** | 1 / 2 | 锥体高度控制 |
| $t$ | 参数时间 | $t \in [0, 2\pi/3]$ | 120°圆锥展开角 |
| num_theta | 截面点数 | **12** | SEG第1环12滚柱 |

### 2.2 数学本质：锥面螺旋的"钟形"

`sqrt(1/(b-t·sinα)² - (t·cosα)²)` 项产生了**非线性的径向缩放**：

- 当$t$接近0时 → $\sqrt{1/b^2} = 1/b$ → 半径为常数（圆柱段）
- 当$t$增大时 → $(b-t\sin\alpha)^2$ 减小 → $\sqrt{}$ 增大 → 半径加速膨胀
- 到达$t\cos\alpha > 1/(b-t\sin\alpha)$ → 开方为负数 → 曲线终止

这种"先圆柱、后膨大"的形状就是**钟形（bell-shaped）**的数学来源——不是抛物线、不是圆锥，而是一个**带奇异点的拟共形映射**。

### 2.3 Frenet 标架 — 螺旋的局部几何

两个版本采用不同的Frenet标架计算方法：

| 版本 | 方法 | 优缺点 |
|:---|:---|:---|
| **底进水版（解析）** | $T=\frac{dr}{dt}/\|\frac{dr}{dt}\|$, $N$由加速度$d^2r/dt^2$归一化, $B=T\times N$ | 精确但参数敏感 |
| **最终版（数值导数）** | NumPy差分→$T, d^2r/dt^2$→$N$, $B=T\times N$ | 稳定、通用、可调 |

```python
# 最终版的Frenet标架（简洁）
dx_dt = 3*f * cos(3*f*t[i]) * sqrt(...)
dy_dt = -3*f * sin(3*f*t[i]) * sqrt(...)
dz_dt = -z_v
T = [dx_dt, dy_dt, dz_dt] / norm    # 切线 = 一阶导归一化
N = [d2x, d2y, 0] / norm             # 法向 = 二阶导归一化
B = cross(T, N) / norm               # 副法向 = T×N
```

### 2.4 Rodrigues 旋转公式 — 阵列复制

沿螺旋线生成叶片的3D曲面：

$$R(\mathbf{v}, \psi) = I + \sin\psi \cdot K + (1-\cos\psi) \cdot K^2$$

其中$K$是$\mathbf{v}$的叉乘矩阵：
$$K = \begin{bmatrix}0&-v_z&v_y\\v_z&0&-v_x\\-v_y&v_x&0\end{bmatrix}$$

- 每片"花瓣"是Frenet标架中沿B方向展开的椭圆面
- 多片花瓣沿螺旋线等距排列 → 产生叶轮效应

## 三、与 PKS 项目的深度关联

### 3.1 钟形水轮的核心创新 — 轴与水旋反向

| 水轮类型 | 水流涡旋方向 | 发电轴旋转方向 | 效率 |
|:---|:---|:---|:---:|
| **传统风扇叶水轮** | 顺时针（或逆时针） | **与水流同向** | 12~15% |
| **Schauberger 钟形水轮** | 可顺可逆（参数可调） | **与水流反向** | **96%** |

**关键机制**：
- 传统水轮：水流直接冲击叶片旋转 → 轴与涡旋同向 → 叶片切出水流时产生湍流→大量能量以涡旋形式脱落在叶片后的尾流中
- Schauberger水轮：进水方向是**自由参数**（顶部进水可顺势/逆势），核心是 **轴转方向与水流内旋方向相反** → 水在锥体中形成**三叶螺旋**（上涡旋进入+下涡旋流出）→ 能量在封闭路径中循环→几乎不产生脱体涡

**代码体现**：
- 三个.py文件的命名说明了一切——`顶逆时针进水锥逆时针转`并非"顶逆时=内爆"的固定法则，而是"顶部逆势进水+锥体逆时针转"的**自由参数组合**
- 底进水版用cw、顶进水版用ccw → 轴的旋转方向可以独立于进水方向来设定


### 3.2 参数自由度

进水方向和旋转方向各自独立，通过Frenet标架中的$T,N,B$三向量和Rodrigues旋转角$\psi$来控制：

- 进水方向由$x_{\text{spiral}}$的正负号决定
- 旋转方向由$\psi$的偏置符号决定
- 轴向速率$z_v$和频率$f$分别控制进水的深度和螺旋的密集程度


### 3.3 num_theta = 12 → SEG 12滚柱

12个截面点沿圆周均布 → 12片"花瓣" → 等同于SEG内环的12个滚柱在流体内的分布。这不是巧合——**12是SEG和Schauberger水轮共享的最优整数分割数**。

### 3.4 b = 1.618 (φ) — 黄金比例的物理角色

`最终.py` 将$b$设为$\phi=1.618$，而非其他版本的1.6。区别：

$$r(t) = \sqrt{\frac{1}{(b - t\sin\alpha)^2}} \approx \frac{1}{b-t\sin\alpha}$$

$b=\phi$ 时：在$t$接近0时初始半径 = $1/\phi \approx 0.618$ → 蛋形尖端的曲率的一个分量。黄金比例进入螺旋的钟形曲线 → 蛋形λ_n谱的一个特解。

## 五、Z轴圆周阵列 — 从单锥到水处理/发电组件

> Python生成的不管是概念验证（matplotlib）还是生产级（ezdxf双轨扫掠），**单个锥体/螺旋线本身不是最终产品**。

### 5.1 工程原则

所有Python生成的螺旋曲线和曲面，都需要**绕Z轴圆周阵列**才能成为实用构件：

```
Python生成单锥 → 绕Z轴旋转复制×N → 形成完整的涡旋核心
```

| 应用场景 | 阵列数N | 代码体现 |
|:---|:---:|:---|
| **水处理螺锥** | 3 | num_instances = 3 |
| **发电水轮核心** | 3~6 | num_instances = 3, 6 |
| 搅拌器涡流叶片 | 2~4 | — |
| 花洒分配板 | 3 | 最简优先系列 |

### 5.2 两种阵列路径

| 方式 | 位置 | 原理 |
|:---|:---|:---|
| **Python内阵列** | 钟形水轮系列 | `rotation_matrix_z(φ)` → 直接生成N个实例3D点阵 |
| **Rhino后阵列** | 02_双轨扫掠系列 | 导出单个DXF → Rhino中`ArrayPolar`绕Z轴阵列 |

### 5.3 与SEG的类比

| 钟形水轮 | SEG |
|:---|:---|
| 圆周阵列数 N=3 | SEG三层环 |
| 每个螺旋锥形成局部涡旋 | 每层滚柱群(12/22/32) |
| 绕Z轴分布 | 滚柱公转轨道 |
