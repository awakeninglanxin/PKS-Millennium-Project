## 五、3-4-5 勾股三角相关经络

以下经络的比率直接来自 3-4-5 勾股三角形的边比：

| 经络 | 比率 | 3-4-5 中的边比 | 角度 |
|:---|:---|:---|:---:|
| 三焦经·整体 | 2:3 | 股:玄≠直接，但 2:3 = (4:6) | 34°/56° |
| **膀胱经** | **3:4** | **勾:股** = 3:4 | **37°/53°** |
| 肺经 | **4:5** | **股:玄** = 4:5 | 39°/51° |
| 心经/任督 | 7:9 | ≈ (3²+4²)/3² − 1 | 38°/52° |

**37° 和 53° 分别是 arctan(3/4) ≈ 36.87° 和 arctan(4/3) ≈ 53.13° 的取整值**——这两个角度是 3-4-5 直角三角形的精确锐角。

## 六、频率计算公式

```
频率 = 256 × 比率

基频 P = C₄ = 256 Hz = 2⁸

例：
  膀胱经 37°: 256 × 3/4 = 192 Hz (G)
  膀胱经 53°: 256 × 4/3 = 341.3 Hz (F)
  肺经 39°:   256 × 4/5 = 204.8 Hz (Ab)
  肺经 51°:   256 × 5/4 = 320 Hz (Eb)
  肝经 45°:   256 × 14/14 = 256 Hz (C) — 基频本身
```

**互补角对特点**：角度1 + 角度2 = 90°，比率1 × 比率2 ≈ 1（互为倒数），频率呈互补关系。
