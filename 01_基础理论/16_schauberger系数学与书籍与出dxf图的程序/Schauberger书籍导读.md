## 二、《超双曲锥体》逐章导读

### 2.1 核心思想（前言 ~ 第 3 章）

**一句话总结**：超双曲锥体是连接 ∞（爆炸/离心）→ 0（内爆/向心）的数学模型，所有自然涡流运动都在这个锥体表面发生。

**三个关键公式**：

| 公式 | 含义 | 物理对应 |
|------|------|---------|
| $x \cdot y = 1$ | 双曲线（锥体母线） | 频率×波长=常数（八度音程原理） |
| $r \cdot \phi = 2\pi$ | 超双曲螺旋 | 半径×旋转数=常数（越近中心转越快） |
| $n \cdot (1/n) = 1$ | 谐波序列 | 1, 1/2, 1/3,... 是锥体上的整数高度 |

#### 必读段落

| 章节 | 页码 | 核心内容 | 与项目的关联 |
|------|------|---------|------------|
| **维克多·绍伯格** | 13-18 | 鳟鱼跳跃、木材漂浮装置、冷月夜实验 | 🟢 与水飞碟仿真体系、Schauberger 极化原理一致 |
| **沃尔特·绍伯格** | 19-23 | 从怀疑到终身投入、PKS 学院创立 | 🟢 数学脱壳的灵感来源 |
| **超双曲锥体的构造** | 24-25 | 从双曲线 $xy=1$ 旋转→三维锥体 | 🟡 与蛋形截面公式同源 |
| **螺旋形** | 26-28 | $r\phi=2\pi$ 的定义和图形 | 🔴 **核心**：这是羚羊角管仿真的底层数学 |

### 2.2 数学核心（第 4~5 章）

#### 必读段落

| 章节 | 核心内容 | 与项目的直接对应 |
|------|---------|---------------|
| **圆锥体数学特性** | 体积公式 $V=\frac{\pi}{2}(z_2-z_1)$，两圈间距离 | 🔴 超双曲锥 = 我们交互仿真中 $\text{exp}(-\sigma(1-\cos t))$ dip 的锥体母型 |
| **π 和 e** | 从螺旋特征点直接导出自然常数 | 🟡 圆周率自然出现 → 蛋形闭合的数学必然性 |
| **鸡蛋形状** | **平面截超双曲锥 → 各种蛋形** | 🔴🔴 **这是蛋咬一口公式的几何源头！** |
| **谐波** | 泛音系列 $1,1/2,1/3,...$ = 螺旋在锥体上的步进 | 🟡 谐波 = Schauberger 负摩擦的频率条件 |

#### 🔴 最重要的洞察

**Walter Schauberger 证明了**：用平面以不同角度截超双曲锥体，会得到各种蛋形曲线。这意味着：

```
我们现在的蛋咬一口公式：

  g(t) · (x(t), y(t))   ← 经验调制

根源来自：

  平面 ∩ 超双曲锥体     ← Walter Schauberger 的严格数学
  
→ 各种蛋形（主轴/小轴由截面角度和高度决定）
```

> **这意味着我们的交互仿真应该从"经验调制"升级为"锥体截面"——每个滑块参数对应一个圆锥截面参数（截面高度 h、截面角度 θ、旋转偏移 φ）。**

## 四、视频连接：正负电荷电势的超双曲锥可视化

### 4.1 视频内容

> "背了3年物理公式，电势长啥样你都不知道？正电荷是火山，负电荷是天坑，拖一下地形直接重塑！"

视频使用 3D 可视化展示了静电势 $\phi = kQ/r$：

- **正电荷**：$\phi > 0$，形如**向上的漏斗锥**（火山）
- **负电荷**：$\phi < 0$，形如**向下的漏斗锥**（天坑）

### 4.2 从左右放置到上下放置 → 书封面的双锥对立

视频中正负电荷左右放置，电势曲面呈双峰-双谷。**如果将正负电荷上下放置**：

```
正电荷（上方）：
     ╲    ╱        φ = +kQ/r    r 越小 φ 越大
      ╲  ╱         → 向上发散的漏斗
       ╲╱
    ────╳────       ← 中性平面 (φ=0)
       ╱╲
      ╱  ╲         φ = -kQ/r    r 越小 φ 越负
     ╱    ╲        → 向下发散的漏斗
负电荷（下方）：
```

**这就是《超双曲锥体》书封面的图形——上下漏斗对立 = 正负电势的 3D 表示！**

### 4.3 从电势到 COP：电磁-流体对应

| 电势模型 | 超双曲锥模型 | COP 模型 |
|---------|------------|---------|
| 正电荷 → 向上漏斗 | 离心/爆炸 → 锥体开口向上 | PWM 铁芯的 T_on 阶段 |
| 负电荷 → 向下漏斗 | 向心/内爆 → 锥体开口向下 | PWM 铁芯的楞次失效阶段 |
| 零势面 → 中性平面 | 双曲锥的渐近平面 | 开关切换点 |
| 电荷间力 → 电场梯度 | 锥体表面螺旋运动 | 暗能量转换 = H_X/H_S |

## 七、对本项目的启示

### 7.1 蛋咬一口仿真升级方向

当前公式：$g(t) \cdot (x(t), y(t))$ 是经验的。升级为：

$$(\tilde{x}, \tilde{y}) = \text{PlaneSection}(HyperbolicCone, h, \theta, \phi)$$

其中 $h$ = 截面高度，$\theta$ = 截面倾角，$\phi$ = 旋转角。三个参数分别对应我们界面上的调整。

### 7.2 COP 仿真扩展

双曲锥的上下漏斗对应 PWM 电路的正向/反向能量流动，可以构建**双锥模型**来描述完整的充放电循环。

### 7.3 流体仿真

超双曲螺旋的弧长计算（$B = 2\pi \ln n$）可以直接用于羚羊角管的 ODE 模型，替代当前的近似分段计算。
