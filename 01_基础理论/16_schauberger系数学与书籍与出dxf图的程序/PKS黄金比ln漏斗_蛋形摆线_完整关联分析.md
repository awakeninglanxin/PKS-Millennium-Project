# PKS 黄金比 φ 漏斗 + 白银比 χ 临界条件 + 蛋形摆线 — 三合一融通分析

> **核心洞察**：将 Schauberger 双曲锥体的三个数学侧面——黄金比 ln 漏斗（φ，连续生长）、白银比临界切面（χ，边界条件）、蛋形摆线轨迹（参数方程，工程实现）——统一为同一几何体系的三种投影。

---

## 零、双曲锥体：一切的本源

### 0.1 构造方式

Schauberger 蛋形曲线的**唯一源头**：

1. 取双曲线 $y = 1/x$
2. 绕 $x = 0$ 中轴旋转
3. 得到一个**旋转双曲圆锥面**（喇叭口曲面）
4. 用平面 $y = kx + b$ 斜切这个锥面
5. 截面在垂直于切割平面的方向看 → **蛋形**

```
y = 1/x 绕 x=0 旋转 → 双曲圆锥面
     ↓
平面 y = kx + b 斜切 → 蛋形截面
     ↓
参数 k, b 决定蛋形的"胖瘦"和曲率
```

### 0.2 三个数学侧面的统一

这个双曲锥体同时承载了三条数学线索：

| 侧面 | 数学签名 | 常数 | 角色 |
|:---|:---|:---|:---|
| **黄金比漏斗** | $r = \phi^n$, $z = \ln n / \ln \phi$ | $\phi \approx 0.618$ | 连续生长/自相似螺旋 |
| **白银比临界** | $c = \frac{z_0\sqrt{2}}{2}\tan(2\alpha)$, $\frac{\|y_{\max}\|}{\|y_{\min}\|} = \chi$ | $\chi = 1+\sqrt{2}$ | 边界条件/临界相切 |
| **蛋形摆线** | $x(t), y(t)$ 参数方程 | $b=5/3, k=2/3$ | 工程实现/CNC 加工 |

**三条线不是三条独立的线——它们在同一个锥体上共存。**

## 一、侧面一：黄金比 φ — ln 函数漏斗（连续生长）

> 对应：8 图 PKS 第 2 页 / pks黄金比ln函数漏斗.jpg / pks黄金比ln函数漏斗出图.jpg

### 1.1 图示要素

| 标注 | 德语原文 | 数学公式 | 物理含义 |
|:---|:---|:---|:---|
| **黄金比例** | Goldener Schnitt | $r = \phi^n$ | 锥体半径按黄金比幂次衰减 |
| **谐波分割** | Harmonikale Teilung | $k = \frac{1}{\ln\phi}$ | 对数尺度下的音程划分常数 |
| **双曲螺旋** | Hyperbolische Spirale | $r = \frac{1}{n}$ | 倒数关系（舒伯格系统） |
| **对数除法** | Logarithmische Teilung | $z \propto \ln r$ | 高度与对数半径成正比 |

### 1.2 谐波分割常数

$$k = \frac{1}{\ln\phi} = \frac{1}{\ln(0.618...)} \approx -2.078 < 0$$

这个负号表示**向内收缩**——从外到内，每一层的半径按黄金比缩减。

### 1.3 ln 函数漏斗 3D 构造（蓝馨推导）

**核心参数**：
```
φ = (√5-1)/2 ≈ 0.618,  ln φ ≈ -0.481
k = 1/ln φ ≈ -2.078,  D = √(1+k²) ≈ 2.311
```

**连续螺旋线参数方程**（$T \in [0, 2\pi M]$, $M=29$ 波段）：

| 步骤 | 计算 | 含义 |
|:---|:---|:---|
| 波段分解 | $n = \lfloor T/2\pi \rfloor + 1$ | 当前所在层 |
| 连续参数 | $t_s = n + \tau/2\pi$ | 平滑过渡 |
| z 坐标 | $z = \ln t_s / \ln \phi$ | 对数高度（阴） |
| 半径 | $r = 1/t_s$ | 倒数半径（阳） |
| 角度 | $\theta = T + \pi$ | 旋转角 |
| 笛卡尔 | $x = r\cos\theta$, $y = r\sin\theta$ | 最终坐标 |

**关键**：当 $t_s = 1$ 时 $(r,z) = (1,0)$，当 $t_s \to \infty$ 时 $(r,z) \to (0,-\infty)$ — 从外向内无限收缩。

## 二、侧面二：白银比 χ — 临界切面条件（边界）

> 来源：`白银比椭圆半焦距和蛋形长短轴比schauberger蛋形曲线形成.docx`（蓝馨）

### 2.1 椭圆投影焦距中的 √2

蛋形截面的投影椭圆的半焦距公式：

$$c = \frac{z_0 \sqrt{2}}{2} \cdot \tan(2\alpha)$$

**√2 直接出现在分子中——不是近似值，是精确因子。** 全焦距 $2c$ 必然包含 √2。

### 2.2 临界相切条件 = 白银比

当切割平面与双曲锥面**刚好碰到但未切入**时，满足：

$$z_0^2 = 4 \tan\alpha$$

此时蛋形曲线在 y 轴上的极值比锁定为：

$$\frac{|y_{\max}|}{|y_{\min}|} = \sqrt{2} + 1 = \chi \approx 2.414$$

### 2.3 物理直觉（刀与喇叭）

> 这个倾斜的平面就像一把刀，喇叭口曲面就像一个实物。
> - **一般情况**：刀"砍"进去，切出截面（蛋形随 k,b 变化）
> - **临界条件 $z_0^2 = 4\tan\alpha$**：刀精准地放在曲面边缘，**刚好碰到但未切入**
> - 此时蛋形展现出极简数学性质：长短轴比固定为 $\chi$

这就是铁律 52 讨论的那个角色：**χ 是"临界"的数学签名——刚好碰到但未越过。**

### 2.4 白银比 χ 在这个体系中的角色总结

| 出现位置 | 公式 | 含义 |
|:---|:---|:---|
| 投影椭圆焦距 | $c = z_0\sqrt{2}\tan(2\alpha)/2$ | $\sqrt{2}$ 是精确因子 |
| 临界蛋形极值比 | $\|y_{\max}\|/\|y_{\min}\| = 1+\sqrt{2}$ | 相切条件下的固定比 |
| 拉糖机 4-棒特征值 | $\lambda = 3+2\sqrt{2} = \chi^2$ | 拉伸-折叠的最优因子 |
| 黎曼归一化 | $X = 1/\sqrt{2\pi}$ | Fresnel 积分的临界线位置 |

## 三、侧面三：蛋形摆线轨迹（工程实现）

> 来源：`运用传统坐标系切截面出蛋形摆线轨迹制造蛋形活水机.docx`（蓝馨完整推导）

### 3.1 核心参数

```
b = 5/3    → Major 6th 音程比，最接近黄金比 φ
k = 2/3    → 纯五度的倒数
m = R/(R+r) = 2/3    → 内外半径比
```

### 3.2 蛋形曲线方程

**原始公式（未平移）**：

$$x = \frac{2 \sin t}{b + \sqrt{b^2 - 4k \cos t}}$$

$$y = \frac{1}{2\sqrt{1+k^2}} \left( \frac{k^2-1}{k} b + \frac{k^2+1}{k} \sqrt{b^2 - 4k \cos t} \right)$$

**平移 + 缩放**（十字聚焦中心）：

$$x_{final} = x \cdot \left(1 + \frac{\sin(ct)}{1.5c}\right)$$

$$y_{final} = y \cdot \left(1 + \frac{\sin(ct)}{1.5c}\right)$$

其中 $c$ 控制振荡幅度。

### 3.3 倒数关系 → 蛋形弹簧

```
z = 1/y = ky + b     (当 z ≥ 0)
z = -1/y = ky + b    (当 z ≤ 0)
```

$y$ 是蛋形横向坐标，$z = 1/y$ 是纵向倒数映射 → 产生 **3D 蛋形弹簧**结构。

### 3.4 与双曲锥体的对应

| 蛋形摆线参数 | 双曲锥对应 | 数学关系 |
|:---|:---|:---|
| $b = 5/3$ | 斜切平面截距 | 决定蛋形"尖度" |
| $k = 2/3$ | 斜切平面斜率 | 决定蛋形"胖瘦" |
| $z = 1/y$ | 双曲螺旋 $r = 1/n$ | 倒数映射 |
| $x(t), y(t)$ | 蛋形截面参数化 | 工程实现接口 |

## 四、三侧面融通：φ 与 χ 在同一个锥体上的共存

### 4.1 统一架构

```
                  双曲锥体（y=1/x 旋转）
                  /        |         \
                 /         |          \
        黄金比φ侧      白银比χ侧      蛋形摆线侧
        (连续生长)    (临界边界)     (工程实现)
            |             |              |
    r=φⁿ, z=ln/lnφ   c=z₀√2/2·tan2α   x(t),y(t)
            |             |              |
      自相似螺旋      相切极值比χ      CNC加工接口
```

### 4.2 φ 和 χ 的关系：不是对立，是分工

| | φ（黄金比） | χ（白银比） |
|:---|:---|:---|
| 在锥体上的角色 | **沿锥面爬升**的速率 | **切割锥面**的临界位置 |
| 数学形式 | $r = \phi^n$（连续衰减） | $\|y_{\max}\|/\|y_{\min}\| = \chi$（固定极值比） |
| 物理类比 | 水沿涡旋从外向内流动 | 水流与壁面的最佳接触角 |
| 方程式 | $x^2 - x - 1 = 0$ | $x^2 - 2x - 1 = 0$ |
| 共同点 | 都属于 $x^2 - Tx + 1 = 0$ 族 | T=1(φ), T=2(χ) |

### 4.3 阴阳统一的物理意义

蓝馨的核心洞察（从 `运用传统坐标系...docx`）：

> **阴**：最大密度压缩效率（黄金比 $\phi^n$）  
> **阳**：$2^n$（隔八度或用 $k\ln(x)$ 公式）  
> 两条切线标记有 H₂O 和 SiO₂（双曲线形成的圆锥横截面面积）  
> **阴阳合一于中道（线）**

**数学翻译**：

| 系统 | 时间观 | 数学 | 对应曲线 | 
|:---|:---|:---|:---|
| 牛顿 | 关于 $t$（线性时间） | $f(t) \propto t$ | — |
| 舒伯格 | 关于 $1/t$（倒数时间） | $f(t) \propto 1/t$ | 双曲螺旋 $r = 1/n$ |
| 中道 | 统一 | $x \cdot (1/x) = 1$ | 蛋形截面 |

## 五、工程应用：蛋形活水机

### 5.1 两种运动模式（蓝馨推导）

**A. 锯齿运动（脉冲式水流）**：
```
x = term_x * (1 - arcsin(sin(c*t)) / (1.5*c))
y = term_y * (1 - arcsin(sin(c*t)) / (1.5*c))
```

**B. 正弦运动（连续涡旋）**：
```
x = term_x * (1 + sin(c*t) / (1.5*c))
y = term_y * (1 + sin(c*t) / (1.5*c))
```

### 5.2 参数调节指南

| 参数 | 取值 | 效果 |
|:---|:---|:---|
| $b = 5/3$ | 固定 | 音程比 Major 6th，接近黄金比 |
| $k = 2/3$ | 固定 | 内外半径比，控制蛋形"胖瘦" |
| $c$ | 可调 (1.0~2.0) | 振荡频率，c 越小振荡越大 |
| $a$ | 可调 (1.0~2.0) | 螺旋扭转率，a=1.5 时首尾错位 180° |
| $t$ 范围 | $[0, \pi/2]$ 或 $[0, 2\pi]$ | 第四象限（半个蛋形）或完整蛋形 |

### 5.3 与 Schauberger 原设计的对比

| 特征 | Schauberger 原始 | 蓝馨改进 |
|:---|:---|:---|
| 坐标系 | 德语复杂参数体系 | 传统笛卡尔坐标系 |
| 蛋形生成 | 双曲锥斜切（隐式） | 参数方程直接生成（显式） |
| 摆线运动 | 未明确给出 | 完整的 x(t), y(t) + 锯齿/sin 两种模式 |
| 3D 弹簧 | 羚羊角螺旋管 | 蛋形弹簧 z = t，参数 a 控错位 |
| CNC 加工 | 需要额外转化 | 直接可生成 DXF |
