## 三、逐类详解

### 3.1 01_直锥螺旋管 — 入门模板

| 文件 | 大小 | 功能 |
|:---|:---:|:---|
| `最简原始模版用于生成直锥螺旋管.py` | 8KB | **所有螺旋程序的共同祖先** |
| `参数可调螺旋锥...dxf` | 0MB | 直锥螺旋管的DXF输出 |

**数学原理**：
- 三维锥面上的螺旋线参数方程：$r(t) = R_0 + (R_1-R_0)·t$, $\theta(t) = 2\pi·n·t$, $z(t) = H·t$
- DXF格式输出采用多段线（POLYLINE）或样条曲线（SPLINE）
- `ezdxf` 库：`msp.add_spline()` 或 `msp.add_lwpolyline()` 

**与PKS关联**：直锥是蛋形$\lambda_n=2\pi\ln n$在螺距恒定的简单情况下的退化版

|:---|:---:|:---|

| `弹簧管中心线与蛋尖双轨...简化.py` | 9KB | DXF → 蛋尖截面的弹簧管 |
| `弹簧旋叶中心线双轨...简化.py` | 7KB | DXF → 旋叶式发电机曲面 |
| `舒式螺旋斜流泵...2的n - 蛋面.py` | 11KB | Schauberger螺旋斜流泵 |
| `舒式螺旋斜流泵...locas外+sun内蛋面.py` | 12KB | 双蛋面复合（Lucas+Sun数列） |
| `旋叶3摆线螺旋向内...阴阳-t平方.py` | 7KB | 摆线螺旋旋叶（向内收敛） |
| `旋叶3摆线螺旋向外...阴阳-t平方.py` | 7KB | 摆线螺旋旋叶（向外扩散） |
| `旋叶发电机...顶上不干涉-丝滑曲线.py` | 8KB | 旋叶发电机优化版 |
| `舒式螺旋漏斗...2的n-蛋面-直管.py` | 10KB | 螺旋漏斗+直管组合 |

**统一的数学框架**：
- **中心线**: 一条3D螺旋曲线（由蛋形截面、$t^2$缩放、黄金比等参数生成）
- **双轨**: 中心线+扭转后的镜像线 → Rhino的`Sweep2`
- **多断面**: 沿中心线在不同参数$t$处放置截面曲线（水滴形/蛋尖/摆线/电子驻波形状）
- **阴阳-t平方**: $r(t) = r_0 ± a·t^2$ — 螺旋半径随$t^2$缩放（阳为正、阴为负）
- **Sun/Lucas数列**: 蛋形截面的两个变体——Sun（外轮廓）/Lucas（内轮廓）

**与PKS关联**：
- 蛋尖截面 = PKS蛋形 $\lambda_n=2\pi\ln n$的3D实现路径
- 双轨+多断面 = EEGT混沌吸引子在3D空间中的物理投影
- Schumann频率叠加 = Schumann 7.83Hz的谐波螺旋管（驻波34个次谐波）
- $t^2$ 缩放 = Don Smith频率平方定律的几何体现

### 3.3 03_蛋形与场约束曲线 — ⭐PKS核心

| 文件 | 大小 | 功能 |
|:---|:---:|:---|
| `anu蛋形场约束曲线.py` | 7KB | **核心** — anu蛋形+电磁场约束 |
| `anu蛋形场约束曲线3.py` | 10KB | 多八度的场约束曲线（迭代3代） |
| `anu蛋形+椭圆场约束曲线.py` | 7KB | 椭圆-蛋形过渡的场约束 |
| `bashar线圈33.3°螺锥.py` | 23KB | **最大文件** — Bashar的33.3°螺旋锥线圈 |
| `指数变换绘制非线性蛋曲线.py` | 1KB | 指数变换生成非线性蛋形 |

**数学原理**：
- **anu蛋形公式**: $r(\theta) = r_0 · f(\theta)$，其中 $f$ 由对数螺线和椭圆修正项组成
- **场约束**: 在蛋形曲线周围生成符合Laplace方程$\nabla^2\Phi=0$约束的场线族
- **多八度迭代**: 从中心蛋形出发，每次向外扩展进行非线性拉伸 → 生成自相似嵌套蛋形
- **33.3°**: Bashar指定的螺旋角——不是随意的，$33.3°=$100°/3（三度音程的几何等价）

**与PKS关联**：
- 这是PKS蛋形$\lambda_n=2\pi\ln n$最直接的Python实现
- 场约束曲线 = TGFT三源项的几何可视化
- 多八度迭代 = EEGT分形吸引子在2D截面上的投影

### 5.1 3D打印材料与壁厚（来源：7天教学DOCX实测数据）

| 材料 | 最小壁厚 | 打印缩放 | 螺纹处理 |
|:---|:---:|:---:|:---|
| **光固化白色树脂** | 1.5mm | 放大1.005倍 | 可直接打印螺纹，打印后用丝攻二次攻丝 |
| **纯铜打印** | **0.6mm** | 放大1.05倍（表面极粗糙） | ❌ 不建议含螺纹，需车床攻丝 |
| **316L不锈钢** | 1.0mm | 放大1.05倍 | ❌ 表面粗糙，淘宝丝攻无效 |

**金属打印教训**（来自实战）：
- 间隙<1mm会被支撑完全堵塞 → 设计时预留>1.5mm间隙
- 螺纹管重且贵 → 建议买标准双四分外丝，车床截半焊接加工
- 悬空平板需加悬链曲线过渡支撑
- Rhino替代方案：FreeCAD (开源) / Fusion 360 (免费个人版)
