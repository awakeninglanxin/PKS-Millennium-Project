## 〇、核心突破：太阳/太阴双螺旋的完整数学推导

### 0.1 两条曲线

| 曲线 | 方程 | 角色 |
|:---|:---|:---|
| **绿色线（太阳）** | $x = \frac{2\pi \sin(t)}{t}, \quad z = \frac{\ln t}{\ln\phi}$ | 正弦波调制螺旋（太阳数列） |
| **红色线（太阴）** | $x = \pm\frac{1}{t}, \quad z = \frac{\ln t}{\ln\phi}$ | 双曲倒数控螺旋（太阴数列） |
| **蓝色直线** | $z = kx + b_n, \quad k = \frac{1}{2\pi \ln\phi}$ | 射影切面（连接太阳/太阴交点） |

其中 $\phi = \frac{\sqrt{5}-1}{2} \approx 0.618034$（小黄金比，$1/\Phi$）

$t \in [0.5, \text{max\_n}]$，max_n 可无限延伸。

### 0.2 为什么用 $\phi = (\sqrt{5}-1)/2$ 而不是 $\Phi = (\sqrt{5}+1)/2$？

$\ln\phi < 0$（因为 $\phi < 1$），所以 $z = \ln t / \ln\phi$ 在 $t > 1$ 时为**负值**——曲线沿 z 轴**向下**螺旋。

这直接对应 Schauberger 的**内爆**（向心吸入）运动：
- $t$ 增大 → $1/t$ 减小（半径缩小）→ $z$ 更负（内爆加深）
- $\phi = 1/\Phi$ 给出 $\ln\phi = -\ln\Phi$，等价于 $z = -\ln t / \ln\Phi$——用 $\Phi$ 时有负号，不如直接用 $\phi$ 自然

### 0.3 两种起始点

| 起始点 | 起点位置 | 对称轴 | 含义 |
|:---|:---|:---|:---|
| **r=1** | $t=1$（谐波分割起点） | $x=1$ 处 | 标准谐波起点（单基准） |
| **r=2** | $t=2$ 开始，$A_1=(2,0,\ln(0.5)/\ln\phi)$ | $x=2$ 处 | 八度谐波起点（2倍频） |

r=2 方案中，$B_n = (-1/n, 0, 0)$ 而 $A_n$ 在正 x 侧——形成交错的阴阳交点对。

### 0.4 关键几何构造

1. **以正红色曲线为中轴旋转 360°** → **黄金比 ln 漏斗曲面**
2. **蓝色直线向 y 正负方向展开** → **切面**（射影平面）
3. **切面与漏斗曲面相交** → **蛋形边缘曲线**
4. **以切面法线为新工作平面** → 2D 曲线方程（可出 DXF 图）

这正是 PKS 8 图中的**黄金比漏斗**的完整数学实现。

## 一、射影几何贯通框架 — 数学深意

### 1.1 核心思想

> 舒伯格 PKS 体系本质上是一个**射影几何体系**——双曲锥体、蛋形截面、谐波分割全部可以用射影几何的语言统一表述。

### 1.2 齐次坐标与双曲锥

PKS 的倒数运算 $y = 1/x$ 在齐次坐标 $(X, Y, Z)$ 中变为：

$$XY = Z^2$$

这是**等轴双曲线的齐次化**——将笛卡尔平面的双曲线自然扩展到射影平面 $\mathbb{P}^2$。

- $Z=0$ 时 $XY=0$ → 无穷远处的**两条渐进线** $X=0$ 和 $Y=0$
- 舒伯格的"超双曲锥"（$z=1/x$）本质上是这个齐次方程在 $\mathbb{P}^3$ 中的推广

### 1.3 交比与谐波比例

交比 $(A,B;C,D) = \frac{AC\cdot BD}{AD\cdot BC}$ 是射影几何最深的**不变量**。

PKS 中的所有比例——黄金比 $\phi$、白银比 $\chi = 1+\sqrt{2}$、3:2、4:3——都是**特定交比的数值实现**：

| PKS 比例 | 交比值 | 射影意义 |
|:---|:---:|:---|
| $\phi = (\sqrt{5}-1)/2$ | 调和点列退化极限 | 内爆/生长的极值 |
| $\chi = 1+\sqrt{2}$ | 临界相切交比 | 蛋形截面临界条件 |
| 3:4（纯四度） | $AB:BC = 3:4$ | Barbara Hero 膀胱经 |
| 2:3（纯五度） | 完全四边形交点 | 音乐和声学 |

### 1.4 绝对形 — 三种几何的统一

**Klein 的 Erlangen 纲领**：用射影几何的固定圆锥曲线（绝对形）来统一欧氏、双曲、椭圆三种几何。

| 绝对形 | 几何类型 | PKS 对应 |
|:---|:---|:---|
| 虚圆（$x^2+y^2+z^2=0$） | 欧氏几何 | 直锥 → 椭圆截面 |
| 实椭圆 | 双曲几何 | **超双曲锥** → 蛋形截面 |
| 虚椭圆 | 椭圆几何 | 球面 → 圆截面 |

**舒伯格的 7 个层级** = 7 个不同的绝对形选择，对应 Cayley-Klein 度量的 7 种度量常数值。

### 1.5 焦散线与波前

Arnold 辛几何中的焦散线奇异性——波前在聚焦点处的**尖点**（cusp）——正是舒伯格工程哲学中要**避免**的奇点。

在射影几何中：
- 焦散线 = 波前在**接触流形**上的投影
- 舒伯格在参数选择中**主动避开**焦散线奇点 → 对应**非临界**参数区域
- 蛋形截面 = 非临界参数区域的稳定波形

### 3.2 统一的数学根源

所有五个体系的共同根源：

$$\Delta\psi + V(r)\psi = E\psi$$

这是球对称势中的 Schrödinger 方程。其本征值壳层闭合给出了**幻数序列**。不同的体系选择不同的 $V(r)$（势函数）：
- 核壳层 → Woods-Saxon 势 → 幻数 2,8,20,28,50,82
- Jellium → 方阱势 → 幻数 2,8,18,20,34,40
- PKS → $\ln(t)/\ln(\phi)$ 对数势 → 交点对 n=1,2,3...

而射影几何的**绝对形**就是势函数 $V(r)$ 的几何对应——不同的绝对形选择 = 不同的 $V(r)$ = 不同的幻数/交点序列。

### 3.3 交叉比 = 壳层间距的不变量

$$\text{CR}_n = \frac{|x_{A_n} - x_{B_n}|}{|x_{A_{n+1}} - x_{B_{n+1}}|}$$

## 四、核心公式总结

### 4.1 PKS 漏斗的显式方程

**红色曲线旋转 360° 的漏斗面**：

$$\begin{cases} x = \frac{\cos\theta}{t} \\ y = \frac{\sin\theta}{t} \\ z = \frac{\ln t}{\ln\phi} \end{cases} \quad t \in [0.5, \text{max}],\; \theta \in [0, 2\pi]$$

**蓝色切面族**：

$$z = \frac{x}{2\pi \ln\phi} + b_n, \quad n = 1, 2, 3, ...$$

其中 $b_n$ 由条件 $z_B = k x_B + b_n$ 确定，$B_n = (-1/n, 0, 0)$。

### 4.2 蛋形边缘曲线（切面 ∩ 漏斗面）

将切面方程代入漏斗面的 $x$= 表达式：

$$\frac{\ln t}{\ln\phi} = \frac{\cos\theta}{2\pi t \ln\phi} + b_n$$

解出 $t = t(\theta; b_n)$ 代入参数方程，得到 2D 蛋形曲线。

### 4.3 关键比率

$$\text{ratio}_n = \left|\frac{x_{A_n}}{x_{B_n}}\right| = \left|\frac{1/t_{A_n}}{1/n}\right| = \frac{n}{t_{A_n}}$$

其中 $t_{A_n}$ 由交点条件确定。ratio_n 的递推关系反映了谐波分割的**非均匀性**——并非简单的等差或等比，而是由 φ 的对数律决定。
