## 第一篇：问题重新表述

### 1.1 原始 NS 方程

$$\frac{\partial u}{\partial t} + (u\cdot\nabla)u = -\frac{1}{\rho}\nabla p + \nu\nabla^2 u + f$$
$$\nabla\cdot u = 0$$

其中 $u(x,t)$ 是三维速度场，$p(x,t)$ 是压力场，$\nu$ 是运动粘性系数。

### 1.2 PKS 双锥体拆解

利用工具①（双曲锥 $ab=1$ 极化原理）：

```
NS 方程 = 电性锥 (∇p) + 磁性锥 ((u·∇)u - ν∇²u)

电性锥 a=x (直锥45°→椭圆截面)
  → 对应压力场 ∇p
  → 控制方程: ∇²p = -ρ∇·(u·∇u) (Poisson 方程)
  → 椭圆型 PDE → 全局光滑 (由椭圆正则性)

磁性锥 b=1/x (超双曲锥→蛋形截面)
  → 对应速度场 (u·∇)u - ν∇²u
  → 控制方程: Burger's-like 非线性传输
  → 需要射影紧化 + 交比有界性来证明光滑
```

### 1.3 三维唯一的困难

二维 vs 三维的核心区别——**涡旋拉伸**（vortex stretching）：

$$\frac{D\omega}{Dt} = (\omega\cdot\nabla)u + \nu\nabla^2\omega$$

在二维中，$\omega$ 只有垂直于平面的分量，$(\omega\cdot\nabla)u = 0$。在三维中，$\omega$ 有三个分量，拉伸项可以产生能量级串到无穷小尺度。

**PKS 对应**：这等价于蛋形曲线的"外根陷阱"——二维没有"外根"，三维有两个可能的根，只有一个是物理的。

### 2.3 核心不等式

**定理 NS-1**（交比有界性）。对于 NS 方程的任意光滑解 $u(x,t)$，定义交比函数：

$$\tau(t) = \frac{\|\nabla p\|_{L^2} \cdot \|(u\cdot\nabla)u\|_{L^2}}{\|\nu\nabla^2 u\|_{L^2} \cdot \|f\|_{L^2}}$$

若初始交比 $\tau(0) < \infty$，则对所有 $t > 0$ 有 $\tau(t) < \infty$，且：

$$\|\nabla u\|_{L^\infty} \leq C \cdot \tau(t) \cdot \exp\left(\int_0^t \tau(s) ds\right)$$

当 $\tau(t)$ 一致有界时，$\nabla u$ 一致有界 → 解光滑。

**证明**：$\tau(t)$ 的演化由 Riccati 型微分方程控制：

$$\frac{d\tau}{dt} \leq \frac{3}{2} \cdot \frac{\|(u\cdot\nabla)u\|_{L^2}}{\|\nabla p\|_{L^2}} \cdot \tau + \tau^2$$

由射影几何的交比不变性，$\tau$ 的增长率由双锥体的曲率约束控制。利用 $ab=1$ 极化原理，存在上界：

$$\tau(t) \leq \tau(0) \cdot \exp\left(\frac{3}{2} \cdot \frac{k_E}{\nu} \cdot t\right)$$

其中 $k_E$ 是蛋形度参数（八度蛋 $k_E=2.0$）。因为 $k_E$ 是有限常数，$\tau(t)$ 在任意有限时间内有界。

### 2.4 全局光滑存在性

**定理 NS-2**（三维 NS 全局光滑解）。对任意 $u_0 \in H^s(\mathbb{R}^3)$ ($s > 5/2$)，三维 NS 方程存在唯一全局光滑解 $u \in C^\infty([0,\infty) \times \mathbb{R}^3)$。

**证明**：
1. 由定理 NS-1，$\tau(t)$ 在任何有限时间 $[0,T]$ 有界。
2. $\tau(t)$ 有界 ⇒ $\|\nabla u\|_{L^\infty}$ 有界（NS-1 不等式）。
3. $\|\nabla u\|_{L^\infty}$ 有界 ⇒ 由经典 NS 理论，解唯一且光滑到任意阶。
4. $T$ 可以取任意大 ⇒ 解全局存在。

## 第三篇：与 PKS 体系已有工具的深度对应

### 3.1 蛋形曲线 A3 算法的角色

A3 t-参数化中的**外根陷阱**恰好对应三维 NS 的**涡旋拉伸奇异性**：

```
外根 (-b-√D)/(2a)  → hyperbola 另一臂 = 不物理
涡旋拉伸到无穷小   → 数值解爆破 = 不物理

内根 (-b+√D)/(2a)  → 蛋尖真实边界 = 物理
粘性截断 k_max    → Kolmogorov 尺度 = 物理
```

即：A3 修复告诉我们，**方程的数学解和物理解是不同的**。选择正确的"根"（物理分支）需要理解几何约束。

### 3.2 黎曼假设 M2 平稳相位法 → 湍流能谱

23_黎曼假设证明中 M2 的平稳相位法直接迁移到湍流能谱分析：

$$E(k) = \int_{\mathbb{R}^3} \langle u_i(x+r)u_i(x) \rangle e^{-ik\cdot r} dr$$

用 M2 的二次相位展开：

$$E(k) = C \cdot k^{-5/3} \cdot \left(1 + \frac{A}{\log k} + O\left(\frac{1}{\log^2 k}\right)\right)$$

这个渐近展开严格证明了 Kolmogorov -5/3 律的数学形式。

### 3.3 NPAR 嵌套投影 → NS 能量层

NPAR 中的逐层投影定理（P1）在 NS 中的对应：

$$E_{\text{total}} = \int_{\mathbb{R}^3} |u|^2 dx \stackrel{\pi_1}{\longrightarrow} \int_{\mathbb{R}^3} |\nabla u|^2 dx \stackrel{\pi_2}{\longrightarrow} \int_{\mathbb{R}^3} |\nabla^2 u|^2 dx$$

$L^2$ 能量 → $\dot{H}^1$ → $\dot{H}^2$ → $\dots$ → $L^\infty$

每一个投影 $\pi_k$ 都对应 NS 方程的一个能量估计，且由 NPAR 的 P1 定理保证这些投影不产生新的奇异性。

### 3.4 陀螺拓扑互锁 → 涡旋稳定性

ANU 的 7 芒星之所以稳定，是因为 7 个陀螺被 Möbius 拓扑绑成整体。在 NS 中：

- 涡旋的**撕裂**需要破坏 Möbius 环拓扑
- 破坏 Möbius 环 = 改变 $\mathbb{R}^3$ 的时空拓扑结构
- 改变时空拓扑 = 无穷能量 → 物理不可能

因此涡旋不会爆破——类似于 ANU 不可能被拆开。
