## 第一篇：$ab=1$ 极化原理的几何展开

### 1.1 核心公式

$$a \cdot b = 1$$

这是Schauberger/PKS体系的**第一性原理**。它等价于：
- 毕达哥拉斯的音律法则：弦长 $\times$ 频率 = 常数
- 开普勒第二定律：$r \cdot v = \text{const}$
- 角动量守恒：$L = m r v_\theta = \text{const}$

但PKS的关键突破在于：**将 $a$ 和 $b$ 分别设为不同的函数，生成两个不同的锥体。**

### 1.2 电性分支：$a = x$（直圆锥）

令 $a = x$，则 $b = 1/x$。

在三维空间中将 $a = x$ 绕轴旋转，得到**直圆锥**（straight circular cone）：
$$z = /sqrt{x^2 + y^2} = r$$

这是一个圆锥角为 $45^/circ$ 的标准圆锥。用平面斜切它，得到**椭圆截面**。

**物理对应**：
- 经典电磁场（高斯定律、库仑定律）
- 线性波动方程
- **NS方程中的压力场**（Laplace/Poisson方程 $\nabla^2 p = -\rho\nabla\cdot(u\cdot\nabla u)$）

### 1.3 磁性分支：$b = 1/x$（超双曲锥）

令 $b = 1/x$，这本身就定义了超双曲螺旋。绕轴旋转得到**超双曲锥**：
$$z = \frac{1}{r} = \frac{1}{/sqrt{x^2+y^2}}$$

用平面斜切这个锥，得到**蛋形截面**。

**物理对应**：
- 非线性涡旋动力学
- 湍流能量级串
- **NS方程中的速度场**（对流项 $(u\cdot\nabla)u$ 和粘性项 $\nu\nabla^2 u$）

### 1.4 核心洞见：椭圆 vs 蛋形 = 压力 vs 速度

| 性质 | 电性（直锥→椭圆） | 磁性（超双曲锥→蛋形） |
|------|-------------------|---------------------|
| 基本方程 | $a = x$（线性） | $b = 1/x$（倒数） |
| 截面形状 | **椭圆**（对称） | **蛋形**（不对称） |
| 几何性质 | 闭合、有界、二次型 | 开放、发散、双曲型 |
| 物理系统 | 电磁学、线性场 | 流体力学、涡旋动力学 |
| NS对应 | **压力场** $p$ | **速度场** $u$ |
| 数学方程 | Laplace $\nabla^2 p = 0$ | NS $u_t + u\cdot\nabla u = \nu\nabla^2 u$ |
| 对称性 | 偶极子（$+$和$-$） | 涡管（左旋和右旋） |

**这解释了NS方程为什么难解**——因为NS方程同时包含两者：
$$\frac{\partial u}{\partial t} + (u\cdot\nabla)u = -\frac{1}{\rho}\nabla p + \nu\nabla^2 u$$

左边是**磁性**（速度场、涡旋、对流—蛋形几何），右边是**电性**（压力场、扩散—椭圆几何）。只有同时"看见"两个锥体的数学家才能找到解。

### 1.5 精细结构常数 α：双锥体的耦合常数（2026-05-30 新增）

> 📐 **完整推导见** → `14_anu结构与精细结构常数的关系/Anu_Alpha_Fine_Structure.md`

双锥体的几何谱系揭示了一个深层事实：**精细结构常数 α 就是两个锥体截面曲率的比值**。

$$α = \frac{R_0}{R_c} = \frac{\text{双曲锥}(\text{磁性})\text{尖端曲率}}{\text{直锥}(\text{电性})\text{基准曲率}}$$

| 量 | 锥体 | 曲率 | 物理含义 |
|:--|:----|:-----|:---------|
| $R_c = ħ/mc$ | 直锥→椭圆截面 | 康普顿半径 = 基准曲率 | 电性分支（线性、对称） |
| $R_0 = e²/4πε₀mc²$ | 双曲锥→蛋形截面 | 经典电子半径 = 尖端曲率 | 磁性分支（倒数、不对称） |
| **α = R₀/R_c** | **两锥耦合比** | **1/137.036** | **极化不对称度** |

这意味着：
- **α 不是电磁学的专属常数**——它是 $ab=1$ 极化原理中电性/磁性两条分支的**耦合力比**
- 电磁力为什么是这个强度？因为 ANU 的蛋形几何锁定了这个曲率比
- 如果双曲锥的截面曲率不同（比如 k_E ≠ 2），α 的值就会不同

> 💡 **双锥统一论的物理版本**：电性锥（直锥→椭圆→电磁力）与磁性锥（双曲锥→蛋形→涡旋力）通过 α 耦合。α 的稳定性（1/137.036）等价于两个锥体的几何自洽性。

### 2.2 一个重要的修改

我之前将椭圆和蛋形混为一谈，现在必须纠正：

| 之前的误解 | 现在的正确理解 |
|-----------|--------------|
| "蛋形曲线是双曲锥的平面截线" ✅ | 但双曲锥的截线不只有蛋形——还有椭圆 |

**修正后的认知**：
- 直圆锥的截面：**椭圆族**（从圆到很扁的椭圆）——电性
- 超双曲锥的截面：**蛋形族**（从近圆到很尖的蛋）——磁性

### 2.3 椭圆方程 vs 蛋形方程的对比

椭圆（来自直锥 $a=x$）：
$$\frac{x^2}{R^2} + \frac{y^2}{r^2} = 1$$

蛋形（来自超双曲锥 $b=1/x$）：
$$/bar{x} = /pm/sqrt{\frac{1}{(z_0 - /bar{y}\sin\alpha)^2} - (/bar{y}\cos\alpha)^2}$$

| 性质 | 椭圆 | 蛋形 |
|------|------|------|
| 对称性 | 完美对称（上下、左右） | 尖端↔钝端不对称 |
| 参数 | 2个（长轴 $R$，短轴 $r$） | 2个（截面高 $z_0$，倾角 $\alpha$） |
| 曲率分布 | 均匀变化 | 尖端曲率极大，钝端曲率极小 |
| 生成方式 | 平面切直圆锥 | 平面切超双曲锥 |
| 物理意义 | 势场等势面 | 涡旋截面 |

$$c_n = \sum_{m,k} C_{\text{nmk}} a_m a_k$$

$$(m,n;k) = \frac{|m-k|}{|m-n|} : \frac{|n-k|}{|n-m|} \in /{0,1,\infty/}$$
