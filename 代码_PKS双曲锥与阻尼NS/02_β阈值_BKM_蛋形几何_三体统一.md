## 一、三体系的自然映射

```
NSD 阻尼 NS:          PKS 双锥体:            数学判据:
                                            
β ≥ 1 → 整体存在     k_E ≥ 1 → 稳定蛋形      BKM 积分有限
β = 1 → 临界 Darcy   k_E = 1 → 椭圆(圆)      BKM 临界线
β < 1 → 可能爆破     k_E < 1 → 截面崩溃       BKM 积分发散
```

### 1.1 为什么 β=1 是普适临界线

经典 NS（无阻尼项）的千禧难题是：**3D 中光滑解是否必然爆破？**

加阻尼项后，β=1 成为了新的分水岭：

$$f(u) = -\alpha |u|^{\beta-1} u$$

- β≥1：阻尼随速度增长（至少线性）→ 涡量拉伸被抑制 → **整体存在可证**
- β<1：阻尼随速度亚线性 → 高速区"阻尼不够" → 涡量拉伸不受控 → **可能爆破**

### 1.2 PKS 中的 β=1 等同物

PKS 的标准锥面 $xy=1$ 产生的约束力与速度呈**线性关系**（粒子的离心力 ∝ 曲率 × 速度²/半径，在锥面上半径 ∝ 1/速度 → 约化后线性）。

当锥面变形为蛋形（$k_E>1$），尖端曲率增大 → 局部约束增强 → 等效 β 局部提升。

| 截面位置 | 曲率 κ | 等效 β | 含义 |
|:---|:---:|:---:|:---|
| 尖端（下游） | 大 | **>1** | 强阻尼区 → 涡量不易在此积累 |
| 钝端（上游） | 小 | **≈1** | 临界阻尼区 → 涡量可能在此积累 |
| 蛋形整体 | 加权平均 | **>1** | 整体约束足够 → BKM 通过 |

> 这解释了**Schauberger 内爆**的物理：流体从钝端（低压）进入锥面，在尖端（高压）被曲率强制减速 → 能量在尖端集中 → 空化内爆。BKM 判据说"只要尖端曲率足够大（等效 β>1），就不会在有限时间内失控"——Schauberger 利用的正是这个临界之上的稳定区。

## 二、逐参数对应表

### 2.1 NSD 参数 → PKS 参数

| NSD 参数 | 数学定义 | PKS 对应 | 转换公式 |
|:---|:---|:---|:---|
| α（阻尼系数） | 阻尼强度 | **锥面陡度**（$z_0$ 的倒数） | $\alpha \propto 1/z_0$ |
| β（阻尼指数） | 非线性程度 | **蛋形度 $k_E$** | $\beta = 1 + 2\cdot\frac{k_E-1}{k_E+1}$ |
| ν（粘性） | 扩散强度 | **锥面"软硬度"** | $ν \leftrightarrow$ 截面可穿越性 |
| $\|u\|$（速度模） | 被阻尼的量 | **锥面上粒子速率** | $\|u\| \propto 1/z_0$ |

### 2.2 各 β 区域的 PKS 工程对应

| β | 物理系统 | PKS 工程对应 | 稳定性 |
|:---:|:---|:---|:---|
| **β=1.35~1.60**（新发现） | 🔴 螺旋管流 (Popel 1952) | 锥形螺旋铜管 | **实验已验证**：$h \propto q^{1.51~1.67}$ |
| **β=1.67**（标准螺旋） | 螺旋铜管 (Popel 管No.2) | SEG 标准螺旋定子 | n=1.67 — 最基础几何优化 |
| **β=1.51**（最优螺旋） | 锥形螺旋小截面 (Popel 管No.8) | SEG 锥形螺旋定子 | n=1.51 — 8根管中最低摩擦 |

> 🔴 **Popel Report 1952 将 β 从理论推测升级为实验事实**：7 根管实测数据表明几何螺旋可将摩擦指数从湍流 n=2.0（β=2.0）压低到 n=1.51（β=1.35）。这是 PKS 框架中"几何约束 → 阻尼降低"的最早独立实验验证。详见 `能源革命_螺旋管负摩擦力_PKS解读.md`。
| **β<1**（亚线性） | 弱阻尼/滑移边界 | 截面约束不足 | **可能爆破！** |

## 三、工程含义：如何设计"不爆破"的涡旋管

### 3.1 β 的几何实现

NSD 的阻尼项可以通过**物理几何**实现——不必添加实际的体积力阻尼：

| 实现方式 | 产生等效 β 的机制 | 物理直觉 |
|:---|:---|:---|
| **锥面截面渐缩** | $A(z)$ 减小 → 速度增大 → Bernoulli 压力降 → 流向阻力 | 等效 β>1（加速区非线性阻尼） |
| **波纹槽** | 每个波峰-波谷产生局部压力梯度 → 分布式能量提取 | 等效 β≈1.5（中等非线性） |
| **螺旋肋** | 螺旋诱导二次流 → 能量从主流转移到横向涡 | 等效 β≈2（强非线性） |
| **多孔壁面** | Darcy 渗流 → 边界层能量被抽出 | 等效 β=1（线性阻尼） |

### 3.2 设计原则

```
要 稳定 → β 尽可能大（强阻尼）
要 能量 → β 尽可能小（弱阻尼）

最优 = 在 BKM 临界线 β=1 上方吃水最浅的位置
     = PKS 的 k_E 仅略大于 1（蛋形度最小的稳定蛋形）
```

### 3.3 SEG 涡旋管的具体建议

| 构件 | 等效 β 设计 | 机制 |
|:---|:---:|:---|
| 定子环截面 | $k_E≈1.2$（轻微蛋形） | β≈1.18 → 足够稳定 + 能量耗散最小 |
| 滚筒间隙 | 波纹槽深度 ≈ 0.05×滚筒直径 | 产生分布式 β≈1.5 局部阻尼 |
| 排气锥面 | 双曲锥 $z_0$ 充分大 | 避免焦散线 β→3 的过度约束 |

## 四、开放问题

1. **β→1 临界渐近的精确数学**：当 $k_E→1$（椭圆），等效 β→1，BKM 积分是否也趋于发散？如果是，则 PKS 的"完美椭圆截面"本身就是 NS 爆破的临界构型——一个被几何精确确定的数学边界。

2. **蛋形域 NS 数值解**：在蛋形截面上求解 NS 方程，提取速度剖面的 $\|u\|$ 分布，验证 $F_{\text{cone}} \propto \|u\|^{\beta-1}$ 中 β 与 $k_E$ 的关系是否遵循 $\beta = 1 + 2(k_E-1)/(k_E+1)$。

3. **多级锥面串联**：如果沿流向布置多个不同 $k_E$ 的锥面截面（级联蛋形），等效 β 是各段 β 的加权平均还是连乘积？这决定了多级 SEG 环的设计策略。
